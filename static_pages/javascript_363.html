charAt

The charAt() method in JavaScript returns the character at a specified index in a string. It takes an integer as an argument and returns the character at that position. If the index is out of range, it returns an empty string. Indices are zero-based, which means the first character of the string is at index 0.

```python
string_value = 'Hello, World!'
char_at_0 = string_value[0]  # 'H'
char_at_7 = string_value[7]  # 'W'
char_at_20 = ''  # Index out of range, will throw error in Python
```
```python
string_value = 'Python Programming'
char_at_5 = string_value[5]  # 'n'
char_at_1 = string_value[1]  # 'y'
try:
    char_at_21 = string_value[21]  # Index out of range
except IndexError as e:
    print(e)  # Will print error message for the out-of-bounds access
```
```python
string_value = 'CharAt Method Example'
char_at_10 = string_value[10]  # 'M'
char_at_0 = string_value[0]  # 'C'
char_at_30 = ''  # Index out of range, will throw error in Python
```
