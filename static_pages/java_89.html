abs

The 'abs' method in Java is used to return the absolute value of a given numerical value. It is a part of the Math class in Java, and can be used with different data types such as int, long, float, and double. The absolute value is the non-negative value of a number without regard to its sign. For instance, the absolute value of -5 is 5, and the absolute value of 5 is also 5. This method can be useful in various mathematical operations where negative values need to be converted to their positive counterparts.

def calculate_absolute_difference(a, b): return abs(a - b)

# Usage:
result = calculate_absolute_difference(-3, 5)
print(result) # Output: 8
def normalize_scores(scores): return [abs(score) for score in scores]

# Usage:
scores = [-10, 5, -7, 8, 0]
normalized_scores = normalize_scores(scores)
print(normalized_scores) # Output: [10, 5, 7, 8, 0]
def find_greatest_magnitude(num1, num2): return abs(num1) if abs(num1) > abs(num2) else abs(num2)

# Usage:
greatest_magnitude = find_greatest_magnitude(-15, 10)
print(greatest_magnitude) # Output: 15
