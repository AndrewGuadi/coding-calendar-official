Add

The 'Add' method is commonly used in various data collection classes in C#. It serves to append an element to a collection such as a List, Dictionary, or any other collection that implements the ICollection interface. When the Add method is called, it increases the size of the collection by one and adds the specified element to the end of the collection.

```python
# Example demonstrating adding items to a list

# Define a list
my_list = []

# Function to add items to the list
def add_to_list(item):
    my_list.append(item)

add_to_list('apple')
add_to_list('banana')
add_to_list('cherry')

print(my_list)  # Output: ['apple', 'banana', 'cherry']
```
```python
# Example demonstrating adding key-value pairs to a dictionary

# Define a dictionary
my_dict = {}

# Function to add items to the dictionary
def add_to_dict(key, value):
    my_dict[key] = value

add_to_dict('name', 'Alice')
add_to_dict('age', 30)
add_to_dict('city', 'New York')

print(my_dict)  # Output: {'name': 'Alice', 'age': 30, 'city': 'New York'}
```
```python
# Example demonstrating adding elements to a set

# Define a set
my_set = set()

# Function to add items to the set
def add_to_set(item):
    my_set.add(item)

add_to_set('cat')
add_to_set('dog')
add_to_set('bird')
add_to_set('cat')  # Adding duplicate item

print(my_set)  # Output: {'cat', 'dog', 'bird'} (Note: 'cat' won't duplicate)
```
