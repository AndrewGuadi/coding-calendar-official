std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that allows you to add a new element to the end of the vector. This method increases the size of the vector by one and copies the value of the added element into the newly allocated space at the end. If the vector's capacity is reached, the function will automatically allocate more memory to accommodate the new element, ensuring that the addition of elements can occur without having to manually resize the vector. This function is often used when building lists of items dynamically as you do not need to know the final size of the vector beforehand.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("apple");
    myVector.push_back("banana");
    myVector.push_back("cherry");

    for (const auto &item : myVector) {
        std::cout << item << std::endl;
    }
    return 0;
}
```
