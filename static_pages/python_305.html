ascii

The `ascii()` function in Python returns a string containing a printable representation of an object, escaping the non-ASCII characters. This function is particularly useful when you want to display characters in a way that is safe for ASCII-only systems. It returns the same string if all characters in the object are ASCII, otherwise it replaces non-ASCII characters with escape sequences in the format '\xHH' or '\uHHHH'.
