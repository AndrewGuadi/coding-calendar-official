abs

The 'abs' method in Java is used to calculate the absolute value of a number. The method can handle various numeric types, including integers, floating-point numbers, and doubles. When a number is passed to the 'abs' method, it returns the non-negative value of that number. If the number is already non-negative, it remains unchanged; if it is negative, the method effectively removes the negative sign. This method is useful in scenarios where the magnitude of a value is more important than its sign, such as in mathematical computations, financial calculations, or any application where negative values should be treated as positive.

def calculate_absolute_value(num):
    return abs(num)

# Example usage:
print(calculate_absolute_value(-5))  # Output: 5
print(calculate_absolute_value(3.7))  # Output: 3.7
def displacement(start, end):
    return abs(start - end)

# Example usage:
print(displacement(20, 10))  # Output: 10
print(displacement(-5, 5))  # Output: 10
def distance_to_origin(x, y):
    return (abs(x), abs(y))

# Example usage:
print(distance_to_origin(-3, -4))  # Output: (3, 4)
print(distance_to_origin(0, 2))  # Output: (0, 2)
