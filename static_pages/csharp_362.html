Add

The 'Add' method is commonly used in C# collections like List, Dictionary, and HashSet to add a new element or key-value pair to the collection. The method modifies the collection by including the specified item, and in cases where uniqueness is required (like in a HashSet or Dictionary), it enforces that policy. If the item to be added already exists in the collection, the behavior may vary depending on the collection type; for example, adding a duplicate key in a Dictionary will throw an exception.

```python
# Example usage of List to add elements
class Example:
    def main(self):
        my_list = []  # Create an empty list
        my_list.append('Hello')  # Add 'Hello' to the list
        my_list.append('World')  # Add 'World' to the list
        print(my_list)  # Output: ['Hello', 'World']

example = Example()  # Create instance
example.main()  # Call the main method
```
```python
# Example usage of Dictionary to add a key-value pair
class Example:
    def main(self):
        my_dict = {}  # Create an empty dictionary
        my_dict['name'] = 'Alice'  # Adding key 'name' with value 'Alice'
        my_dict['age'] = 30  # Adding key 'age' with value 30
        print(my_dict)  # Output: {'name': 'Alice', 'age': 30}

example = Example()  # Create instance
example.main()  # Call the main method
```
```python
# Example usage of set to add unique elements
class Example:
    def main(self):
        my_set = set()  # Create an empty set
        my_set.add('Python')  # Add 'Python' to the set
        my_set.add('Java')  # Add 'Java' to the set
        my_set.add('Python')  # Attempt to add 'Python' again (will not change the set)
        print(my_set)  # Output: {'Python', 'Java'} (order may vary due to set nature)

example = Example()  # Create instance
example.main()  # Call the main method
```
