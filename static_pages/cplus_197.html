std::vector::push_back

The push_back method in C++ is used to add a new element to the end of a vector, which is a dynamic array that can grow in size. When elements are added using push_back, the vector may reallocate memory to accommodate the new element if its current capacity is exceeded. The new element is added at the end and its value is copied into the vector. This method is useful for building up a collection of objects without needing to specify the size of the vector in advance.

In this example, we create a vector of strings and use push_back to append various Python code snippets representing basic functions.

```cpp
#include 
#include 
#include 

int main() {
    std::vector python_codes;
    python_codes.push_back("def hello():\n    print('Hello, World!')");
    python_codes.push_back("def add(a, b):\n    return a + b");
    python_codes.push_back("def subtract(a, b):\n    return a - b");

    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```

```cpp
int main() {
    std::vector python_data_structures;
    python_data_structures.push_back("my_list = [1, 2, 3, 4]");
    python_data_structures.push_back("my_dict = {'key1': 'value1', 'key2': 'value2'}");
    python_data_structures.push_back("my_set = {1, 2, 3}");

    for (const auto& data_struct : python_data_structures) {
        std::cout << data_struct << std::endl;
    }
    return 0;
}
```
This example demonstrates using push_back to collect Python code snippets that illustrate control flow statements such as if-else and loops.

```cpp
#include 
#include 
#include 

int main() {
    std::vector python_control_flow;
    python_control_flow.push_back("if a > b:\n    print('A is greater than B')");
    python_control_flow.push_back("for i in range(5):\n    print(i)");
    python_control_flow.push_back("while count < 5:\n    count += 1\n    print(count)");

    for (const auto& control_flow : python_control_flow) {
        std::cout << control_flow << std::endl;
    }
    return 0;
}
```
