charAt

The `charAt()` method in JavaScript is used to return the character at a specified index (position) in a string. The index is zero-based, meaning the first character is at index 0. If the specified index is out of range, `charAt()` returns an empty string. This method is useful for extracting specific characters from a string for further processing or analysis.

def get_character(string, index):
    # Using JavaScript's charAt-like approach in Python
    if index < 0 or index >= len(string):
        return ''
    return string[index]

# Example usage:
result = get_character('hello', 1)  # result would be 'e'
print(result)
def get_first_character(string):
    # Getting the first character using a charAt-like method
    return string[0] if string else ''

# Example usage:
result = get_first_character('world')  # result would be 'w'
print(result)
def get_character_or_empty(string, index):
    # Mimicking JavaScript's charAt behavior
    return string[index] if 0 <= index < len(string) else ''

# Example usage:
result = get_character_or_empty('Python', 10)  # result would be '' (empty string)
print(result)
