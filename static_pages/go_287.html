close

The 'close' method in Go is used to close resources such as files, network connections, or any I/O resources that were opened during the execution of a program. Closing these resources is important to free up system resources and prevent memory leaks. When a resource is closed, it can no longer be used. If an attempt is made to use a closed resource, a panic typically occurs, signaling that the resource is no longer available for operations.

import os

def close_file_example():
    file_path = 'example.txt'
    # Open a file
    f = open(file_path, 'w')
    try:
        # Write to the file
        f.write('Hello, World!')
    finally:
        # Close the file
        f.close()

import socket

def close_socket_example():
    # Create a TCP/IP socket
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server_address = ('localhost', 6789)
    # Connect the socket to the server
    s.connect(server_address)
    try:
        # Send data
        s.sendall(b'Hello, Server!')
    finally:
        # Close the socket
        s.close()

import requests

def close_response_example():
    # Send a request to a URL
    response = requests.get('https://api.github.com')
    try:
        # Process response
        print(response.json())
    finally:
        # Close the response
        response.close()
