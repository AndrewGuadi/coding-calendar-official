ascii

The `ascii()` function in Python is used to return a string containing a printable representation of an object. It takes an object as an argument and converts it into a string that can be safely printed, replacing non-ASCII characters with their escape sequences. This is useful when you want to ensure that the representation of the object only contains ASCII characters.

my_string = 'Hello, 世界'
print(ascii(my_string))  # Output: 'Hello, \u4e16\u754c'
data = {'name': 'Alice', 'age': 30, 'city': 'New York'}
print(ascii(data))  # Output: "{'name': 'Alice', 'age': 30, 'city': 'New York'}"
complex_obj = ['data', (1, 2), {'key': 'value', 'ñ': 'non-ascii'}]
print(ascii(complex_obj))  # Output: "['data', (1, 2), {'key': 'value', 'ñ': 'non-ascii'}]"
