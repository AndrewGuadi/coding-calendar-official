std::vector::push_back

The 'std::vector::push_back' method is used to add a new element at the end of a vector in C++. When this method is called, it increases the size of the vector by one and adds the specified value to the last position. If the current capacity of the vector is exceeded, it automatically allocates more memory to accommodate the new element. The 'push_back' method is typically used when we need to dynamically append items to a vector, making it a flexible container for managing arrays or lists of data.

```cpp
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    for (const auto& lang : languages) {
        std::cout << lang << " ";
    }
    return 0;
}
``` This example demonstrates creating a vector of strings and adding three programming languages to it using 'push_back' and then printing out the contents of the vector.

    std::cout << "Fruits in the vector:\n";
    for (size_t i = 0; i < fruits.size(); ++i) {
        std::cout << fruits[i] << '\n';
    }
    return 0;
}
