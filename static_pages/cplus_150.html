std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one, and the new element is initialized with the provided value. If the vector's current capacity is not sufficient to accommodate the additional element, the vector will allocate new storage with a larger capacity, copy the existing elements to the new storage, and then add the new element. This allows for dynamic resizing of the vector. The operation has an average time complexity of O(1) but can be O(n) when a reallocation occurs.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Python");
    myVector.push_back("Java");
    myVector.push_back("C++");

    for (const auto& language : myVector) {
        std::cout << language << std::endl;
    }
    return 0;
}
```

```cpp
        numbers.push_back(i * 10);
    }
    for (const auto& num : numbers) {

    }

}
```
