Aggregate

The Aggregate method in C# is used to apply an accumulator function over a sequence. It is particularly useful for performing calculations across a collection, allowing you to roll up data into a single result. The method takes an initial seed value and an accumulator function, which specifies how to combine the values in the collection. Optionally, a result selector function can also be provided to transform the final accumulated result. The result is the final accumulated value after applying the function to the entire sequence.

numbers = [1, 2, 3, 4, 5]
result = reduce(lambda x, y: x + y, numbers)
# result will be the sum of the numbers in the list, which is 15.
prices = [10, 20, 30, 40]
total_cost = reduce(lambda x, y: x + y, prices)
# total_cost will be the sum of all prices, which equals 100.
words = ['hello', 'world']
combined = reduce(lambda x, y: x + ' ' + y, words)
# combined will result in 'hello world'.
