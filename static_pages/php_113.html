strlen

The `strlen` function in PHP calculates the length of a given string. It returns the number of characters present in the string, including spaces and punctuation. This function is useful for validating input, handling strings of specific lengths, and ensuring that strings meet certain size requirements.

```python
# Example of using strlen to validate user input
user_input = 'Hello, World!'
if len(user_input) > 10:
    print('Input is too long!')
else:
    print('Input is within the acceptable length.')
```
```python
# Example of using strlen to check for empty strings
user_input = ''
if len(user_input) == 0:
    print('Input cannot be empty!')
else:
    print('Input is valid.')
```
```python
# Example of using strlen to extract a substring
user_input = 'Programming is fun'
if len(user_input) > 5:
    substring = user_input[:5]
    print('Substring:', substring)
else:
    print('Input is too short for a substring.')
```
