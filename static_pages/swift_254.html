uppercased

The 'uppercased' method is a string method in Swift that returns a new string with all characters converted to uppercase. It is useful for normalizing string data to ensure that comparisons or displays are case-insensitive. When calling this method on a string, it creates a copy of the original string with all lowercase letters converted to their uppercase equivalents. If the string contains characters that are already uppercase or non-alphabetical, those characters remain unchanged in the resulting string.

let originalString = "hello, world!" 
let uppercasedString = originalString.uppercased() 
print(uppercasedString) // Output: HELLO, WORLD!
let greeting = "Hello There!" 
let uppercasedGreeting = greeting.uppercased() 
print(uppercasedGreeting) // Output: HELLO THERE!
let mixedCase = "Python is Awesome!" 
let uppercasedMixed = mixedCase.uppercased() 
print(uppercasedMixed) // Output: PYTHON IS AWESOME!
