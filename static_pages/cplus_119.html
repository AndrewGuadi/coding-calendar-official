std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of a vector. This operation increases the size of the vector by one and extends the capacity if necessary to accommodate the new element. The element that is pushed back can be of any data type that is compatible with the vector's declared type. It's important to note that if the vector's capacity is reached, the push_back method will allocate additional memory to ensure that there is enough space available for new elements.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("apple");
    myVector.push_back("banana");
    myVector.push_back("cherry");

    for (const auto &fruit : myVector) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
```
