ascii

The `ascii()` function in Python is used to return a string containing a printable representation of an object. This representation is similar to that returned by the `repr()` function but will escape non-ASCII characters in the string by using backslash escapes. The `ascii()` function is particularly useful when you want to visualize non-ASCII characters in a readable format. It is commonly used for debugging or logging purposes.

original_string = 'Caf√©'
print(ascii(original_string))  # Output: 'Caf\xe9'
data = {'name': 'Jos√©', 'age': 30}
print(ascii(data))  # Output: '{'name': 'Jos\xe9', 'age': 30}'
