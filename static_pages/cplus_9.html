std::vector::push_back

The `push_back` method in C++ is used to add a new element at the end of a vector, which is a dynamic array that can grow in size. This method increases the size of the vector by one and initializes the new element with the value provided to `push_back`. It is important to note that if the current capacity of the vector is exceeded, the `push_back` method will automatically resize the vector, which may involve allocating new memory and moving existing elements to the new space.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");
    
    for (const auto& fruit : fruits) {
        std::cout << fruit << " ";
    }
    return 0;
}
```

int main() {

        numbers.push_back(i * 10);
    }

        std::cout << number << " ";
    }
