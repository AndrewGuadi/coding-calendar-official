Add

The 'Add' method is typically used in collections like lists, dictionaries, or sets in C#. It allows you to add an element to a collection. The method may vary depending on the type of collection being used. In a List, it adds an item to the end of the list; in a Dictionary, it adds a key-value pair; and in a Set, it adds an element if it doesn't already exist. This method is fundamental for managing dynamic collections where the size can change during runtime.

```python
# Example of adding elements to a list
my_list = []
my_list.append(1)
my_list.append(2)
my_list.append(3)
print(my_list)  # Output: [1, 2, 3]
```
```python
# Example of adding key-value pairs to a dictionary
my_dict = {}
my_dict['a'] = 1
my_dict['b'] = 2
my_dict['c'] = 3
print(my_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
```python
# Example of adding elements to a set
my_set = set()
my_set.add(1)
my_set.add(2)
my_set.add(3)
my_set.add(2)  # This will not add a duplicate
print(my_set)  # Output: {1, 2, 3}
```
