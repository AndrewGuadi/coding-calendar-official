std::vector::push_back

The 'push_back' method is used to add a new element to the end of a std::vector in C++. When an element is added using 'push_back', the size of the vector increases by one. If the vector's capacity has been reached, it automatically reallocates memory to accommodate the new element. This method does not return any value.

```cpp
#include 
#include 

int main() {
    std::vector pythonExamples;
    pythonExamples.push_back("print('Hello, World!')");
    pythonExamples.push_back("for i in range(5): print(i)");

    for (const auto &example : pythonExamples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
