std::vector::insert

The `std::vector::insert` method in C++ is used to insert elements into a vector at a specified position. It can insert a single element, fill a range with multiple copies of a single value, or insert a range of elements from another container or a raw array. This insertion operation can shift existing elements to the right to make space for the new ones, and the vector will automatically resize if necessary. The method can vary in usage according to its parameters: - `insert(const_iterator pos, const T& val)`: Inserts a single element `val` before the position defined by the iterator `pos`. - `insert(const_iterator pos, size_t count, const T& val)`: Inserts `count` copies of the element `val` before the position defined by the iterator `pos`. - `insert(const_iterator pos, InputIterator first, InputIterator last)`: Inserts elements from the range `[first, last)` before the position defined by the iterator `pos`.

```cpp
#include 
#include 

int main() {
    std::vector v = {"apple", "banana", "cherry"};
    auto it = v.begin() + 1; // position to insert
    v.insert(it, "orange"); // insert a single element

    for (const auto& fruit : v) {
        std::cout << fruit << ' '; // Outputs: apple orange banana cherry 
    }
    return 0;
}
```

int main() {
    std::vector v = {"apple", "banana", "orange"};
    auto it = v.begin() + 2; // position to insert
    v.insert(it, 2, "kiwi"); // insert two copies of "kiwi"

    for (const auto& fruit : v) {
        std::cout << fruit << ' '; // Outputs: apple banana kiwi kiwi orange 
    }
    return 0;

#include 
#include 

int main() {
    std::vector v1 = {"apple", "banana"};
    std::vector v2 = {"cherry", "date", "elderberry"};
    auto it = v1.end(); // insert at the end
    v1.insert(it, v2.begin(), v2.end()); // insert elements from v2 into v1

    for (const auto& fruit : v1) {
        std::cout << fruit << ' '; // Outputs: apple banana cherry date elderberry 
    }
    return 0;
}
