strlen

The strlen function in PHP computes the length of a given string. It returns the number of bytes in the string. This function is commonly used when you need to determine the size of a string for validation, manipulation, or display purposes. If the string is empty, the return value will be 0. It's important to note that the length returned may differ from the number of characters in multibyte character sets.

def get_string_length(s):
    return len(s)

# Example usage:
string_example = 'Hello, World!'
length = get_string_length(string_example)
print(length)  # Output: 13
def check_string_length(s):
    length = len(s)
    if length > 0:
        print(f'The string has {length} characters.')
    else:
        print('The string is empty.')

# Example usage:
check_string_length('Python is great!')  # Output: The string has 17 characters.
check_string_length('')  # Output: The string is empty.
def is_string_too_long(s, max_length):
    return len(s) > max_length

# Example usage:
max_length_allowed = 10
result = is_string_too_long('Hello!', max_length_allowed)
print(result)  # Output: False
result = is_string_too_long('Hello, World!', max_length_allowed)
print(result)  # Output: True
