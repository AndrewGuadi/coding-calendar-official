append

In Go, the 'append' function is used to add elements to the end of a slice. It takes one or more elements as arguments and returns a new slice with the elements appended. If the slice has enough capacity, the original slice is modified; otherwise, a new slice is allocated to accommodate the new elements. It is essential for working with dynamic data since the size of slices in Go can change at runtime.

```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3}
    slice = append(slice, 4, 5)
    fmt.Println(slice) // Output: [1, 2, 3, 4, 5]
}
```
```go
package main

import "fmt"

func main() {
    names := []string{"")
    names = append(names, "Alice", "Bob")
    names = append(names, "Charlie")
    fmt.Println(names) // Output: [Alice Bob Charlie]
}
```
```go
package main

import "fmt"

func main() {
    var nums []int
    nums = append(nums, 10)
    nums = append(nums, 20, 30)
    fmt.Println(nums) // Output: [10 20 30]
```
