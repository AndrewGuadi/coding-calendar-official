abs

The 'abs' method in Java is a static method provided by the Math class that returns the absolute value of a given number. This method can take various numeric types as parameters, such as int, long, float, and double, and will return the same type of value as the argument it was called with. The absolute value of a number is its non-negative value, which means it removes any negative sign if present.

```python
# Example 1: Finding the absolute value of a negative integer
negative_integer = -10
absolute_value = abs(negative_integer)
print(f'Absolute value of {negative_integer} is {absolute_value}')  # Output: Absolute value of -10 is 10
```
```python
# Example 2: Finding the absolute value of a float
negative_float = -3.14
absolute_value = abs(negative_float)
print(f'Absolute value of {negative_float} is {absolute_value}')  # Output: Absolute value of -3.14 is 3.14
```
```python
# Example 3: Comparing absolute values
numbers = [-5, 3, -8, 2]
absolute_values = [abs(num) for num in numbers]
print(f'Absolute values: {absolute_values}')  # Output: Absolute values: [5, 3, 8, 2]
```
