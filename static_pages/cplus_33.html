std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector. When an element is added, if the current capacity of the vector is insufficient to accommodate the new element, the vector will automatically allocate more memory to accommodate the new size. This makes 'push_back' a dynamic and flexible way to build a vector as you can keep adding elements without needing to manage the underlying storage yourself.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Hello");
    vec.push_back("World");
    vec.push_back("from");
    vec.push_back("C++");
    
    // Display elements in the vector
    for (const auto& str : vec) {
        std::cout << str << ' ';  // Output: Hello World from C++ 
    }
    return 0;
}
```
