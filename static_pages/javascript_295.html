charAt

The 'charAt' method in JavaScript is used to return the character at a specified index (position) from a string. The index is zero-based, meaning that the first character has an index of 0. If the index is out of range, it returns an empty string. This method is particularly useful when you need to manipulate or examine specific characters within strings.

def get_char_at_index(string, index):
    if index < 0 or index >= len(string):
        return ""
    return string[index]

# Example usage:
result = get_char_at_index("Hello, World!", 7)
print(result)  # Output: W
def find_first_char(string):
    return get_char_at_index(string, 0)

# Example usage:
result = find_first_char("Python")
print(result)  # Output: P
def get_last_char(string):
    return get_char_at_index(string, len(string) - 1)

# Example usage:
result = get_last_char("JavaScript")
print(result)  # Output: t
