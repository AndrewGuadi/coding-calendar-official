append

The 'append' method in Java is commonly used in the context of certain classes like StringBuilder and ArrayList. For StringBuilder, 'append' is used to add strings or objects to the end of the current StringBuilder object. For ArrayList, it is used to add an element at the end of the list. This method increases the size of the collection when adding a new element. For string concatenation, 'append' is more efficient than using the plus operator (+) as it avoids creating multiple immutable String instances.

```java
public class Example {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();
        sb.append('Hello');
        sb.append(' World!');
        System.out.println(sb.toString()); // Output: Hello World!
    }
}
```
```java
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList list = new ArrayList<>();
        list.append('First');
        list.append('Second');
        System.out.println(list); // Output: [First, Second]
    }
}
```
```java
import java.util.LinkedList;

public class Example {
    public static void main(String[] args) {
        LinkedList linkedList = new LinkedList<>();
        linkedList.append('Node1');
        linkedList.append('Node2');
        System.out.println(linkedList); // Output: [Node1, Node2]
    }
}
```
