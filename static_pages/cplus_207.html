std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element at the end of a vector. When an element is pushed back, if there is not enough space in the vector, it automatically resizes the vector to accommodate the new element. This method is important for dynamically managing arrays as it allows you to build vectors whose size is not known at compile time. The push_back operation runs in amortized constant time, although it may occasionally take longer when the vector must be resized.

```cpp
#include 
#include 

int main() {
    std::vector python_versions;
    python_versions.push_back("Python 3.8");
    python_versions.push_back("Python 3.9");
    python_versions.push_back("Python 3.10");

    for (const std::string& version : python_versions) {
        std::cout << version << std::endl;
    }
    return 0;
}
```
