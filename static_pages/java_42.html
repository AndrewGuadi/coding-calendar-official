abs

The abs method in Java is used to return the absolute value of a given number, which is the non-negative value of that number regardless of its sign. This method is often used in mathematical calculations where negative numbers are not acceptable, and a positive representation is required. In Java, this method is part of the Math class and can be applied to different numeric primitive types such as int, long, float, and double.

def calculate_distance(x1, y1, x2, y2):
    return abs(x2 - x1) + abs(y2 - y1)

distance = calculate_distance(2, 3, -5, -1)  # Expected output: 10
def find_positive_difference(num1, num2):
    return abs(num1 - num2)

result = find_positive_difference(-10, 5)  # Expected output: 15
def absolute_temperature_change(initial_temp, final_temp):
    return abs(final_temp - initial_temp)

change = absolute_temperature_change(20.5, 15.0)  # Expected output: 5.5
