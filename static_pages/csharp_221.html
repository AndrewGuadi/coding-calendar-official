Add

The 'Add' method is commonly used to insert a new element into a collection, such as a list, dictionary, or database in C#. This method typically accepts an element as a parameter and appends it to the end of the collection or updates the collection accordingly. The behavior of the Add method may vary depending on the type of collection being used. For example, in a List, it simply adds the element to the end, while in a Dictionary, it adds a key-value pair to the collection.

```python
# Example demonstrating the usage of 'Add' with a List in C#
# C# Code:

List myList = new List();
myList.Add("Example 1");
myList.Add("Example 2");
Console.WriteLine(string.Join(", ", myList));  # Output: Example 1, Example 2
```
```python
# Example demonstrating the usage of 'Add' with a Dictionary in C#
# C# Code:

Dictionary myDict = new Dictionary();
myDict.Add(1, "One");
myDict.Add(2, "Two");
foreach (var item in myDict)
{
    Console.WriteLine(item.Key + ": " + item.Value);
}  # Output:
# 1: One
# 2: Two
```
```python
# Example demonstrating the usage of 'Add' in a custom class in C#
# C# Code:

class MyCollection
{
    private List items = new List();

    public void Add(string item)
    {
        items.Add(item);
    }

    public void Display()
    {
        Console.WriteLine(string.Join(", ", items));
    }
}

MyCollection collection = new MyCollection();
collection.Add("Item 1");
collection.Add("Item 2");
collection.Display();  # Output: Item 1, Item 2
```
