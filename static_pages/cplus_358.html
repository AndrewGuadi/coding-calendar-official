std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of a vector. This operation increases the size of the vector by one, appending the new element at the last position. If the current capacity of the vector is exceeded, it may allocate more memory to accommodate the new element, ensuring that continuous memory is available for all elements. It is important to notice that the element being pushed back must be of the type contained by the vector or convertible to that type.

```cpp
#include 
#include 

int main() {
    std::vector my_vector;
    my_vector.push_back("First element");
    my_vector.push_back("Second element");

    for (const auto& str : my_vector) {
        std::cout << str << std::endl;
    }
    return 0;
}
```
