abs

The 'abs' method in Java is used to obtain the absolute value of a number. The absolute value of a number is its non-negative value, regardless of its sign. This method can be used with different numeric types, such as integers, floats, and doubles. Itâ€™s a static method of the Math class, and its signature varies depending on the type of input: Math.abs(int a), Math.abs(double a), Math.abs(float a), and so on.

def abs_example_1(x):
    return abs(x)

# Usage
result = abs_example_1(-10)
print('The absolute value of -10 is:', result) # Outputs: The absolute value of -10 is: 10
