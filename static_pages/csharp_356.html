Add

The 'Add' method is typically used to insert a new element into a collection, such as a list or a dictionary, in C#. It takes one or more parameters representing the value(s) to be added and updates the collection accordingly. In the context of a list, this method appends the new element at the end of the list, while in the context of dictionaries, it adds a key-value pair where the specified key must be unique within the dictionary.

```python
# Using the Add method with a List

# Create an empty list
my_list = []

# Add elements to the list
my_list.append('apple')
my_list.append('banana')

print(my_list) # Output: ['apple', 'banana']```
```python
# Using the Add method with a Dictionary

# Create an empty dictionary
my_dict = {}

# Add key-value pairs to the dictionary
my_dict['name'] = 'John'
my_dict['age'] = 30

print(my_dict) # Output: {'name': 'John', 'age': 30}```
```python
# Using the Add method in a custom class

class CustomList:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

# Create an instance of CustomList
my_custom_list = CustomList()

# Add items to the custom list
my_custom_list.add('carrot')
my_custom_list.add('tomato')

print(my_custom_list.items) # Output: ['carrot', 'tomato']```
