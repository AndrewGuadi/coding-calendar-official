abs

The 'abs' method in Java is used to return the absolute value of a given number. This method can be applied to different data types such as integers, floats, and doubles. The absolute value of a number is its non-negative value without regard to its sign. For instance, the absolute value of -5 is 5, and the absolute value of 3 is still 3. The 'abs' method is part of the java.lang.Math class.

def calculate_profit_loss(amount):
    return abs(amount)

# Example: If the amount is negative, it indicates a loss.
profit_loss = calculate_profit_loss(-150)
print(profit_loss)  # Output: 150
def measure_distance(point_a, point_b):
    return abs(point_a - point_b)

# Example: Calculate the distance between two points on a line.
distance = measure_distance(5, -3)
print(distance)  # Output: 8
def temperature_difference(temp_a, temp_b):
    return abs(temp_a - temp_b)

# Example: The absolute temperature difference.
difference = temperature_difference(30, 20)
print(difference)  # Output: 10
