abs

The abs() function in Python is a built-in function that returns the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, disregarding whether itâ€™s positive or negative. This function works with both integers and floating-point numbers.

# Example 1: Absolute value of a positive integer
number = 42
absolute_value = abs(number)
print(absolute_value)  # Output: 42
# Example 2: Absolute value of a negative integer
number = -42
absolute_value = abs(number)
print(absolute_value)  # Output: 42
# Example 3: Absolute value of a floating-point number
number = -3.14
absolute_value = abs(number)
print(absolute_value)  # Output: 3.14
