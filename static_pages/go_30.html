append

The append function in Go is used to add elements to the end of a slice. It takes a slice and the elements to be added as arguments, and it returns a new slice that includes the original elements along with the newly appended elements. If the slice has enough capacity to accommodate the new elements, they are added in place; otherwise, a new slice is allocated with a larger capacity.

my_slice = []string{"apple", "banana"}
my_slice = append(my_slice, "orange")
# Result: my_slice == ["apple", "banana", "orange"]
my_numbers = []int{1, 2, 3}
my_numbers = append(my_numbers, 4, 5, 6)
# Result: my_numbers == [1, 2, 3, 4, 5, 6]
my_data = []interface{}{}
my_data = append(my_data, "text", 42, true)
# Result: my_data == ["text", 42, true]
