abs

The `abs` method in Java is a static method available in the Math class used to return the absolute value of a number. It can take different data types such as int, long, float, or double as its arguments, and it returns the non-negative value of the input number. This method effectively allows you to eliminate any negative sign from a number, simplifying calculations where only magnitude is needed.

def calculate_absolute_difference(num1, num2):
    return abs(num1 - num2)

result = calculate_absolute_difference(-5, 3)
print(result)  # Output: 8
def max_absolute_value(a, b):
    return max(abs(a), abs(b))

result = max_absolute_value(-10, 4)
print(result)  # Output: 10
def total_distance(points):
    total = 0
    for point in points:
        total += abs(point)
    return total

result = total_distance([-3, 5, -2])
print(result)  # Output: 10
