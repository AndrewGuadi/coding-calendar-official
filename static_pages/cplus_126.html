std::vector::push_back

The 'push_back' method in C++ is a member function of the std::vector class. It is used to add a new element to the end of the vector. This function automatically resizes the vector if necessary, allowing users to manage dynamic arrays with ease. The 'push_back' method takes a single argument, which is the value to be added to the vector. The time complexity of this operation is amortized O(1), meaning that although resizing might take longer, most insertions occur in constant time.

```python
# Initialize an empty list to simulate a vector in C++
my_vector = []

# Adding elements to the end of the list
my_vector.append('Python')
my_vector.append('Java')
my_vector.append('C++')

# Print the current state of the list
print(my_vector)  # Output: ['Python', 'Java', 'C++']
```
```python
# Initialize an empty list
my_vector = []

# A loop to add multiple programming languages to the list
for language in ['JavaScript', 'Ruby', 'Go']:
    my_vector.append(language)

# Print the final list of languages
print(my_vector)  # Output: ['JavaScript', 'Ruby', 'Go']
```
```python
# Example of using push_back logic when reading from a data source
my_vector = []

# Simulating reading lines of data from a file or input
input_data = ['Swift', 'Kotlin', 'Rust']

# Adding each line of data to the list
for language in input_data:
    my_vector.append(language)

# Display the contents of the list
print(my_vector)  # Output: ['Swift', 'Kotlin', 'Rust']
```
