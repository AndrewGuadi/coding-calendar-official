upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. It does not modify the original string; instead, it returns a new string with the changes. This method is useful when you need to standardize input, such as user names or any other string data, to a consistent upper case format for comparison or display purposes.

original_string = 'hello'
upcased_string = original_string.upcase
print(upcased_string)  # Outputs: 'HELLO'
name = 'jane doe'
uppercase_name = name.upcase
print(uppercase_name)  # Outputs: 'JANE DOE'
message = 'welcome to ruby world'
formatted_message = message.upcase
print(formatted_message)  # Outputs: 'WELCOME TO RUBY WORLD'
