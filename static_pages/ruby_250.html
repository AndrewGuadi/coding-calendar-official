upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. It does not change non-alphabetic characters, nor does it affect characters that are already in uppercase. This method is called on a string object and returns a new string with the desired transformation. It is useful in scenarios where uniform string casing is required, such as when processing user input, preparing strings for comparisons, or formatting output data.

original_string = 'hello world'
uppercase_string = original_string.upcase
print(uppercase_string)  # Output: 'HELLO WORLD'
mixed_string = 'Hello, Ruby!'
uppercase_mixed = mixed_string.upcase
print(uppercase_mixed)  # Output: 'HELLO, RUBY!'
num_string = '123abc'
uppercase_num = num_string.upcase
print(uppercase_num)  # Output: '123ABC'
