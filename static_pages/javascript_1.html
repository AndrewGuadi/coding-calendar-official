charCodeAt

The `charCodeAt` method in JavaScript returns an integer between 0 and 65535 representing the Unicode of the character at the specified index in a string. The index is zero-based, meaning that the first character of the string is at index 0. If the index provided is out of the string's range, it will return NaN. This method is useful for converting a character to its corresponding Unicode value.

string_value = 'hello'
index = 0
unicode_value = string_value.charCodeAt(index)
# Result is 104, which is the Unicode for 'h'
string_value = 'Python'
index = 1
unicode_value = string_value.charCodeAt(index)
# Result is 121, which is the Unicode for 'y'
string_value = 'A'
index = 0
unicode_value = string_value.charCodeAt(index)
# Result is 65, which is the Unicode for 'A'
