charAt

The `charAt()` method in JavaScript returns the character at a specified index in a string. The index is zero-based, meaning that the first character in the string is at index 0. If the index is out of range (less than 0 or greater than or equal to the string length), it returns an empty string. This method is useful for retrieving individual characters from a string for various purposes such as parsing, validation, or simply manipulation of string data.

str = 'Hello, world!';
character = str.charAt(4);

print(character)  # Output: 'o'  # Retrieves the 5th character from the string
str = 'abcdef';
character = str.charAt(0);

print(character)  # Output: 'a'  # Retrieves the first character from the string
str = 'Programming';
character = str.charAt(11);

print(character)  # Output: ''  # Index out of range; returns an empty string
