std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of a vector. This function increases the size of the vector by one and assigns the value passed to it to the new element. If the vector's current capacity is not sufficient to accommodate the new element, it will automatically allocate more memory to ensure that the operation can be completed. Pushing back elements in a vector is a common operation when you want to store a collection of items that may change in size during the execution of your program.

```cpp
#include 
#include 

int main() {
    std::vector programmingLanguages;
    programmingLanguages.push_back("Python");
    programmingLanguages.push_back("C++");
    programmingLanguages.push_back("Java");

    for (const auto& lang : programmingLanguages) {
        std::cout << lang << std::endl;
    }
    return 0;
}```
In this example, we create a vector of type string, add three programming languages using push_back, and then print each language to the console.

#include 
#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i);
    }

    std::cout << "The numbers are:" << std::endl;
    for (const auto& num : numbers) {
        std::cout << num << " ";
    }
    std::cout << std::endl;
    return 0;
}```
In this example, we use a loop to add the numbers 1 to 5 to the vector using push_back and then print them out.
```cpp
#include 
#include 

int main() {
    std::vector fruits;
    std::string fruitInput;
    std::cout << "Enter fruits (type 'end' to stop):\n";

    while (true) {
        std::getline(std::cin, fruitInput);
        if (fruitInput == "end") {
            break;
        }
        fruits.push_back(fruitInput);
    }

    std::cout << "You entered:\n";
    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}```
In this example, we continuously read string inputs from the user to add to a vector of fruits until the user types 'end'. Then we print out all entered fruits.
