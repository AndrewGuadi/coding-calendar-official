std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. It resizes the vector to accommodate the new element and copies or moves the value into the new location. This method can be used with any data type that is compatible with vector, including user-defined types. The vector ensures that there is sufficient capacity to store the new element, and if not, it allocates more memory accordingly. Using this method helps in managing dynamic arrays easily, maintaining the order of elements, and allowing for efficient appends.

```cpp
#include 
#include 

int main() {
    std::vector pythonVersions;
    pythonVersions.push_back("Python 2.7");
    pythonVersions.push_back("Python 3.6");
    pythonVersions.push_back("Python 3.9");

    for (const auto &version : pythonVersions) {
        std::cout << version << std::endl;
    }
    return 0;
}```
