acos

The 'acos' method in Java is a static method of the Math class that computes the arc cosine (inverse cosine) of a given value. The input should be a double value in the range of -1.0 to 1.0, and the method returns the angle in radians whose cosine is the specified value. If the argument is outside the range [-1, 1], the result is NaN (not-a-number). This method is useful in various mathematical calculations where you need to determine the angle from a cosine value.

import math

# Calculate the angle whose cosine is 0.5
cos_value = 0.5
angle_in_radians = math.acos(cos_value)

# Convert radians to degrees
angle_in_degrees = math.degrees(angle_in_radians)

print('The angle is:', angle_in_radians, 'radians or', angle_in_degrees, 'degrees')
