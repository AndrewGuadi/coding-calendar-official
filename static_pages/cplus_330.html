std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a vector. This method increases the vector's size by one and may cause a reallocation of memory if the current capacity is exceeded. It works efficiently with a time complexity of average O(1). The added element can be of any type that is compatible with the vector's type declaration.

# Creating a vector of strings
my_vector = []
# Adding elements to the vector using push_back
my_vector.append('element1')
my_vector.append('element2')
my_vector.append('element3')
# Displaying the contents of the vector
print(my_vector)  # Output: ['element1', 'element2', 'element3']
# Creating a vector of integers
my_vector = []
# Adding elements to the vector using push_back
my_vector.append(10)
my_vector.append(20)
my_vector.append(30)
# Displaying the contents of the vector
print(my_vector)  # Output: [10, 20, 30]
# Creating a vector of mixed types
my_vector = []
# Adding elements of different types to the vector using push_back
my_vector.append('hello')
my_vector.append(42)
my_vector.append(3.14)
# Displaying the contents of the vector
print(my_vector)  # Output: ['hello', 42, 3.14]
