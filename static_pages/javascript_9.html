charAt

The 'charAt' method in JavaScript is used to retrieve the character at a specified index from a string. The index is zero-based, meaning that the first character of the string is at index 0, the second character at index 1, and so on. If the specified index is out of range (i.e., less than 0 or greater than or equal to the string's length), 'charAt' will return an empty string.

```python
# Example 1: Get the first character of a string
my_string = 'Hello, World!'
first_char = my_string[0]  # Using Python's indexing
print(first_char)  # Output: 'H'
```
```python
# Example 2: Access a character from the middle of a string
my_string = 'Python Programming'
middle_char = my_string[7]  # 'P' is at index 7
print(middle_char)  # Output: 'P'
```
```python
# Example 3: Access a character with an out-of-range index
my_string = 'Out of Range'
out_of_range_char = my_string[20] if len(my_string) > 20 else ''  # Handling out-of-range
print(out_of_range_char)  # Output: ''
```
