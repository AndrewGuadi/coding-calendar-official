std::vector::push_back

The `std::vector::push_back` method in C++ adds a new element to the end of a vector, which is a dynamic array that can grow in size. When the new element is added, if the current capacity of the vector is not sufficient to accommodate the new element, it automatically increases its size. This method takes a single parameter, which is the value to be added to the vector. After the element is added, the size of the vector is incremented by one. It is worth noting that the added value will be copied into the vector, which means that if the value is a large object, it may incur a performance cost due to copying.

```cpp
#include 
#include 

int main() {
    std::vector strings;
    strings.push_back("Hello");
    strings.push_back("World");
    for (const auto& str : strings) {
        std::cout << str << ' '; // Output: Hello World 
    }
    return 0;
}
```
