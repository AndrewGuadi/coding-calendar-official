Add

The 'Add' method in C# is commonly used across various classes and collections to insert a new element into a collection or add a new entry to a data structure. It is a way to append or include new data, such as adding an item to a list, adding a key-value pair to a dictionary, or adding an element to a set. The method typically accepts one or more parameters, which represent the item or items to be added. This method can usually be found in collections such as List, Dictionary, HashSet, and others. When using the 'Add' method, itâ€™s essential to ensure that the data structure allows duplicates or that the same key is not already present (in the case of a dictionary).

```python
class CustomList:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

my_list = CustomList()
my_list.add('Python')
my_list.add('Java')
print(my_list.items)  # Output: ['Python', 'Java']
```
```python
class CustomDictionary:
    def __init__(self):
        self.items = {}

    def add(self, key, value):
        self.items[key] = value

my_dict = CustomDictionary()
my_dict.add('language', 'Python')
my_dict.add('version', '3.10')
print(my_dict.items)  # Output: {'language': 'Python', 'version': '3.10'}
```
```python
class CustomSet:
    def __init__(self):
        self.items = set()

    def add(self, item):
        self.items.add(item)

my_set = CustomSet()
my_set.add('Python')
my_set.add('Python')  # Adding duplicate
my_set.add('Java')
print(my_set.items)  # Output: {'Python', 'Java'}
```
