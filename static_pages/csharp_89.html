Add

The 'Add' method in C# is typically used to add an item to a collection, such as a list, dictionary, or other types of collections. It modifies the collection by including the new item at the end or into a specified key-value pair in the dictionary. This method is essential for dynamically managing data as it allows for expansion of the collection size. In generic collections like List, the item added can be of any type defined by T.

```python
items = []
# Using the Add method to add elements to the list
items.append('Python')
items.append('C#')
print(items)  # Output: ['Python', 'C#']
```
```python
user_data = {}
# Using the Add method to add key-value pairs in a dictionary
user_data['name'] = 'Alice'
user_data['age'] = 30
print(user_data)  # Output: {'name': 'Alice', 'age': 30}
```
```python
unique_numbers = set()
# Using the Add method to add elements to a set
unique_numbers.add(1)
unique_numbers.add(2)
unique_numbers.add(2)
print(unique_numbers)  # Output: {1, 2}
```
