upcase

The 'upcase' method in Ruby returns a copy of the string with all lowercase letters converted to uppercase. It does not modify the original string, ensuring that the original data remains unchanged. This method is particularly useful when you want to perform case-insensitive comparisons or present text in a standardized format.

def convert_to_uppercase(text):
    return text.upper()

# Example usage
result = convert_to_uppercase('hello world')
print(result)  # Output: HELLO WORLD
def shout(text):
    return text.upper() + '!!!'

# Example usage
result = shout('hello')
print(result)  # Output: HELLO!!!
def filter_and_upcase(words):
    return [word.upper() for word in words if len(word) > 3]

# Example usage
result = filter_and_upcase(['a', 'is', 'ruby', 'code'])
print(result)  # Output: ['RUBY', 'CODE']
