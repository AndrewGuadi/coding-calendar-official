upcase

The 'upcase' method in Ruby is used to convert all the characters in a string to uppercase. This method does not modify the original string but instead returns a new string with all characters transformed to uppercase. It's commonly used in scenarios where case-insensitivity is important, such as when comparing strings or when formatting output for better readability.

```
# Example of using upcase to convert user input to uppercase
user_input = 'hello world'
uppercased = user_input.upcase
print(uppercased)  # Output: HELLO WORLD
```
```
# Example of using upcase in string comparisons
string_a = 'Ruby'
string_b = 'ruby'
if string_a.upcase == string_b.upcase:
    print('Strings are equal ignoring case.')  # Output: Strings are equal ignoring case.
```
```
# Example of formatting string output
message = 'welcome to the coding platform'
formatted_message = message.upcase
print(formatted_message)  # Output: WELCOME TO THE CODING PLATFORM
```
