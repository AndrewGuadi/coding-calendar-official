Add

The 'Add' method is a commonly used function in various collections and data structures in C#. It is generally used to insert an element into a collection, such as a List, Dictionary, or a Set. The method modifies the collection by adding a new item without returning a value. In the context of Lists and Dictionaries, the element is appended to the end of the List or added as a key-value pair in the Dictionary.

```python
# Example of using the Add method with a List in C#
# This code declares a list of strings and adds items to it.

class Program:
    def __init__(self):
        self.my_list = []

    def add_to_list(self, item):
        self.my_list.append(item)  # Using the Add method equivalent

# Usage
program = Program()
program.add_to_list('Python')
program.add_to_list('C#')
program.add_to_list('Java')
print(program.my_list)  # Output: ['Python', 'C#', 'Java']
```
```python
# Example of using the Add method with a Dictionary in C#
# This code creates a dictionary and adds key-value pairs to it.

class Program:
    def __init__(self):
        self.my_dict = {}

    def add_to_dict(self, key, value):
        self.my_dict[key] = value  # Using the Add method equivalent

# Usage
program = Program()
program.add_to_dict('language', 'Python')
program.add_to_dict('year', 2023)
print(program.my_dict)  # Output: {'language': 'Python', 'year': 2023}
```
```python
# Example of using the Add method with a Set in C#
# This code demonstrates how to add unique items to a set.

class Program:
    def __init__(self):
        self.my_set = set()

    def add_to_set(self, item):
        self.my_set.add(item)  # Using the Add method equivalent

# Usage
program = Program()
program.add_to_set('Python')
program.add_to_set('C#')
program.add_to_set('Python')  # Duplicates will not be added
print(program.my_set)  # Output: {'Python', 'C#'}
```
