abs

The 'abs' method in Java is used to calculate the absolute value of a given number. It is available in the java.lang.Math class and can be applied to various numeric types such as int, long, float, and double. The absolute value of a number is its distance from zero on the number line, disregarding the sign. For example, both -5 and 5 have an absolute value of 5.

import java.lang.Math;

public class AbsoluteValueExample {
    public static void main(String[] args) {
        int number = -10;
        int absValue = Math.abs(number);
        System.out.println("The absolute value of " + number + " is: " + absValue);
    }
}
