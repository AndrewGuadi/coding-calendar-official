ascii

The `ascii()` function in Python takes an object as an argument and returns a string that represents the object's printable ASCII characters. Non-ASCII characters are escaped using the Unicode escape sequence. This function is particularly useful for safely displaying strings that may contain non-ASCII characters, making them easily readable in an ASCII-only format.

def display_ascii(string):
    return ascii(string)

print(display_ascii('Hello, World!'))  # Output: 'Hello, World!'
print(display_ascii('Café'))  # Output: 'Caf\xe9'

def json_safe_string(data):
    import json
    return json.dumps(data, ensure_ascii=False)

print(json_safe_string({'name': 'José', 'city': 'São Paulo'}))  # Output: '{"name": "José", "city": "São Paulo"}'

def filter_non_ascii(lst):
    return [ascii(item) for item in lst]

print(filter_non_ascii(['Hello', 'Café', 'привет']))  # Output: ['Hello', 'Caf\xe9', '\u043f\u0440\u0438\u0432\u0435\u0442']
