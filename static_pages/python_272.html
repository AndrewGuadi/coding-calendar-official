ascii

The ascii() method in Python returns a string containing a printable representation of an object, but in a format that only contains ASCII characters. Non-ASCII characters are escaped using their Unicode code points. This method is useful when you want to create a readable string representation of data that might contain non-ASCII characters, such as strings with special characters or characters from other languages.

text = 'Hello, π!'    
result = ascii(text)    
print(result)    # Output: 'Hello, \u03c0!'
text = 'Café Münsterländer'    
result = ascii(text)    
print(result)    # Output: 'Caf\u00e9 M\u00fcnsterl\u00e4nder'
text = '你好'    
result = ascii(text)    
print(result)    # Output: '\u4f60\u597d'
