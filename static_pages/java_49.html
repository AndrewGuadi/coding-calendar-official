abs

The 'abs' method in Java is used to compute the absolute value of a numeric value. It can be applied to different data types such as int, long, float, and double, returning the magnitude of the value without its sign. If the value is positive, the method will return it unchanged; if the value is negative, it will return the positive equivalent. This method is particularly useful in mathematical calculations where the sign of the number is not needed, such as in distance calculations or when determining the non-negative difference between two values.

## Calculating the absolute value of an integer

```java
public class Main {
    public static void main(String[] args) {
        int negativeInt = -10;
        int absoluteValue = Math.abs(negativeInt);
        System.out.println("The absolute value of " + negativeInt + " is: " + absoluteValue);
    }
}
```
## Using abs with floating-point numbers

```java
public class Main {
    public static void main(String[] args) {
        double negativeDouble = -15.75;
        double absoluteValue = Math.abs(negativeDouble);
        System.out.println("The absolute value of " + negativeDouble + " is: " + absoluteValue);
    }
}
```
## Calculating absolute difference

```java
public class Main {
    public static void main(String[] args) {
        int x = 5;
        int y = -8;
        int absoluteDifference = Math.abs(x - y);
        System.out.println("The absolute difference between " + x + " and " + y + " is: " + absoluteDifference);
    }
}
```
