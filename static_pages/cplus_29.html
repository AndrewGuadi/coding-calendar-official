std::vector::push_back

The `push_back` method is used to add a new element to the end of a vector in C++. This function increases the size of the vector by one and copies or moves the element to the newly allocated position at the end of the vector. If the vector's capacity is exceeded, it automatically resizes the vector to accommodate the new element. This method is essential for dynamically growing an array-like structure provided by `std::vector`.

```cpp
#include 
#include 

int main() {
    std::vector programmingLanguages;
    programmingLanguages.push_back("Python");
    programmingLanguages.push_back("C++");
    programmingLanguages.push_back("JavaScript");

    for (const auto &lang : programmingLanguages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
```cpp
#include 
#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i * 10);
    }

    for (const auto &num : numbers) {
        std::cout << num << " "; // Outputs: 10 20 30 40 50 
    }
    return 0;
}
```
