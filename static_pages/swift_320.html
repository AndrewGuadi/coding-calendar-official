uppercased

The 'uppercased' method in Swift is used to convert all the characters of a given string to uppercase. It is a part of the Swift Standard Library and applies to String types. This method returns a new string with all the original string's characters transformed to their uppercase counterparts, making it as useful for formatting, case normalization, or preparing user input for comparison tasks.

def convert_to_uppercase(input_string):
    return input_string.upper()

# Example usage
input_str = 'hello world'
output_str = convert_to_uppercase(input_str)
# output_str will be 'HELLO WORLD'
def shout(sentence):
    return sentence.upper() + '!!!'

# Example usage
response = shout('i am excited')
# response will be 'I AM EXCITED!!!'
def normalize_input(user_input):
    return user_input.strip().upper()

# Example usage
user_name = normalize_input('   alice   ')
# user_name will be 'ALICE'
