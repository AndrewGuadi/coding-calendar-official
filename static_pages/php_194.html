strlen

The 'strlen' function in PHP is used to get the length of a string. It counts the number of characters in the string, including spaces and special characters, but it does not count NULL bytes. This function is very useful for validating input data, checking if strings are empty, or simply knowing the size of a string for formatting purposes.

def get_string_length(input_string):
    return len(input_string)

# Example usage:
my_string = 'Hello, World!'
length = get_string_length(my_string)
print(f'The length of the string is: {length}')  # Output: The length of the string is: 13
def is_string_empty(input_string):
    return len(input_string) == 0

# Example usage:
my_string = ''
is_empty = is_string_empty(my_string)
print(f'Is the string empty? {is_empty}')  # Output: Is the string empty? True
def string_char_count(input_string):
    print(f'The string "{input_string}" has {len(input_string)} characters.')

# Example usage:
my_string = 'Python programming'
string_char_count(my_string)  # Output: The string "Python programming" has 18 characters.
