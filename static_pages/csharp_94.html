Add

The 'Add' method is commonly used in C# to insert an element into a collection, such as a list, dictionary, or any class implementing the ICollection interface. When using the 'Add' method, you typically provide a single parameter, which is the element to be added. This method modifies the collection in place and can be used to maintain dynamic lists, manage key-value pairs, and add new data entries efficiently.

```python
# Example: Adding elements to a list using C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List names = new List();
        // Using the 'Add' method to add names to the list
        names.Add("Alice");
        names.Add("Bob");
        names.Add("Charlie");

        // Displaying the names
        foreach (var name in names)
        {
            Console.WriteLine(name);
        }
    }
}
```
```python
# Example: Adding key-value pairs to a dictionary using C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary capitals = new Dictionary();
        // Using the 'Add' method to add capitals to the dictionary
        capitals.Add("USA", "Washington, D.C.");
        capitals.Add("France", "Paris");
        capitals.Add("Japan", "Tokyo");

        // Displaying the capitals
        foreach (var kvp in capitals)
        {
            Console.WriteLine($"The capital of {kvp.Key} is {kvp.Value}. ");
        }
    }
}
```
```python
# Example: Using Add method in a custom collection class in C#

using System;
using System.Collections.Generic;

class CustomCollection
{
    private List numbers = new List();

    public void Add(int number)
    {
        numbers.Add(number);
    }

    public void Display()
    {
        foreach (var num in numbers)
        {
            Console.WriteLine(num);
        }
    }
}

class Program
{
    static void Main()
    {
        CustomCollection collection = new CustomCollection();
        // Adding numbers to the custom collection
        collection.Add(10);
        collection.Add(20);
        collection.Add(30);

        // Displaying the numbers
        collection.Display();
    }
}
```
