std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. This operation increases the size of the vector by one and adds the specified value at the last position. If the vector's capacity is exceeded, it automatically reallocates memory to accommodate the new element. This method is commonly used when you want to dynamically append elements to a list-like structure during runtime.

```cpp
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    // Print the vector contents
    for(const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```

```cpp
int main() {
    std::vector techStack;
    techStack.push_back("Django");
    techStack.push_back("Flask");
    techStack.push_back("FastAPI");

    // Initialize a counter for counting occurrences of specified frameworks
    int count = 0;
    for(const auto& tech : techStack) {
        if (tech == "Flask") {
            count++;
        }
    }
    std::cout << "Flask appears: " << count << " times" << std::endl;
    return 0;
}
```
