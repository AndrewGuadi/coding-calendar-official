std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that allows you to add a new element to the end of a vector. This method modifies the vector by increasing its size by one and inserting the new element at the last position. If the underlying array needs to be resized to accommodate additional elements, the vector automatically handles memory allocation and copying of existing elements to the new storage location. This method is commonly used when you want to incrementally build a vector as you gather data during program execution.

```cpp
#include 
#include 

int main() {
    std::vector pythonCodes;
    pythonCodes.push_back("print('Hello, World!')");
    pythonCodes.push_back("for i in range(5): print(i)");

    for (const auto &code : pythonCodes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
