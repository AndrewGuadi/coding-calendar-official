charAt

The `charAt` method in JavaScript is used to retrieve a character from a specified index in a string. The index is zero-based, meaning that the first character of the string is at index 0. If the index is out of bounds (i.e., negative or greater than the string length), the method returns an empty string. This method is useful for accessing specific characters in a string for various operations such as parsing, checking for specific characters, or even modifying the string based on certain conditions.

my_string = 'abcde'; char_at_0 = my_string.charAt(0); // Retrieves 'a'
my_string = 'Hello, World!'; char_at_7 = my_string.charAt(7); // Retrieves 'W'
my_string = 'Python'; char_at_6 = my_string.charAt(6); // Since index is out of range, it returns '' (empty string)
