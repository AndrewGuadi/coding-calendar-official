std::vector::push_back

The `push_back` method is used to add a new element to the end of a `std::vector` in C++. This method increases the size of the vector by one and constructs the new element at the end. The `push_back` method can accept various data types, and in this context, Python code snippets will be used as values to demonstrate its functionality. It is important to note that when adding complex data types, the vector must be capable of storing them properly (e.g., using `std::vector` for strings).

```cpp
#include 
#include 
#include 

int main() {
    std::vector examples;
    examples.push_back("print('Hello, World!')");
    examples.push_back("for i in range(5): pass");
    examples.push_back("def hello(): return 'Hello'");

    for (const auto& example : examples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
