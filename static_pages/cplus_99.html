std::vector::push_back

The method `push_back` is a member function of the C++ Standard Library's `std::vector` class. It is used to add a new element to the end of a vector, increasing its size by one. This method allows dynamic resizing and is a way to store data in a contiguous memory location. When an element is pushed back, the vector's capacity is checked, and if it is at full capacity, the vector will automatically allocate more memory to accommodate additional elements. The time complexity for `push_back` is amortized constant time, O(1).

```python
# C++ Code
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");
    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}

# Example Output: 
# Python
# Java
# C++
```

int main() {
    std::vector fruits;
    std::string fruit;
    while (true) {
        std::cout << "Enter a fruit (or type 'quit' to stop): ";
        std::cin >> fruit;
        if (fruit == "quit") {
            break;
        }
        fruits.push_back(fruit);
    }
    std::cout << "You entered: ";
    for (const auto& f : fruits) {
        std::cout << f << " ";
    }
    std::cout << std::endl;
    return 0;
}

# Example Output:
# Enter a fruit (or type 'quit' to stop): Apple
# Enter a fruit (or type 'quit' to stop): Banana
# Enter a fruit (or type 'quit' to stop): quit
# You entered: Apple Banana
