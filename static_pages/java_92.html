abs

The 'abs' method in Java is a static method that belongs to the Math class. It is used to return the absolute value of a given number. The absolute value of a number is its non-negative value, regardless of its sign. This method is commonly used when you need to ensure that a value is positive, or when performing calculations where negative values may lead to errors or meaningful discrepancies. The 'abs' method can handle various types of numerical data types, including integers (int), floating-point numbers (float and double), and long integers (long).
