std::vector::push_back

The method 'std::vector::push_back' is used in C++ to add a new element to the end of a vector. A vector is a dynamic array that can grow in size, and 'push_back' allows you to efficiently append an item by taking advantage of the vector's underlying dynamic array management. When using this method, the vector automatically handles memory reallocations if necessary, ensuring that the new element is always added at the end of the collection. This method does not return any value and has an average time complexity of O(1), although it can be O(n) in cases where a reallocation is needed.

```python
# Initialize an empty list to represent a vector
vector = []

# Using push_back equivalent to add elements
vector.append('element_1')
vector.append('element_2')

print(vector)  # Output: ['element_1', 'element_2']
```
```python
# Initialize a list with some elements
vector = ['element_1', 'element_2']

# Add more elements using append (equivalent to push_back)
vector.append('element_3')
vector.append('element_4')

print(vector)  # Output: ['element_1', 'element_2', 'element_3', 'element_4']
```
```python
# Initialize an empty list
vector = []

# Function to add multiple elements
def add_elements(vec, elements):
    for element in elements:
        vec.append(element)

add_elements(vector, ['element_1', 'element_2', 'element_3'])

print(vector)  # Output: ['element_1', 'element_2', 'element_3']
```
