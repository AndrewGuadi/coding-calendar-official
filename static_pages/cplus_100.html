std::vector::push_back

The `push_back` method in C++ is used to append a new element at the end of a vector. This operation increases the size of the vector by one. The method takes a single argument, which is the value to be added. If the vector's capacity is reached, it will automatically allocate more memory to accommodate the new element.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("first item");
    myVector.push_back("second item");
    myVector.push_back("third item");

    for (const auto &item : myVector) {
        std::cout << item << std::endl;
    }
    return 0;
}
```
