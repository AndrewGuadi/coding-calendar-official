std::vector::push_back

The `push_back` function is a member of the `std::vector` class in C++. It is used to add an element to the end of a vector, increasing its size by one. This function automatically handles memory allocation to accommodate the new element. It works efficiently and is commonly used when the final size of the vector is not known at compile time. However, it's worth noting that frequent calls to `push_back` may lead to performance implications due to potential reallocations.

# Example of using std::vector::push_back to store integers
#include 
#include 

int main() {
    std::vector nums;
    
    // Using push_back to add elements
    nums.push_back(1);
    nums.push_back(2);
    nums.push_back(3);

    // Display the contents of the vector
    for (int num : nums) {
        std::cout << num << " "; // Output: 1 2 3
    }
    return 0;
}
