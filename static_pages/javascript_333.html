charAt

The 'charAt' method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning that the first character of the string is at index 0, the second character is at index 1, and so on. If the specified index is out of range, 'charAt' returns an empty string. This method is useful for extracting individual characters from a string for processing, comparison, or display.

input_string = "Hello, World!"
character_at_7 = input_string.charAt(7)
# character_at_7 will be 'W'
input_string = "Python Programming"
character_at_0 = input_string.charAt(0)
# character_at_0 will be 'P'
input_string = "JavaScript"
character_at_5 = input_string.charAt(5)
# character_at_5 will be 'S'
