Add

The 'Add' method in C# is commonly used to add an element to a collection, such as a list, dictionary, or any other enumerable container. This method increases the size of the collection by one and allows users to insert new data at the end of the existing collection for list-type structures. For dictionaries, it associates a key with a specific value.

from collections import defaultdict

d = defaultdict(list)
d['fruits'].append('apple')
d['fruits'].append('banana')
print(d)  # Output: defaultdict(, {'fruits': ['apple', 'banana']})
fruits = []
fruits.append('orange')
fruits.append('grape')
print(fruits)  # Output: ['orange', 'grape']
fruits_dict = {}
fruits_dict['banana'] = 'yellow'
fruits_dict['apple'] = 'red'
print(fruits_dict)  # Output: {'banana': 'yellow', 'apple': 'red'}
