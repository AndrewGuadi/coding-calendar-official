Add

The 'Add' method is commonly used in collections, such as lists, dictionaries, and sets in C#. It allows you to append a new element to the collection. In lists, it adds the element to the end of the list; in dictionaries, it adds a new key-value pair; and in sets, it adds a unique element. This method is crucial for dynamic data management, enabling developers to modify collections during runtime.

```python
# Example of using Add in a List
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list) # Output: ['Hello', 'World']
```
```python
# Example of using Add in a Dictionary
my_dict = {}
my_dict['name'] = 'Alice'
my_dict['age'] = 30
print(my_dict) # Output: {'name': 'Alice', 'age': 30}
```
```python
# Example of using Add in a Set
my_set = set()
my_set.add('Apple')
my_set.add('Banana')
print(my_set) # Output: {'Apple', 'Banana'}
```
