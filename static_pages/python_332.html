ascii

The `ascii()` function in Python returns a string containing a printable representation of an object, escaping the non-ASCII characters in the string so that they can be read as ASCII. This function is particularly useful for converting objects that may have non-ASCII characters into a format that can be understood in a more limited character set. This is different from `repr()`, which returns a string that would yield the same object when passed to `eval()`. Note that it outputs the string in a format suitable for ASCII encoding by replacing the non-ASCII characters with Unicode escape sequences.

data = 'Café'
print(ascii(data))  # Output: 'Caf\xe9'
data = 'Hello, 世界'
print(ascii(data))  # Output: 'Hello, \u4e16\u754c'
data = {'name': 'Björk', 'age': 55}
print(ascii(data))  # Output: "{'name': 'Bj\xf6rk', 'age': 55}"
