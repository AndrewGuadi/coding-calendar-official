abs

The abs method in Java is used to return the absolute value of a given number. It is a static method that belongs to the Math class. The absolute value of a number is its non-negative value regardless of its sign. The method can handle integers, floating-point numbers (floats and doubles), and even complex data types. It is commonly used in mathematical calculations where the magnitude of a number is needed, without regard to its direction or sign.

def calculate_absolute_value(num):
    return abs(num)

print(calculate_absolute_value(-10))  # Output: 10
def positive_distance(x, y):
    return abs(x - y)

print(positive_distance(5, 10))  # Output: 5
def temperature_difference(temp1, temp2):
    return abs(temp1 - temp2)

print(temperature_difference(-20.5, 15.5))  # Output: 36.0
