std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of the vector. This method dynamically increases the size of the vector and requires that the vector has enough memory allocated to accommodate the new element. If the vector's capacity is reached, the method will automatically allocate a larger block of memory to accommodate additional elements. This method can be used for any data type that the vector holds.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("first");
    vec.push_back("second");
    vec.push_back("third");

    for (const auto& str : vec) {
        std::cout << str << std::endl;
    }
    return 0;
}
```

int main() {

    std::string values[] = {"apple", "banana", "cherry"};

    for (const auto& val : values) {
        vec.push_back(val);
    }

    for (const auto& fruit : vec) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
