std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a vector. A vector is a dynamic array that can resize itself automatically when an element is added or removed. 'push_back' increases the size of the vector by one and initializes the added element with the provided value. If necessary, the capacity of the vector is increased to accommodate the new element.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Python");
    myVector.push_back("Java");
    myVector.push_back("C++");

    for (const auto& lang : myVector) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
