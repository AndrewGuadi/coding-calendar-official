std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class. It is used to add a new element at the end of the vector, effectively increasing its size by one. This method dynamically resizes the vector if necessary to accommodate the new element. It is particularly useful for building a list of elements in a flexible manner. The performance of `push_back` is generally amortized constant time, but can take longer if the vector needs to be resized to accommodate the new element. In this method, the argument can be of any type that is stored in the vector. For example, if the vector is of type `std::vector`, the argument should be an integer. Usage examples include building a list of elements or accumulating results in competitive programming.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");

    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}```, This example initializes an empty vector of strings, adds three fruit names using `push_back`, and then prints each fruit name to the console.

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i * 10);
    }

    for (const auto& number : numbers) {
        std::cout << number << " ";
    }
    return 0;
}```, This example creates an empty vector of integers and uses a loop to add multiples of 10 to the vector, finally printing the values.
