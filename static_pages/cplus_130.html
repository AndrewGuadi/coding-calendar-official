std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++. It is used to add an element to the end of the vector. Each time an element is added, if the current capacity of the vector is exceeded, the vector allocates new memory, copies existing elements to the newly allocated space, and then adds the new element. This operation could involve reallocating the buffer, which can be an expensive operation in terms of performance if done repeatedly on large vectors.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Adding 'Hello' to the vector
    myVector.push_back("World"); // Adding 'World' to the vector
    
    for(const auto& str : myVector) {
        std::cout << str << " "; // Output should display: Hello World
    }
    return 0;
}
```
