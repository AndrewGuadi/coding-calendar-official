uppercased

The 'uppercased' method is a String method in Swift that returns a new string with all the characters converted to uppercase. This method does not modify the original string but produces a new instance. It is useful for normalizing text input, ensuring case-insensitivity in comparisons, or displaying strings in a specific format.

swift_str = "hello world"
uppercased_str = swift_str.uppercased()
print(uppercased_str)  # Output: HELLO WORLD
greeting = "Good Morning"
uppercased_greeting = greeting.uppercased()
print(uppercased_greeting)  # Output: GOOD MORNING
phrase = "This is a Test"
uppercased_phrase = phrase.uppercased()
print(uppercased_phrase)  # Output: THIS IS A TEST
