std::vector::push_back

The `push_back` method is a member function of the `std::vector` class in C++. It is used to add an element to the end of the vector, thereby increasing the container's size by one. The element is copied or moved into the vector. This method may cause reallocation of the internal data structure to accommodate the new element, if the current capacity is exceeded. `push_back` works efficiently on most occasions, and it's a commonly used method for dynamically managing collections of data in C++. 

// Example usage of std::vector::push_back
#include 
#include 

int main() {
    std::vector myVector;
    
    // Using push_back to add elements to the vector
    myVector.push_back("Python");
    myVector.push_back("Java");
    myVector.push_back("C++");
    
    // Display the elements in the vector
    for (const auto& lang : myVector) {
        std::cout << lang << std::endl;
    }
    return 0;
}
// Example usage of std::vector::push_back with integer values
#include 
#include 

int main() {
    std::vector numbers;
    
    // Adding integer values
    for (int i = 1; i <= 5; i++) {
        numbers.push_back(i * 10);
    }
    
    // Display the numbers
    for (const auto& number : numbers) {
        std::cout << number << std::endl;
    }
    return 0;
}
