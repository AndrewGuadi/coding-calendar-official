uppercased

The `uppercased` method is a String method in Swift that returns a new string, where all the characters in the original string are converted to uppercase. This method does not modify the original string but creates a new one with the desired transformation. It is useful for formatting text, ensuring that user input adheres to certain casing standards, and for cases where textual comparison is needed regardless of case. The method is generally called on a String instance and returns an uppercase String.

def to_uppercase(input_string):
    return input_string.upper()

result = to_uppercase('hello world')
print(result)  # Output: 'HELLO WORLD'
def emphasize_text(input_string):
    return input_string.upper() + '!!!'

result = emphasize_text('warning')
print(result)  # Output: 'WARNING!!!'
def format_name(first_name, last_name):
    return (first_name + ' ' + last_name).upper()

result = format_name('john', 'doe')
print(result)  # Output: 'JOHN DOE'
