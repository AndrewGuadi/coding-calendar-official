std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector, increasing its size by one. This method is particularly useful for building dynamic arrays that can grow in size as needed. When using push_back, if the capacity of the vector is exceeded, the vector automatically reallocates memory to accommodate the new element. The performance of push_back is amortized to O(1), meaning that while individual calls can take longer during reallocation, the average time spent per call remains constant over multiple calls. It's important to note that push_back requires that the object being pushed is either copyable or movable if it is not a primitive type.

In this example, we create a vector to store different programming languages. We use push_back to add languages and then print the contents of the vector.

#include 
#include 

int main() {
    std::vector languages;
    
    languages.push_back("Python");
    languages.push_back("C++");
    languages.push_back("Java");
    
    std::cout << "The vector contains: ";
    for (const auto& lang : languages) {
        std::cout << lang << " ";
    }
    std::cout << std::endl;
    return 0;
}
In this example, we demonstrate how to build a vector of integers by using push_back to add values one by one and then sum them up.

#include 
#include 

int main() {
    std::vector numbers;
    int sum = 0;
    
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i);
    }
    
    for (const auto& num : numbers) {
        sum += num;
    }
    std::cout << "The sum of the numbers is: " << sum << std::endl;
    return 0;
}

        names.push_back(input);

    }
    std::cout << std::endl;
