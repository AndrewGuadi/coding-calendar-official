strlen

The 'strlen' function in PHP is used to count the number of bytes in a string. It returns the length of the string as an integer. This function is particularly useful when you need to validate the length of input data, such as passwords or usernames, or when you want to manipulate strings based on their length.

string_data = 'Hello, World!'
length = len(string_data)
print(f'The length of the string is: {length}')  # Output: The length of the string is: 13
string_message = 'PHP is great'
message_length = len(string_message)
if message_length > 10:
    print('The message is longer than 10 characters.')
else:
    print('The message is 10 characters or shorter.')  # Output: The message is longer than 10 characters.
user_input = 'abc'
if len(user_input) < 5:
    print('Input must be at least 5 characters long.')
else:
    print('Valid input.')  # Output: Input must be at least 5 characters long.
