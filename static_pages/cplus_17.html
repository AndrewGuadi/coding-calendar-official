std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector. When an element is pushed back, the size of the vector increases by one. If there is not enough space in the vector to accommodate the new element, the vector will automatically allocate more memory to ensure that the push_back operation can be performed. The new element is added to the last position of the vector. The time complexity of this operation is amortized O(1).

```cpp
#include 
#include 

int main() {
    std::vector pythonVersions;
    pythonVersions.push_back("Python 3.6");
    pythonVersions.push_back("Python 3.7");
    pythonVersions.push_back("Python 3.8");

    for (const auto& version : pythonVersions) {
        std::cout << version << std::endl;
    }
    return 0;
}
```
