abs

The 'abs' method in Java is used to return the absolute value of a given number. It can take integers, floating-point numbers, and other numeric types as arguments. The absolute value of a number is its value without regard to its sign, meaning it will always return a non-negative number regardless of whether the input number is negative or positive.

The absolute value of -5 can be found using: abs_value = abs(-5) which returns 5.
For a floating-point number like -3.14, you can use: abs_value = abs(-3.14) to get 3.14.
If you want to find the absolute value of a large negative integer, say -100000, you can do: abs_value = abs(-100000) resulting in 100000.
