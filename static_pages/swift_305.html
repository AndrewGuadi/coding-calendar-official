uppercased

The 'uppercased' method in Swift is a string method that returns a new string with all the characters transformed to uppercase. It is commonly used when there's a need to standardize string format for comparison, display, or processing purposes. This method does not alter the original string; instead, it creates a new string with the uppercased version. This feature is particularly useful in scenarios where case sensitivity can affect logic or result in string mismatches. It's important to note that this method can handle localization and will use the appropriate rules for the current language settings.

original_string = 'hello world'
uppercased_string = original_string.uppercased()

print(uppercased_string)  # Output: 'HELLO WORLD'
username = 'johnDoe'
normalized_username = username.uppercased()

if normalized_username == 'JOHNDOE':
    print('Username matches!')
else:
    print('Username does not match!')
greeting = 'good morning'
shouted_greeting = greeting.uppercased()

print('Saying:', shouted_greeting)  # Output: 'SAYING: GOOD MORNING'
