std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element at the end of a vector. This function increases the size of the vector by one and initializes the new element with the given value. If the vector's capacity is reached, it automatically reallocates memory to accommodate the new element, ensuring that the vector can grow dynamically. This method does not return a value and is commonly used for managing dynamic collections of data.

```cpp
#include 
#include 

int main() {
    std::vector words;
    words.push_back("Hello"); // Adding 'Hello' to the vector
    words.push_back("World"); // Adding 'World' to the vector
    
    for (const auto& word : words) {
        std::cout << word << ' '; // Output: Hello World
    }
    return 0;
}
```
```cpp
#include 
#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i); // Adding integers 1 to 5
    }
    
    std::cout << "Numbers: ";
    for (const auto& number : numbers) {
        std::cout << number << ' '; // Output: Numbers: 1 2 3 4 5
    }
    return 0;
}
```
