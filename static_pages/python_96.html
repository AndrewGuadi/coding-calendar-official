abs

The `abs` method in Python returns the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, without considering direction. This means that `abs` converts negative numbers to their positive counterparts and leaves positive numbers unchanged. The `abs` method can be used with integers, floating-point numbers, and complex numbers. For complex numbers, it returns the magnitude (or modulus).

# Example 1: Using abs with an integer
number = -10
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')
# Example 2: Using abs with a floating-point number
number = -7.25
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')
# Example 3: Using abs with a complex number
number = 3 + 4j
absolute_value = abs(number)
print(f'The magnitude of {number} is {absolute_value}')
