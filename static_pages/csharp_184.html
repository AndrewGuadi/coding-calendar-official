Add

The 'Add' method is a common function used in various data structures in C#. It allows for the insertion of a new element into a collection, such as a List, Dictionary, or Set. When using the 'Add' method, the element is appended to the end of the collection or added based on the rules of the specific collection type. This method typically requires one parameter that specifies the new element to be added.

```python
# Example of using Add method with a List in C#
# Create a new List
numbers = []

# Use Add method to insert numbers
numbers.append(1)  # Adding first element
numbers.append(2)  # Adding second element

print(numbers)  # Output: [1, 2]
```
```python
# Example of using Add method with a Dictionary in C#
# Create a new Dictionary
person = {}

# Use Add method to insert key-value pairs
person['name'] = 'Alice'  # Adding name
person['age'] = 30       # Adding age

print(person)  # Output: {'name': 'Alice', 'age': 30}
```
```python
# Example of using Add method with a HashSet in C#
# Create a new set
fruits = set()

# Use Add method to insert fruits
fruits.add('apple')  # Adding first fruit
fruits.add('banana')  # Adding second fruit

print(fruits)  # Output: {'apple', 'banana'}
```
