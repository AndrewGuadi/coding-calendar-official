uppercased

The `uppercased` method is a function in Swift that transforms all the characters in a specified string to their uppercase equivalents. This method is called on a string instance and returns a new string containing the original string's characters in uppercase. It does not modify the original string, as strings in Swift are value types. This method is particularly useful when you need to standardize string input for comparisons or for display purposes, where consistent casing is important.

input_string = 'hello world'
uppercased_string = input_string.upper()
input_string = 'Python Programming'
uppercased_string = input_string.upper()
input_string = 'swift language'
uppercased_string = input_string.upper()
