charAt

The `charAt()` method in JavaScript is a built-in function that returns the character at a specified index (position) in a string. The index is zero-based, which means that the first character has an index of 0. If the index is out of range (less than 0 or greater than the string length), `charAt()` will return an empty string.

In the example below, we use `charAt()` to get the first character of the string 'hello'. The code will return 'h'.

```python
string = 'hello'
char_at_first = string[0]  # Accessing the character using index directly in Python
print(char_at_first)  # Output: 'h'
```
This example shows how to access a character from the string 'world' using its index. Here, we get the third character, which is 'r'.

```python
string = 'world'
char_at_third = string[2]  # Again using index directly, as Python uses 0-based indexing
print(char_at_third)  # Output: 'r'
```
In this example, we demonstrate accessing a character using an invalid index. Since Python does not throw an error for out-of-bounds access in this manner, it will simply return 'None' if an index is truly out of range.

```python
string = 'hello'
char_at_invalid = string[10] if len(string) > 10 else None  # Check if index 10 is valid
print(char_at_invalid)  # Output: None
```
