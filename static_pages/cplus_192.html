std::vector::push_back

The 'std::vector::push_back' method is used to add a new element at the end of a vector in C++. This method resizes the vector if it has reached its capacity to accommodate the new element. It performs the following operations: If the current capacity is reached, it allocates new memory for more elements, copies the existing elements to the new memory, and then adds the new element. This makes it a dynamic way to handle data collections of varying sizes. The complexity of this operation is amortized constant time, meaning that while it may take longer during the resize operation, on average, it is quick.

```cpp
#include 
#include 

int main() {
    std::vector codeExamples;

    // Adding Python code examples using push_back
    codeExamples.push_back("print('Hello, World!')");  // Example 1
    codeExamples.push_back("for i in range(5): print(i)"); // Example 2
    codeExamples.push_back("def add(a, b): return a + b"); // Example 3

    // Displaying the contents of the vector
    for(const auto& example : codeExamples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
