upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method does not modify the original string but returns a new string with all uppercase letters. This can be particularly useful for normalizing input data or for making string comparisons case-insensitive.

original_string = 'hello world'
uppercase_string = original_string.upcase
print(uppercase_string)  # Output: 'HELLO WORLD'
name = 'ada lovelace'
formatted_name = name.upcase
print(formatted_name)  # Output: 'ADA LOVELACE'
sentence = 'Ruby is fun!'
upper_sentence = sentence.upcase
print(upper_sentence)  # Output: 'RUBY IS FUN!'
