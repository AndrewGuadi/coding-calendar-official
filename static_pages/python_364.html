ascii

The `ascii()` function in Python is used to return a string containing a printable representation of an object. It is primarily used to represent non-ASCII characters in a string in a way that can be printed. If the object has a custom `__repr__()` method, that will be called to generate the string. The output of the `ascii()` function is the same as the built-in `repr()` function, but with non-ASCII characters escaped using the ` `, ` `, `\`, etc. notation and is enclosed in quotes.
