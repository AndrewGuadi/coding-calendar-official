Add

The 'Add' method in C# is commonly used in collections, such as lists or dictionaries, to insert a new element into the collection. For example, when dealing with a list of items, the Add method appends a new item to the end of the list. In a dictionary, it adds a new key-value pair if the key does not already exist. This method is essential for managing dynamic collections in C#.

```python
my_list = []
my_list.append('Item 1')
my_list.append('Item 2')
print(my_list)
# Output: ['Item 1', 'Item 2']
```
```python
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)
# Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
class ItemCollection:
    def __init__(self):
        self.items = []

    def add_item(self, item):
        self.items.append(item)

collection = ItemCollection()
collection.add_item('Item A')
collection.add_item('Item B')
print(collection.items)
# Output: ['Item A', 'Item B']
```
