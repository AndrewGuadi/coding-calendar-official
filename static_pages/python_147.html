ascii

The `ascii()` function in Python takes a single argument, which can be any object, and returns a string representation of that object. This string will contain only printable ASCII characters. Any non-ASCII characters in the original object will be replaced with their corresponding escape sequences.

print(ascii('Hello, world!'))  # Output: 'Hello, world!'
print(ascii('Café'))  # Output: 'Caf\xe9'
print(ascii(['apple', 'œuf', 'café']))  # Output: "['apple', '\xf5uf', 'caf\xe9']"
