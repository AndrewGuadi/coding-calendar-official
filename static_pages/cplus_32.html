std::vector::push_back

The 'push_back' method in C++ is a member function of the std::vector class that allows you to add a new element to the end of a vector. This method dynamically increases the size of the vector and allocates new memory if necessary. It takes a single argument, which is the element to be added. The element can be of any data type that the vector is capable of storing. After using 'push_back', the size of the vector increases by one, and the new element is accessible using the indexing operator.

# Example 1: Using std::vector to store a list of strings in C++.

#include 
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello, World!"); // Adding a string to the vector
    myVector.push_back("Learning C++");

    for (const auto& str : myVector) {
        std::cout << str << std::endl; // Output: Hello, World! Learning C++
    }
    return 0;
}
