charAt

The `charAt` method in JavaScript returns the character at a specified index (position) in a string. The index starts at 0, which means that the first character in the string is at index 0, the second character at index 1, and so on. If the index provided is out of bounds (e.g., negative or greater than the length of the string), it returns an empty string. This method is useful for retrieving specific characters from a string, especially when looping through the characters or validating certain conditions based on character positions.

def get_char_at_index(string, index):
    result = string[index] if 0 <= index < len(string) else ''
    return result

string_example = 'Hello, world!'
char_at_7 = get_char_at_index(string_example, 7)
print(char_at_7)  # This will print 'w'
def find_first_character(string):
    if len(string) > 0:
        return string[0]
    return ''

string_example = 'Example'
first_char = find_first_character(string_example)
print(first_char)  # This will print 'E'
def get_char_or_empty(string, index):
    if 0 <= index < len(string):
        return string[index]
    return ''

string_example = 'Python'
char_at_negative = get_char_or_empty(string_example, -1)
print(repr(char_at_negative))  # This will print an empty string ('')
