Add

The 'Add' method is commonly used in C# to add an item to a collection, such as a list or a dictionary. When using the 'Add' method on a List, it appends the item to the end of the list. In the case of a Dictionary, it adds a new key-value pair. The method alters the existing collection to include the new item and may throw exceptions if the item cannot be added (for example, if a duplicate key is added to a dictionary).

```python
# Example of adding items to a list
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
# Example of adding key-value pairs to a dictionary
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
# Example of adding multiple items to a list using a loop
my_list = []
for i in range(3):
    my_list.append(f'Item {i}')
print(my_list)  # Output: ['Item 0', 'Item 1', 'Item 2']
```
