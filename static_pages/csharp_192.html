Add

The 'Add' method is used to add an element to a collection, such as a list, dictionary, or any other data structure that supports adding elements. This method typically modifies the original collection by including the specified element. In C#, the 'Add' method is commonly found in collections like List, Dictionary, and HashSet.

```python
# Using a list to collect items
items = []

# Adding a python string representation of data
items.append('Example 1: Add a string')

print(items)  # Output: ['Example 1: Add a string']
```
```python
# Using a dictionary to store key-value pairs
my_dict = {}

# Adding a key-value pair (where value is a python code)
my_dict['key1'] = 'print("Hello from Example 2")'

# Accessing and executing the value
exec(my_dict['key1'])  # Output: Hello from Example 2
```
```python
# Using a set to store unique elements
my_set = set()

# Adding a code snippet as a unique element
my_set.add('print("This is example 3")')

# Iterating over the set and executing the code
for code in my_set:
    exec(code)  # Output: This is example 3
```
