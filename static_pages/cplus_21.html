std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. It resizes the vector as needed to accommodate the new element and ensures that the vector maintains its capacity. This method is commonly used to dynamically increase the size of the vector while storing data continuously.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    
    // Pushing back multiple fruit names
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");
    
    // Displaying contents of the vector
    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
```
