ascii

The `ascii()` function in Python takes an object as an argument and returns a string containing a printable representation of the object. In the resulting string, any non-ASCII characters (characters with a numerical value above 127) are escaped using the format '\uXXXX', where 'XXXX' is the Unicode code point in hexadecimal. This method is particularly useful for making data string representations safe for output to environments that only support ASCII encoding.

my_string = 'Café'
result = ascii(my_string)
print(result)  # Output: 'Caf\xe9'
my_dict = {'name': 'José', 'age': 30}
result = ascii(my_dict)
print(result)  # Output: '{'name': 'Jos\xe9', 'age': 30}'
my_list = ['apple', 'banana', 'chéri']
result = ascii(my_list)
print(result)  # Output: "['apple', 'banana', 'ch\xe9ri']"
