abs

The `abs` function in Python returns the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, regardless of direction. This means `abs` can handle integers, floating-point numbers, and even complex numbers (in which case it returns the magnitude). The syntax is simple: `abs(x)`, where `x` can be any numeric type.

 # Example 1: Using abs with an integer
number = -5
absolute_value = abs(number)
print(absolute_value)  # Output: 5
 # Example 2: Using abs with a floating-point number
number = -3.14
absolute_value = abs(number)
print(absolute_value)  # Output: 3.14
 # Example 3: Using abs with a complex number
number = complex(-3, 4)
absolute_value = abs(number)
print(absolute_value)  # Output: 5.0
