abs

The 'abs' method in Python returns the absolute value of a given number. It takes a single argument (an integer or a floating-point number) and returns the non-negative value of the number without regard to its sign. If the number is a positive number or zero, the abs() method returns the number itself. If the number is a negative number, abs() returns the positive value corresponding to that number.

number = -5
positive_number = abs(number)
print(f'The absolute value of {number} is {positive_number}')
# Output: The absolute value of -5 is 5
floating_number = -3.14
positive_floating_number = abs(floating_number)
print(f'The absolute value of {floating_number} is {positive_floating_number}')
# Output: The absolute value of -3.14 is 3.14
mixed_numbers = [-10, 0, 5, -20.5, 7.1]
absolute_values = [abs(num) for num in mixed_numbers]
print(f'The absolute values of {mixed_numbers} are {absolute_values}')
# Output: The absolute values of [-10, 0, 5, -20.5, 7.1] are [10, 0, 5, 20.5, 7.1]
