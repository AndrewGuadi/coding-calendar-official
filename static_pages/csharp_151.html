Add

The 'Add' method in C# is typically used to insert an element into a collection such as a List, Dictionary, or any other data structure that supports adding items. This method usually takes one or more parameters that define what to add, and it modifies the existing collection by including the new element(s). The behavior of the 'Add' method may vary based on the type of collection. For example, when adding a new item to a List, it increases the size of the List and places the new item at the end, while in a Dictionary, a key-value pair is being added where the key must be unique.

```python
class ShoppingCart:
    def __init__(self):
        self.items = []

    def add_item(self, item):
        self.items.append(item)  # This emulates a collection's Add method.

    def show_cart(self):
        return self.items

# Usage
cart = ShoppingCart()
cart.add_item('apple')
cart.add_item('banana')
print(cart.show_cart())  # Outputs: ['apple', 'banana']
```
```python
class StudentRecord:
    def __init__(self):
        self.records = {}

    def add_record(self, student_id, name):
        self.records[student_id] = name  # Simulates Dictionary Add method where student_id is key.

    def show_records(self):
        return self.records

# Usage
record_system = StudentRecord()
record_system.add_record(1, 'John Doe')
record_system.add_record(2, 'Jane Doe')
print(record_system.show_records())  # Outputs: {1: 'John Doe', 2: 'Jane Doe'}
```

class TodoList:
    def __init__(self):
        self.tasks = []

    def add_task(self, task):
        self.tasks.append(task)  # This simulates adding a task to the list of todos.

    def show_tasks(self):
        return self.tasks

# Usage
todo = TodoList()
todo.add_task('Complete assignment')
todo.add_task('Read a book')
print(todo.show_tasks())  # Outputs: ['Complete assignment', 'Read a book']
