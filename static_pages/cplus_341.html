std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. This method automatically resizes the vector if it has reached its current capacity. The new element is added as the last element of the vector, allowing for dynamic management of the collection of elements. It is important to note that the `push_back` method works with various data types, including built-in types and user-defined types. With this method, you can maintain a growing list of elements without worrying about the underlying array's size.

```cpp
#include 
#include 

int main() {
    std::vector code_snippets;
    
    // Adding Python code snippets to the vector
    code_snippets.push_back("print('Hello, World!')");
    code_snippets.push_back("for i in range(5): print(i)");
    code_snippets.push_back("def add(a, b): return a + b");
    
    // Displaying the contents of the vector
    for (const auto& snippet : code_snippets) {
        std::cout << snippet << std::endl;
    }
    return 0;
}
```
