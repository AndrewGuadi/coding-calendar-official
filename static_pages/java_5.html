abs

The 'abs' method in Java is used to return the absolute value of a given numeric type. It is particularly useful when you want to ensure that the outcome of a numerical operation is non-negative. Java provides 'abs' methods for various numeric types including int, long, float, and double. When using 'abs', if the input is a negative number, it returns the corresponding positive number; if the input is already positive (including zero), it simply returns that number.

To demonstrate the use of 'abs' with an integer, you can define a method that takes an integer and prints its absolute value. For instance:

```java
public class Main {
    public static void main(String[] args) {
        int number = -10;
        System.out.println(Math.abs(number)); // Output will be 10
    }
}
```
