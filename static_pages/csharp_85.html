Add

The 'Add' method in C# is typically used to insert an element into a collection such as lists, dictionaries, or sets. The method appends a new item to the end of the collection, making it straightforward to grow the data structure dynamically. In the context of lists, for example, 'Add' increases the size of the list by one, and the new item can be accessed through its index.

```python
# Example of using Add with a List in C#
# Create a List to hold integers.
numbers = []

# Function to add a number to the list.
def add_number(num):
    numbers.append(num)

# Adding numbers to the list.
add_number(5)
add_number(10)
add_number(20)

# Display the list
print(numbers)  # Output will be [5, 10, 20]
```
```python
# Example of using Add with a Dictionary in C#
# Create a dictionary to hold key-value pairs.
data = {}

# Function to add a key-value pair to the dictionary.
def add_item(key, value):
    data[key] = value

# Adding items to the dictionary.
add_item('first', 'Python')
add_item('second', 'Java')
add_item('third', 'C#')

# Display the dictionary
print(data)  # Output will be {'first': 'Python', 'second': 'Java', 'third': 'C#'}
```
```python
# Example of using Add with a Set in C#
# Create a Set to hold unique items.
unique_numbers = set()

# Function to add a number to the set.
def add_unique_number(num):
    unique_numbers.add(num)

# Adding numbers to the set.
add_unique_number(1)
add_unique_number(2)
add_unique_number(2)  # This will not add a duplicate

# Display the set
print(unique_numbers)  # Output will be {1, 2}
```
