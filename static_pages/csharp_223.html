Add

The 'Add' method is commonly used in C# to insert an element into collections such as lists, dictionaries, or sets. It appends the specified element to the end of the collection and increases the size of the collection accordingly. The behavior of the method varies slightly based on the type of collection it is being applied to. For example, in a List, the element is added at the next available index, while in a Dictionary, a key-value pair is added.

```python
# Example of adding an item to a List in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        List programmingLanguages = new List();
        programmingLanguages.Add("Python"); // Adding Python to the list
        programmingLanguages.Add("C#");   // Adding C# to the list
        Console.WriteLine(string.Join(", ", programmingLanguages)); // Output: Python, C#
    }
}
```
```python
# Example of adding a key-value pair to a Dictionary in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        Dictionary capitals = new Dictionary();
        capitals.Add("USA", "Washington, D.C."); // Adding key "USA" with value "Washington, D.C."
        capitals.Add("Germany", "Berlin");     // Adding key "Germany" with value "Berlin"
        Console.WriteLine(capitals["USA"]); // Output: Washington, D.C.
    }
}
```
```python
# Example of adding an item to a HashSet in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        HashSet uniqueItems = new HashSet();
        uniqueItems.Add("Apple"); // Adding "Apple" to the HashSet
        uniqueItems.Add("Banana"); // Adding "Banana" to the HashSet
        uniqueItems.Add("Apple"); // Adding "Apple" again will not change the set
        Console.WriteLine(string.Join(", ", uniqueItems)); // Output: Apple, Banana
    }
}
```
