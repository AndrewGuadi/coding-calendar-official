upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase. This method returns a new string with all the alphabetic characters transformed to their uppercase equivalents, while non-alphabetic characters remain unchanged. The 'upcase' method is particularly useful when you need to standardize string data for comparison or display purposes.

original_string = 'hello world'
# Use the upcase method
upcased_string = original_string.upcase
puts upcased_string # Output: 'HELLO WORLD'
original_string = 'Ruby is fun!'
# Convert to uppercase
upcased_string = original_string.upcase
puts upcased_string # Output: 'RUBY IS FUN!'
original_string = '1234 and abcd'
# Using upcase on a string with numbers and letters
upcased_string = original_string.upcase
puts upcased_string # Output: '1234 AND ABCD'
