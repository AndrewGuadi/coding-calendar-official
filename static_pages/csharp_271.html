Add

The `Add` method is typically used to insert a new item into a collection, such as a list, dictionary, or any other collection type in C#. This method generally modifies the original collection by adding the specified element to it. It is important to note that the functionality may vary slightly depending on the collection type.

collection = []
collection.append('item')
# After this operation, collection becomes ['item']
collection = {}
collection['key'] = 'value'
# After this operation, collection becomes {'key': 'value'}
import pandas as pd

data = pd.DataFrame(columns=['A', 'B'])
data = data.append({'A': 1, 'B': 2}, ignore_index=True)
# After this operation, data contains one row with A=1 and B=2.
