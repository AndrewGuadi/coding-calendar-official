std::vector::push_back

The 'std::vector::push_back' method appends a new element to the end of a vector in C++. This operation increases the size of the vector by one. If the vector needs to grow its capacity to accommodate the new element, it will allocate more memory. This method does not return a value. It is a part of the C++ Standard Library and requires including the header.

#include 
#include 

int main() {
    std::vector myVector;
    // Using push_back to add elements
    myVector.push_back("apple");
    myVector.push_back("banana");
    myVector.push_back("cherry");

    // Display the contents of the vector
    for (const auto& fruit : myVector) {
        std::cout << fruit << " ";
    }
    return 0;
}
