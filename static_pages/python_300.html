ascii

The `ascii()` function in Python takes a single argument, which can be a string or any object that can be converted to a string using its `__str__()` method. It returns a string containing a printable representation of the object, where non-ASCII characters are escaped with their Unicode code points. This can be useful for debugging or logging purposes, as it ensures that all characters in the output are ASCII characters. If the input is already composed only of ASCII characters, the result is the same as calling `str()` on the object.

input_str = 'Hello, 世界'
print(ascii(input_str))  # Output: 'Hello, \u4e16\u754c'
input_str = 'Python 3.9'
print(ascii(input_str))  # Output: 'Python 3.9'
input_list = ['sample', 'test', 'data', '日本語']
print([ascii(item) for item in input_list])  # Output: ['sample', 'test', 'data', '\u65e5\u672c\u8a9e']
