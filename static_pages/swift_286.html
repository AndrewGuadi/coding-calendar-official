uppercased

The 'uppercased' method in Swift is used to convert all the characters in a given string to uppercase letters. This method is particularly useful when you want to standardize text input, for comparison purposes, or to format output consistently. The 'uppercased()' method returns a new string where all the lowercase letters have been converted to their uppercase counterparts. It is important to note that this method does not modify the original string but instead returns a new string with the changes applied. If there are characters in the string that are already uppercase or are non-alphabetic, they remain unaffected. The method does not require any parameters and is called directly on an instance of a string.

string_input = 'hello, world!'
result = string_input.uppercased()
print(result)  # Output: 'HELLO, WORLD!'
name = 'john doe'
result = name.uppercased()
print(result)  # Output: 'JOHN DOE'
message = 'Welcome to the world of Swift.'
result = message.uppercased()
print(result)  # Output: 'WELCOME TO THE WORLD OF SWIFT.'
