abs

The `abs` method in Python returns the absolute value of a given number. When the input is a positive number or zero, the output is the same number. When the input is negative, the output is the number converted to positive.

number = -10
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')
# Output: The absolute value of -10 is 10
number = 25.5
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')
# Output: The absolute value of 25.5 is 25.5
number = -3.14
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')
# Output: The absolute value of -3.14 is 3.14
