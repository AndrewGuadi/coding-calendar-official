abs

The 'abs' method in Java is a static method of the java.lang.Math class that returns the absolute value of a specified number. This method can be used for different numeric types such as int, long, float, and double. The absolute value of a number is its distance from zero on the number line, regardless of its sign. For example, the absolute value of both -5 and 5 is 5.

Using abs to get the absolute value of a negative integer:

int number = -10;
int absoluteValue = Math.abs(number);
System.out.println(absoluteValue);  // Output: 10
Getting the absolute value of a negative double:

double negativeValue = -99.99;
double absoluteDouble = Math.abs(negativeValue);
System.out.println(absoluteDouble);  // Output: 99.99
Calculating the absolute difference between two numbers:

int num1 = -20;
int num2 = 30;
int difference = Math.abs(num1 - num2);
System.out.println(difference);  // Output: 50
