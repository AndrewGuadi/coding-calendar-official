std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that adds a new element to the end of the vector, effectively increasing its size by one. This method allows you to dynamically append elements to a vector, making it a flexible choice for managing collections of data. When using `push_back`, if the vectorâ€™s capacity is reached, it will automatically allocate more memory to accommodate additional elements. This method has an average time complexity of O(1) for appending elements.

In this example, we create a vector to store a collection of strings. Using `push_back`, we append Python code snippets to illustrate the dynamic resizing of the vector:

```cpp
#include 
#include 

int main() {
    std::vector pythonCodes;
    pythonCodes.push_back("print('Hello, world!')");
    pythonCodes.push_back("for i in range(5): print(i)");

    for (const auto& code : pythonCodes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
