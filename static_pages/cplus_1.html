std::vector::pop_back

The method 'std::vector::pop_back' removes the last element from the vector. This method effectively reduces the size of the vector by one but does not necessarily free the memory associated with the removed element. It's important to note that using this method on an empty vector leads to undefined behavior. Typically, this method is used when you need to remove the most recently added element from the list of elements stored in a vector.

```python
# Initializing a list to simulate a vector
vector = [1, 2, 3, 4, 5]

# Function to pop back the last element
def pop_back(lst):
    if len(lst) > 0:
        lst.pop()  # Removes the last element

# Using the function
pop_back(vector)
print(vector)  # Output: [1, 2, 3, 4]
```
