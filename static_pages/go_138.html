cap

The 'cap' function in Go is used to determine the capacity of a given slice. A slice in Go is a dynamically-sized array, and its capacity is the number of elements it can hold without allocating more memory. This is especially useful when working with dynamic datasets where you want to control the amount of memory being used and to optimize performance when adding elements to slices.

my_slice = []int{1, 2, 3}; print(cap(my_slice))  # This will output: 3, as it contains three elements and capacity equals length.
empty_slice = make([]int, 0, 5); print(cap(empty_slice))  # This will output: 5, as it is created with a capacity of 5.
larger_slice = append(my_slice, 4, 5, 6); print(cap(larger_slice))  # Depending on what the initial capacity was, this can show how appending affects capacity.
