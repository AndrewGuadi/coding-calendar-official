uppercased

The 'uppercased' method in Swift is used to convert all lowercase letters in a string to uppercase letters. This method returns a new string in which all the characters are in uppercase form. It does not modify the original string but provides a transformed version of it. The 'uppercased' method is commonly used when you need to standardize string data for comparisons, display purposes, or formatting. It is part of the 'String' type in Swift.

def convert_to_uppercase(s):
    return s.upper()

input_string = 'hello world'
output_string = convert_to_uppercase(input_string)
print(output_string)
# Output: HELLO WORLD
def convert_and_compare(s1, s2):
    return s1.upper() == s2.upper()

string1 = 'Python'
string2 = 'python'
is_equal = convert_and_compare(string1, string2)
print(is_equal)
# Output: True
def format_name(name):
    return name.upper()

full_name = 'john doe'
formatted_name = format_name(full_name)
print(formatted_name)
# Output: JOHN DOE
