std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add an element to the end of the vector, effectively increasing its size by one. When an element is added, if the current capacity of the vector is insufficient to store the new element, the vector will automatically allocate new memory to accommodate the additional items. The new element is initialized by making a copy of the provided value. The push_back operation has an amortized time complexity of O(1).

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Python");
    myVector.push_back("Java");
    myVector.push_back("JavaScript");

    for (const auto &language : myVector) {
        std::cout << language << " \n";
    }
    return 0;
}
```

```cpp
    programmingLanguages.push_back("Go");
    programmingLanguages.push_back("Rust");

    std::cout << "Programming Languages:\n";
    for (std::size_t i = 0; i < programmingLanguages.size(); ++i) {
        std::cout << "Language " << (i + 1) << ": " << programmingLanguages[i] << "\n";
    }
    return 0;
}
```
