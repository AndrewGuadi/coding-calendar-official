append

In Go, the 'append' function is used to add elements to the end of a slice. The function takes a slice as its first argument and one or more elements to be appended. If the capacity of the slice is exceeded, 'append' will automatically allocate a new underlying array, making the method efficient and flexible. This is a widely used feature for dynamic array manipulation in Go.

```go
package main
import (
	"fmt"
)

func main() {
	// Start with an initial slice
	numbers := []int{1, 2, 3}
	// Append values 4 and 5
	numbers = append(numbers, 4, 5)
	fmt.Println(numbers) // Output: [1 2 3 4 5]
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Start with an empty slice
	names := []string{}
	// Append several names to the slice
	names = append(names, "Alice", "Bob", "Charlie")
	fmt.Println(names) // Output: [Alice Bob Charlie]
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Create a slice of integers
	initial := []int{10, 20}
	// Append more integers
	newSlice := append(initial, 30, 40)
	// Show that the original slice remains unchanged
	fmt.Println("Original Slice:", initial) // Output: Original Slice: [10 20]
	fmt.Println("New Slice:", newSlice) // Output: New Slice: [10 20 30 40]
}
```
