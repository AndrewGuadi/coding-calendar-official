close

In Go, the 'close' method is used to close channels, which prevents any further values from being sent on the channel. It is important to close a channel when it is no longer needed to free up resources and to signal to other goroutines that no more values will be sent. When a channel is closed, any attempt to send values to it will cause a panic. However, receiving from a closed channel will yield the zero value of the channel's type, and will also return a second boolean value indicating whether the channel is still open or not.

```go
package main

import (
    "fmt"
)

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 5; i++ {
            ch <- i
        }
        close(ch)  // Closing the channel after sending values
    }()

    for val := range ch {
        fmt.Println(val)  // Receiving values until the channel is closed
    }
}
```
```go
package main

import (
    "fmt"
)

func main() {
    done := make(chan bool)
    go func() {
        // Simulating some work
        for i := 0; i < 3; i++ {
            fmt.Println("Working...")
        }
        done <- true  // Signaling completion
        close(done)  // Closing the done channel
    }()

    <-done  // Waiting for the signal that work is done
    fmt.Println("Work is done!")
}
```
```go
package main

import (
    "fmt"
)

func main() {
    data := make(chan int)
    go func() {
        for i := 1; i <= 5; i++ {
            data <- i
        }
        close(data)  // Closing channel after sending
    }()

    for num := range data {
        fmt.Println(num)  // Printing numbers received from the channel
    }
    fmt.Println("All values received. Channel closed.")
}
```
