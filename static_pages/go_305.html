close

In Go, the 'close' method is used to close a channel, which ensures that no more values will be sent to that channel. Once a channel is closed, any attempts to send values on it will cause a panic, and the channel will return a boolean false when trying to receive from it if there are no values left. It is essential to only close a channel from the goroutine that created it to prevent concurrent closing issues and race conditions.

```python
ch = []
def send(ch):
    for i in range(5):
        ch.append(i)
        print('Sent:', i)
    close(ch)

def receive(ch):
    while True:
        try:
            print('Received:', ch.pop(0))
        except IndexError:
            break

send(ch)
receive(ch)
```
```python
ch = []

def producer(ch):
    for i in range(3):
        ch.append(i)
        print('Produced:', i)
    close(ch)


def consumer(ch):
    while ch:
        value = ch.pop(0)
        print('Consumed:', value)

producer(ch)
consumer(ch)
```
```python
ch = []

def number_generator(ch):
    for number in range(10):
        ch.append(number)
        print('Generated:', number)
    close(ch)


def number_collector(ch):
    while ch:
        try:
            number = ch.pop(0)
            print('Collected:', number)
        except IndexError:
            break

number_generator(ch)
number_collector(ch)
```
