std::vector::push_back

In C++, the std::vector::push_back method is used to add a new element to the end of a vector. This method can dynamically increase the size of the vector by allocating more memory as needed. The element being added is passed as an argument to the method. It is important to note that push_back takes a copy of the provided element, which may incur an overhead if the element is large or complex. This method is part of the Standard Template Library (STL) and allows for efficient management of dynamic arrays through automatic memory management, ensuring that the user does not have to worry about memory allocation details.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World!");

    // Display the contents of the vector
    for (const auto& str : myVector) {
        std::cout << str << ' ';
    }
    return 0;
}
```
