std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element at the end of a vector. If the vector's current capacity is exhausted, it will automatically increase its size to accommodate the new element. This method is widely utilized for dynamically managing an array of objects, allowing for easy appending and management of a collection of items. The method modifies the vector in place and returns no value.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");

    // Print fruits
    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
```
