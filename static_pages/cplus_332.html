std::vector::push_back

The `push_back` method of the `std::vector` class in C++ adds a new element at the end of the vector. This method increases the size of the vector by one and initializes the new element to the specified value. If the vector's capacity is insufficient to accommodate the new element, `push_back` will allocate a new memory block, copy the existing elements to the new block, and then add the new element. This makes `push_back` a dynamic method that allows for efficient appending of elements and maintains the order of elements within the vector.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("orange");

    for(const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
```
