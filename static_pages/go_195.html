cap

The 'cap' method in Go is used to retrieve the capacity of a slice. The capacity is the number of elements the slice can hold without requiring a reallocation of memory. This is particularly useful when working with dynamic arrays in Go, as it helps understand how much extra space is available in the slice before needing to allocate more memory. The capacity can be equal to or greater than the length of the slice but never less than it.

slice = []int{1, 2, 3}

length = len(slice)    # Retrieves the length of the slice, which is 3.
capacity = cap(slice)  # Retrieves the capacity of the slice.

# Example of adding to the slice
slice = append(slice, 4)
length_after_append = len(slice)    # Length becomes 4.
capacity_after_append = cap(slice)  # Capacity remains the same unless the underlying array is exhausted.
slice = make([]int, 2, 5)  # Creates a slice of length 2 and capacity 5.

length = len(slice)  # Length is 2.
capacity = cap(slice)  # Capacity is 5.

# Appending elements
slice = append(slice, 6)
length_after_append = len(slice)  # Length becomes 3.
capacity_after_append = cap(slice)  # Capacity remains 5.
slice = []string{"apple", "banana"}

length = len(slice)    # Length is 2.
capacity = cap(slice)  # Initial capacity is also 2 unless modified.

# Extending the slice with more elements
slice = append(slice, "cherry", "date")  # Adding more elements

length_after_append = len(slice)          # Length is now 4.
capacity_after_append = cap(slice)        # Capacity may increase if needed.
