charAt

The `charAt()` method returns the character at a specified index (position) in a string. The index is zero-based, meaning that the first character is at index 0, the second character at index 1, and so on. If the index is out of range, it returns an empty string. This method is useful for extracting individual characters from strings, allowing for manipulation or analysis of string contents.

let text = 'Hello, World!';
let firstChar = text.charAt(0); // firstChar will be 'H'
console.log(firstChar);
let text = 'Hello, World!';
let fifthChar = text.charAt(4); // fifthChar will be 'o'
console.log(fifthChar);
let text = 'Hello, World!';
let lastChar = text.charAt(text.length - 1); // lastChar will be '!'
console.log(lastChar);
