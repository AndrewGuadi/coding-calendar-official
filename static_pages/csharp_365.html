Add

The 'Add' method is used to add an element to a collection in C#. It is commonly used in lists, dictionaries, and sets. When using 'Add', the element is appended to the end of the collection in the case of lists or added as a key-value pair in dictionaries. If attempting to add a duplicate key to a dictionary, an exception will be thrown. This method is essential for dynamic data structures that grow as new elements are added during the execution of a program.

```python
# Example of using 'Add' with a list in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonExamples = new List();
        pythonExamples.Add("print('Hello, World!')");
        pythonExamples.Add("for i in range(5): print(i)");

        foreach (string example in pythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
```
```python
# Example of using 'Add' with a dictionary in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary pythonCodeSnippets = new Dictionary();
        pythonCodeSnippets.Add("greeting", "print('Hello, World!')");
        pythonCodeSnippets.Add("loop", "for i in range(5): print(i)");

        foreach (var snippet in pythonCodeSnippets)
        {
            Console.WriteLine(snippet.Key + ": " + snippet.Value);
        }
    }
}
```
```python
# Example of using 'Add' with a HashSet in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet uniquePythonExamples = new HashSet();
        uniquePythonExamples.Add("print('Hello, World!')");
        uniquePythonExamples.Add("for i in range(5): print(i)");
        uniquePythonExamples.Add("print('Hello, World!')");  // Duplicate, will not be added

        foreach (string example in uniquePythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
```
