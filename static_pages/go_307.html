close

In Go, the 'close' method is used to close a channel, which signals that no more values will be sent on that channel. This is particularly important for goroutines that are waiting to receive values from the channel; closing the channel can help them to exit gracefully. It's crucial to ensure that no other goroutine is attempting to send values on a closed channel, as this will cause a panic.

import time

# Define a channel
ch = make(chan string)

# Function to send data to the channel
func sendData(ch chan string) {
    ch <- "Hello"
    ch <- "World"
    close(ch) // Close the channel after sending
}

// Function to receive data from the channel
func receiveData(ch chan string) {
    for msg := range ch {
        fmt.Println(msg) // Print the message received from the channel
    }
}

// Main function to run goroutines
func main() {
    go sendData(ch)
    receiveData(ch)
    time.Sleep(1 * time.Second) // Wait to allow messages to be printed
}
