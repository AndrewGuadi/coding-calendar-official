uppercased

The `uppercased` method in Swift is a string method that returns a new string in which all lowercase letters in the original string have been converted to uppercase. This method does not modify the original string, as strings in Swift are value types. It is commonly used when you want to standardize data for comparison or formatting purposes.

original_string = 'Hello, World!'
uppercased_string = original_string.uppercased()
print(uppercased_string)  # Output: 'HELLO, WORLD!'
text = 'swift programming'
result = text.uppercased()
print(result)  # Output: 'SWIFT PROGRAMMING'
greeting = 'good morning'
uppercase_greeting = greeting.uppercased()
print(uppercase_greeting)  # Output: 'GOOD MORNING'
