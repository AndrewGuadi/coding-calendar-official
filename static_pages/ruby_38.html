upcase

The 'upcase' method in Ruby is used to convert all lowercase letters of a string to uppercase. It returns a new string with all the characters converted to uppercase while leaving non-alphabetic characters unchanged. This method is particularly useful for normalizing string input and ensuring case insensitivity when comparing strings. The method does not alter the original string; rather, it produces a new string with the changes applied.

def convert_to_uppercase(text): return text.upper()

# Usage
input_text = 'hello world'
output_text = convert_to_uppercase(input_text)
print(output_text)  # Output: 'HELLO WORLD'
def shout(text): return text.upper() + '!!!'

# Usage
input_text = 'watch out'
output_text = shout(input_text)
print(output_text)  # Output: 'WATCH OUT!!!'
def format_name(first_name, last_name): return (first_name + ' ' + last_name).upper()

# Usage
first = 'John'
last = 'Doe'
full_name = format_name(first, last)
print(full_name)  # Output: 'JOHN DOE'
