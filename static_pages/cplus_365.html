std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that adds a new element to the end of the vector. It is used to dynamically add items to the vector as it automatically manages memory allocation. If the vector's current capacity is exceeded, it will allocate new memory with a larger capacity to accommodate the new element. This method effectively increases the vector's size by one with each call.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Add 'Hello'
    myVector.push_back("World"); // Add 'World'

    for (const auto& str : myVector) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}```
