std::vector::push_back

The `push_back` method in C++ is used to add an element at the end of a vector. A vector is a part of the C++ Standard Template Library (STL) and is a dynamic array that can resize itself automatically when an element is added. Calling `push_back` copies or moves the given element to the end of the vector. If the vector's current capacity is not enough to accommodate the new element, it will reallocate memory to accommodate the new sizes. This process ensures that the vector can always hold the elements added to it.

Below is a complete example demonstrating how to create a vector of strings, add Python code snippets using `push_back`, and then print the vector's contents:

```cpp
#include 
#include 
#include 

int main() {
    std::vector pythonCode;
    pythonCode.push_back("print('Hello, World!')");
    pythonCode.push_back("def add(a, b): return a + b");
    pythonCode.push_back("for i in range(5): print(i)");

    for (const auto& code : pythonCode) {
        std::cout << code << std::endl;
    }
    return 0;
}
```

```cpp
int main() {

    mathOperations.push_back("result = 10 * 2");

    }
    return 0;
}
```
Here we show a more interactive example where we read user inputs into the vector using `push_back`, allowing for dynamic entry of Python code:

```cpp
#include 
#include 
#include 

int main() {
    std::vector userPythonCode;
    std::string input;
    std::cout << "Enter Python code (type 'exit' to stop): \n";
    while (true) {
        std::getline(std::cin, input);
        if (input == "exit") break;
        userPythonCode.push_back(input);
    }

    std::cout << "You entered the following Python code:\n";
    for (const auto& code : userPythonCode) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
