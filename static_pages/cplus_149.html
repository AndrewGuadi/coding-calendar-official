std::vector::push_back

The std::vector::push_back method is a member function of the C++ Standard Library's vector class that allows you to add an element to the end of the vector. When you call push_back() with a value, the vector checks if there is enough capacity to hold the new element. If there isn't, it allocates more memory. The time complexity of push_back() is amortized constant time, making it efficient for dynamic arrays.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");
    
    for(const auto& fruit : fruits) {
        std::cout << fruit << std::endl; // Output: apple, banana, cherry
    }
    return 0;
}
```
