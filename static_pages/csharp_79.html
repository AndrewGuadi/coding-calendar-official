Add

The 'Add' method is commonly used in collections and data structures in C#. It is designed to add an element to a specific collection, such as a list or a dictionary. In a List, for instance, 'Add' will append the item to the end of the list, while in a Dictionary, 'Add' will insert a new key-value pair. This method does not return any value. When the collection is full or if there is a duplicate key in the case of a dictionary, it may throw an exception.

```python
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
class MyList:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

my_list = MyList()
my_list.add('Python')
my_list.add('Java')
print(my_list.items)  # Output: ['Python', 'Java']
```
