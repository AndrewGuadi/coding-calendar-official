strlen

The `strlen()` function in PHP is used to get the length of a string. It counts the number of characters in a string, including spaces and punctuation. If the input is not a string, it converts it implicitly to a string before measuring its length. This function is useful for validating user input, processing string data, and ensuring that a string meets certain length criteria.

string_input = 'Hello, World!'
length = strlen(string_input)  # This will return 13
mixed_input = [1, 2, 3]
length = strlen(str(mixed_input))  # This will return 9 because the string representation is '[1, 2, 3]'
file_content = 'This is a file content.'
length = strlen(file_content)  # This will return 25
