acos

The 'acos' method is a part of the Java Math class and is used to compute the arc cosine of a given value. The input to this method must be a double value in the range of -1.0 to 1.0, and it returns the angle in radians. The 'acos' function is useful in various mathematical applications, including trigonometry, physics, and computer graphics, where you may need to calculate angles based on cosine values.

import math

# Calculate the arc cosine of 0.5
result = math.acos(0.5)
print(result)  # Output: 1.0471975511965979 (which is pi/3 radians)

import math

# Calculate the arc cosine of -1
result = math.acos(-1)
print(result)  # Output: 3.141592653589793 (which is pi radians)

import math

# Calculate the arc cosine of 1
result = math.acos(1)
print(result)  # Output: 0.0 (which is 0 radians)
