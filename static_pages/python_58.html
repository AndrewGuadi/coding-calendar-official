abs()

The abs() function in Python returns the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, regardless of direction. For example, the absolute value of both -10 and 10 is 10. This function can take an integer, a float, or a complex number as an argument. For complex numbers, the abs() function returns the magnitude of the number.

***Example 1: Using abs() with an integer***
# Get the absolute value of an integer
num = -20
absolute_value = abs(num)
print(absolute_value)
# Output: 20

***Example 2: Using abs() with a float***
# Get the absolute value of a float
num = -15.5
absolute_value = abs(num)
print(absolute_value)
# Output: 15.5

***Example 3: Using abs() with a complex number***
# Get the absolute value (magnitude) of a complex number
num = 3 + 4j
absolute_value = abs(num)
print(absolute_value)
# Output: 5.0
