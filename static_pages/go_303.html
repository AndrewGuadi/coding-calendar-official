close

In Go, the 'close' method is primarily used to close channels. When you close a channel, you indicate that no more values will be sent on it. Close is a built-in function that takes a channel as its argument and closes it, allowing any listeners that are waiting for values to be aware that no more are expected. Closing a channel is essential for gracefully terminating goroutines that communicate via channels and is also important for preventing deadlocks in certain concurrent scenarios.

import (\n    "fmt"\n    "sync"\n)\n\nfunc main() {\n    var wg sync.WaitGroup\n    ch := make(chan string)\n\n    wg.Add(1)\n    go func() {\n        defer wg.Done()\n        for msg := range ch {\n            fmt.Println(`Received:`, msg)\n        }\n    }()\n\n    ch <- "Hello"\n    ch <- "World"\n    close(ch) // Closing the channel\n    wg.Wait()\n}
