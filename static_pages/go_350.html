close

In Go, the 'close' function is used to close a channel, which indicates that no more values will be sent on it. Closing a channel allows other goroutines to detect that they should stop waiting for values. It's important to note that once a channel is closed, attempting to send a value on it will cause a panic. However, receiving from a closed channel will return the zero value of the channel's type without blocking.

# This is a simple example of using close in Go.

import ( 
    "fmt" 
    "time" 
)

func main() { 
    ch := make(chan string)
    
    go func() { 
        time.Sleep(1 * time.Second) 
        close(ch) 
    }()
    
    for msg := range ch { 
        fmt.Println(msg) 
    }
    fmt.Println("Channel closed") 
}
# Example demonstrating the effect of closing a channel.

import ( 
    "fmt" 
)

func main() { 
    ch := make(chan int)
    
    go func() { 
        for i := 0; i < 5; i++ { 
            ch <- i 
        } 
        close(ch) 
    }()
    
    for msg := range ch { 
        fmt.Println(msg) 
    }
    fmt.Println("All messages received and channel closed") 
}
# Example showing panic when sending to a closed channel.

import ( 
    "fmt" 
)

func main() { 
    ch := make(chan string)
    close(ch)
