Add

The 'Add' method is commonly used in collections to insert elements into a data structure such as a list, dictionary, or any collection that implements an 'Add' method. In C#, this method is typically followed by specifying the item to be added, and it increases the size of the collection by one. It can be applied to various types of collections like List, HashSet, and Dictionary. When adding an item, the method performs several checks such as ensuring that duplicates are not allowed (in the case of a HashSet) or updating an entry in a dictionary if the key already exists.

```python
# Example of using the Add method in a List in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonCodeExamples = new List();

        // Adding Python code examples to the list
        pythonCodeExamples.Add("print('Hello World')");
        pythonCodeExamples.Add("for i in range(5): print(i)");
        pythonCodeExamples.Add("def hello(): print('Hello from function')");

        // Displaying added elements
        foreach (var example in pythonCodeExamples)
        {
            Console.WriteLine(example);
        }
    }
}
```

# Example of using the Add method in a HashSet in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet uniquePythonCodes = new HashSet();

        // Adding unique Python code snippets to the HashSet
        uniquePythonCodes.Add("if x > 0:");
        uniquePythonCodes.Add("    print('Positive')");
        uniquePythonCodes.Add("else:");
        uniquePythonCodes.Add("    print('Non-positive')");
        uniquePythonCodes.Add("if x > 0:"); // this won't be added again

        // Displaying the HashSet elements
        foreach (var code in uniquePythonCodes)
        {
            Console.WriteLine(code);
        }
    }
}
