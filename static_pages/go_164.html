cap

The 'cap' function in Go is used to return the capacity of a slice or channel. The capacity of a slice is the number of elements the slice can hold (i.e., the number of elements in the underlying array that can be used without needing to allocate a new backing array). If the slice is declared with a specific length or includes a specific slice of a larger array, 'cap' will return that capacity. For channels, 'cap' returns the maximum number of elements that the channel can hold.

Here's an example demonstrating how to use 'cap' with a slice:

```go
package main
import (
	"fmt"
)

func main() {
	// Creating a slice with an initial capacity of 5
	numbers := make([]int, 0, 5)
	fmt.Println("Capacity of slice before append:", cap(numbers)) // Output: 5

	// Appending elements to the slice
	numbers = append(numbers, 1, 2, 3)
	fmt.Println("Capacity of slice after append:", cap(numbers)) // Output: 5
	fmt.Println(numbers)
}
```
Here's an example with a channel:

```go
package main
import (
	"fmt"
)

func main() {
	// Creating a buffered channel with a capacity of 3
	ch := make(chan int, 3)
	fmt.Println("Capacity of channel:", cap(ch)) // Output: 3

	// Sending elements to the channel
	ch <- 1
	ch <- 2
	ch <- 3
	// Uncommenting the next line would cause a deadlock on a buffered channel if the buffer is full
	// ch <- 4

	fmt.Println("Elements in channel have been sent")
}
```
This example shows using 'cap' with a slice derived from an array:

```go
package main
import (
	"fmt"
)

func main() {
	arr := [5]int{0, 1, 2, 3, 4}
	// Slicing the array from index 1 to 4
	slice := arr[1:4]
	fmt.Println("Capacity of the slice from an array:", cap(slice)) // Output: 4
}
```
