upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. It returns a new string that is a copy of the original string with all lowercase letters converted. This method does not modify the original string but returns a new one.

def convert_to_uppercase(input_string):
    # Convert the input string to uppercase using upcase method
    return input_string.upper()

# Example usage:
result = convert_to_uppercase('hello world')
print(result)  # Output: HELLO WORLD
def convert_titles_to_uppercase(titles):
    # Convert a list of titles to uppercase
    return [title.upper() for title in titles]

# Example usage:
titles_list = ['ruby programming', 'python coding', 'java basics']
result = convert_titles_to_uppercase(titles_list)
print(result)  # Output: ['RUBY PROGRAMMING', 'PYTHON CODING', 'JAVA BASICS']
def shouting_message(message):
    # Convert the message to uppercase to simulate shouting
    return message.upper() + '!!!'

# Example usage:
message = 'hello there'
result = shouting_message(message)
print(result)  # Output: HELLO THERE!!!
