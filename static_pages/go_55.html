cap

The 'cap' function in Go is used to retrieve the capacity of a slice, array, or channel. The capacity is the total number of elements that the underlying data structure can hold, which is important for efficient memory management. In contrast to the length of the slice (which counts the number of currently stored elements), the capacity indicates how many elements can be stored without needing to allocate additional memory.

```go
package main
import (
	"fmt"
)

func main() {
	// Creating a slice with an initial length and capacity
	slice := make([]int, 3, 5)

	// Showing length and capacity
	fmt.Println("Length of slice:", len(slice))  // Output: Length of slice: 3
	fmt.Println("Capacity of slice:", cap(slice)) // Output: Capacity of slice: 5
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Creating an array
	array := [5]int{1, 2, 3, 4, 5}

	// Using cap on an array (implicitly converted to a slice)
	fmt.Println("Capacity of array slice:", cap(array[:])) // Output: Capacity of array slice: 5
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Creating a channel with a capacity of 2
	channel := make(chan int, 2)

	// Showing the capacity of the channel
	fmt.Println("Capacity of channel:", cap(channel)) // Output: Capacity of channel: 2
}
```
