uppercased

The 'uppercased' method in Swift is a string method that returns a new string with all the characters converted to uppercase. It does not modify the original string but instead produces a new one. This method is useful when you need to ensure that a string is in uppercase format for display purposes, comparisons, or storage.

let originalString = "Hello, World!"
let uppercasedString = originalString.uppercased()
print(uppercasedString)  // Prints: HELLO, WORLD!
let greeting = "good morning"
let upperGreeting = greeting.uppercased()
print(upperGreeting)  // Prints: GOOD MORNING
let message = "Swift is fun!"
let loudMessage = message.uppercased()
print(loudMessage)  // Prints: SWIFT IS FUN!
