append

The 'append' function in Go is used to add elements to the end of a slice. It takes a slice and one or more elements as arguments and returns a new slice containing the original elements plus the added items. The original slice is not modified (though if there is capacity, it may just use the existing memory without allocation). If there is not enough capacity in the slice, a larger slice will be allocated, and the elements will be copied over.

s := []string{"a", "b"}
    s = append(s, "c")
    fmt.Println(s) // Output: [a b c]
s := []int{1, 2, 3}
    s = append(s, 4, 5)
    fmt.Println(s) // Output: [1 2 3 4 5]
s := []interface{}{}
    s = append(s, "hello", 3.14, true)
    fmt.Println(s) // Output: [hello 3.14 true]
