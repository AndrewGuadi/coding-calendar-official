Add

The 'Add' method is commonly used to add elements to a collection, such as a list, set, or dictionary in C#. It usually takes a single parameter, which is the element to be added. If the collection already contains the element (in the case of a set or dictionary), the behavior may vary depending on the collection type â€“ for instance, a set will ignore duplicates, while a list will allow them. The 'Add' method typically returns void, meaning it does not return a value but rather modifies the collection directly.

```python
# Usage of 'Add' method in a List
my_list = []
my_list.append('Python')  # Instead of Add
my_list.append('Java')
print(my_list)  # Output: ['Python', 'Java']
```
```python
# Usage of 'Add' method in a Set
my_set = set()
my_set.add('Python')  # Instead of Add
my_set.add('Java')
my_set.add('Python')  # Duplicate will be ignored
print(my_set)  # Output: {'Python', 'Java'}
```
```python
# Usage of 'Add' method in a Dictionary
my_dict = {}
my_dict['key1'] = 'Python'
my_dict['key2'] = 'Java'
# Instead of Add, we assign values to keys
print(my_dict)  # Output: {'key1': 'Python', 'key2': 'Java'}
```
