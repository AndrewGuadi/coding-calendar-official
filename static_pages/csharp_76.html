Add

The 'Add' method is a commonly used function in C# that belongs to various collections like List, Dictionary, and others. It is utilized to insert new elements into a collection. The specifics of how 'Add' works can vary depending on the collection type, but its primary role is to allow you to expand your collection by appending new elements to it.

```python
# Example using a List class to add elements
class List:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

my_list = List()
my_list.add('Python')
my_list.add('C#')
print(my_list.items)  # Output: ['Python', 'C#']```
```python
# Example using a Dictionary class to add key-value pairs
class Dictionary:
    def __init__(self):
        self.items = {}

    def add(self, key, value):
        self.items[key] = value

my_dict = Dictionary()
my_dict.add('language', 'Python')
my_dict.add('framework', 'Django')
print(my_dict.items)  # Output: {'language': 'Python', 'framework': 'Django'}```
```python
# Example using a custom Set class to add unique elements
class Set:
    def __init__(self):
        self.items = set()

    def add(self, item):
        self.items.add(item)

my_set = Set()
my_set.add('Python')
my_set.add('Python')  # This will not add a duplicate
my_set.add('C#')
print(my_set.items)  # Output: {'Python', 'C#'}```
