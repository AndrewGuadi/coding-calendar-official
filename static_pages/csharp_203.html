Add

The 'Add' method in C# is used to add an item to a collection, such as a List, Dictionary, or HashSet. When used with a List, it inserts a new element at the end of the list. For a Dictionary, it adds a key-value pair. Using it with a HashSet will add the item only if it is not already present in the set. This method typically alters the state of the collection it is called upon and can be useful for dynamic data manipulation.

```
# Example of List Usage in C#
using System;
using System.Collections.Generic;

class Program
{
    public static void Main()
    {
        List scripts = new List();
        scripts.Add("print('Hello World')");  // Adding a Python code string
        scripts.Add("for i in range(5): print(i)");

        foreach (string script in scripts)
        {
            Console.WriteLine(script);
        }
    }
}
```
```
# Example of Dictionary Usage in C#
using System;
using System.Collections.Generic;

class Program
{
    public static void Main()
    {
        Dictionary pythonScripts = new Dictionary();
        pythonScripts.Add("script1", "print('Hello World')");
        pythonScripts.Add("script2", "for i in range(3): print(i)");

        foreach (var item in pythonScripts)
        {
            Console.WriteLine(item.Key + ": " + item.Value);
        }
    }
}
```
```
# Example of HashSet Usage in C#
using System;
using System.Collections.Generic;

class Program
{
    public static void Main()
    {
        HashSet uniqueScripts = new HashSet();
        uniqueScripts.Add("print('Hello World')");
        uniqueScripts.Add("print('Hello World')");  // Duplicate, won't be added again

        foreach (string script in uniqueScripts)
        {
            Console.WriteLine(script);
        }
    }
}
```
