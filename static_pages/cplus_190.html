std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a std::vector. This method increases the size of the vector by one and the new element is initialized with the value provided as an argument. If the capacity of the vector is exceeded, it automatically reallocates memory to accommodate the new data. This method is commonly used when you want to dynamically add items to a list-like structure. It's important to note that the 'push_back' method operates in amortized constant time, making it efficient for adding elements.

```cpp
#include 
#include 

int main() {
    // Create a vector of integers
    std::vector numbers;
    // Using push_back to add elements
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    // Displaying the contents of the vector
    for (int num : numbers) {
        std::cout << num << ' '; // Output: 1 2 3
    }
    return 0;
}
```
