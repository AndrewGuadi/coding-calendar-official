Add

The 'Add' method is commonly used in C# within collections such as List, Dictionary, and sets. It is used to insert or add a new element into the collection. When adding an element, it is important to ensure that the collection supports duplicates or not, based on its implementation.

```python
# Example: Using Add method with a List in C#
# Initialize a list and add some elements using the Add method.

numbers = []  # Create an empty list
numbers.append(10)  # Use 'Add' equivalent in Python to add elements
numbers.append(20)
numbers.append(30)

# Print the list to see the elements added
print(numbers)  # Output: [10, 20, 30]
```
```python
# Example: Using Add method with a Dictionary in C#
# Create a dictionary and use the Add method to insert key-value pairs.

my_dict = {}  # Initialize an empty dictionary
my_dict['name'] = 'Alice'  # Use 'Add' equivalent in Python
my_dict['age'] = 30

# Print the dictionary to see the added key-value pairs
print(my_dict)  # Output: {'name': 'Alice', 'age': 30}
```
```python
# Example: Using Add method with a Set in C#
# Create a set and add elements ensuring uniqueness.

my_set = set()  # Initialize an empty set
my_set.add(1)  # Use 'Add' equivalent in Python
my_set.add(2)
my_set.add(1)  # Adding a duplicate, which won't change the set

# Print the set to see the unique elements added
print(my_set)  # Output: {1, 2}
```
