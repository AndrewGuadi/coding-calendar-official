abs

The 'abs' method in Java is used to calculate the absolute value of a given number. It can take an integer, a long, a float, or a double as a parameter and returns the absolute value of that number. The absolute value of a number is its non-negative value without regard to its sign.

import java.lang.Math; 
public class AbsoluteValueExample {
    public static void main(String[] args) {
        int number = -10;
        int absoluteValue = Math.abs(number);
        System.out.println("The absolute value of " + number + " is: " + absoluteValue);
    }
}
