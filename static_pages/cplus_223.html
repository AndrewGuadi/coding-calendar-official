std::vector::push_back

The method 'std::vector::push_back' is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one and allows you to store more elements. It takes a single argument, which is the value to be added to the vector. This method automatically handles memory allocation, so you do not have to manage the size of the vector manually. If the size exceeds the current capacity, it allocates a new array with a larger capacity, copies the existing elements to the new array, and then adds the new element.

```cpp
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    for(const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
