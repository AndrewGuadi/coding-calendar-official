close

In Go, the 'close' method is used to close a channel, preventing any further sends on that channel. It is a built-in function that takes a channel as an argument. Closing a channel is important because it signals to the receiving goroutine that no more values will be sent on that channel. After a channel is closed, it can still be read, but attempts to send on it will cause a panic. It is also a good practice to always check for the 'ok' boolean when receiving from a channel that may be closed, to avoid unwanted behavior.

import time
import random

def producer(ch):
    for i in range(5):
        ch.append(i)
        print(f'Produced: {i}')
        time.sleep(random.uniform(0.5, 1.5))
    close(ch)
