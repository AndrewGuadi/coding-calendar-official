uppercased

The 'uppercased' method is a String method in Swift that returns a new string with all the characters of the original string converted to uppercase. It is useful for formatting strings in a uniform uppercase style, often required for display, searching, or comparison tasks. This method does not modify the original string but returns a new instance containing the uppercased characters.

original_string = 'hello world'  
uppercased_string = original_string.uppercased()  
print(uppercased_string)  # Output: 'HELLO WORLD'
original_string = 'hello Python programmer'  
uppercased_string = original_string.uppercased()  
print(uppercased_string)  # Output: 'HELLO PYTHON PROGRAMMER'
original_string = 'Swift String Methods'  
uppercased_string = original_string.uppercased()  
print(uppercased_string)  # Output: 'SWIFT STRING METHODS'
