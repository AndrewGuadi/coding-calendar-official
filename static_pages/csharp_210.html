Add

The 'Add' method is typically used to add an element to a collection, such as a list, set, or dictionary in C#. This method allows you to append a new item to the existing collection, thereby increasing its size. When using the 'Add' method, the type of the elements in the collection must be compatible with the collectionâ€™s defined type.

```python
# Example of using the Add method with a List in C#

class Program:
    def main():
        # Defining a list
        numbers = []  # This will be our list

        # Adding numbers using the Add method
        numbers.append(10)
        numbers.append(20)
        numbers.append(30)

        # Displaying the list contents
        print(numbers)  # Output will be [10, 20, 30]

    if __name__ == '__main__':
        main()
```
```python
# Example of using the Add method with a Dictionary in C#

class Program:
    def main():
        # Defining a dictionary
        my_dict = {}  # This will be our dictionary

        # Adding key-value pairs using the Add method
        my_dict['a'] = 1
        my_dict['b'] = 2
        my_dict['c'] = 3

        # Displaying the dictionary contents
        print(my_dict)  # Output will be {'a': 1, 'b': 2, 'c': 3}

    if __name__ == '__main__':
        main()
```
```python
# Example of using the Add method with a Set in C#

class Program:
    def main():
        # Defining a set
        my_set = set()  # This will be our set

        # Adding elements to the set using the Add method
        my_set.add('apple')
        my_set.add('banana')
        my_set.add('cherry')

        # Displaying the set contents
        print(my_set)  # Output will be {'apple', 'banana', 'cherry'}

    if __name__ == '__main__':
        main()
```
