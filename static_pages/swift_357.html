uppercased

The 'uppercased' method in Swift is used to transform a string into its uppercase equivalent. When this method is called on a string, it returns a new string where all the characters are in uppercase. This is particularly useful for cases where case consistency is important, such as when comparing strings, formatting output, or preparing data for display. The 'uppercased' method does not modify the original string but provides a new string with all uppercase characters.

```python
text = 'Hello, World!'
uppercased_text = text.upper()
print(uppercased_text)  # Output: HELLO, WORLD!
```
```python
normal_string = 'python programming'
upper_string = normal_string.upper()
print(upper_string)  # Output: PYTHON PROGRAMMING
```
```python
mixed_case = 'sWiFt Is Fun!'
upper_case_result = mixed_case.upper()
print(upper_case_result)  # Output: SWIFT IS FUN!
```
