charAt

The 'charAt' method in JavaScript is used to retrieve a character at a specified index from a string. The index is zero-based, meaning that the first character is at index 0, the second character at index 1, and so on. If the index is out of bounds (i.e., greater than or equal to the string length or less than 0), 'charAt' returns an empty string. This method is useful for string manipulation and character extraction.

def get_character_at_index(s, index):
    return s[index] if index < len(s) else ''

# Example usage
string_example = 'Hello, World!'
index_example = 4
print(get_character_at_index(string_example, index_example))  # Output: 'o'
def first_character(s):
    return s[0] if len(s) > 0 else ''

# Example usage
string_example = 'Python'
print(first_character(string_example))  # Output: 'P'
def last_character(s):
    return s[-1] if len(s) > 0 else ''

# Example usage
string_example = 'Data Science'
print(last_character(string_example))  # Output: 'e'
