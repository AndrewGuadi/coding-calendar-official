strlen

The `strlen` function in PHP is used to determine the length of a given string. It returns the number of bytes in a string. Note that multibyte characters may take more than one byte; for counting characters in a multibyte string, consider using `mb_strlen` instead. This function takes one parameter, which is the string you want to measure, and it returns an integer that represents the length of the string.

string_value = 'Hello, World!'
length = strlen(string_value)
print(length)  # Output: 13
string_value = 'Python'
length = strlen(string_value)
print(length)  # Output: 6
