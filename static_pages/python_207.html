capitalize

The 'capitalize' method in Python is a string method that returns a new string where the first character is converted to uppercase and the rest of the characters are converted to lowercase. This method is particularly useful when you want to ensure that a string has a standard format with only the first letter capitalized. If the first character is not alphabetic, it just returns the string with the first character unchanged and the rest of the letters in lowercase.

text = 'hello world'
capitalized_text = text.capitalize()
print(capitalized_text)  # Output: 'Hello world'

text = 'pyTHon PrOgrAMminG'
capitalized_text = text.capitalize()
print(capitalized_text)  # Output: 'Python programming'
