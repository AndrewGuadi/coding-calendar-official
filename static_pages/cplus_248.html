std::vector::push_back

The 'push_back' method in C++'s std::vector class is used to add a new element at the end of the vector. This increases the size of the vector by one and assigns the new element the value provided. If the capacity of the vector is reached, it automatically reallocates the memory to accommodate the new element, ensuring that all operations are managed efficiently. This is particularly useful for dynamically-sized arrays where the number of elements is not known beforehand.

```cpp
#include 
#include 

int main() {
    std::vector pythonExamples;
    pythonExamples.push_back("print('Hello, World!')");
    pythonExamples.push_back("def add(a, b): return a + b");

    for (const auto &example : pythonExamples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
