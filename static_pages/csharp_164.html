Add

The 'Add' method in C# is commonly used to add an item to a collection, such as a list or a dictionary. In the context of a list, it appends the specified object to the end of the list. In the context of a dictionary, it adds a key-value pair. This method is crucial for dynamically building collections during runtime.

list = []
list.append('value1')
print(list)  # Output: ['value1']
dict_values = {}
dict_values['key1'] = 'value1'
print(dict_values)  # Output: {'key1': 'value1'}
my_list = []
my_list.append('value2')
my_list.append('value3')
print(my_list)  # Output: ['value2', 'value3']
