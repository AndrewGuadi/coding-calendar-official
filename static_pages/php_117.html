strlen

The 'strlen' function in PHP is used to calculate the length of a given string, returning the number of characters it contains. This function is useful when you need to perform validations or manipulations based on the string length, such as checking if an input meets certain requirements (e.g., password length). It operates by taking a single parameter â€“ the string for which the length is to be determined.

$input = "Hello, World!";
$length = strlen($input);
echo "The length of the input string is: $length"; // Output: The length of the input string is: 13
$password = "SecureP@ssw0rd";
if(strlen($password) < 8) {
    echo "Password must be at least 8 characters long.";
} else {
    echo "Password accepted.";
} // Output: Password accepted.
$username = "user123";
if(strlen($username) > 15) {
    echo "Username is too long. Maximum length is 15 characters.";
} else {
    echo "Username is valid.";
} // Output: Username is valid.
