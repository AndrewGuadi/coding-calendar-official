std::vector::push_back

The 'push_back' method of the std::vector class in C++ is used to add a new element at the end of the vector. This method increases the size of the vector by one, and the new element will contain the value passed to the push_back method. If the vector's allocated storage is full, the vector will automatically increase its capacity to accommodate the new element. This method is useful when you need to dynamically add elements to a vector during runtime. The time complexity is amortized constant time O(1).

```
#include 
#include 

int main() {
    std::vector languages;
    
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    for(const auto &lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
