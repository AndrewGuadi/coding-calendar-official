Add

The 'Add' method in C# is used to append elements to a collection, such as a List or a Dictionary. It increases the size of the collection and allows the user to manage data dynamically. This method plays a crucial role when dealing with collections in many applications, enabling the addition of new items at runtime without needing to know the size of the collection beforehand.

```python
# Example of using List in C# to add elements

my_list = []  # Create an empty list
my_list.append('First Item')  # Add first item
my_list.append('Second Item')  # Add second item
print(my_list)  # Output: ['First Item', 'Second Item']
```
```python
# Example of using Dictionary in C# to add key-value pairs

my_dict = {}  # Create an empty dictionary
my_dict['key1'] = 'value1'  # Add first key-value pair
my_dict['key2'] = 'value2'  # Add second key-value pair
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
# Example of using a custom collection in C# to add objects

class Item:
    def __init__(self, name, value):
        self.name = name
        self.value = value

class ItemCollection:
    def __init__(self):
        self.items = []  # Initialize an empty list

    def add(self, item):
        self.items.append(item)  # Add an item to the list

# Create a collection and add multiple items
collection = ItemCollection()
collection.add(Item('Item1', 100))
collection.add(Item('Item2', 200))

for item in collection.items:
    print(item.name, item.value)  # Output: Item1 100
                  #         Item2 200
```
