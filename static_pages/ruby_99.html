upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method does not change the original string; instead, it returns a new string with all lowercase letters changed to uppercase. The method is part of Ruby's String class.

original_string = 'hello world' 
 
 uppercased_string = original_string.upcase
 print(uppercased_string)
greeting = 'good morning' 
 
 uppercased_greeting = greeting.upcase
 print(uppercased_greeting)
phrase = 'ruby is fun' 
 
 uppercased_phrase = phrase.upcase
 print(uppercased_phrase)
