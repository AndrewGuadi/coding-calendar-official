strlen

The strlen() function in PHP is used to get the length of a given string. It returns the number of characters in the string passed to it, including spaces and special characters. If the input is not a string, it is converted to a string before the length is calculated. This function is particularly helpful when validating input lengths, counting characters for user-generated content, or parsing strings.

def string_length(string):
    return len(string)

# Example usage
example_string = 'Hello, World!'
length = string_length(example_string)
print(f'The length of the string is: {length}')
def get_string_length(text):
    return len(text)

# Example usage
example_text = 'Python Programming'
length = get_string_length(example_text)
print(f'Length of the provided text is: {length}')
def calculate_length(string_input):
    return len(string_input)

# Example usage
input_string = 'OpenAI GPT-3'
length_value = calculate_length(input_string)
print(f'The length of the input string is: {length_value}')
