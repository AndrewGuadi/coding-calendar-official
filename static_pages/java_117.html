acos

The `acos` method is part of the Java `Math` class, and it returns the arc cosine of a specified value. The arc cosine function is the inverse of the cosine function and returns the angle in radians whose cosine is the specified value. The argument must be in the range of -1.0 to 1.0, as these are the valid inputs to the cosine function. If the input is outside this range, the method will return NaN (Not a Number). This method is often used in scenarios involving trigonometry, such as calculating angles in various geometric calculations.

import math

# Function to calculate the angle in radians for a cosine value

def calculate_angle(cos_value):
    if cos_value < -1 or cos_value > 1:
        return 'Invalid input. Input must be between -1 and 1.'
    angle_radians = math.acos(cos_value)
    return angle_radians

# Usage
print(calculate_angle(0.5))  # Expected output: 1.0471975511965979 (in radians)
import math

# Function to demonstrate arc cosine usage

def angle_from_cosine(cos_value):
    angle_degrees = math.degrees(math.acos(cos_value))
    return angle_degrees

# Usage
print(angle_from_cosine(0.0))  # Expected output: 90.0 (in degrees)
import math

# Function to find the angle for a list of cosine values

def angles_for_multiple_cosine(cos_values):
    return [math.degrees(math.acos(cos_val)) for cos_val in cos_values if -1 <= cos_val <= 1]

# Usage
print(angles_for_multiple_cosine([1, 0, -1]))  # Expected output: [0.0, 90.0, 180.0] (as degrees)
