std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one, effectively allowing you to append new data to the container. The element added can be of any data type that the vector is designed to hold, and it adjusts the allocated storage of the vector if the current capacity is exceeded.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Add "Hello" to the vector
    myVector.push_back("World"); // Add "World" to the vector

    for (const auto& str : myVector) {
        std::cout << str << " "; // Output: Hello World
    }
    return 0;
}
```
