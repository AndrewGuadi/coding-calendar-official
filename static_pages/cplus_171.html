std::vector::push_back

The 'push_back' method in C++ is a member function of the std::vector class that adds a new element at the end of the vector. This method increases the size of the vector by one. It can take any value type that the vector is defined to hold. When using 'push_back', the vector automatically handles memory management behind the scenes, resizing as necessary to accommodate additional elements.

# Example: Adding elements to an integer vector
#include 
#include 

int main() {
    std::vector numbers;
    
    // Adding elements using push_back
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);
    
    // Print the vector.
    for (int num : numbers) {
        std::cout << num << ' '; // Output: 1 2 3
    }
    return 0;
}

    fruits.push_back("apple");

    fruits.push_back("cherry");

}
