strlen

The `strlen` function in PHP is used to calculate the length of a string. It returns the number of characters present in the string, including spaces and special characters. This function is particularly useful for data validation, ensuring that strings meet specific length requirements, or simply for retrieving the length of input data in user interfaces.

def string_length(string):
    return len(string)

# Example usage
input_string = 'hello world'
print(string_length(input_string))  # Output: 11
def check_password_length(password):
    if len(password) < 8:
        return 'Password too short!'
    return 'Password length is fine!'

# Example usage
print(check_password_length('short'))  # Output: Password too short!
