std::vector::push_back

The 'std::vector::push_back' method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one, placing the new element at the last position. If the vector has reached its capacity, it automatically reallocates memory to accommodate the new element, ensuring that the vector remains contiguous in memory. The push_back method is typically used in scenarios where dynamic arrays are needed, and flexibility to grow in size is required.

```python
# Example code to demonstrate the use of std::vector::push_back in C++.

#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Python");  // Adding element 'Python'
    myVector.push_back("Java");    // Adding element 'Java'
    myVector.push_back("C++");     // Adding element 'C++'

    // Displaying the elements of the vector
    for (const std::string& language : myVector) {
        std::cout << language << std::endl;
    }
    return 0;
}
```

```python
# Another example demonstrating dynamically adding elements to a vector.

#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("Apple");  // Adding 'Apple'
    fruits.push_back("Banana"); // Adding 'Banana'
    fruits.push_back("Cherry"); // Adding 'Cherry'

    std::cout << "Fruits List:" << std::endl;
    for (std::size_t i = 0; i < fruits.size(); ++i) {
        std::cout << i + 1 << ". " << fruits[i] << std::endl;
    }
    return 0;
}
```

```python
# Example showing how to push back elements in a growing vector and print its size.

int main() {
    std::vector countries;
    countries.push_back("Canada");
    countries.push_back("USA");
    countries.push_back("Mexico");
    countries.push_back("Brazil");

    std::cout << "Number of countries: " << countries.size() << std::endl; // Prints size of vector.

    // Print all countries
    std::cout << "Countries in the vector:" << std::endl;
    for (const auto& country : countries) {
        std::cout << country << std::endl;
    }
    return 0;
}
```
