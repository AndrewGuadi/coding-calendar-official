std::vector::push_back

The `push_back` method of the C++ Standard Library's `std::vector` class is used to add a new element at the end of the vector. This method increases the size of the vector by one and inserts the new element in the last position, which can be of any built-in type or user-defined type. If the vector needs to expand its capacity to accommodate the new element, it will automatically reserve more memory for growth. The operation is average-case O(1), but can sometimes be O(n) in the worst case when the vector resizes.

```cpp
#include 
#include 

int main() {
    std::vector stringVector;
    stringVector.push_back("Hello");
    stringVector.push_back("World");

    for (const auto& str : stringVector) {
        std::cout << str << ' ';
    }
    return 0;
} // Output: Hello World
```
```cpp
#include 
#include 

int main() {
    std::vector intVector;
    for (int i = 0; i < 5; ++i) {
        intVector.push_back(i * 10);
    }

    for (const auto& num : intVector) {
        std::cout << num << ' ';
    }
    return 0;
} // Output: 0 10 20 30 40
```
