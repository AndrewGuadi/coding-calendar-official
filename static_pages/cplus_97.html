std::vector::push_back

The `std::vector::push_back` method is used to add an element to the end of a vector in C++. It increases the size of the vector by one and constructs the new element in place. If the vector's capacity is reached, it will automatically allocate more memory to accommodate the new element. This method is useful for dynamically building a list of items without needing to preallocate storage for a fixed number of items.

```python
# Example to demonstrate adding elements to a vector using push_back

import random

# Function to create a vector and add random integers

def create_vector():
    vec = []  # Initialize an empty vector
    for _ in range(10):  # Loop 10 times to add 10 random numbers
        num = random.randint(1, 100)  # Get a random number between 1 and 100
        vec.append(num)  # Using push_back equivalent append in Python
    return vec

# Call the function and print the resulting vector
resulting_vector = create_vector()
print("Resulting vector:", resulting_vector)
```

```python
# Example to append strings to a vector using push_back

# Function to create a vector of strings

def create_string_vector():
    strings = []  # Initialize an empty vector of strings
    for i in range(5):  # Loop 5 times
        strings.append(f'Item {i + 1}')  # Create a string and append it
    return strings

# Call the function and print the resulting vector
resulting_string_vector = create_string_vector()
print("Resulting string vector:", resulting_string_vector)
```

```python
# Example to append dictionaries to a vector using push_back

# Function to create a vector of dictionaries

def create_dict_vector():
    dicts = []  # Initialize an empty vector of dictionaries
    for i in range(3):  # Loop 3 times
        itemDict = {'index': i, 'value': i ** 2}  # Create a dictionary with index and square value
        dicts.append(itemDict)  # Append dictionary to the vector
    return dicts

# Call the function and print the resulting vector
resulting_dict_vector = create_dict_vector()
print("Resulting dictionary vector:", resulting_dict_vector)
```
