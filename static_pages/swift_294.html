uppercased

The `uppercased()` method is a function in Swift that returns a new string with all the characters in uppercase. This method is a property of the `String` type and can be particularly useful for normalizing string input or comparing strings in a case-insensitive manner. It does not modify the original string but instead generates a new string instance with the transformed characters.

let originalString = "hello world";
let uppercasedString = originalString.uppercased();
print(uppercasedString)  // Output: "HELLO WORLD"
let greeting = "Good Morning";
let loudGreeting = greeting.uppercased();
print(loudGreeting)  // Output: "GOOD MORNING"
let message = "welcome to swift programming!";
let loudMessage = message.uppercased();
print(loudMessage)  // Output: "WELCOME TO SWIFT PROGRAMMING!"
