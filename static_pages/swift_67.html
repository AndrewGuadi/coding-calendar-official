uppercased

The `uppercased` method in Swift is used to convert all characters in a string to uppercase. This is particularly useful for standardizing textual data, ensuring consistency in case representation, or when implementing features that require case-sensitive comparisons.

let originalString = "hello world"
let uppercasedString = originalString.uppercased()
print(uppercasedString)  // Output: "HELLO WORLD"
let mixedCaseString = "Swift Programming"
let resultString = mixedCaseString.uppercased()
print(resultString)  // Output: "SWIFT PROGRAMMING"
let name = "Alice Smith"
let uppercasedName = name.uppercased()
print(uppercasedName)  // Output: "ALICE SMITH"
