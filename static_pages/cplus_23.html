std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of a vector. This method dynamically increases the size of the vector and copies the given value into the newly created space at the end. The time complexity of this operation is amortized O(1), meaning that although it may take longer during certain reallocations, on average, it operates in constant time. This function is commonly used to build up a list of items dynamically as they become available during program execution.

```c++
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("C++");
    languages.push_back("JavaScript");

    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
