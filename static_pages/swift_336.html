uppercased

The 'uppercased' method is a built-in function in Swift that converts all the characters in a String to uppercase. This method returns a new string where all alphabetic characters have been converted to uppercase, leaving non-alphabetic characters unchanged. It is particularly useful when you need to standardize text input for comparison or presentation purposes.

original_string = 'hello world'
uppercased_string = original_string.uppercased()
print(uppercased_string)  # Output: 'HELLO WORLD'
sentence = 'Swift is a powerful language'
uppercased_sentence = sentence.uppercased()
print(uppercased_sentence)  # Output: 'SWIFT IS A POWERFUL LANGUAGE'
greeting = 'Good Morning!'
uppercased_greeting = greeting.uppercased()
print(uppercased_greeting)  # Output: 'GOOD MORNING!'
