append

The `append` function in Go is used to add elements to the end of a slice. The `append` function takes a slice and one or more elements to add, returning a new slice that includes the original elements along with the newly added ones. If the backing array of the original slice does not have enough capacity to accommodate the new elements, a new array will be allocated, and the original elements will be copied over.

```go
package main

import "fmt"

func main() {
    // Create an initial slice of integers
    nums := []int{1, 2, 3}

    // Append additional numbers to the slice
    nums = append(nums, 4, 5, 6)

    // Print the modified slice
    fmt.Println(nums) // Output: [1 2 3 4 5 6]
}
```
```go
package main

import "fmt"

func main() {
    // Create a slice of strings
    fruits := []string{"apple", "banana"}

    // Append a single item
    fruits = append(fruits, "cherry")

    // Append multiple items
    fruits = append(fruits, "date", "elderberry")

    // Print the modified slice
    fmt.Println(fruits) // Output: [apple banana cherry date elderberry]
}
```
```go
package main

import "fmt"

func main() {
    // Create a slice of slices (2D)
    matrix := [][]int{{1, 2}, {3, 4}}

    // Append a new row to the matrix
    newRow := []int{5, 6}
    matrix = append(matrix, newRow)

    // Print the modified matrix
    fmt.Println(matrix) // Output: [[1 2] [3 4] [5 6]]
}
```
