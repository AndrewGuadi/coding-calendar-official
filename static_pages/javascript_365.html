charAt

The `charAt` method in JavaScript is used to retrieve the character at a specified index from a string. It takes one argument, which is the index of the character you want to access, and returns the character at that index. If the index is out of bounds (less than 0 or greater than or equal to the string's length), it returns an empty string. This method is useful for string manipulation and accessing specific characters within a string.

string = 'hello';
char = string.charAt(1);
# Output: 'e' # The character at index 1 is 'e'
string = 'programming';
char = string.charAt(0);
# Output: 'p' # The character at index 0 is 'p'
string = 'data';
char = string.charAt(3);
# Output: 'a' # The character at index 3 is 'a'
