std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This method dynamically increases the size of the vector to accommodate the new element, making it a convenient way to build a vector incrementally. When the element is added, the total number of elements in the vector increases by one. If the vector runs out of space, it automatically allocates more memory to ensure that the new element can be added.

```cpp
#include 
#include 

int main() {
    std::vector myVec;
    myVec.push_back("Hello");
    myVec.push_back("World");
    for(const auto& str : myVec) {
        std::cout << str << " ";
    }
    return 0;
}```
