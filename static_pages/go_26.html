append

The 'append' function in Go is used to add elements to the end of a slice. It can dynamically increase the size of the slice, making it a flexible data structure for handling collections of data. You can append single elements or another slice. When appending another slice, the elements of the second slice will be added to the end of the first slice.

```go
package main
import (
	"fmt"
)
func main() {
	// Initial slice of integers
	numbers := []int{1, 2, 3}
	// Append a single value
	numbers = append(numbers, 4)
	fmt.Println(numbers) // Output: [1 2 3 4]
}
```
```go
package main
import (
	"fmt"
)
func main() {
	// Initial slice of strings
	names := []string{"Alice", "Bob"}
	// Append multiple strings
	names = append(names, "Charlie", "Dave")
	fmt.Println(names) // Output: [Alice Bob Charlie Dave]
}
```
```go
package main
import (
	"fmt"
)
func main() {
	// Initial slice
	numbers := []int{10, 20, 30}
	// Another slice to append
	moreNumbers := []int{40, 50}
	// Append another slice
	numbers = append(numbers, moreNumbers...)
	fmt.Println(numbers) // Output: [10 20 30 40 50]
}
```
