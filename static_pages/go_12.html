append

The 'append' function in Go is used to add elements to slices. It takes a slice and one or more elements as arguments and returns a new slice that includes the original elements with the new ones appended. It is a variadic function, meaning it can accept a variable number of arguments. If the capacity of the slice is exceeded, a new array is allocated and the elements copied over.

s := []int{1, 2, 3}
 s = append(s, 4)
 fmt.Println(s)  // Output will be [1, 2, 3, 4]
s := []string{"apple", "banana"}
 s = append(s, "cherry", "date")
 fmt.Println(s) // Output will be ["apple", "banana", "cherry", "date"]
initialSlice := []string{"Python"}
 newSlice := append(initialSlice, "Go", "Java")
 fmt.Println(newSlice) // Output will be ["Python", "Go", "Java"]
