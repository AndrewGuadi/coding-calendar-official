close

In Go, the `Close()` method is typically used to close resources that need to be released, such as files, network connections, or database connections. It is an important method to ensure that resources do not leak, which can lead to performance issues or system resource exhaustion. The `Close()` method generally returns an error to indicate if there was any issue while closing the resource, which should be checked in applications to handle potential errors appropriately.

import os

# Open a file and ensure it's closed after use.
file, err = os.Open('example.txt')
if err != nil:
    print(f'Error opening file: {err}')
else:
    # Perform file operations here
    print(file.read())
    # Close the file to free resources
    file.close()
import sqlite3

# Connect to a SQLite database
connection = sqlite3.connect('example.db')
# Ensure the connection is closed properly after operations
try:
    cursor = connection.cursor()
    cursor.execute('SELECT * FROM table_name')
    results = cursor.fetchall()
    print(results)
finally:
    # Close the connection
    connection.close()
import socket

# Create a TCP/IP socket
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
# Attempt to connect to a server
try:
    sock.connect(('localhost', 8080))
    sock.sendall(b'Hello, World!')
finally:
    # Always close the socket
    sock.close()
