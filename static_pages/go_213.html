close

In Go, the 'close' method is used to close a channel. Closing a channel informs all receivers that no more values will be sent on that channel. It is important to note that attempting to send a value on a closed channel will panic, and receiving from a closed channel will yield the zero value of the channel's type along with a boolean that indicates whether the channel is still open. Closing a channel should be done only by the sender and only when no more values will be sent. The syntax for closing a channel is `close(channelName)`. It is a good practice to just close the channel when all data has been sent to avoid any panics.

```go
package main
import (
    "fmt"
)

func main() {
    ch := make(chan string)
    
    go func() {
        ch <- "Hello"
        ch <- "World"
        close(ch) // Closing the channel here
    }()
    
    for msg := range ch { // This loop will receive until the channel is closed
        fmt.Println(msg)
    }
}
```
```go
package main
import (
    "fmt"
)

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 5; i++ {
            ch <- i
        }
        close(ch) // Close channel after sending all values
    }()
    
    for val := range ch {
        fmt.Println(val) // Receives values until channel is closed
    }
}
```
```go
package main
import (
    "fmt"
)

func main() {
    ch := make(chan int)
    
    go func() {
        for i := 1; i <= 3; i++ {
            ch <- i * 10
        }
        close(ch) // Close the channel after sending
    }()
    
    for val, ok := <-ch; ok; val, ok = <-ch { // Receiving from a closed channel
        fmt.Println(val)
    }
}
```
