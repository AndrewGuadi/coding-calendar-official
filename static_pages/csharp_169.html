Add

The Add method is commonly used in C# collections (like List, Dictionary, HashSet, etc.) to insert a new element into the collection. The specific implementation and behavior can vary depending on the type of collection. For instance, in a List, it adds the given element at the end of the list, while in a Dictionary, it adds a key-value pair. The Add method typically does not return a value.

```python
# Using List to demonstrate the Add method equivalent in Python
my_list = []

# Equivalent Add operation: appending an element
my_list.append('New element')

# Display the list
print(my_list)  # Output: ['New element']
```
```python
# Using a dictionary in Python to demonstrate the Add method equivalent
my_dict = {}

# Equivalent Add operation: adding a key-value pair
my_dict['key1'] = 'value1'

# Display the dictionary
print(my_dict)  # Output: {'key1': 'value1'}
```
```python
# Using a set in Python to demonstrate the Add method equivalent
my_set = set()

# Equivalent Add operation: adding an element
my_set.add('New item')

# Display the set
print(my_set)  # Output: {'New item'}
```
