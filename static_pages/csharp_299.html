Add

The 'Add' method is commonly used to insert elements into a collection such as a list, dictionary, or any other data structure that supports addition of items. In C#, this method can be found in several types like List, Dictionary, HashSet, etc. The method typically takes one or more parameters depending on the underlying data structure. For example, in a List, you might use `Add(T item)` to append an item to the end of the list.

```python
# Example 1: Adding items to a list in C#
myList = [1, 2, 3]
myList.Add(4)  # C# equivalent: myList.Add(4)
print(myList)  # Output: [1, 2, 3, 4]
```
```python
# Example 2: Adding a key-value pair to a dictionary in C#
myDict = {'a': 1, 'b': 2}
myDict.Add('c', 3)  # C# equivalent: myDict.Add('c', 3)
print(myDict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
```python
# Example 3: Adding items to a HashSet in C#
mySet = {1, 2, 3}
mySet.Add(4)  # C# equivalent: mySet.Add(4)
print(mySet)  # Output: {1, 2, 3, 4}
```
