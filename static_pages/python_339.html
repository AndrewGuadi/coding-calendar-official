ascii

The 'ascii' function in Python returns a string containing a printable representation of an object. This representation is a string that represents the object as a sequence of ASCII characters. If the object contains non-ASCII characters, they are replaced with escape sequences (e.g., ' ' for newline, ' ' for tab, etc.). This method is useful for debugging and logging, ensuring that the output is correctly encoded and does not contain unprintable or non-ASCII characters.

s = 'Hello, World!'
print(ascii(s))  # Output: 'Hello, World!'
