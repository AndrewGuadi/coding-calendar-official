acos

The 'acos' method in Java is part of the 'java.lang.Math' class and is used to compute the arc cosine (inverse cosine) of a given number. The return value is in radians, which means if you want the result in degrees, you'll need to convert it. The valid input range for this method is between -1.0 and 1.0. If the input is outside this range, the method will return NaN (Not a Number). This method is useful in various mathematical computations, particularly in trigonometry, where determining angles is necessary based on given cosine values.

import math

# Calculate the angle (in radians) whose cosine is 0.5
cosine_value = 0.5
angle_in_radians = math.acos(cosine_value)

# Print the result
print(f'The arc cosine of {cosine_value} is:', angle_in_radians)
import math

# Calculate the angle (in degrees) whose cosine is -1
cosine_value = -1
angle_in_radians = math.acos(cosine_value)
angle_in_degrees = math.degrees(angle_in_radians)

# Print the result
print(f'The arc cosine of {cosine_value} in degrees is:', angle_in_degrees)
import math

# Example with a value out of the acos range
cosine_value = 2.0
angle_in_radians = math.acos(cosine_value)

# Handle NaN case
if math.isnan(angle_in_radians):
    print(f'The value {cosine_value} is out of range for acos.')
else:
    print(f'The arc cosine of {cosine_value} is:', angle_in_radians)
