Add

The 'Add' method in C# is used to add elements to a collection such as a List, Dictionary, or HashSet. When using the 'Add' method, a new entry is created and appended to the end of the collection if it is a List or added to the collection if it is a Dictionary or HashSet, which maintains a unique set of keys. This method allows for dynamic manipulation of collections during runtime.

```python
# Example of using 'Add' method with a List
my_list = []
my_list.append('Python')
my_list.append('C#')
print(my_list)  # Output: ['Python', 'C#']
```
```python
# Example of using 'Add' method with a Dictionary
my_dict = {}
my_dict['language'] = 'Python'
my_dict['year'] = 2023
print(my_dict)  # Output: {'language': 'Python', 'year': 2023}
```
```python
# Example of using 'Add' method with a Set
my_set = set()
my_set.add('Python')
my_set.add('C#')
my_set.add('Python')  # Duplicates are ignored
print(my_set)  # Output: {'Python', 'C#'}
```
