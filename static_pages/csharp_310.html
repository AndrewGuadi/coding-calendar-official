Add

The `Add` method is used to insert an element into a collection such as a list, dictionary, or set in C#. When adding an element, the element will be placed at the end of the collection for a list, or as a key-value pair in a dictionary. It's particularly useful for dynamically managing collections where the size may not be known beforehand. When using a dictionary, you must provide both a key and a value to add an entry, while for lists, only the value needs to be provided. In C#, the `Add` method is defined in various classes, including List, Dictionary, and HashSet.

```python
# Example of using Add with a List
my_list = []  # Define an empty list
my_list.append('Python')  # Using Add to append 'Python' to the list
my_list.append('C#')
print(my_list)  # Output: ['Python', 'C#']
```
```python
# Example of using Add with a Dictionary
my_dict = {}  # Define an empty dictionary
my_dict['language'] = 'Python'  # Using Add to insert key-value pair
my_dict['version'] = '3.10'
print(my_dict)  # Output: {'language': 'Python', 'version': '3.10'}
```
```python
# Example of using Add with a Set
my_set = set()  # Define an empty set
my_set.add('Python')  # Using Add to insert an element into the set
my_set.add('C#')
my_set.add('Python')  # Adding duplicate, will not change the set
print(my_set)  # Output: {'Python', 'C#'}
```
