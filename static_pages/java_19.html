abs

The 'abs' method in Java is a static method used to compute the absolute value of a given number. It can handle various numeric types, including integers, long integers, floats, and doubles. The method returns the non-negative value of the number, which means that if the input is negative, it returns its positive counterpart, and if the input is already positive or zero, it returns the value as is. This method is part of the java.lang.Math class.

num = -5; result = Math.abs(num); // result will be 5
num = 4.5; result = Math.abs(num); // result will be 4.5
num = -3.14; result = Math.abs(num); // result will be 3.14
