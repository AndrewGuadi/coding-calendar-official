std::vector::push_back

The 'push_back' method is used to add a new element at the end of a vector. This increases the size of the vector by one and allows for dynamic sizing, as the vector can automatically allocate more memory to accommodate new elements. If the vector needs to resize, it typically reallocates its memory and copies the existing elements to the new memory location, which can incur some overhead. The complexity of the operation is amortized O(1), making it an efficient way to append elements.

```
#include 
#include 

int main() {
    std::vector python_codes;
    python_codes.push_back("print('Hello World')");
    python_codes.push_back("for i in range(5): print(i)");

    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
