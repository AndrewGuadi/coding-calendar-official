Add

The 'Add' method is used to insert a new element into a collection such as a list, dictionary, or any other data structure that supports adding new items. In the context of a collection, this method typically takes an item as a parameter and appends it to the end of a list or adds it to a dictionary with a specified key. The 'Add' method is commonly utilized when dynamically managing data in applications where data can change during runtime.

```python
# Example of using the Add method with a list in C#
# This code snippet illustrates adding integers to a list.
class Program:
    def main(self):
        my_list = []  # Initialize an empty list.
        my_list.append(1)  # Add the integer 1 to the list.
        my_list.append(2)  # Add the integer 2 to the list.
        my_list.append(3)  # Add the integer 3 to the list.
        print(my_list)  # Output: [1, 2, 3]

if __name__ == '__main__':
    Program().main()
```
```python
# Example of using the Add method with a dictionary in C#
# This code snippet demonstrates adding key-value pairs to a dictionary.
class Program:
    def main(self):
        my_dict = {}  # Initialize an empty dictionary.
        my_dict['a'] = 1  # Add a key-value pair 'a': 1.
        my_dict['b'] = 2  # Add a key-value pair 'b': 2.
        my_dict['c'] = 3  # Add a key-value pair 'c': 3.
        print(my_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}

if __name__ == '__main__':
    Program().main()
```
```python
# Example of using the Add method with a list of custom objects in C#
# This code snippet illustrates adding instances of a custom class to a list.
class Item:
    def __init__(self, name):
        self.name = name

class Program:
    def main(self):
        items = []  # Initialize an empty list for items.
        items.append(Item('Apple'))  # Add an Item with name 'Apple'.
        items.append(Item('Banana'))  # Add an Item with name 'Banana'.
        items.append(Item('Cherry'))  # Add an Item with name 'Cherry'.
        for item in items:
            print(item.name)  # Output: Apple, Banana, Cherry

if __name__ == '__main__':
    Program().main()
```
