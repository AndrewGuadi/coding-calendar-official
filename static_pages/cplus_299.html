std::vector::push_back

The 'push_back' method adds a new element to the end of a vector in C++. This method increases the size of the vector by one and initializes the new element with the given value. It is commonly used for dynamically growing vectors, where elements are added on-the-fly as needed. The 'push_back' method ensures that the vector manages memory efficiently, reallocating space if the current capacity is exceeded.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto &str : myVector) {
        std::cout << str << " ";
    }
    return 0;
}
```
```cpp
#include 
#include 

int main() {
    std::vector myNumbers;
    for (int i = 1; i <= 5; ++i) {
        myNumbers.push_back(i);
    }
    for (const auto &num : myNumbers) {
        std::cout << num << " ";
    }
    return 0;
}
```
