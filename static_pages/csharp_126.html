Add

The 'Add' method in C# is commonly used to insert an element into a collection, such as a list, dictionary, or any custom collection class that implements this method. It typically takes one or more parameters and adds them to the collection, increasing the size of the collection by one. The 'Add' method does not return a value; however, it modifies the state of the collection.

# Example 1: Adding an element to a List

list = []  # Create an empty list
list.append('First Element')  # Add an element to the list
print(list)  # Output: ['First Element']
# Example 2: Adding multiple elements to a List

list = []  # Create an empty list
list.append('First Element')
list.append('Second Element')  # Add another element to the list
print(list)  # Output: ['First Element', 'Second Element']
# Example 3: Using a custom object with the 'Add' method

class MyCollection:
    def __init__(self):
        self.items = []

    def Add(self, item):  # Custom add method
        self.items.append(item)  # Add item to the collection

my_collection = MyCollection()  # Create a new collection
my_collection.Add('Custom Item 1')  # Add an item to the collection
print(my_collection.items)  # Output: ['Custom Item 1']
