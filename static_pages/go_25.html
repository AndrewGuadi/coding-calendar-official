append

In Go, the 'append' function is used to add elements to a slice. It can take one or more elements as arguments and returns a new slice that includes the original elements of the slice along with the new elements. This method is particularly useful for dynamic slice management as it allows modifications to the length of the slice without the need for manual resizing.

```go
package main
import "fmt"

func main() {
    // Create a slice of integers
    numbers := []int{1, 2, 3}
    
    // Append a single integer to the slice
    numbers = append(numbers, 4)
    // Append multiple integers to the slice
    numbers = append(numbers, 5, 6, 7)
    
    // Print the modified slice
    fmt.Println(numbers) // Output: [1 2 3 4 5 6 7]
}
```
```go
package main

func main() {
    // Create a slice of strings
    fruits := []string{"apple", "banana"}
    
    // Append a single string to the slice
    fruits = append(fruits, "orange")
    // Append multiple strings to the slice
    fruits = append(fruits, "grape", "kiwi")
    
    // Print the modified slice
    fmt.Println(fruits) // Output: [apple banana orange grape kiwi]
}
```
