ascii

The built-in `ascii` function in Python returns a string containing a printable representation of an object, but escapes the non-ASCII characters in the string using (e.g. U+xxxx). This is useful for debugging or when you need a string safe for representation, instead of the raw character representation that might not be printable. It is a simple alternative to the `repr` function that only includes printable ASCII characters in the output.
