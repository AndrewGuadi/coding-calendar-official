std::vector::push_back

The 'std::vector::push_back' method is a member function of the 'std::vector' class in C++. It is used to add a new element to the end of a vector, dynamically increasing its size. When using 'push_back', the vector will allocate more memory if needed to accommodate the new element. This function does not return a value and can throw exceptions if memory allocation fails. It is commonly used in scenarios where the size of the data collection is not known in advance, allowing for dynamic storage of elements.

In this example, we will create a vector of strings and use 'push_back' to add multiple lines of Python code that will be printed out later:

```cpp
#include 
#include 
#include 

int main() {
    std::vector codeLines;
    codeLines.push_back("print(\"Hello, World!\")");
    codeLines.push_back("for i in range(5):");
    codeLines.push_back("    print(i)");

    // Output the saved lines
    for (const auto &line : codeLines) {
        std::cout << line << std::endl;
    }
    return 0;
}
```

int main() {
    std::vector functionSnippets;
    functionSnippets.push_back("def add(a, b):");
    functionSnippets.push_back("    return a + b");
    functionSnippets.push_back("print(add(2, 3))");

    // Print each function snippet stored
    for (const auto &snippet : functionSnippets) {
        std::cout << snippet << std::endl;
    }
    return 0;
}
