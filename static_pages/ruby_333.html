upcase

The 'upcase' method in Ruby is used to convert all the lowercase letters in a string to their uppercase equivalents. It returns a new string with the transformation applied, leaving the original string unchanged. This method is often used when you want to standardize text input or prepare data for case-sensitive comparisons.

name = 'ruby programming'
upcase_name = name.upcase
print(upcase_name)  # Output: 'RUBY PROGRAMMING'
phrase = 'Hello World!'
upcase_phrase = phrase.upcase
print(upcase_phrase)  # Output: 'HELLO WORLD!'
text = 'data science'
upcase_text = text.upcase
print(upcase_text)  # Output: 'DATA SCIENCE'
