Add

The 'Add' method is typically used to insert a new element into a collection such as a List, Dictionary, or Set in C#. This method allows the user to specify the value to be added, and it manages the underlying data structure to accommodate the new item appropriately. In the case of a List, it appends the item to the end, while a Dictionary pairs the value with a specific key. This method returns no value but modifies the collection it is called on.

```python
# Example of using the Add method with a List in C#
class Program:
    def __init__(self):
        self.numbers = []  # Initialize an empty list

    def add_numbers(self, number):
        self.numbers.append(number)  # Using Add method equivalent in Python (append)

my_program = Program()
my_program.add_numbers(5)
my_program.add_numbers(10)
print(my_program.numbers)  # Output: [5, 10]
```
```python
# Example of using the Add method with a Dictionary in C#
class Program:
    def __init__(self):
        self.data = {}  # Initialize an empty dictionary

    def add_entry(self, key, value):
        self.data[key] = value  # Using Add method equivalent in Python (key-value assignment)

my_program = Program()
my_program.add_entry('name', 'Alice')
my_program.add_entry('age', 30)
print(my_program.data)  # Output: {'name': 'Alice', 'age': 30}
```
```python
# Example of using the Add method with a Set in C#
class Program:
    def __init__(self):
        self.unique_values = set()  # Initialize an empty set

    def add_value(self, value):
        self.unique_values.add(value)  # Using Add method equivalent in Python (add)

my_program = Program()
my_program.add_value(1)
my_program.add_value(2)
my_program.add_value(1)  # Duplicate value, will not be added
print(my_program.unique_values)  # Output: {1, 2}
```
