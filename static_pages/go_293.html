close

The 'close' method in Go is primarily used to shut down resources, typically net connections, files, or channels. It signals that the resource is no longer needed, allowing the underlying resources to be freed or terminated. Properly using 'close' is crucial for preventing resource leaks and ensuring that files or network connections are gracefully terminated.

import os

def read_file(file_path):
    file = open(file_path, 'r')
    try:
        data = file.read()
        print(data)
    finally:
        file.close()

read_file('example.txt')

def process_channel(ch):
    try:
        while True:
            item = next(ch)
            print(item)
    except StopIteration:
        pass
    finally:
        ch.close()

my_channel = iter(range(5))
process_channel(my_channel)
