acos

The 'acos' method in Java is a part of the java.lang.Math class and is used to return the arccosine of a number. The value passed to this method should be in the range of -1 to 1. The result is returned in radians, which can then be converted to degrees if necessary. The method is a static method, meaning it is called on the Math class rather than on an instance of the class. The acosh value corresponds to the angle whose cosine is the specified number, hence it is called arccosine.

import math

# Example to demonstrate the usage of acos
# x is the cosine value for which we need to find the angle
x = 0.5

# Calculate arccosine in radians
angle_radians = math.acos(x)

# Convert radians to degrees
angle_degrees = math.degrees(angle_radians)

print('Arccosine of', x, 'in radians is:', angle_radians)
print('Arccosine of', x, 'in degrees is:', angle_degrees)

import math

# Example with edge cases
x1 = 1.0
x2 = 0.0
x3 = -0.5

# Calculate arccosine in radians for each case
angles_radians = [math.acos(x) for x in (x1, x2, x3)]

# Convert radians to degrees
angles_degrees = [math.degrees(angle) for angle in angles_radians]

for i in range(len(angles_radians)):
    print('Arccosine of', (x1, x2, x3)[i], 'in radians is:', angles_radians[i])
    print('Arccosine of', (x1, x2, x3)[i], 'in degrees is:', angles_degrees[i])
