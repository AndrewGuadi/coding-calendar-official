charAt

The `charAt` method in JavaScript is used to return the character at a specified index (position) in a string. The index is zero-based, meaning that the first character has index 0, the second character has index 1, and so on. If the specified index is out of range, `charAt` returns an empty string.

text = 'Hello, World!';
character = text.charAt(0); // returns 'H'

# Using charAt to get the first character of the string from a PHP code logic
text = 'abcdef';
character = text.charAt(3); // returns 'd'

text = 'JavaScript';
character = text.charAt(5); // returns 'S'
