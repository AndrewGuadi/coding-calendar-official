charAt

The `charAt()` method in JavaScript is used to return the character at a specified index (position) in a string. The index is zero-based, meaning the first character has an index of 0, the second an index of 1, and so on. If the index is out of bounds (less than 0 or greater than or equal to the string's length), it returns an empty string. This method is useful when you need to retrieve a single character from a string and manipulate it for various purposes.

def get_character_at_index_example_1():\n    example_string = 'Hello, world!'\n    index = 4  # Index of the character 'o'\n    character = example_string[index]  # Getting the character at index 4\n    return character  # Returns 'o'\n\nresult_1 = get_character_at_index_example_1()\nprint(result_1)
