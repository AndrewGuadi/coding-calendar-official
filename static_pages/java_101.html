abs

The abs method in Java is used to return the absolute value of a given number. This method is part of the Math class and can handle various data types including integer, double, and float. The absolute value of a number is its distance from zero on the number line, regardless of its sign. For instance, abs(-10) returns 10, whereas abs(10) also returns 10. It is useful in scenarios where the magnitude of a number is required without consideration of its sign.

def calculate_absolute_difference(x, y):
    difference = x - y
    return abs(difference)

result = calculate_absolute_difference(-5, 5)
print(result)  # Output: 10
def is_positive(x):
    return abs(x) == x

print(is_positive(-7))  # Output: False
print(is_positive(7))   # Output: True
def distance_to_origin(x, y):
    return abs(x) + abs(y)

print(distance_to_origin(-3, 4))  # Output: 7
print(distance_to_origin(3, -4))  # Output: 7
