Add

The 'Add' method in C# is typically used to add an element to a collection or a list data structure. It can be applied to various collection types such as List, Dictionary, HashSet, etc. When using the 'Add' method, you pass the item you want to add as a parameter. This method modifies the collection by inserting the specified item at the end of the collection if there is space, or it may increase the capacity if itâ€™s a list.

```python
class C:
    def __init__(self):
        self.numbers = []  
    
    def add(self, number):
        self.numbers.append(number)
        
my_list = C()
my_list.add(1)
my_list.add(2)
print(my_list.numbers)  # Output: [1, 2]
```
```python
class Contacts:
    def __init__(self):
        self.contact_list = {}
    
    def add_contact(self, name, phone):
        self.contact_list[name] = phone
        
my_contacts = Contacts()
my_contacts.add_contact('Alice', '123-4567')
my_contacts.add_contact('Bob', '765-4321')
print(my_contacts.contact_list)  # Output: {'Alice': '123-4567', 'Bob': '765-4321'}
```
```python
class UniqueItems:
    def __init__(self):
        self.items = set()
    
    def add_item(self, item):
        self.items.add(item)
        
my_items = UniqueItems()
my_items.add_item('apple')
my_items.add_item('banana')
my_items.add_item('apple')  # Duplicate will not be added
print(my_items.items)  # Output: {'banana', 'apple'}
```
