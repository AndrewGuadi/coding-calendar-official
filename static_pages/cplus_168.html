std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class (part of the C++ Standard Library). This method is used to add a new element at the end of a vector. The `push_back` function will automatically increase the size of the vector to accommodate the new element if necessary. It takes a single argument, which is the value to be added to the vector. This operation is typically performed in constant time, although it may take longer under certain conditions (e.g., when the vector needs to allocate more memory).

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    // Adding Python code as strings
    myVector.push_back("print('Hello, World!')");
    myVector.push_back("a = 5");
    myVector.push_back("print(a)");

    // Output the contents of the vector
    for (const auto& code : myVector) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
