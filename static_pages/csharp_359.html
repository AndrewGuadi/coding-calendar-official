Add

The 'Add' method in C# is typically used to insert a new element into a collection, such as a List, Dictionary, or any other collection that implements the ICollection interface. It takes the element to be added as an argument and appends it to the end of the collection or, in the case of a Dictionary, creates a new key-value entry. This method is fundamental for managing collections of data dynamically at runtime.

```python
my_list = []

def add_element(element):
    my_list.append(element)

add_element('Hello')
add_element('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}

def add_key_value(key, value):
    my_dict[key] = value

add_key_value('name', 'John Doe')
add_key_value('age', 30)
print(my_dict)  # Output: {'name': 'John Doe', 'age': 30}
```
```python
class ItemCollection:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

collection = ItemCollection()
collection.add('Item 1')
collection.add('Item 2')
print(collection.items)  # Output: ['Item 1', 'Item 2']
```
