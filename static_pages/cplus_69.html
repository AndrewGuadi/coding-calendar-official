std::vector::push_back

The method 'std::vector::push_back' is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one, and the new element is initialized with the value passed to the method. If the vector's current capacity is exceeded, 'push_back' may also trigger a reallocation of the underlying array to accommodate the new element. It is important to note that 'push_back' operates in amortized constant time, but the worst-case time can be linear in the case of reallocation. The element added is copied into the vector, which may involve invoking copy constructors if the type of the vector is a class. This method does not return a value.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Python");
    vec.push_back("Java");
    vec.push_back("C++");

    for (const std::string &value : vec) {
        std::cout << value << std::endl;
    }
    return 0;
}
```
