Add

The 'Add' method in C# is commonly used to insert a new item into a collection, such as a list or a dictionary. In most cases, the 'Add' method allows you to append a single element to the end of the collection or add a key-value pair in the case of dictionaries. This method is fundamental for dynamically managing collections as it alters the size or contents of the collection.

```python
my_list = []
my_list.append('new_item')
print(my_list)  # Output: ['new_item']
```
```python
dictionary = {}
dictionary['key'] = 'value'
print(dictionary)  # Output: {'key': 'value'}
```
```python
class CustomList:
    def __init__(self):
        self.items = []
    
    def add(self, item):
        self.items.append(item)

my_custom_list = CustomList()
my_custom_list.add('item1')
my_custom_list.add('item2')
print(my_custom_list.items)  # Output: ['item1', 'item2']
```
