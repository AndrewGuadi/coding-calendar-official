abs

The abs() method in Python is used to return the absolute value of a number. The absolute value is the non-negative value of a number without regard to its sign. This method can be used with integers, floating-point numbers, and complex numbers. For complex numbers, it returns the magnitude of the number. The syntax for the abs() method is: abs(number), where 'number' is the number you want to find the absolute value of.

print(abs(-10))  # Output: 10. Finds the absolute value of a negative integer.
print(abs(-3.14))  # Output: 3.14. Finds the absolute value of a negative float.
print(abs(3 + 4j))  # Output: 5.0. Finds the magnitude of a complex number.
