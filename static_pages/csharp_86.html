Add

The 'Add' method is commonly used in C# to add an element to a collection, such as a list or a dictionary. In a list, it appends the element to the end, while in a dictionary, it adds a key-value pair. This method is vital for dynamically managing data as it allows modification of collections at runtime.

```python
# Example of adding an element to a list
my_list = []
my_list.append('Python')
my_list.append('C#')
print(my_list)  # Output: ['Python', 'C#']
```
```python
# Example of adding a key-value pair to a dictionary
my_dict = {}
my_dict['language'] = 'Python'
my_dict['version'] = '3.9'
print(my_dict)  # Output: {'language': 'Python', 'version': '3.9'}
```
```python
# Example of adding elements to a set
my_set = set()
my_set.add('Python')
my_set.add('C#')
print(my_set)  # Output: {'Python', 'C#'}  # (Unordered set output)
```
