std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one, thereby allowing for dynamic array resizing. It takes a single argument, which is the value of the element to be added. If the vector's capacity is exceeded, it automatically reallocates memory to accommodate the new elements, ensuring there is enough space for the new item. It is commonly used to build lists or collections of elements in a flexible manner without needing to specify an initial size.

```python
# create an empty list to hold strings
my_list = []

# use push_back logic to add elements
my_list.append('Hello')
my_list.append('World')

# final output
print(my_list)  # Output: ['Hello', 'World']
```
```python
# Creating an empty list to hold integers
numbers = []

# use push_back logic to add integers
for i in range(5):
    numbers.append(i)

# final output
print(numbers)  # Output: [0, 1, 2, 3, 4]
```
```python
# Initializing an empty list for characters
char_list = []

# Using push_back logic to insert characters
for char in 'hello':
    char_list.append(char)

# final output
print(char_list)  # Output: ['h', 'e', 'l', 'l', 'o']
```
