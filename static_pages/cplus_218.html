std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector, increasing the vector's size by one. The element is copied or moved into the vector, and the capacity may be reallocated if necessary to accommodate the new size. This method is particularly useful for building dynamic arrays where the size is not known at compile time.

```cpp
#include 
#include 

int main() {
    std::vector myVector;

    // Adding some Python code as strings to the vector
    myVector.push_back("print('Hello, World!')");
    myVector.push_back("x = 5");
    myVector.push_back("y = x + 3");

    for (const auto &code : myVector) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
