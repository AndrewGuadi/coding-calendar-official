Add

The 'Add' method in C# is commonly used to insert an item into a collection such as a List, Dictionary, or Set. For example, in a List, the 'Add' method appends an element to the end of the list. This method is crucial for dynamically-building collections where elements can be added during runtime, ensuring that code can accommodate varying amounts of data without needing to define fixed sizes at compile time.

```python
# Example of using Add method in a List

class CustomList:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

# Create an instance of CustomList
my_list = CustomList()

# Add items
my_list.add("Hello")
my_list.add("World")

# View the added items
print(my_list.items)  # Output: ['Hello', 'World']
```
```python
# Example of using Add method in a Dictionary

class CustomDictionary:
    def __init__(self):
        self.items = {}

    def add(self, key, value):
        self.items[key] = value

# Create an instance of CustomDictionary
my_dict = CustomDictionary()

# Add items
my_dict.add("name", "John")
my_dict.add("age", "30")

# View the added items
print(my_dict.items)  # Output: {'name': 'John', 'age': '30'}
```
```python
# Example of using Add method in a Set

class CustomSet:
    def __init__(self):
        self.items = set()

    def add(self, item):
        self.items.add(item)

# Create an instance of CustomSet
my_set = CustomSet()

# Add items
my_set.add("Python")
my_set.add("C#")
my_set.add("Python")  # Duplicates are ignored

# View the added items
print(my_set.items)  # Output: {'Python', 'C#'}
```
