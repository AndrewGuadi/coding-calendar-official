uppercased

The `uppercased` method in Swift is used to convert all the characters of a given string to their uppercase equivalents. This method is a part of the String class and does not mutate the original string but instead returns a new string that contains the uppercase version of the original string.

let originalString = "hello world";
let uppercasedString = originalString.uppercased();
print(uppercasedString)  // Output: "HELLO WORLD"
let pythonCode = "print('Hello, world!')";
let uppercasedPythonCode = pythonCode.uppercased();
print(uppercasedPythonCode)  // Output: "PRINT('HELLO, WORLD!')"
let message = "This is a test message.";
let uppercasedMessage = message.uppercased();
print(uppercasedMessage)  // Output: "THIS IS A TEST MESSAGE."
