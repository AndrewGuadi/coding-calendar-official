std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector, expanding the vector's size by one. When an element is added, if the vector's capacity is reached, the vector will allocate new memory for a larger size and then copy the existing elements over. This method effectively allows for dynamic resizing of vectors, which is one of their most useful features.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    // Adding elements to the vector
    vec.push_back("print('Hello, World!')");
    vec.push_back("for i in range(5): print(i)");
    vec.push_back("def add(a, b): return a + b");

    // Display the contents of the vector
    for (const auto& code : vec) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
