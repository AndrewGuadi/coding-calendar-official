std::vector::push_back

The `std::vector::push_back` method is used to add an element at the end of a vector in C++. This method increases the size of the vector by one and initializes the new element to the value provided as an argument. If the vector needs to allocate additional memory to accommodate the new element, it will do so automatically. It is a crucial method for dynamically building a collection of items in C++. As of C++11, the method allows for move semantics, enabling efficient passing of temporary objects.

```cpp
#include 
#include 

int main() {
    std::vector python_codes;
    python_codes.push_back("print('Hello, World!')");
    python_codes.push_back("for i in range(5): \n    print(i)");

    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
