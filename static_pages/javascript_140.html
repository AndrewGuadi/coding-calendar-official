charAt

The 'charAt' method in JavaScript is used to return the character at a specified index (position) in a string. If the index is out of bounds (less than 0 or greater than the length of the string minus one), it returns an empty string. The first character in a string is at index 0, the second at index 1, and so on.

string = 'hello';
result = string.charAt(0);
# result: 'h'  # Getting first character
string = 'world';
result = string.charAt(1);
# result: 'o'  # Getting second character
string = 'javascript';
result = string.charAt(10);
# result: 't'  # Getting last character of 'javascript'
