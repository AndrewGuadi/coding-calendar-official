acos

The `acos` method in Java is part of the `java.lang.Math` class and is used to return the arc cosine of a value. This method takes a single argument, which must be in the range of -1.0 to 1.0 inclusive, and returns the angle in radians whose cosine is the specified value. If the argument is outside of this range, the method will return NaN (not a number). This method is primarily used in mathematical calculations, particularly in geometry and trigonometry, when one needs to determine angles based on cosine values.

import math

# Calculate the arc cosine of a value
value = 0.5
result = math.acos(value)
print('The arc cosine of', value, 'is', result)  # Expected output: 1.0471975511965979 (which is π/3 in radians)
import math

# Calculate the arc cosine of another value
value = -1.0
result = math.acos(value)
print('The arc cosine of', value, 'is', result)  # Expected output: 3.141592653589793 (which is π in radians)
import math

# Check for an invalid input value
value = 2.0
result = math.acos(value)
print('The arc cosine of', value, 'is', result)  # Expected output: nan (not a number, because 2 is out of the range [-1, 1])
