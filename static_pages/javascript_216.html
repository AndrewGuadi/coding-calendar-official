charAt

The 'charAt' method in JavaScript is a string method that returns the character located at a specified index in a given string. The index is zero-based, meaning that the first character in the string is at index 0. If the index specified is out of bounds (negative or greater than the length of the string), 'charAt' will return an empty string. This method is often used when you need to access specific characters within a string for processing, manipulation, or analysis.

my_string = 'Python Programming';
char = my_string[0]; 
# char will be 'P' - accessing the first character
my_string = 'Data Science';
char = my_string[5]; 
# char will be ' ' - accessing the sixth character which is a space
my_string = 'Artificial Intelligence';
char = my_string[19]; 
# char will be 'e' - accessing the twentieth character
