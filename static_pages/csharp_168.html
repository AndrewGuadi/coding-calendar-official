Add

The 'Add' method in C# is used to add a new element to a collection, such as a list, dictionary, or set. The method typically takes one or more parameters that represent the item(s) to be added to the collection. This method is commonly associated with collection types that implement the ICollection interface, allowing for the expansion of the collection by adding new items dynamically.

```python
# Example 1: Adding an element to a List in C#
# Create a list of integers
numbers = new List();
# Add numbers to the list
numbers.Add(10);
numbers.Add(20);
numbers.Add(30);
// Output: 10, 20, 30
```
```python
# Example 2: Adding key-value pairs to a Dictionary in C#
# Create a dictionary to hold string keys and integer values
personAge = new Dictionary();
# Add key-value pairs to the dictionary
personAge.Add("Alice", 30);
personAge.Add("Bob", 25);
// Output: Alice -> 30, Bob -> 25
```
```python
# Example 3: Adding items to a HashSet in C#
# Create a HashSet of strings
uniqueNames = new HashSet();
# Add names to the HashSet
uniqueNames.Add("Alice");
uniqueNames.Add("Bob");
uniqueNames.Add("Alice"); // Duplicates will not be added
// Output: Alice, Bob
```
