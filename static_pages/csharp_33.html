Add

The 'Add' method is commonly used to add an element to a collection in C#. This can refer to adding items to lists, dictionaries, or other data structures. This method typically modifies the collection in place and ensures that the added item meets the requirements of the collection (e.g., unique keys in dictionaries). The 'Add' method is a straightforward way to extend the size of the collection by appending new items to it.

To add an element to a List in C#:
```C#
List fruits = new List();
fruits.Add("Apple");
fruits.Add("Banana");
fruits.Add("Cherry");
// fruits will now contain "Apple", "Banana", "Cherry"
```
To add a key-value pair to a Dictionary in C#:
```C#
Dictionary ages = new Dictionary();
ages.Add("Alice", 30);
ages.Add("Bob", 25);
// ages will now contain the pairs "Alice": 30 and "Bob": 25
```
To add an element to a HashSet in C#:
```C#
HashSet uniqueNames = new HashSet();
uniqueNames.Add("John");
uniqueNames.Add("Doe");
uniqueNames.Add("John"); // This will not be added again since HashSet does not allow duplicates.
// uniqueNames will now contain "John", "Doe"
```
