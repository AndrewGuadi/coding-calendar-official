std::vector::push_back

The 'push_back' method is used in C++ to add a new element to the end of a vector. A vector is a dynamic array, and 'push_back' allows for efficient growth of the array by adding new items one at a time. When an element is added with 'push_back', the vector automatically resizes if necessary to accommodate the new element. This method does not return a value.

```cpp
#include 
#include 

int main() {
    // Declare a vector of strings
    std::vector my_vector;

    // Using push_back to add string elements
    my_vector.push_back("element_1");
    my_vector.push_back("element_2");
    my_vector.push_back("element_3");

    // Output the vector elements
    for (const auto& elem : my_vector) {
        std::cout << elem << std::endl;
    }
    return 0;
}
```
