abs

The 'abs' method in Python is used to return the absolute value of a given number. The absolute value of a number is its distance from zero without considering its direction (i.e., it is always non-negative). The method works with integers, floating-point numbers, and complex numbers (for complex numbers, it returns the magnitude).

number = -10
absolute_value = abs(number)
print('The absolute value of -10 is:', absolute_value)
# Output: The absolute value of -10 is: 10
number = -3.14
absolute_value = abs(number)
print('The absolute value of -3.14 is:', absolute_value)
# Output: The absolute value of -3.14 is: 3.14
complex_number = 3 - 4j
absolute_value = abs(complex_number)
print('The magnitude of 3 - 4j is:', absolute_value)
# Output: The magnitude of 3 - 4j is: 5.0
