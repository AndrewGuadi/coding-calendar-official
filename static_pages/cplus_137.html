std::vector::push_back

The std::vector::push_back method in C++ is used to add an element at the end of a vector. This method increases the size of the vector by one and initializes the new element with the value provided. It is an efficient way to append new items to a vector, as it is generally implemented to achieve amortized constant time complexity. However, when the vector's capacity is exceeded, the vector must allocate more memory, which can lead to a performance hit.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Adding string 'Hello'
    myVector.push_back("World"); // Adding string 'World'

    for (const auto& str : myVector) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}
```
