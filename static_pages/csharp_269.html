Add

The 'Add' method is typically used in C# collections to add a new element to an existing collection, such as a list, set, or dictionary. This method is essential for dynamically modifying a collection's contents during runtime. The specific behavior of the 'Add' method depends on the type of collection being used. In general, the 'Add' method modifies the collection by inserting a new element at the end (in the case of lists) or by associating a key and value (in the case of dictionaries).

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonExamples = new List();
        pythonExamples.Add("print('Hello, World!')");
        pythonExamples.Add("for i in range(5):\n    print(i)");
        pythonExamples.Add("def greet(name):\n    return f'Hello, {name}!'");

        foreach (var example in pythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
```
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary pythonFunctions = new Dictionary();
        pythonFunctions.Add("len", "length of an object");
        pythonFunctions.Add("str", "convert to string");
        pythonFunctions.Add("int", "convert to integer");

        foreach (var item in pythonFunctions)
        {
            Console.WriteLine($"Function: {item.Key}, Description: {item.Value}");
        }
    }
}
```
