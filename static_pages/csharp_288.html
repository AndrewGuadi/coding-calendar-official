Add

The 'Add' method in C# is typically used for adding an item to a collection, such as a List, Dictionary, or HashSet. It allows you to insert a new element into the collection. In List collections, the new item is added at the end, while in a Dictionary, a key-value pair is added. This method is commonly used in scenarios where dynamic data storage is required, enabling you to build collections that can grow as needed during the execution of a program.

```python
# Example of using Add in a List in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        List languages = new List();
        languages.Add("Python");  # Adding a new language to the list
        languages.Add("C#");
        languages.Add("Java");

        Console.WriteLine(string.Join(", ", languages));  # Output: Python, C#, Java
    }
}
```
```python
# Example of using Add in a Dictionary in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        Dictionary scores = new Dictionary();
        scores.Add("Alice", 85);  # Adding a score for Alice
        scores.Add("Bob", 92);    # Adding a score for Bob

        foreach (var entry in scores) {
            Console.WriteLine(entry.Key + ": " + entry.Value);  # Outputs:
            # Alice: 85
            # Bob: 92
        }
    }
}
```
```python
# Example of using Add in a HashSet in C#

using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        HashSet uniqueNames = new HashSet();
        uniqueNames.Add("Alice");  # Adding a unique name
        uniqueNames.Add("Bob");
        uniqueNames.Add("Alice");  # Duplicate, will not be added

        foreach (var name in uniqueNames) {
            Console.WriteLine(name);  # Outputs:
            # Alice
            # Bob
        }
    }
}
```
