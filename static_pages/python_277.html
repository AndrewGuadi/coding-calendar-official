ascii

The `ascii()` function in Python returns a string containing a printable representation of an object. The representation is similar to what `repr()` would return, but it converts non-ASCII characters to their escape sequences. This is particularly useful when dealing with strings that may contain non-ASCII characters, enabling the output to be safely viewed and processed in ASCII only environments.

my_string = 'Café'
result = ascii(my_string)
print(result)  # Output: 'Caf\xe9'
my_dict = {'name': 'Alice', 'city': 'São Paulo'}
result = ascii(my_dict)
print(result)  # Output: '{'name': 'Alice', 'city': 'S\xa3o Paulo'}'
example_list = ['Python', 'ü', 'こんにちは']
result = ascii(example_list)
print(result)  # Output: "['Python', '\xfc', '\u3053\u306e\u3072\u3068']"
