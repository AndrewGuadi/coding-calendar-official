strlen

The 'strlen' function in PHP is used to calculate the length of a string, returning the number of characters in it. This includes all types of characters, such as letters, numbers, spaces, punctuation, and special characters. It is a commonly used function to validate string lengths in applications, helping to ensure that user inputs meet specific criteria, like minimum or maximum lengths.

string_value = 'Hello, World!'
length = strlen(string_value)
print('Length of string:', length)
string_value = 'Python programming'
length = strlen(string_value)
if length > 20:
    print('String too long')
else:
    print('String length is fine')
string_value = '1234567890'
length = strlen(string_value)
if length == 10:
    print('Valid number of digits')
else:
    print('Invalid number of digits')
