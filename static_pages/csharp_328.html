Add

The `Add` method in C# is used to add a new element to a collection, such as a list, dictionary, or any other collection that implements the `ICollection` or `IDictionary` interfaces. This method modifies the collection by inserting a new item, and the specific behavior may vary depending on the type of collection being modified. If the collection has constraints on its contents (like keys for dictionaries), the `Add` method will throw exceptions if the constraints are violated.

```python
# Example of using the Add method with a List in C#[
# This example demonstrates creating a List and adding Python code snippets.

from System.Collections.Generic import List

# Create a new List of strings.
code_snippets = List[str]()

# Adding Python code snippets to the List.
code_snippets.Add("print('Hello, World!')")  
code_snippets.Add("for i in range(5): print(i)")

# Display the code snippets added to the list.
for snippet in code_snippets:
    print(snippet)
```
```python
# Example of using the Add method with a Dictionary in C#[
# This example demonstrates creating a Dictionary and adding key-value pairs of Python code snippets.

from System.Collections.Generic import Dictionary

# Create a new Dictionary with string keys and values.
code_snippets_dict = Dictionary[str, str]()

# Add Python code snippets to the Dictionary with descriptive keys.
code_snippets_dict.Add("print_hello", "print('Hello, World!')")
code_snippets_dict.Add("print_range", "for i in range(5): print(i)")

# Display the code snippets in the dictionary.
for key in code_snippets_dict.Keys:
    print(f'{key}: {code_snippets_dict[key]}')
```
```python
# Example of using the Add method with a HashSet in C#[
# This example demonstrates creating a HashSet and adding unique Python code snippets.

from System.Collections.Generic import HashSet

# Create a new HashSet of strings.
code_set = HashSet[str]()

# Adding Python code snippets to the HashSet.
code_set.Add("print('Hello, World!')")
code_set.Add("for i in range(5): print(i)")
code_set.Add("print('Hello, World!')") # This will not add a duplicate.

# Display the unique code snippets in the HashSet.
for snippet in code_set:
    print(snippet)
```
