acos

The 'acos' method in Java is a part of the 'Math' class and is used to calculate the arc cosine (inverse cosine) of a given value. The input should be a double value between -1.0 and 1.0, and it returns the angle in radians between 0 and Ï€ (which is approximately 3.14159). If the input is outside this range, Math.acos will return NaN (Not-a-Number). This method is often used in applications involving trigonometry and geometry, such as calculating angles based on the sides of a triangle.

import math

# Calculate the angle whose cosine is 0.5
cosine_value = 0.5
angle_in_radians = math.acos(cosine_value)
angle_in_degrees = math.degrees(angle_in_radians)
print(f'The angle in radians is: {angle_in_radians}')
print(f'The angle in degrees is: {angle_in_degrees}')
