std::vector::push_back

The 'push_back' method in C++ is used to add an element to the end of a vector, effectively increasing its size by one. This method allows for dynamic arrays where elements can be added sequentially without manually managing memory. The new element is copied or moved into the vector, depending on its type. If the vectorâ€™s capacity is exceeded, it will automatically allocate more memory to accommodate the new element.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("hello");
    vec.push_back("world");

    for (const auto &str : vec) {
        std::cout << str << " ";
    }
    return 0;
}
```

```cpp
    numbers.push_back(3);

    int sum = 0;
    for (int num : numbers) {
        sum += num;
    }
    std::cout << "Sum: " << sum << std::endl;
```
