std::vector::push_back

The std::vector::push_back method is a member function of the C++ Standard Library's std::vector class that adds a new element at the end of the vector, increasing its size by one. This method accepts a single argument, which is the value to be added. If the current capacity of the vector is exceeded, it automatically increases the storage capacity to accommodate the new element, ensuring that the vector can store more elements as needed. This method is useful when you want to dynamically grow the vector based on the application requirements.

def populate_vector_with_numbers(n: int) -> List[int]:
