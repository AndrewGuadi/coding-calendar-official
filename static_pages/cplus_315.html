std::vector::push_back

The std::vector::push_back method is a member function of the C++ Standard Library's vector class. It is used to add a new element at the end of the vector, increasing its size by one. This method is useful for dynamically growing the vector when new data is being collected. If necessary, the method automatically allocates additional memory to accommodate the new element. The complexity of this operation is amortized constant time; however, if allocation is needed, then it can be linear. Note that the element being added is typically passed by value.

    myVector.push_back("First item");
    myVector.push_back("Second item");
    myVector.push_back("Third item");
    
    // Output the contents of the vector

    }
    return 0;
}
