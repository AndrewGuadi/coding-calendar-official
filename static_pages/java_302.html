acos

The 'acos' method in Java is a static method of the Math class that returns the arc cosine (inverse cosine) of a specified value in radians. The argument must be in the range of -1.0 to 1.0, and the return value is the angle in radians whose cosine is the specified number. If the argument is outside this range, NaN (not a number) is returned. This method is useful for scenarios where you need to calculate angles in trigonometric calculations.

import math

# Calculate the arc cosine of 0.5
value = 0.5
angle_in_radians = math.acos(value)
angle_in_degrees = math.degrees(angle_in_radians)
print(f'The angle whose cosine is {value} is {angle_in_degrees} degrees.')
import math

# Calculate the arc cosine of -1

value = 2
angle_in_radians = math.acos(value)
