charAt

The `charAt()` method in JavaScript is used to return the character at a specified index in a string. If the index is out of bounds (i.e., less than zero or greater than the length of the string), it returns an empty string. This method is particularly useful when you need to access individual characters in a string at specific positions.

const greeting = 'Hello, World!';
const charAt3 = greeting.charAt(3);
console.log(charAt3);  // Output: 'l'
const word = 'JavaScript';
const firstChar = word.charAt(0);
const lastChar = word.charAt(word.length - 1);
console.log(firstChar);  // Output: 'J'
console.log(lastChar);  // Output: 't'
const emptyString = '';
const charAtIndex5 = emptyString.charAt(5);
console.log(charAtIndex5);  // Output: '' (empty string)
