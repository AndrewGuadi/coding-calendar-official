std::vector::push_back

The std::vector::push_back() method is used to add a new element to the end of a vector in C++. It automatically increases the size of the vector to accommodate the new element. If the capacity of the vector is exceeded, it will allocate more memory to ensure that there is enough space for the new elements. This method effectively allows dynamic resizing of the vector as elements are added. The time complexity of push_back is amortized constant time, although occasionally it can take linear time when resizing occurs.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    for (const auto &str : myVector) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}
```
