charAt

The 'charAt' method in JavaScript is used to target a specific index of a string, retrieving the character located there. It takes one parameter, which is the index of the character you want to return. The index is zero-based, meaning that the first character in the string is at index 0. If the index provided is out of bounds, the method will return an empty string. This method is useful for string manipulation and extraction tasks.

string = 'Hello World'; char = string.charAt(0); print(char) # Output: 'H'
string = 'Python is fun'; char = string.charAt(7); print(char) # Output: 'i'
string = 'abcdef'; char = string.charAt(10); print(char) # Output: ''
