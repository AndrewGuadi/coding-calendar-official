charAt

The 'charAt' method in JavaScript is used to access the character at a specific index in a string. It takes a single parameter, which is the index of the character you want to retrieve. The index is zero-based, meaning that the first character is at index 0, the second character at index 1, and so on. If the index provided is out of range, 'charAt' will return an empty string. This method is useful for retrieving characters from strings for comparison or processing tasks.

string_value = 'Hello World';

char_at_index_0 = string_value.charAt(0);  // H
char_at_index_4 = string_value.charAt(4);  // o
char_at_out_of_range = string_value.charAt(15);  // ''
message = 'OpenAI is amazing!';

first_character = message.charAt(0);  // O
ninth_character = message.charAt(8);  // a
greater_than_length_character = message.charAt(100);  // ''
name = 'JavaScript';

second_character = name.charAt(1);  // a
seventh_character = name.charAt(6);  // c
negative_index_character = name.charAt(-1);  // ''
