uppercased

The `uppercased` method is used in Swift to return a new string that contains the same characters as the original string, but with all letters converted to uppercase. This method does not modify the original string but instead creates a new string with the uppercase characters. It is commonly used for formatting text for display or comparison purposes.

let greeting = "Hello, World!"
let uppercasedGreeting = greeting.uppercased()
print(uppercasedGreeting)  // Output: "HELLO, WORLD!"
let password = "securePass123"
let uppercasedPassword = password.uppercased()
if uppercasedPassword == "SECUREPASS123" {
    print("Password matches in upper case!")
} else {
    print("Password does not match.")
}
let userInput = "swift programming"
let finalInput = userInput.uppercased()
print("The input in uppercase is: \(finalInput)")  // Output: "The input in uppercase is: SWIFT PROGRAMMING"
