Add

The 'Add' method is commonly used in collections such as lists, dictionaries, or custom collections in C#. It allows you to add a new element or key-value pair to the collection. Depending on the context, it can handle different data types such as integers, strings, or complex objects. The method modifies the collection in place and usually doesn't return a value. It is important to note that attempting to add a duplicate key to a dictionary will result in an exception.

```python
# Example of using the Add method with a list in C#
# Initialize a list
my_list = []
# Adding elements using Add method
my_list.Add('Hello')
my_list.Add('World')
# Resulting list: ['Hello', 'World']
```
```python
# Example of using the Add method with a dictionary in C#
# Initialize a dictionary
my_dict = {}
# Adding key-value pairs using Add method
my_dict.Add('name', 'John Doe')
my_dict.Add('age', 30)
# Resulting dictionary: {'name': 'John Doe', 'age': 30}
```
```python
# Example of using the Add method with a custom collection in C#
# Custom class with an Add method
class CustomCollection:
    def __init__(self):
        self.items = []
    def Add(self, item):
        self.items.append(item)

# Usage of CustomCollection
my_collection = CustomCollection()
my_collection.Add('Item 1')
my_collection.Add('Item 2')
# Resulting items: ['Item 1', 'Item 2']
```
