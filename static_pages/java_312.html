acos

The `acos` method in Java is a part of the `java.lang.Math` class and is used to compute the arc cosine (inverse cosine) of a value. It takes a single argument (a double) and returns the arc cosine in radians. The input value must be within the range of -1.0 to 1.0. If the input is outside this range, it will return NaN (Not a Number). This method is particularly useful in trigonometry, geometry, and physics calculations where you need to determine angles from their cosine values.

import math

# Example: Calculate the angle in radians for cos(theta) = 0.5
cos_value = 0.5
angle_radians = math.acos(cos_value)
# Output: The angle in radians is 1.0471975511965979 (which is 60 degrees)

cos_value = -1
