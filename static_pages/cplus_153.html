std::vector::push_back

The 'push_back' method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one and adds the specified element to the last position. If the vector's capacity needs to be increased to accommodate the new element, the method takes care of that automatically, ensuring that memory is allocated as necessary. The complexity of 'push_back' is amortized constant time, which means that most of the time it operates in constant time, although occasionally it may take longer when resizing is required.

C++ code demonstrating the use of push_back to fill a vector with integer values:
```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("example_1");
    myVector.push_back("example_2");
    myVector.push_back("example_3");

    for (const auto& str : myVector) {
        std::cout << str << std::endl;
    }
    return 0;
}
```

```cpp
int main() {
    std::vector codeSnippets;
    codeSnippets.push_back("print('Hello, World!')");
    codeSnippets.push_back("for i in range(5): print(i)");
    codeSnippets.push_back("def add(a, b): return a + b");

    for (const auto& snippet : codeSnippets) {
        std::cout << snippet << std::endl;
    }
    return 0;
}
```
