uppercased

The `uppercased` method in Swift is used to convert all the characters in a string to uppercase. This method is called on a String instance and returns a new string where all the letters are represented in upper case. It is particularly useful for formatting strings for display purposes, ensuring uniformity in text representation, or when case sensitivity is a factor in comparisons or data processing.

def uppercased_example_1():
    original_string = 'hello world'
    uppercased_string = original_string.upper()
    return uppercased_string
def uppercased_example_2():
    original_string = 'Python programming'
    uppercased_string = original_string.upper()
    return uppercased_string
def uppercased_example_3():
    original_string = 'Swift and Python'
    uppercased_string = original_string.upper()
    return uppercased_string
