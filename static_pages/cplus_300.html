std::vector::push_back

The 'std::vector::push_back' method is used to add a new element at the end of a vector in C++. This method increases the size of the vector by one and places the element at the new end of the vector. Vectors manage their own memory, and if the current capacity is not enough to fit the new element, the vector will automatically allocate more memory to accommodate the new size. This method provides an efficient way to append elements to the vector.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    myVector.push_back("Python");

    for (const auto& str : myVector) {
        std::cout << str << std::endl;
    }
    return 0;
}
```
