std::vector::push_back

The std::vector::push_back method in C++ is used to add an element to the end of a vector. This method dynamically increases the size of the vector by one and assigns the newly added value to the last position. It is an essential method for building vectors as it allows for easy addition of elements. If the vector's current capacity is exceeded, the vector will automatically allocate more memory to accommodate the new size, which may involve copying existing elements to the new memory location. It is important to note that the push_back method operates in amortized constant time (O(1)), making it efficient for dynamic array implementations.

```cpp
#include 
#include 

int main() {
    std::vector vec;

    vec.push_back("apple");
    vec.push_back("banana");
    vec.push_back("cherry");

    for (const auto &item : vec) {
        std::cout << item << std::endl;
    }
    return 0;
}
```

```cpp
#include

int main() {
    std::vector fruits;

    std::string fruit;
    std::cout << "Enter fruits (type 'stop' to end):\n";
    while (true) {
        std::cin >> fruit;
        if (fruit == "stop") break;
        fruits.push_back(fruit);
    }

    std::cout << "You have entered: \n";
    for (const auto &f : fruits) {
        std::cout << f << std::endl;
    }
    return 0;
}
```
