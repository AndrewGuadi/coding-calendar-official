ascii

The `ascii()` function in Python takes an object (like a string, list, or dictionary) and returns a string containing a printable representation of that object, but with non-ASCII characters escaped. This is useful for ensuring that all characters in the output are ASCII characters, which makes it easier to handle and display in environments that may not support non-ASCII characters.

s = 'Hello, 世界'
print(ascii(s))  # Output: 'Hello, \u4e16\u754c'
my_dict = {'name': 'Alice', 'age': 30, 'language': 'français'}
print(ascii(my_dict))  # Output: '{'name': 'Alice', 'age': 30, 'language': 'fr\u00e7ais'}'
list_with_unicode = ['Python', 'C++', 'Swift', 'こんにちは']
print(list(ascii(x) for x in list_with_unicode))  # Output: ['Python', 'C++', 'Swift', '\u3053\u306e\u3072\u306a\u307b']
