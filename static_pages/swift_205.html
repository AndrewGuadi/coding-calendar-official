uppercased

The 'uppercased' method is a string method in Swift that converts all lowercase letters in a string to their uppercase equivalents. This method is useful for formatting text to ensure consistency, especially in cases where you want to display strings in a uniform manner regardless of their original casing. The method returns a new string and does not modify the original string.

let originalString = "hello world"
let uppercasedString = originalString.uppercased()
print(uppercasedString) // Output: "HELLO WORLD"
let greeting = "Good Morning"
let uppercasedGreeting = greeting.uppercased()
print(uppercasedGreeting) // Output: "GOOD MORNING"
let mixedCase = "Swift Programming is Fun!"
let uppercasedMixed = mixedCase.uppercased()
print(uppercasedMixed) // Output: "SWIFT PROGRAMMING IS FUN!"
