std::vector::push_back

The 'push_back' method in std::vector is used to add a new element at the end of the vector. This method increases the size of the vector by one and assigns the new element to the last position. If the vector needs to resize (i.e., if its capacity is exceeded), the new capacity is typically double the old capacity. This method is efficient and commonly used for dynamic arrays in C++. Note that while 'push_back' modifies the vector, it returns nothing.

```cpp
#include 
#include 

int main() {
    std::vector programmingLanguages;
    programmingLanguages.push_back("Python");
    programmingLanguages.push_back("Java");
    programmingLanguages.push_back("C++");

    for(const auto &lang : programmingLanguages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
