std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one, and the new element is a copy of the value provided as an argument. If the vector needs to grow to accommodate the new element, memory is allocated accordingly. It is important to ensure that the vector can hold additional elements; otherwise, it might lead to performance overhead due to memory reallocation. This method is a member of the `std::vector` class and is part of the C++ Standard Template Library (STL).

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& str : myVector) {
        std::cout << str << ' ';
    }
    return 0;
}
```
