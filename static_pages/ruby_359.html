upcase

The 'upcase' method in Ruby is used to convert all the lowercase letters in a string to uppercase. This method returns a new string with the original casing unaffected, keeping the original string intact. If there are any characters in the string that are not letters (like numbers or punctuation), they are left unchanged. This method is particularly useful when you need to standardize the casing of strings for comparisons or display purposes.

text = 'hello world'
uppercase_text = text.upcase
puts uppercase_text  # Outputs: HELLO WORLD
greeting = 'Good Morning!'
upper_greeting = greeting.upcase
puts upper_greeting  # Outputs: GOOD MORNING!
data_entry = 'Data123!'
modified_entry = data_entry.upcase
puts modified_entry  # Outputs: DATA123!
