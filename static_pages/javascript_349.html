charAt

The 'charAt' method in JavaScript is used to retrieve a character from a string at a specified index. The index is zero-based, meaning that the first character is at index 0, the second character at index 1, and so on. If the specified index is out of range (not between 0 and string.length - 1), the method returns an empty string. This method is particularly useful when you need to access individual characters within a string.

string_value = 'Hello, World!';
char_at_index_0 = string_value.charAt(0);  // 'H'
char_at_index_7 = string_value.charAt(7);  // 'W'
char_at_index_out_of_range = string_value.charAt(20);  // '' (empty string)
greeting = 'Good Morning';
first_character = greeting.charAt(0);  // 'G'
last_character = greeting.charAt(greeting.length - 1);  // 'g'
negative_index = greeting.charAt(-1);  // '' (empty string)
phrase = 'Use charAt to get character';
for (let i = 0; i < phrase.length; i++) {
  console.log('Character at index ' + i + ': ' + phrase.charAt(i));
}  // Logs each character of the phrase one by one
