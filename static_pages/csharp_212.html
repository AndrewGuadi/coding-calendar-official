Add

The 'Add' method in C# is commonly used in various collections, including lists, dictionaries, and other data structures, to insert new elements into the collection. When calling this method, you typically pass the item that you want to add to the collection. The 'Add' method does not return a value and will modify the existing collection by introducing the new element. Its behavior may vary slightly depending on the specific collection being used, such as whether it allows duplicates or maintains order.

```python
# Example of using Add method to add an element to a list
my_list = []
my_list.append('Hello, C#')  # Using Python list append as equivalent to C#'s Add
print(my_list)  # Output: ['Hello, C#']
```
```python
# Example of using Add method to add a key-value pair to a dictionary
dictionary = {}
dictionary['key'] = 'value'  # Using Python dict assignment as equivalent to C#'s Add
print(dictionary)  # Output: {'key': 'value'}
```
```python
# Example of using Add method in a custom class implementing a list
class CustomList:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)  # Using Python's list method append

custom_list = CustomList()
custom_list.add('Item 1')
print(custom_list.items)  # Output: ['Item 1']
```
