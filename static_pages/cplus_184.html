std::vector::push_back

The `push_back` method is a member function of the C++ Standard Template Library (STL) container `std::vector`. This method is used to add a new element to the end of the vector. When calling `push_back`, the vector's size increases by one. If the vector's capacity is reached, it automatically reallocates its storage to accommodate the new element, ensuring that it can hold more elements. It is important to note that the `push_back` method takes its argument by value, meaning it can accept any type that is compatible with the vector. During this operation, the vector may need to manage memory allocations to accommodate the new element, which can have an impact on performance if done repeatedly in a loop without reservation.

In this example, we will create a vector of strings and use `push_back` to add python code snippets to it:

```cpp
#include 
#include 
#include 

int main() {
    std::vector pythonCodes;
    pythonCodes.push_back("print('Hello World')");  // Adds the first python snippet
    pythonCodes.push_back("for i in range(5): print(i)"); // Adds the second snippet
    pythonCodes.push_back("def greet(name): return f'Hello, {name}!'"); // Adds the third snippet

    // Output the snippets to verify
    for (const std::string& code : pythonCodes) {
        std::cout << code << std::endl;
    }

    return 0;
}
```
In this example, we'll demonstrate how to dynamically build a vector of python code examples using a loop:

```cpp
#include 
#include 
#include 

int main() {
    std::vector pythonCodes;
    std::string basicCode[3] = {
        "print('This is a loop in Python')",
        "for j in range(3): print(j)",
        "total = sum([1, 2, 3])"
    };

    for(int i = 0; i < 3; ++i) {
        pythonCodes.push_back(basicCode[i]);  // Adds each python code snippet from the array to the vector
    }

    // Print all stored python codes
    for (const std::string& code : pythonCodes) {
        std::cout << code << std::endl;
    }

    return 0;
}
```
