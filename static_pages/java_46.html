abs

The `abs` method in Java is a static method that is used to return the absolute value of a given number. This method can be applied to different numerical types like integers, floats, doubles, and long. The absolute value of a number is its distance from zero on the number line, regardless of direction; in other words, it is always a non-negative value. The `Math` class provides this method, and its syntax varies slightly depending on the data type being passed to it.

import math

# Using abs to find the absolute value of a negative integer
negative_integer = -5
result = abs(negative_integer)
print(result)  # Output: 5
import math

# Using abs to find the absolute value of a floating-point number
negative_float = -3.14
result = abs(negative_float)
print(result)  # Output: 3.14
import math

# Using abs with zero
zero_value = 0
result = abs(zero_value)
print(result)  # Output: 0
