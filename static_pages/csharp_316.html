Add

The Add method is commonly used in collection classes in C# (such as List, Dictionary, etc.) to insert elements into the collection. This method allows you to specify the item you want to add, and it will place the item in the appropriate location within the collection. The behavior may vary slightly depending on the type of collection, but it generally increases the size of the collection by one and inserts the specified item. In Dictionary, it will add a key-value pair, where the key must be unique.

```python
# Example 1: Adding elements to a List

my_list = []
my_list.append('Hello')  # Using the Add method's equivalent in Python
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
# Example 2: Adding key-value pairs to a Dictionary

dictionary = {}
dictionary['name'] = 'John Doe'  # Using the Add method's equivalent in Python
dictionary['age'] = 30
dictionary['city'] = 'New York'
print(dictionary)  # Output: {'name': 'John Doe', 'age': 30, 'city': 'New York'}
```
```python
# Example 3: Adding user-defined objects to a List

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

people = []
people.append(Person('Alice', 28))  # Using the Add method's equivalent in Python
people.append(Person('Bob', 34))

for person in people:
    print(f'Name: {person.name}, Age: {person.age}')  # Output: Name: Alice, Age: 28
                  # Output: Name: Bob, Age: 34
```
