std::vector::push_back

The 'push_back' method is used to add a new element to the end of a std::vector in C++. This method increases the size of the vector by one and initializes the added element with the value provided. If the size of the vector exceeds its capacity, the vector will automatically allocate more memory to accommodate the new element. This function has amortized constant time complexity, meaning that most of the time it will be a constant time operation, but it may occasionally take longer if the vector needs to grow.

The following code demonstrates how to create a vector of strings and add Python code snippets for printing 'Hello, World!' using the 'push_back' method.

```cpp
#include 
#include 
#include 

int main() {
    std::vector codeSnippets;
    codeSnippets.push_back("print('Hello, World!')");
    codeSnippets.push_back("for i in range(5): print(i)");
    codeSnippets.push_back("def greet(name): print(f'Hello, {name}')");

    for (const auto &snippet : codeSnippets) {
        std::cout << snippet << std::endl;
    }
    return 0;
}
```
