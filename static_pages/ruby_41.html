upcase

The 'upcase' method in Ruby is used to convert all the lowercase letters in a string to uppercase letters. This method does not modify the original string; instead, it returns a new string with all lowercase letters replaced by their uppercase counterparts. It is commonly used in situations where uniformity in text case is needed, such as formatting, search functions, and user input validation.

original_string = 'hello world'
upcase_string = original_string.upcase
print(upcase_string)  # Output: 'HELLO WORLD'
user_input = 'ruby programming'
formatted_input = user_input.upcase
print(formatted_input)  # Output: 'RUBY PROGRAMMING'
greeting = 'Good Evening'
shouted_greeting = greeting.upcase
print(shouted_greeting)  # Output: 'GOOD EVENING'
