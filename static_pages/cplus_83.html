std::vector::push_back

The 'push_back' method in C++ is used to add a new element to the end of a std::vector. This method dynamically increases the size of the vector by one, placing the new element at the last position. The 'push_back' method is an efficient way to append data, making it a common choice when building collections of data that can change in size. It operates with a time complexity of O(1) for average cases, but can be O(n) in the worst cases if the vector needs to allocate new memory.

```cpp
#include 
#include 

int main() {
    std::vector stringVector;
    stringVector.push_back("Python");
    stringVector.push_back("Java");
    stringVector.push_back("C++");

    for (const auto &language : stringVector) {
        std::cout << language << " ";
    }
    return 0;
}
``` 
This example initializes a vector of strings, adds three programming languages using 'push_back', and then prints them to the console.

```cpp
    }
    return 0;
}
``` 
In this example, a vector called 'languages' is created and filled with programming languages using the 'push_back' method. It also prints the contents of the vector.
