uppercased

The `uppercased` method in Swift is used to convert all characters of a given string into their uppercase equivalents. It's a simple yet powerful string transformation that can be useful in various scenarios, such as normalizing user input or ensuring consistent formatting for comparisons. The method returns a new string with all alphabetic characters converted to uppercase. It's important to note that the `uppercased` method does not mutate the original string; instead, it returns a new string with the applied transformation.

def to_uppercase(input_string):
    return input_string.upper()

print(to_uppercase('hello world'))  # Outputs: 'HELLO WORLD'
def convert_and_compare(original, to_compare):
    return original.upper() == to_compare.upper()

print(convert_and_compare('hello', 'HELLO'))  # Outputs: True
def shout(text):
    return text.upper() + '!!!'

print(shout('watch out'))  # Outputs: 'WATCH OUT!!!'
