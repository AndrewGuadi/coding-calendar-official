abs

The 'abs' method in Java is part of the 'Math' class, which returns the absolute value of a number. Absolute value is defined as the non-negative value of a number without regard to its sign. For example, the absolute value of both -5 and 5 is 5. The 'abs' method can take different types of arguments such as int, long, float, and double, and returns the absolute value in the same numeric type. This method is particularly useful in various mathematical calculations and algorithms where distance or non-negative values are required.

import math

number = -42
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
import math

number = 3.14
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
import math

numbers = [-1, 0, 2, -3]
absolute_values = [abs(num) for num in numbers]
