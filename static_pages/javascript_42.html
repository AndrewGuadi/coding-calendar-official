charAt

The `charAt()` method in JavaScript is used to retrieve the character at a specified index (position) in a string. Strings in JavaScript are zero-indexed, meaning the first character is at index 0. If the specified index is out of range, an empty string is returned. This method is useful when needing to access individual characters from a string for various purposes like validation, parsing, or formatting.

let str = 'Python';
let firstChar = str.charAt(0);
console.log(firstChar); // Output: 'P'
let str = 'Python';
let lastChar = str.charAt(str.length - 1);
console.log(lastChar); // Output: 'n'
let str = 'Python';
let thirdChar = str.charAt(2);
console.log(thirdChar); // Output: 't'
