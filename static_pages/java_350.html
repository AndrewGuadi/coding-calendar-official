acos

In Java, the `acos` method is part of the `java.lang.Math` class. It is used to calculate the arc cosine (inverse cosine) of a given value. The result of `acos` is the value in radians whose cosine is the specified value. The input must be within the range of -1.0 to 1.0, inclusive. If the input value is outside this range, the method will return NaN (Not a Number). This method can be helpful in various applications, including physics, engineering, or any scenario where trigonometric calculations involving angles are necessary.

import math

# Example 1: Calculate the arc cosine of a value within the valid range
value = 0.5
radians = math.acos(value)
print(f'The arc cosine of {value} is {radians} radians.')
import math

# Example 2: Calculate the arc cosine of a negative value
value = -1.0
radians = math.acos(value)
print(f'The arc cosine of {value} is {radians} radians.')
import math

# Example 3: Attempt to calculate the arc cosine of a value outside the valid range
value = 2.0
try:
    radians = math.acos(value)
    print(f'The arc cosine of {value} is {radians} radians.')
except ValueError as e:
    print(f'Error: {e}')
