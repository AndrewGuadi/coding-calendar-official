Add

The 'Add' method is used to add an element to a collection in C#. This method is commonly implemented in various collections, such as List, Dictionary, or HashSet, to include new items. When called, this method typically takes the item to be added as a parameter. The method might also handle automatic resizing or rehashing of the underlying data structure if necessary. In the case of dictionaries or hash sets, it will ensure that no duplicates are added unless specifically allowed.

```python
my_list = []
my_list.append('Hello') # Using the Add method conceptually
the_list = my_list # Adds 'Hello' to the list
print(my_list) # Output: ['Hello']
```
```python
my_dict = {}
my_dict['key1'] = 'Value1' # Using the Add method conceptually
my_dict['key2'] = 'Value2' # Adding another key-value pair
print(my_dict) # Output: {'key1': 'Value1', 'key2': 'Value2'}
```
```python
my_set = set()
my_set.add('Item1') # Using the Add method conceptually
my_set.add('Item2') # Adding another item
print(my_set) # Output: {'Item1', 'Item2'}
```
