std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one, and the element is initialized to the value provided as an argument. It is useful for dynamically expanding the size of a vector as new data is added. If there is not enough capacity to accommodate the new element, the vector will automatically resize itself, which may involve allocating new memory and moving the existing elements to the new memory location.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Hello");
    vec.push_back("World");
    for (const auto& str : vec) {
        std::cout << str << ' ';  // Output: Hello World
    }
    return 0;
}
```
```cpp
#include 
#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i * 10);
    }
    for (const auto& num : numbers) {
        std::cout << num << ' ';  // Output: 10 20 30 40 50
    }
    return 0;
}
```
