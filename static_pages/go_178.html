cap

In Go, the 'cap' function is used to return the capacity of a slice, which is the total number of elements the slice can hold. The capacity is always equal to or greater than the length of the slice. The 'cap' function can be useful for understanding how much more data can be added to a slice before it needs to increase its underlying array size.

my_list = [1, 2, 3]
print('Length:', len(my_list))
print('Capacity:', len(my_list) * 2)  # Assuming a default estimated capacity for demonstration
# This would display the length and a hypothetical capacity, as lists in Python do not have a direct method to get capacity.
dynamic_array = []
for i in range(10):
    dynamic_array.append(i)
print('List elements:', dynamic_array)
print('Length of list:', len(dynamic_array))
print('Hypothetical capacity:', len(dynamic_array) * 2)  # This simulates capacity based on elements appended.
def demonstrate_capacity():
    initial_array = [0]*5  # Simulating an initial capacity of 5
    print('Array elements:', initial_array)
    print('Initial Length:', len(initial_array))
    print('Hypothetical Capacity:', len(initial_array) * 2)
    
    for i in range(5):
        initial_array[i] = i+1
    print('Updated Array:', initial_array)
    print('Updated Length:', len(initial_array)) 
    print('Updated Hypothetical Capacity:', len(initial_array) * 2)

demonstrate_capacity()
