acos

The 'acos' method in Java is a part of the Math class and is used to compute the arc cosine of a value. It takes a single argument (a double value) within the range of -1 to 1 and returns the angle in radians whose cosine is the specified value. The result is in the range of 0 to π (0 to 3.14159...). If the value is outside the range of -1 and 1, the method returns NaN (Not a Number). This method is useful in trigonometric calculations, particularly in scenarios involving angles and distances.

import math

# Given a cosine value of 0.5, find the angle in radians
cos_value = 0.5
angle_radians = math.acos(cos_value)
angle_degrees = math.degrees(angle_radians)
print('Angle in radians:', angle_radians)
print('Angle in degrees:', angle_degrees)
