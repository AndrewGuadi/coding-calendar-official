strlen

The strlen function in PHP is used to calculate the length of a given string. It returns an integer representing the number of characters in the string. The function counts all characters in the string, including letters, numbers, symbols, and spaces. If the provided input is not a string, the behavior of the function could be unpredictable, so it is crucial to ensure the input is a string before using this function.

string_example = 'Hello, World!'
length = strlen(string_example)
print(f'The length of the string is: {length}')
input_string = 'Programming is fun!'
length_of_input = strlen(input_string)
print('Length of input_string:', length_of_input)
text = 'Data Science 101'
length_of_text = strlen(text)
print('Total characters in the text:', length_of_text)
