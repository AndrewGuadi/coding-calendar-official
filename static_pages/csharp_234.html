Add

The 'Add' method is a member of various collections in C#. It is commonly used with data structures such as List, Dictionary, and HashSet. This method allows you to add new elements to a collection, thereby increasing its size. When using List, for example, the Add method appends an item to the end of the list. For dictionaries, it associates a key with a value, and for HashSets, it adds an element if it is unique. It is important to note that the behavior can vary slightly based on the collection type, particularly concerning duplicate entries and key/value pairs.

using System;
using System.Collections.Generic;

class Program {
    public static void Main(string[] args) {
        List pythonExamples = new List();
        pythonExamples.Add("print('Hello, World!')");
        pythonExamples.Add("for i in range(5): print(i)");
        pythonExamples.Add("def greet(name): return f'Hello {name}'");
        foreach(var example in pythonExamples) {
            Console.WriteLine(example);
        }
    }
}
using System;
using System.Collections.Generic;

class Program {
    public static void Main(string[] args) {
        Dictionary pythonCode = new Dictionary();
        pythonCode.Add(1, "import math");
        pythonCode.Add(2, "def add(a, b): return a + b");
        pythonCode.Add(3, "x = math.pi");
        foreach(var kvp in pythonCode) {
            Console.WriteLine(kvp.Key + ": " + kvp.Value);
        }
    }
}
