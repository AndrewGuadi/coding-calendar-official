abs

The `abs` method in Python returns the absolute value of a given number. The absolute value of a number is the non-negative value of that number without regard to its sign. This means that `abs(-5)` will return `5`, and `abs(5)` will also return `5`. The method can take an integer, a floating-point number, or even a complex number as arguments. For complex numbers, `abs` returns the magnitude (also known as modulus) of the complex number.

print(abs(-10))  # Output: 10
print(abs(3.14))  # Output: 3.14
complex_num = 3 + 4j
print(abs(complex_num))  # Output: 5.0
