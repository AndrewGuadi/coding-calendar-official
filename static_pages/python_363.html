ascii

The `ascii()` function in Python returns a string containing a printable representation of an object, escaping the non-ASCII characters in the string using the ` `, ` `, and ` ` representations accordingly. It is particularly useful for displaying strings that may contain unprintable characters or to ensure a string is safe for ASCII-only output. This method helps to visualize the actual contents of an object, especially useful during debugging.

obj = 'Caf√©'
result = ascii(obj)
print(result)  # Output: 'Caf\xe9'
obj = 'Hello, world!'
result = ascii(obj)
print(result)  # Output: 'Hello, world!'
obj = 'Python 3.9: 
New features'
result = ascii(obj)
print(result)  # Output: 'Python 3.9: \nNew features'
