ascii

The 'ascii()' function in Python returns a string containing a printable representation of an object. The result is the same as the output of the built-in 'repr()' function, but it is limited to ASCII characters. Any non-ASCII characters are replaced with escape sequences, making it suitable for display in environments that can only handle ASCII characters.
