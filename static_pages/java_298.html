acos

The `acos` method in Java is a part of the `java.lang.Math` class. It is used to compute the arc cosine of a given value. This method takes a single argument, which is the cosine of the desired angle, and returns the angle in radians. The input must be in the range of -1.0 to 1.0, where values outside of this range will result in a `NaN` (Not-a-Number) response.
