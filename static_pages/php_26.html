strlen

The 'strlen' function in PHP is used to get the length of a string. It returns the number of characters in the string provided as its argument. This is helpful in various applications, such as validating user input, dynamically adjusting the display of strings, or simply for informational purposes where the length of a string may matter.

my_string = 'Hello, world!'
string_length = strlen(my_string)
print('Length of the string:', string_length)  # Outputs Length of the string: 13
password = 'secret123'
password_length = strlen(password)
if password_length < 8:
    print('Password must be at least 8 characters long.')  # Outputs if password is shorter than 8 characters
text = 'The quick brown fox jumps over the lazy dog'
text_length = strlen(text)
print(f'The length of the text is: {text_length}')  # Outputs The length of the text is: 43
