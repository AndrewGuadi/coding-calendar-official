abs

The `abs` method in Python is used to return the absolute value of a number. This means that it takes a single argument, which can be an integer or a float, and returns the non-negative value of that number. If the original number is positive or zero, it remains unchanged. If it is negative, the sign is removed.

number = -15
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')  # Output: The absolute value of -15 is 15
float_number = -12.34
absolute_value = abs(float_number)
print(f'The absolute value of {float_number} is {absolute_value}')  # Output: The absolute value of -12.34 is 12.34
positive_number = 42
absolute_value = abs(positive_number)
print(f'The absolute value of {positive_number} is {absolute_value}')  # Output: The absolute value of 42 is 42
