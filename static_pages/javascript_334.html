charAt

The 'charAt' method in JavaScript is used to return the character at a specified index in a string. This method takes one argument, which is the index of the desired character. Note that indexing is zero-based; the first character has an index of 0, the second an index of 1, and so on. If the specified index is out of range (negative or greater than the string length), it returns an empty string.

```python
# Example usage of charAt method in JavaScript
string_value = 'hello'
character = string_value.charAt(1)
print(character)  # Output: 'e'
```
```python
# Example usage of charAt method in JavaScript for last character
string_value = 'world'
character = string_value.charAt(4)
print(character)  # Output: 'd'
```
```python
# Example usage of charAt method with out of range index
string_value = 'test'
character = string_value.charAt(10)
print(character)  # Output: '' (an empty string)
```
