std::vector::push_back

The `push_back` method of the C++ Standard Library's `std::vector` class is used to add an element to the end of the vector. This method increases the vector's size by one and constructs the new element in place at the end of the existing elements. If the vector's current capacity is exceeded, it automatically reallocates memory to accommodate the new element. `push_back` can take an argument of any type that is compatible with the vector's type, including primitive types, structs, or user-defined classes.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Hello");
    vec.push_back("World");

    for (const auto& str : vec) {
        std::cout << str << " "; // Output: Hello World
    }
    return 0;
}
```
