std::vector::push_back

The `push_back` method in C++ STL (Standard Template Library) is used to add a new element to the end of a vector. It increases the size of the vector by one and the new element is initialized to the value provided in the `push_back` method. This method can be used with any data type that the vector holds, including user-defined types. `push_back` may also trigger a reallocation of the vector's storage if it exceeds its current capacity, which typically involves copying the existing elements to a new location.

```cpp
#include 
#include 
#include 

int main() {
    // Create a vector of strings
    std::vector stringVector;

    // Add elements using push_back
    stringVector.push_back("Hello");
    stringVector.push_back("World");

    // Print elements of the vector
    for (const auto &str : stringVector) {
        std::cout << str << " "; // Output: Hello World 
    }
    return 0;
}
```

#include

int main() {
    // Create a vector of integers
    std::vector intVector;

    // Add elements using push_back
    for (int i = 0; i < 5; ++i) {
        intVector.push_back(i * 10); // Adding multiples of 10
    }

    // Print elements of the vector
    for (const auto &num : intVector) {
        std::cout << num << " "; // Output: 0 10 20 30 40 
    }
    return 0;
}
