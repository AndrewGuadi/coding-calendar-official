Add

The 'Add' method is a commonly used function in various collections in C#, including lists and dictionaries. This method allows you to insert new elements into the collection. When using a list, the 'Add' method appends the specified object to the end of the list. For dictionaries, it adds a key-value pair into the collection. This method is essential for dynamically modifying the contents of collections at runtime.

my_list = []
my_list.append('print("Hello, World!")')

# Now `my_list` contains one element which is a Python code as a string and can be executed later
for code in my_list:
    exec(code)
my_dict = {}
my_dict['greet'] = 'print("Hello from dictionary!")'

# The key 'greet' has been added to `my_dict` and can be executed as follows
exec(my_dict['greet'])
my_list = []
my_list.append('for i in range(3):\n    print(\\"Hello from a loop!\\")')

# The code within the list can be executed to show its functionality
for code in my_list:
    exec(code)
