std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a 'std::vector'. This method increases the size of the vector by one and can take any value or object type that the vector is designed to hold. The new element is copied or moved into the vector. If the vector's current capacity is exceeded, a new memory allocation occurs to accommodate the additional elements.
