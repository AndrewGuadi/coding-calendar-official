Add

The 'Add' method is used to add an element to a collection in C#. It is commonly found in data structures such as Lists, Dictionaries, and Sets. The method typically takes one or more parameters that represent the value(s) to be added to the collection. When adding an item, the collection may grow dynamically (e.g., in lists) or require specific handling for duplicates (e.g., in sets or dictionaries). This method is essential for building and managing dynamic data structures that require modification during runtime.

```python
# Example using a List in C#
colors = List()
colors.Add('red')
colors.Add('blue')

# Output the colors list
print(colors)  # Output: ['red', 'blue']
```
```python
# Example using a Dictionary in C#
mapping = Dictionary()
mapping.Add('apples', 5)
mapping.Add('bananas', 3)

# Output the dictionary
for key, value in mapping.items():
    print(key, value)  # Output: apples 5, bananas 3
```
```python
# Example using a HashSet in C#
unique_numbers = HashSet()
unique_numbers.Add(1)
unique_numbers.Add(2)
unique_numbers.Add(1)  # This will not be added again

# Output the unique numbers set
print(unique_numbers)  # Output: {1, 2}
```
