strlen

The strlen() function in PHP is used to get the length of a string, which is the number of characters it contains. This function returns the length of the string as an integer. It's useful for validating input, determining the size of data being processed, and manipulating strings based on their size.

name = 'Hello World';
length = len(name)
print(length)  # Output will be 11
message = 'PHP is great!';
message_length = len(message)
print(message_length)  # Output will be 15
text = 'OpenAI ChatGPT';
text_length = len(text)
print(text_length)  # Output will be 16
