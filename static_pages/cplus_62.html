std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add an element to the end of a vector. The vector automatically resizes if the new element causes the capacity to be exceeded, ensuring that all elements are stored in contiguous memory. This method is useful when needing to append data dynamically to a vector without worrying about the current size or capacity of the vector.

```cpp
#include 
#include 

int main() {
    std::vector pythonScripts;
    pythonScripts.push_back("print('Hello from script 1')");
    pythonScripts.push_back("print('Hello from script 2')");
    
    for (const auto& script : pythonScripts) {
        std::cout << script << std::endl;
    }
    return 0;
}
```
