acos

The 'acos' method in Java is used to calculate the arc cosine (inverse cosine) of a value. This method is a static method of the Math class and accepts a double argument that should be in the range of -1.0 to 1.0. It returns the angle in radians whose cosine is the specified value. If the argument is outside of this range, it will return NaN (Not a Number). The result can be converted to degrees if necessary by multiplying by (180/PI).

import math
result = math.acos(0.5)
# This will return the angle in radians whose cosine is 0.5
import math
result = math.acos(-1)
# This will return the angle in radians whose cosine is -1, which is pi radians
import math
result = math.acos(1)
# This will return the angle in radians whose cosine is 1, which is 0 radians
