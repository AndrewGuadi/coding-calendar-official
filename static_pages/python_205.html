ascii

The `ascii()` method in Python is used to return a string containing a printable representation of an object. It converts non-ASCII characters in the string to their Unicode escape sequences, and it is particularly useful when you want to ensure that the string can be displayed in environments that support only ASCII characters. The method is often used with strings, but it can also be applied to various data types. This function is a built-in function and is available in all versions of Python.

s = 'Hello, 世界'
print(ascii(s))  # Output: 'Hello, \u4e16\u754c'
data = {'name': 'Alice', 'city': 'München'}
print(ascii(data))  # Output: '{'name': 'Alice', 'city': 'M\u00fcnchen'}'
text = 'Café'
print(ascii(text))  # Output: 'Caf\xe9'
