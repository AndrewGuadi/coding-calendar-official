std::vector::push_back

The `std::vector::push_back` method adds a new element to the end of a vector, increasing its size by one. This method is typically used when you want to add data dynamically to a vector as it manages memory automatically. Before using `push_back`, ensure the vector is properly initialized. If the new size exceeds the current capacity, `push_back` will handle memory reallocation internally. Note that the complexity of this operation can be amortized O(1).

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Adding a string to the vector
    myVector.push_back("World"); // Adding another string
    for (const std::string& str : myVector) {
        std::cout << str << " "; // Outputs: Hello World
    }
    return 0;
}
```
```cpp
#include 
#include 

int main() {
    std::vector myVector;
    std::string element;
    for (int i = 0; i < 5; ++i) {
        element = "Python" + std::to_string(i); // Creating dynamic strings
        myVector.push_back(element); // Pushing the strings to the vector
    }
    for (const auto& str : myVector) {
        std::cout << str << " "; // Outputs: Python0 Python1 Python2 Python3 Python4
    }
    return 0;
}
```
