Add

The 'Add' method is commonly used in various collections in C#, such as lists and dictionaries, to insert an element into the data structure. When using a List, the Add method adds an object to the end of the List, while in the case of a Dictionary, it adds a new key-value pair. This method typically handles the underlying resizing of the collection as necessary to accommodate new elements.

```python
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}
my_dict['a'] = 'Apple'
my_dict['b'] = 'Banana'
print(my_dict)  # Output: {'a': 'Apple', 'b': 'Banana'}
```
```python
my_set = set()
my_set.add('unique_item')
my_set.add('another_item')
print(my_set)  # Output: {'unique_item', 'another_item'}
```
