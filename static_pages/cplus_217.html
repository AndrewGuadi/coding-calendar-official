std::vector::push_back

The `std::vector::push_back` method is used to add a new element at the end of a vector in C++. It dynamically increases the size of the vector and adds the provided element as the new last element. This method is efficient in appending single elements, and it automatically manages the underlying memory allocation to accommodate the new size of the vector. If the vector has a reserved capacity that is already filled, the `push_back` method will allocate more memory to ensure successful insertion.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("first item");
    vec.push_back("second item");

    for (const auto& item : vec) {
        std::cout << item << std::endl;
    }
    return 0;
}```  This example demonstrates creating a vector of strings, adding two string items using `push_back`, and then printing each item to the console.
```cpp
#include 
#include 

int main() {
    std::vector vec;
    std::string data[] = {"item 1", "item 2", "item 3"};

    for (const auto& item : data) {
        vec.push_back(item);
    }

    for (const auto& item : vec) {
        std::cout << item << std::endl;
    }
    return 0;
}```  In this example, we use an array of strings to populate the vector using `push_back` within a loop, where each element from the array is added to the vector, followed by printing each element.
```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.reserve(5); // Reserve space for 5 elements

    vec.push_back("Python");
    vec.push_back("Java");
    vec.push_back("C++");

    std::cout << "Current size of the vector: " << vec.size() << std::endl;

    vec.push_back("JavaScript");
    vec.push_back("Go");

    std::cout << "Updated size of the vector: " << vec.size() << std::endl;

    return 0;
}```  This example illustrates reserving space in the vector before adding elements. It prints the vector size before and after additional elements are added, showing how `push_back` dynamically resizes the vector if needed.
