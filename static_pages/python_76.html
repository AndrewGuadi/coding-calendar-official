abs

The 'abs' function in Python returns the absolute value of a number. This means it converts negative numbers to their positive counterparts and leaves positive numbers, including zero, unchanged. The 'abs' function can handle integers, floats, and complex numbers (in which case it returns the magnitude).


# Example 1: Working with an integer
number = -10
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')


# Example 2: Working with a float
number = -3.14
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')


# Example 3: Working with a complex number
number = 3 + 4j
absolute_value = abs(number)
print(f'The absolute value (magnitude) of {number} is {absolute_value}')
