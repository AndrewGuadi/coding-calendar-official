abs

The `abs` method in Java is used to return the absolute value of a given number. It can be used with various numeric types, including integer (`int`), long (`long`), float (`float`), and double (`double`). The absolute value is the non-negative value of a number without regard to its sign. For instance, the absolute value of both -5 and 5 is 5. The `abs` method is available in the `Math` class in Java and is commonly used in mathematical calculations where the non-negative value is required.
