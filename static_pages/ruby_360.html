upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method returns a new string with the changes applied, leaving the original string unchanged. It is a commonly used method for string manipulation, particularly when you need to ensure that string comparisons are case-insensitive or when formatting output for better readability.

def convert_to_uppercase(input_str):
    # Ensure the input is a string
    if not isinstance(input_str, str):
        return 'Input must be a string'
    
    # Convert to upper case
    upper_str = input_str.upper()
    return upper_str

# Example usage:
result = convert_to_uppercase('hello world')
print(result)  # Output: HELLO WORLD
def titles_to_uppercase(titles):
    # Convert a list of titles to uppercase
    return [title.upper() for title in titles]

# Example usage:
titles_list = ['the great gatsby', 'to kill a mockingbird', '1984']
upper_titles = titles_to_uppercase(titles_list)
print(upper_titles)  # Output: ['THE GREAT GATSBY', 'TO KILL A MOCKINGBIRD', '1984']
def shout_message(message):
    # Convert the message to uppercase to simulate shouting
    return message.upper() + '!'

# Example usage:
shouted = shout_message('hello everyone')
print(shouted)  # Output: HELLO EVERYONE!
