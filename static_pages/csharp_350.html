Add

The 'Add' method is commonly used in C# to add elements to a collection, such as a list, dictionary, or other data structures. In the case of lists, 'Add' appends an item to the end of the list. For dictionaries, it adds a key-value pair. This method alters the state of the collection by increasing its size or adding new entries.

```python
# Adding elements to a list
fruits = []
fruits.append('apple')
fruits.append('banana')
print(fruits)  # Output: ['apple', 'banana']
```
```python
# Adding key-value pairs to a dictionary
student_grades = {}
student_grades['Alice'] = 'A'
student_grades['Bob'] = 'B'
print(student_grades)  # Output: {'Alice': 'A', 'Bob': 'B'}
```
```python
# Adding elements in a custom class that uses Add-like behavior
class NumberList:
    def __init__(self):
        self.numbers = []

    def add(self, number):
        self.numbers.append(number)

number_list = NumberList()
number_list.add(10)
number_list.add(20)
print(number_list.numbers)  # Output: [10, 20]
```
