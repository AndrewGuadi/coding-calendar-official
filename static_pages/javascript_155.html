charAt

The `charAt` method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning the first character is at index 0, the second character is at index 1, and so on. If the index is out of range (less than 0 or greater than or equal to the string length), it returns an empty string.

```python
# Example usage of charAt in JavaScript

my_string = 'Hello, World!'  # sample string

# Function to retrieve character at a certain index
def get_char_at_index(my_string, index):
    return my_string[index] if 0 <= index < len(my_string) else ''

# Getting character at index 7
result = get_char_at_index(my_string, 7)
print(result)  # Output: W
```
