abs

The 'abs' method in Python is used to return the absolute value of a number. This number can be an integer, a floating point number, or a complex number. The absolute value is the distance of a number from zero on a number line, without considering which direction from zero the number lies.

number = -10
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
# Output: The absolute value of -10 is 10
floating_number = -20.5
absolute_value = abs(floating_number)
print('The absolute value of', floating_number, 'is', absolute_value)
# Output: The absolute value of -20.5 is 20.5
complex_number = 3 - 4j
absolute_value = abs(complex_number)
print('The absolute value of', complex_number, 'is', absolute_value)
# Output: The absolute value of (3-4j) is 5.0
