Add

The Add method is commonly used in data structures such as lists, dictionaries, or any collection type in C#. It is primarily used to add a new element to the collection. In the context of a List, it appends an item to the end of the list, while in a Dictionary, it adds a key-value pair if the key does not already exist. This method is essential for dynamically managing datasets where the size can change at runtime.

```python
# Example of adding elements to a list in C#
my_list = []
my_list.append('Example 1')
my_list.append('Example 2')
print(my_list)  # Output: ['Example 1', 'Example 2']
```
```python
# Example of adding key-value pairs to a dictionary in C#
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
# Example of combining lists and adding multiple elements
my_list = []
items_to_add = ['Example 3', 'Example 4']
for item in items_to_add:
    my_list.append(item)
print(my_list)  # Output: ['Example 3', 'Example 4']
```
