abs

The `abs` method in Python returns the absolute value of a given number. The absolute value of a number is the non-negative value of the numerical value without regard to its sign. For example, `abs(-3)` and `abs(3)` will both return 3. The `abs` function can handle integers, floating-point numbers, and complex numbers. When dealing with complex numbers, `abs` returns the magnitude (or modulus) of the complex number, which is the distance from the origin to the point in the complex plane.

# Example 1: Absolute value of an integer
num = -5
absolute_value = abs(num)
print(f'The absolute value of {num} is {absolute_value}')
# Example 2: Absolute value of a floating-point number
num = -7.25
absolute_value = abs(num)
print(f'The absolute value of {num} is {absolute_value}')
# Example 3: Absolute value (magnitude) of a complex number
num = 3 - 4j
absolute_value = abs(num)
print(f'The absolute value (magnitude) of {num} is {absolute_value}')
