uppercased

The `uppercased` method in Swift is used to convert a string to uppercase. This method is called on a string instance and returns a new string with all the characters converted to uppercase. This is particularly useful when you want to ensure that string comparisons are case-insensitive or when you want to format strings for consistency.

let originalString = "hello world"
let uppercasedString = originalString.uppercased()
print(uppercasedString)  // Output: "HELLO WORLD"
let greeting = "Good Morning!"
let uppercasedGreeting = greeting.uppercased()
print(uppercasedGreeting)  // Output: "GOOD MORNING!"
let variableName = "swift programming"
let uppercasedVariable = variableName.uppercased()
print(uppercasedVariable)  // Output: "SWIFT PROGRAMMING"
