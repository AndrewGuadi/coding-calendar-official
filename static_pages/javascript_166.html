charAt

The charAt() method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning that the first character in the string is at index 0. If the specified index is out of bounds, charAt() returns an empty string.

let str = 'Hello, World!';
let charAtIndex5 = str.charAt(5);
console.log(charAtIndex5); // Outputs: ','
Example logic: The method charAt(5) retrieves the character at the 5th position in the string 'Hello, World!', which is a comma.
let str = 'JavaScript';
let charAtIndex3 = str.charAt(3);
console.log(charAtIndex3); // Outputs: 'a'
Example logic: The method charAt(3) retrieves the character at the 3rd position in the string 'JavaScript', which is 'a'.
let str = 'OpenAI';
let charAtIndex10 = str.charAt(10);
console.log(charAtIndex10); // Outputs: ''
Example logic: The method charAt(10) is called on the string 'OpenAI', but since the index 10 is out of bounds, it returns an empty string.
