std::vector::push_back

The 'push_back' method in C++ is a member function of the std::vector class that adds a new element to the end of the vector. After the element is added, the size of the vector increases by one. If the current capacity of the vector is exceeded, it automatically allocates more memory to accommodate the new element, ensuring that the vector remains dynamic and can grow as needed. This method is commonly used when we want to dynamically build a collection of items, as it provides a simple way to append new data without needing to manage memory manually.

In this example, we create a vector of strings and use 'push_back' to append several Python code examples:

```cpp
#include 
#include 
#include 

int main() {
    std::vector codeExamples;

    codeExamples.push_back("print('Hello, World!')");
    codeExamples.push_back("for i in range(5):\n    print(i)");
    codeExamples.push_back("def add(a, b):\n    return a + b");

    for (const std::string& code : codeExamples) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
