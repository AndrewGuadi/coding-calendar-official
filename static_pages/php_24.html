strlen

The `strlen` function in PHP is used to get the length of a given string. It returns the number of characters present in the string, including spaces and special characters. This method is useful for validating input lengths, manipulating strings, or for simple tasks where the length of the text is needed.

def get_string_length(s):
    return len(s)

string = 'Hello, world!'
length = get_string_length(string)
print('Length of the string:', length)
def count_characters(s):
    return len(s)

example_string = 'Python Programming'
character_count = count_characters(example_string)
print('Number of characters:', character_count)
def string_length_with_spaces(s):
    return len(s)

test_string = 'String with spaces'
spaces_length = string_length_with_spaces(test_string)
print('Length including spaces:', spaces_length)
