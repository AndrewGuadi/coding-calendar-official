abs

The 'abs' method in Java is a static method provided by the Math class to compute the absolute value of a given number. It can handle different data types including integers, floats, and doubles. The absolute value of a number is its non-negative value, which means that it will return the original number if it is zero or positive, or its negation if it is negative.

def absolute_value(x):
    return abs(x)

# Example usage:
print(absolute_value(-5))  # Output: 5
print(absolute_value(3.14)) # Output: 3.14
def compare_abs_values(a, b):
    return abs(a) > abs(b)

# Example usage:
print(compare_abs_values(-10, 6)) # Output: True
print(compare_abs_values(4, -5))  # Output: False
def distance_from_origin(x, y):
    return abs(x) + abs(y)

# Example usage:
print(distance_from_origin(-3, 4))  # Output: 7
print(distance_from_origin(6, -8))  # Output: 14
