uppercased

The 'uppercased' method is used to convert all the characters in a string to uppercase. It is typically called on a string instance and returns a new string with all lowercase letters converted to uppercase. This method is non-destructive and does not modify the original string. It is a useful function when you want to standardize the casing of string data, such as preparing input for comparisons or for display purposes.

def convert_string(string):
    return string.upper()

input_string = 'hello world'
output_string = convert_string(input_string)
print(output_string)  # Output: 'HELLO WORLD'
def shout_string(string):
    return string.upper() + '!'

input_string = 'this is amazing'
output_string = shout_string(input_string)
print(output_string)  # Output: 'THIS IS AMAZING!'
def check_user_input(input_string):
    if input_string.upper() == 'YES':
        return 'User confirmed.'
    else:
        return 'User denied.'

input_string = 'yes'
output_string = check_user_input(input_string)
print(output_string)  # Output: 'User confirmed.'
