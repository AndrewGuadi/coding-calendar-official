charAt

The `charAt()` method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning the first character is located at index 0. If the index is out of range (greater than or equal to the string length or negative), it returns an empty string. This method is useful for retrieving specific characters from a string for further manipulation or examination.

```python
# Example 1: Accessing characters in a string
text = 'Hello, World!'
character = text[7]  # Using the index 7
print(character)  # Output: 'W'
```
```python
# Example 2: Accessing the first and last characters
text = 'OpenAI'
first_character = text[0]  # First character at index 0
last_character = text[-1]  # Last character using negative index
print(first_character, last_character)  # Output: 'O I'
```
```python
# Example 3: Handling out-of-range index
text = 'Python'
character = text[10] if len(text) > 10 else ''  # Out of range
print(character)  # Output: ''
```
