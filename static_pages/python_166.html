ascii

The `ascii()` method in Python takes a string as an argument and returns a string containing only the ASCII characters from the given input. Non-ASCII characters are represented as escape sequences. This method is often used when you need to sanitize or encode strings to ensure they only contain characters in the printable ASCII range.
