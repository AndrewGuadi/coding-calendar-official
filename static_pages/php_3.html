substr

The 'substr' function in PHP is used to extract a portion of a string. It takes three parameters: the string from which to extract, the starting index, and an optional length that specifies how many characters to extract. If the length is not provided, it will extract from the start index to the end of the string. Indices are zero-based, which means that the first character of the string is at index 0.

```php
// Extracting a substring from position 1 to the end
$string = 'Hello, World!';
$substring = substr($string, 1);
echo $substring; // Output: 'ello, World!'
```
```php
// Extracting a substring of specific length
$string = 'Hello, World!';
$substring = substr($string, 0, 5);
echo $substring; // Output: 'Hello'
```
```php
// Extracting a substring from the end
$string = 'Hello, World!';
$substring = substr($string, -6);
echo $substring; // Output: 'World!'
```
