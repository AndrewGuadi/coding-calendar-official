std::vector::push_back

The 'push_back' method in C++'s std::vector class is used to add a new element at the end of the vector. When an element is pushed back, the size of the vector increases by one. If the current capacity of the vector is exceeded, the vector automatically allocates more memory to accommodate the new element. This method takes a single argument, which is the element to be added, and it is generally efficient as it can amortize its time complexity to O(1).

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    
    // Adding elements to the vector
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");
    
    // Displaying the contents of the vector
    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
```

```cpp
    // Displaying the contents of the vector

}
```
