std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one, allowing dynamic array resizing. The new element is a copy of the value passed to 'push_back'. It is beneficial for when you need to store a collection of items where the total number of items is not known in advance. If the underlying array needs to be expanded to accommodate the new element, the old data will be copied to a new, larger array, and the original array will be freed. The time complexity of 'push_back' can be amortized constant time (O(1)), but occasionally it may take linear time (O(n)) when the capacity needs to be increased.

# C++ usage of std::vector::push_back
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("hello"); // Adding 'hello'
    myVector.push_back("world"); // Adding 'world'

    for (const auto &str : myVector) {
        std::cout << str << std::endl;
    }
    return 0;
}

int main() {
    std::vector fruits;
    fruits.push_back("apple");  // Add 'apple'
    fruits.push_back("banana"); // Add 'banana'
    fruits.push_back("cherry"); // Add 'cherry'

    std::cout << "Fruits:
";
    for (size_t i = 0; i < fruits.size(); ++i) {
        std::cout << fruits[i] << std::endl;
    }
    return 0;
}
