Add

The 'Add' method is typically used in collections to append an element to the end of a list, array, or any collection implementing a similar functionality. In C#, the 'Add' method is commonly associated with the List class, allowing users to dynamically increase the size of the list by adding elements. If you attempt to add an item that already exists in the collection, it will be added again as lists can contain duplicate items.

```python
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_numbers = []
my_numbers.append(1)
my_numbers.append(2)
my_numbers.append(2)
print(my_numbers)  # Output: [1, 2, 2]
```
```python
my_fruits = ['Apple']
my_fruits.append('Banana')
my_fruits.append('Cherry')
print(my_fruits)  # Output: ['Apple', 'Banana', 'Cherry']
```
