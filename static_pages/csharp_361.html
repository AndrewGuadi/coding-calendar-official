Add

The 'Add' method in C# is commonly used in collections to insert an element into a list, dictionary, or any other data structure that supports addition of new items. The method modifies the collection by adding a new entry, which can be particularly useful when dynamically managing data. For instance, in a List, the Add method appends an item to the end of the collection. In a Dictionary, the method adds a key-value pair while ensuring the key is unique. This method can significantly simplify data manipulation operations in applications.

using System;
using System.Collections.Generic;

public class Program
{
    public static void Main()
    {
        List pythonExamples = new List();
        pythonExamples.Add("print('Hello, World!')");
        pythonExamples.Add("def add(a, b): return a + b");
        pythonExamples.Add("for i in range(5): print(i)");

        foreach (var example in pythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
using System;
using System.Collections.Generic;

public class Program
{
    public static void Main()
    {
        Dictionary functionExamples = new Dictionary();
        functionExamples.Add("Hello World", "print('Hello, World!')");
        functionExamples.Add("Addition", "def add(a, b): return a + b");
        functionExamples.Add("Looping", "for i in range(5): print(i)");

        foreach (var item in functionExamples)
        {
            Console.WriteLine($"{item.Key}: {item.Value}");
        }
    }
}
