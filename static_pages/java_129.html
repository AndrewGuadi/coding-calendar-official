acos

The 'acos' method in Java is part of the 'java.lang.Math' class. It returns the arc cosine (inverse cosine) of a value. The input value should be in the range of -1.0 to 1.0, and the result is returned in radians. If the input value is outside the range of -1.0 to 1.0, it will throw an IllegalArgumentException. The result can also be converted to degrees if necessary by using the formula degrees = radians * (180.0 / Math.PI).

import java.lang.Math;

public class Main {
    public static void main(String[] args) {
        double value = 0.5;
        double result = Math.acos(value);
        System.out.println("The arc cosine of " + value + " is " + result + " radians.");
        // Convert to degrees
        double degrees = result * (180.0 / Math.PI);
        System.out.println("In degrees: " + degrees);
    }
}

public class Main {
    public static void main(String[] args) {
        try {
            double value = 1.5; // This will throw an exception since it is outside the valid range
            double result = Math.acos(value);
            System.out.println("The arc cosine of " + value + " is " + result + " radians.");
        } catch (IllegalArgumentException e) {
            System.out.println("Error: The input value is out of range for acos. Please provide a value between -1 and 1.");
        }
    }
}
