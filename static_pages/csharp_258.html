Add

The 'Add' method is typically used to add an item to a collection, such as a list or a dictionary, in C#. It allows for dynamic expansion of data structures by inserting new elements while maintaining the current elements. The method is commonly found in classes like List and Dictionary. The 'Add' method requires the specific value to be added as a parameter, and it can handle various data types depending on the collection being used.

```python
# Example of adding an item to a list in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List myList = new List();
        myList.Add("Hello");  # Add a string to the list
        myList.Add("World");  # Add another string

        foreach (var item in myList)
        {
            Console.WriteLine(item);  # Output: Hello
                                   #         World
        }
    }
}
```
```python
# Example of adding key-value pairs to a dictionary in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary myDict = new Dictionary();
        myDict.Add(1, "One");  # Add key-value pair to the dictionary
        myDict.Add(2, "Two");  # Add another key-value pair

        foreach (var kvp in myDict)
        {
            Console.WriteLine($"Key: {kvp.Key}, Value: {kvp.Value}");  # Output: Key: 1, Value: One
                                                               #         Key: 2, Value: Two
        }
    }
}
```
