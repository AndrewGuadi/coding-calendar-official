Add

The 'Add' method is commonly used in collections in C#. It allows you to insert or append an item to a collection such as a List, Dictionary, or Set. When adding an item to a List, for example, it increases the size of the list and stores the item at the next available index. This method is essential for dynamic data manipulation where the size of the collection can change during runtime.

This example demonstrates how to use the 'Add' method with a List in C# to store Python code snippets. // C# code: 
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonSnippets = new List();
        pythonSnippets.Add("print('Hello, World')");
        pythonSnippets.Add("for i in range(5):\n    print(i)");

        foreach (string snippet in pythonSnippets)
        {
            Console.WriteLine(snippet);
        }
    }
} 
// This will output:
// print('Hello, World')
// for i in range(5):
//     print(i)
This example shows how to add items to a Dictionary in C#. We will associate keys with Python code snippets. // C# code:
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary pythonCodeDict = new Dictionary();
        pythonCodeDict.Add("Simple Print", "print('Hello, World!')");
        pythonCodeDict.Add("For Loop", "for i in range(5):\n    print(i)");

        foreach (var entry in pythonCodeDict)
        {
            Console.WriteLine(entry.Key + ": " + entry.Value);
        }
    }
}
// This will output:
// Simple Print: print('Hello, World!')
// For Loop: for i in range(5):
//     print(i)
In this example, we use the 'Add' method with a HashSet to ensure unique Python code snippets. // C# code:
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet pythonSnippetsSet = new HashSet();
        pythonSnippetsSet.Add("print('Unique Snippet')");
        pythonSnippetsSet.Add("print('Another Unique Snippet')");
        pythonSnippetsSet.Add("print('Unique Snippet')"); // This will not be added again

        foreach (string snippet in pythonSnippetsSet)
        {
            Console.WriteLine(snippet);
        }
    }
}
// This will output:
// print('Unique Snippet')
// print('Another Unique Snippet')
