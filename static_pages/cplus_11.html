std::vector::push_back

The `push_back` method in C++ is used to add an element to the end of a `std::vector`. This method increases the size of the vector by one and allows you to create a dynamic array that can grow as needed. When you call `push_back`, the vector automatically manages memory, allocating more space if necessary, and copies or moves the new element to the end of the vector. It is important to note that if the vector's capacity is reached, the vector will automatically allocate more memory to accommodate the additional elements.

```python
# Example of using push_back to store a list of Python scripts in a C++ vector
script_files = []

# Simulate the C++ std::vector push_back method
def push_back(vec, item):
    vec.append(item)

# Adding Python scripts to the vector
push_back(script_files, 'script1.py')
push_back(script_files, 'script2.py')
push_back(script_files, 'script3.py')

# Display the contents of the vector
print(script_files)  # Output: ['script1.py', 'script2.py', 'script3.py']
```
