charAt

The `charAt` method in JavaScript returns the character at a specified index in a string. This method takes one parameter, which is the zero-based index of the character you want to retrieve. If the index is out of range (less than 0 or greater than the length of the string), it returns an empty string. This method is useful when you need to access individual characters in a string for processing or manipulation.

const str = 'Hello, World!';
const char = str.charAt(0);
console.log(char); // Output: H
 // This code retrieves the first character of the string, which is 'H'.
const str = 'JavaScript';
const char = str.charAt(4);
console.log(char); // Output: S
 // This code retrieves the character at index 4, which is 'S' from the string 'JavaScript'.
const str = 'OpenAI';
const char = str.charAt(6);
console.log(char); // Output:  
 // This code tries to retrieve the character at index 6, which is out of range, resulting in an empty string.
