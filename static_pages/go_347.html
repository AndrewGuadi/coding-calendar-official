close

In Go, the 'close' method is used to close a channel. Closing a channel is important because it signals that no more values will be sent on the channel. This can help prevent deadlocks and allow goroutines to terminate gracefully. Once a channel is closed, no more values can be sent to it, but values can still be received until all values have been drained. Itâ€™s important to note that you should only close a channel from the goroutine that created it to avoid panics.

import time

ch = make(chan int)

# Start a goroutine to send data
std.Thread(target=lambda: [ch.send(i) for i in range(3)])

# In main goroutine, wait for sending to finish and then close
std.Thread(target=lambda: [time.sleep(1), ch.close()])

# Read loop that receives until channel is closed
while True:
    try:
        val = ch.receive()
        print(f'Received: {val}')
    except ClosedChannel:
        print('Channel closed.')
        break
