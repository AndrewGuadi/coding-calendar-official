std::vector::push_back

The `push_back` method in C++ is used to add a new element at the end of a vector. This operation increases the size of the vector by one and places the new element in the last position. The `push_back` method is commonly used to store a dynamic number of elements in a vector without the need to specify an initial size. A key feature of this method is that it can automatically manage the memory allocation when the vector exceeds its current capacity. If the vector needs more space, it will allocate additional memory, allowing us to keep adding elements seamlessly.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    myVector.push_back("from C++!");

    for (const auto& str : myVector) {
        std::cout << str << ' ';
    }
    return 0;
}
```
