Add

The 'Add' method is commonly found in various context types in C#, such as in collections (lists, dictionaries) or mathematical operations (like for numeric types). The primary purpose of the 'Add' method is to insert a new element into a collection or add a value to a specific data structure. For instance, in a List, the Add method appends an element to the end of the list. In a Dictionary, it adds a key-value pair to the dictionary. This method is crucial for dynamically managing collections of data as the program runs.

```python
# Example of using the Add method with a list in C#
# This example adds a Python string to a C# List.

# C# code:
"""
List myList = new List();
myList.Add("Hello, World!");
"""
# Result: myList now contains one item: 'Hello, World!'
```
```python
# Example of using the Add method with a dictionary in C#
# This example adds a Python string as a value with a key to a dictionary in C#.

# C# code:
"""
Dictionary myDictionary = new Dictionary();
myDictionary.Add(1, "First Item");
"""
# Result: myDictionary now contains one entry: {1: 'First Item'}
```
```python
# Example of adding integers with a custom class in C#
# This uses Add to create a method that adds two numbers together and returns the result.

# C# code:
"""
public class Calculator
{
    public int Add(int a, int b)
    {
        return a + b;
    }
}
Calculator calc = new Calculator();
int result = calc.Add(15, 10);
"""
# Result: result is 25, which is the sum of 15 and 10.
```
