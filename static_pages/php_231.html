strlen

The `strlen` function in PHP is used to calculate and return the length of a given string. The length is measured in bytes. This function is particularly useful for determining how many characters or bytes are present in a string which can help in various operations such as validation, formatting, and processing of user input.

str_length = strlen('Hello, World!')
print(f'The length of the string is: {str_length}')
input_string = 'Data Science'
str_length = strlen(input_string)
if str_length > 10:
    print('The string is too long.')
else:
    print('The string length is acceptable.')
phrase = 'Welcome to the world of programming!'
length = strlen(phrase)
print(f'This phrase contains {length} characters.')
