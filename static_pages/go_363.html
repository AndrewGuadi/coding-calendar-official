close

The 'close' method in Go is used to close an open resource, typically a file, network connection, or any other I/O operation that requires cleanup after use. Properly closing resources is crucial to avoid memory leaks and ensure that system resources are released. In Go, it is common to call 'defer' before the 'close' method when opening a resource, so that it is guaranteed to execute at the end of the function, even if an error occurs.

import (  
    "os"  
)  

func main() {  
    file, err := os.Open("example.txt")  
    if err != nil {  
        panic(err)  
    }  
    defer file.Close()  
    // Perform operations on the file  
}

    response, err := http.Get(url)  
    if err != nil {  
        panic(err)  
    }  
    defer response.Body.Close()

}
