ascii

The `ascii()` function in Python takes an object and returns a string containing a printable representation of that object, where non-ASCII characters are represented using escape sequences. It's particularly useful when you want to ensure that the output of a string is human-readable and contains only ASCII characters, which can be helpful in data processing and debugging.

s = 'Héllo Wörld'
result = ascii(s)
print(result)  # Output: 'H\xe9llo W\xf6rld'
s = 'Python is fun!'
result = ascii(s)
print(result)  # Output: 'Python is fun!'
s = 'Café'
result = ascii(s)
print(result)  # Output: 'Caf\xe9'
