Add

The 'Add' method is commonly used in many C# collection classes, including List, Dictionary, and HashSet, to insert new elements into the collection. This method typically accepts an item as a parameter and appends it to the end of the collection or adds a key-value pair in the case of a dictionary. The operation usually has a constant time complexity, O(1), for lists and hash sets, although it can be O(n) if the underlying array needs to be resized. When using dictionaries, the key must be unique; attempting to add a duplicate key will throw an exception. The 'Add' method provides a straightforward way to build collections dynamically during the runtime of a program.

```python
# Example of using a list to store Python code snippets
code_snippets = []

# Using Add method to append a new snippet
code_snippets.append("print('Hello, World!')")
code_snippets.append("for i in range(5): print(i)")

# Print the list of snippets
print(code_snippets)  # Outputs: ["print('Hello, World!')", "for i in range(5): print(i)"]
```
```python
# Example of using a dictionary to store Python code snippets with titles
code_snippets = {}

# Using Add method to insert new code snippets
code_snippets['Hello World'] = "print('Hello, World!')"
code_snippets['Looping'] = "for i in range(5): print(i)"

# Print the dictionary of code snippets
print(code_snippets)  # Outputs: {'Hello World': "print('Hello, World!')", 'Looping': "for i in range(5): print(i)"}
```

# Example of using a set to store unique Python keywords
