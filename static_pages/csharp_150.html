Add

The 'Add' method in C# is used for adding elements to various collections such as lists, dictionaries, and sets. It is a common operation that allows you to enhance a data structure by including new items. Depending on the collection type, the behavior of the 'Add' method can differ, such as how duplicates are handled or whether the order of elements is preserved.

In this example, we use the 'Add' method to append Python code as a string to a list:

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonExamples = new List();
        pythonExamples.Add("def hello_world():\n    print('Hello, World!')");
        pythonExamples.Add("def add(a, b):\n    return a + b");
        pythonExamples.Add("for i in range(5):\n    print(i)");

        foreach (var example in pythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
```
In this example, we demonstrate using the 'Add' method with a dictionary to add key-value pairs where the values are Python code:

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary pythonCodeSnippets = new Dictionary();
        pythonCodeSnippets.Add("factorial", "def factorial(n):\n    return 1 if n == 0 else n * factorial(n - 1)");
        pythonCodeSnippets.Add("fibonacci", "def fibonacci(n):\n    if n <= 0:\n        return 0\n    elif n == 1:\n        return 1\n    else:\n        return fibonacci(n - 1) + fibonacci(n - 2)");

        foreach (var snippet in pythonCodeSnippets)
        {
            Console.WriteLine($"Function: {snippet.Key}\nCode: {snippet.Value}\n");
        }
    }
}
```

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet uniquePythonExamples = new HashSet();
        uniquePythonExamples.Add("def greet(name):\n    print(f'Hello, {name}!')");
        uniquePythonExamples.Add("def square(x):\n    return x * x");
        uniquePythonExamples.Add("def greet(name):\n    print(f'Hello, {name}!')"); // Duplicate example added

        foreach (var example in uniquePythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
