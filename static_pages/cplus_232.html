std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element at the end of a vector, which is a dynamic array that can grow in size. This method allows you to append a single element to the vector, effectively increasing its size by one. The new element is initialized using a copy or move of the value provided. If the vector needs to resize to accommodate the new element, it will allocate a new larger array, copy the existing elements to this new array, and then deallocate the old array. It is important to note that the `push_back` operation has an average time complexity of O(1), but can take O(n) time in worst-case scenarios when the underlying array has to be reallocated.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World!");

    for(const auto& str : myVector) {
        std::cout << str << " "; // Output: Hello World! 
    }
    return 0;
}
```
