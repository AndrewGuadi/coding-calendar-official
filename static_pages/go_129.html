cap

The 'cap' method in Go is used to retrieve the capacity of a slice. The capacity of a slice is the number of elements in the underlying array that the slice can hold, which may be larger than the number of elements currently stored in the slice. Understanding the capacity is important for optimizing memory usage and performance, especially when appending elements to a slice.

s = []int{1, 2, 3}  # Create a slice with 3 elements.
print('Capacity of s:', cap(s))  # Output will display the capacity of slice s (which is 3).
s = make([]int, 0, 5)  # Create a slice with a capacity of 5.
s.append(1)
s.append(2)
print('Capacity of s:', cap(s))  # Output will display the capacity (which is 5).
s = []int{1, 2}
s = append(s, 3)  # Append an element to the slice.
print('New Capacity of s:', cap(s))  # Output will display the updated capacity after appending an element.
