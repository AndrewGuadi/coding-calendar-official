std::vector::push_back

The 'std::vector::push_back' method in C++ is used to add a new element at the end of the vector. This operation increases the size of the vector by one and allows you to append data to the end of your dynamic array. The new element is constructed in place at the end of the vector, and if the capacity of the vector is exceeded, it automatically allocates new memory to accommodate the additional element. The method does not return any value and is typically used when you need to build a collection of items dynamically. It is essential to keep in mind that this method is only applicable to vectors defined in the C++ Standard Library.

# Define a vector of strings
std::vector myVector;

// Add a Python code snippet
myVector.push_back("print('Hello, World!')");

// Display the contents of the vector
for (const std::string &code : myVector) {
    std::cout << code << std::endl;
}
# Initialize an empty vector for multiple Python function names
std::vector funcNames;

// Add Python function names to the vector
funcNames.push_back("def factorial(n):");
funcNames.push_back("return 1 if n == 0 else n * factorial(n - 1)");

// Print the function names
for (const std::string &name : funcNames) {
    std::cout << name << std::endl;
}
# Using push_back in a loop to create a vector of Python for-loops
std::vector loops;

// Push multiple for-loop structures into the vector
for (int i = 0; i < 5; ++i) {
    loops.push_back("for j in range(" + std::to_string(i) + "): print(j)");
}

// Print each loop structure
for (const std::string &loop : loops) {
    std::cout << loop << std::endl;
}
