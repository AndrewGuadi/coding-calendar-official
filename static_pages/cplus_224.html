std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class, which adds a new element to the end of the vector container. This method increases the size of the vector by one and initializes the new element to the value provided as a parameter. If the vector's capacity is reached, it automatically reallocates memory to accommodate the new element, ensuring that the operation is safe and efficient. It is particularly useful when dynamically building lists of items, as it allows for ease of adding elements without worrying about manual memory management.

In this example, we create a vector of strings to hold Python code snippets. We then use `push_back` to add three different Python print statements to the vector. Finally, we iterate through the vector and print each statement:

```cpp
#include 
#include 
#include 

int main() {
    std::vector python_code;
    python_code.push_back("print('Hello, World!')");
    python_code.push_back("print('This is a vector example.')");
    python_code.push_back("for i in range(5): print(i)");

    for (const auto& code : python_code) {
        std::cout << code << std::endl;
    }
    return 0;
}
```
In this example, we create a vector of integers to store the results of a simple mathematical operation. We push back the squares of integers from 0 to 4 into the vector and then display the results:

```cpp
#include 
#include 
#include 

int main() {
    std::vector squares;
    for (int i = 0; i < 5; ++i) {
        squares.push_back(pow(i, 2));
    }

    for (const auto& square : squares) {
        std::cout << square << std::endl;
    }
    return 0;
}
```
In this example, we utilize a vector to dynamically collect user input for a list of programming languages. After collecting inputs, we print the list:

```cpp
#include 
#include 
#include 

int main() {
    std::vector languages;
    std::string input;
    std::cout << "Enter programming languages (type 'done' to finish):" << std::endl;
    while (true) {
        std::getline(std::cin, input);
        if (input == "done") break;
        languages.push_back(input);
    }

    std::cout << "You entered the following languages:" << std::endl;
    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
