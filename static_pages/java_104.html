abs

The 'abs' method in Java is used to calculate the absolute value of a given number. The absolute value of a number is its non-negative value, meaning it is the distance of that number from zero on a number line, regardless of direction. This method can be used with different numeric types including integers, floating-point numbers, and even big integers. In Java, the 'abs' method is part of the java.lang.Math class.

def find_absolute_value(num):
    return abs(num)

# Example usage:
print(find_absolute_value(-10))  # Output: 10
def calculate_net_income(expenses, income):
    return abs(expenses - income)

# Example usage:
income = 50000
expenses = 70000
print(calculate_net_income(expenses, income))  # Output: 20000
def difference_in_heights(height1, height2):
    return abs(height1 - height2)

# Example usage:
height1 = 6.2
height2 = 5.9
print(difference_in_heights(height1, height2))  # Output: 0.3
