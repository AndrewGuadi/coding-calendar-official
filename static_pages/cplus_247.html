std::vector::push_back

The `push_back` method in C++ adds a new element to the end of a `std::vector`. This method increases the size of the vector by one and allows you to push new items onto the back of the vector. The added element is initialized with a copy of the value passed to `push_back`. If the current allocated memory is not sufficient to accommodate the new element, the `vector` automatically increases its capacity. It provides an efficient way to build a collection of items dynamically as the size requirement grows during runtime.

```python
# Initialize an empty list to store values
vector = []

# Define some Python code snippets as strings
code_snippets = ["print('Hello, World!')", "for i in range(5): print(i)", "def add(a, b): return a + b"]

# Push each code snippet into the list
for snippet in code_snippets:
    vector.append(snippet)

# Display the contents of the vector
print(vector)
```
This example shows how to create a dynamic list of code snippets using `append`, which corresponds to the `push_back` method in C++.
```python
# Simulating the addition of items to a list
vector = []

def add_to_vector(item):
```

In this example, we define a function that mimics the behavior of `push_back` by appending items to a Python list.
```python
# Initialize an empty list
vector = []

# Function to simulate building a vector of integers
def add_integers(nums):
    for num in nums:
        vector.append(num)  # Mimicking push_back behavior

# Add a series of integers to the vector
add_integers([1, 2, 3, 4, 5])

# Output the list to check its contents
print(vector)
```
This example demonstrates creating a vector and utilizing a function to add multiple integers, similar to how `push_back` is used in C++.
