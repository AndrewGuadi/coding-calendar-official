append

The `append` function in Go is used to add elements to a slice. It takes a slice and one or more elements to be added and returns a new slice containing the original contents plus the new elements. If there is enough capacity in the slice, it will add the new elements to the existing slice; otherwise, it will allocate a new slice with sufficient capacity.

```python
my_list = [1, 2, 3]
new_element = "4"
my_list.append(new_element)
print(my_list)  # Output: [1, 2, 3, '4']
```
```python
fruits = ["apple", "banana"]
new_fruit = "cherry"
fruits.append(new_fruit)
print(fruits)  # Output: ['apple', 'banana', 'cherry']
```
```python
numbers = [10, 20, 30]
numbers_to_add = [40, 50]
for num in numbers_to_add:
    numbers.append(num)
print(numbers)  # Output: [10, 20, 30, 40, 50]
```
