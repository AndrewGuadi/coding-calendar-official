acos

The 'acos' method in Java is part of the Math class and is used to compute the arc cosine (inverse cosine) of a given value. This method takes a single argument in the range of -1.0 to 1.0 and returns the angle in radians whose cosine is the specified value. It's important to note that if the input is outside of this range, the method will return NaN (not a number). The value returned from the method can be converted into degrees if needed by multiplying the result by (180/Math.PI).

import math

# Example 1: Finding the angle whose cosine is 0.5
cos_value = 0.5
angle_radians = math.acos(cos_value)
angle_degrees = angle_radians * (180 / math.pi)
print('Angle in radians:', angle_radians)
print('Angle in degrees:', angle_degrees)

# Example 3: Attempting to find the arc cosine of a value outside the range -1 to 1
cos_value = 2.0
angle_radians = math.acos(cos_value)
if math.isnan(angle_radians):
    print('The input is outside the range for acos: NaN')
else:

    print('Angle in degrees:', angle_degrees)
