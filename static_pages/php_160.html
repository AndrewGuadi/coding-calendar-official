strlen

The `strlen` function in PHP is used to calculate the length of a given string. It returns the number of bytes contiguous in the string. This can be particularly useful for validating input data or managing string manipulations where the length is a factor. The function takes a single argument, which is the string whose length you want to determine, and returns an integer representing the length of that string. Note that `strlen` counts all characters including spaces, punctuation, and special characters.

def check_string_length(s):
    return len(s)

# Example usage
input_string = 'Hello, World!'
print(check_string_length(input_string))  # Output: 13
def is_empty_string(s):
    return len(s) == 0

# Example usage
input_string = ''
print(is_empty_string(input_string))  # Output: True
def format_string_length(s):
    return f'The length of the string is {len(s)} characters.'

# Example usage
input_string = 'Programming in Python'
print(format_string_length(input_string))  # Output: The length of the string is 21 characters.
