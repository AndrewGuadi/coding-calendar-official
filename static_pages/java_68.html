abs

The 'abs' method in Java is a built-in function that returns the absolute value of a given number. It is available in the 'Math' class, and it can take integers, floats, and double types as arguments. The absolute value of a number is the non-negative value of that number, meaning that if the number is negative, the 'abs' function will return the positive equivalent of that number. If the number is already positive or zero, it will return the same number.

import math

number = -5
absolute_value = abs(number)
print(absolute_value)  # Output: 5
import math

number = 3.14
absolute_value = abs(number)
print(absolute_value)  # Output: 3.14
import math

number = -1000.0
absolute_value = abs(number)
print(absolute_value)  # Output: 1000.0
