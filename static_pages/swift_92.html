uppercased

The `uppercased` method in Swift is used to convert all the characters in a string to uppercase. This method returns a new string where all lowercase letters in the original string have been converted to their uppercase equivalents, and any characters that are already uppercase, or non-alphabetic characters, remain unchanged. This is particularly useful in scenarios where case-insensitive comparisons are needed, or when formatting strings for display purposes.

original_string = 'Hello, World!'
uppercase_string = original_string.uppercased()
print(uppercase_string)  # Output: 'HELLO, WORLD!'
email = 'user@example.com'
uppercase_email = email.uppercased()
print(uppercase_email)  # Output: 'USER@EXAMPLE.COM'
greeting = 'welcome to the jungle'
uppercase_greeting = greeting.uppercased()
print(uppercase_greeting)  # Output: 'WELCOME TO THE JUNGLE'
