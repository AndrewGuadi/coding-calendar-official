cap

The 'cap' method in Go is used to retrieve the capacity of a slice, which is the total number of elements the slice can hold without needing to allocate more memory. The capacity of a slice is determined when it is created and can grow as needed when elements are appended. Understanding the capacity of a slice can help optimize memory usage and performance when working with slices in Go.

```python
my_slice = [1, 2, 3]
my_slice.extend([4, 5])
print(len(my_slice))  # Outputs: 5
```
```python
num_slice = [10] * 10
num_slice.append(11)
print(len(num_slice))  # Outputs: 11
```
```python
chars = ['a', 'b']
chars += ['c', 'd', 'e', 'f']
print(len(chars))  # Outputs: 6
```
