std::vector::push_back

The 'push_back' method in C++ is a member function of the std::vector class used to add a new element to the end of the vector. This method allows for dynamic resizing of the vector as it grows, automatically allocating more memory when necessary. It takes a single argument of the type stored in the vector and doesn't return a value. If the vector needs to increase its capacity, 'push_back' may involve the memory reallocation of existing elements to fit the new size.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");

    for(const auto& fruit : fruits) {
        std::cout << fruit << " ";
    }
    // Output: apple banana cherry
}
```
