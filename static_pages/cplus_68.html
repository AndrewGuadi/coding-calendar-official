std::vector::push_back

The std::vector::push_back method is used to add an element to the end of a vector in C++. This method increases the size of the vector by one and initializes the new element with the value provided. It is a convenient way to build a list of elements dynamically as the size of the vector can change during runtime. When an element is added, if sufficient capacity is available in the vector, the new element is simply added; otherwise, the vector will allocate additional memory to accommodate the new element.

```python
# Creating a list to simulate std::vector behavior
class Vector:
    def __init__(self):
        self.data = []

    def push_back(self, value):
        self.data.append(value)  # Simulates push_back by appending to the list

# Example of using Vector class
vector = Vector()
vector.push_back('hello')
vector.push_back('world')
print(vector.data)  # Output: ['hello', 'world']
```
```python
# Using Vector class to store multiple Python strings
class Vector:
    def __init__(self):
        self.data = []

    def push_back(self, value):
        self.data.append(value)

# Create an instance of Vector
my_vector = Vector()

# Add strings into the vector
my_vector.push_back('apple')
my_vector.push_back('banana')
my_vector.push_back('cherry')

# Print the contents of the vector
print(my_vector.data)  # Output: ['apple', 'banana', 'cherry']
```
```python
# Demonstrating Vector to store and concatenate strings
class Vector:
    def __init__(self):
        self.data = []

    def push_back(self, value):
        self.data.append(value)

# Initialize the Vector
string_vector = Vector()

# Adding strings to the vector
string_vector.push_back('Hello')
string_vector.push_back('from')
string_vector.push_back('OpenAI')

# Concatenate all strings in the vector
concatenated_result = ' '.join(string_vector.data)
print(concatenated_result)  # Output: 'Hello from OpenAI'
```
