std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one and initializes the new element with the provided value. If the vector's current capacity is exceeded, it will automatically allocate more memory to accommodate the new element. This method is particularly useful for dynamically growing arrays in C++, allowing for efficient addition of elements.

```cpp
#include 
#include 

int main() {
    std::vector languages;  // Create an empty vector for strings
    languages.push_back("Python");      // Add "Python" to the vector
    languages.push_back("Java");        // Add "Java" to the vector

    for (const auto &lang : languages) {
        std::cout << lang << std::endl; // Print all languages in the vector
    }
    return 0;
}
```
