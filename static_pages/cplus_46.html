std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. A vector is a dynamic array that can grow and shrink in size as needed. When an element is added using `push_back`, the vector's size increases by one, and the new element is placed at the end. If the current capacity of the vector is not sufficient to hold the new element, the vector automatically allocates more memory to accommodate the additional element.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    // Print the vector elements
    for (const auto& str : myVector) {
        std::cout << str << " ";
    }
    return 0;
}
```

#include

int main() {
    std::vector fruits;
    fruits.push_back("Apple");
    fruits.push_back("Banana");
    fruits.push_back("Cherry");

    // Use a for loop to display fruit names
    std::cout << "Fruits: ";
    for (int i = 0; i < fruits.size(); ++i) {
        std::cout << fruits[i];
        if (i < fruits.size() - 1) std::cout << ", ";
    }
    std::cout << std::endl;
    return 0;
}
