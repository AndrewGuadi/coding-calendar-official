Add

The 'Add' method in C# is commonly used in collections, such as lists or dictionaries, to insert a new element into the collection. Depending on the type of collection, the method can add an object, key-value pair, or any other data type. When used in the context of a List, for example, it appends the specified object to the end of the List. In a Dictionary, it adds a key-value pair, where the key must be unique. This method helps in dynamically managing collections by allowing users to increase the size of the collection on demand.

```python
# Adding elements to a list
my_list = []
my_list.append(1)  # Adding an integer
my_list.append('Hello')  # Adding a string
my_list.append(3.14)  # Adding a float
print(my_list)  # Output: [1, 'Hello', 3.14]
```
```python
# Adding key-value pairs to a dictionary
my_dict = {}
my_dict['name'] = 'Alice'  # Adding a string as value
my_dict['age'] = 30  # Adding an integer as value
my_dict['is_student'] = True  # Adding a boolean as value
print(my_dict)  # Output: {'name': 'Alice', 'age': 30, 'is_student': True}
```
```python
# Appending multiple values to a list using loop
my_list = []
for i in range(5):
    my_list.append(i * 2)  # Adding even numbers from 0 to 8
print(my_list)  # Output: [0, 2, 4, 6, 8]
```
