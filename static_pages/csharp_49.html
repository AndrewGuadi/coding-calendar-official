Add

The 'Add' method is commonly used in various collections in C#, including lists, dictionaries, and sets. It allows users to insert or append a new element to the collection, increasing its size by one. The specific implementation can vary according to the type of collection being used. When adding an element, the method often accepts a parameter that represents the item to be added, and it might throw exceptions if the addition violates the collection's constraints (such as exceeding capacity or adding a duplicate in the case of sets).

# Example of adding an item to a List in C#

using System;
using System.Collections.Generic;

public class Program
{
    public static void Main()
    {
        // Create a new list of integers
        List numbers = new List();

        // Add elements to the list
        numbers.Add(1);
        numbers.Add(2);
        numbers.Add(3);

        // Print the numbers
        Console.WriteLine(string.Join(", ", numbers)); // Output: 1, 2, 3
    }
}

# Example of adding a key-value pair to a Dictionary in C#

using System;
using System.Collections.Generic;

public class Program
{
    public static void Main()
    {
        // Create a new dictionary
        Dictionary myDictionary = new Dictionary();

        // Add key-value pairs to the dictionary
        myDictionary.Add("key1", "value1");
        myDictionary.Add("key2", "value2");

        // Print each key-value pair
        foreach (var kvp in myDictionary)
        {
            Console.WriteLine($"Key: {kvp.Key}, Value: {kvp.Value}");
        }
    }
}

# Example of adding an item to a HashSet in C#

using System;
using System.Collections.Generic;

public class Program
{
    public static void Main()
    {
        // Create a new HashSet
        HashSet mySet = new HashSet();

        // Add elements to the HashSet
        mySet.Add("apple");
        mySet.Add("banana");
        mySet.Add("apple"); // Adding a duplicate does not throw an exception, but it will not be added

        // Print the contents of the HashSet
        Console.WriteLine(string.Join(", ", mySet)); // Output: apple, banana
    }
}
