ascii

The `ascii()` method in Python is used to convert an object into a string that contains a printable representation of that object. The output will be similar to `repr()` but with non-ASCII characters replaced with escape sequences. This method is particularly useful for debugging and displaying strings that may contain non-ASCII characters. It takes one argument, which is the object you want to convert, and returns a string in ASCII format.

my_string = 'Café'
print(ascii(my_string))  # Output: 'Caf\xe9'
my_dict = { 'name': 'José', 'age': 30 }
print(ascii(my_dict))  # Output: '{\'name\': \'Jos\xe9\', \'age\': 30}'
my_list = ['Müller', 'Niño', 'López']
print(ascii(my_list))  # Output: ['M\xfcller', 'Ni\xf1o', 'L\xf3pez']
