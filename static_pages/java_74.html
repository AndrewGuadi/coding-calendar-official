abs

The 'abs' method in Java is utilized to return the absolute value of a given number. It can be applied to various numeric data types, including int, long, float, and double. The absolute value is the non-negative value of a number irrespective of its sign; that is, it represents the distance of the number from zero on the number line. Using the 'abs' method is essential in mathematical computations where the magnitude of the number is more relevant than its sign.

def absolute_value_example():
    num = -15
    absolute = abs(num)
    print(f'The absolute value of {num} is {absolute}.')  
# Output: The absolute value of -15 is 15.
def float_absolute_value_example():
    num = -3.14
    absolute = abs(num)
    print(f'The absolute value of {num} is {absolute}.')  
# Output: The absolute value of -3.14 is 3.14.
def list_absolute_values_example():
    nums = [-3, -4.5, 2, 6, -7.8]
    absolute_values = [abs(x) for x in nums]
    print(f'The absolute values are: {absolute_values}.')  
# Output: The absolute values are: [3, 4.5, 2, 6, 7.8].
