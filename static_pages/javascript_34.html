charAt

The `charAt` method in JavaScript is used to access a specific character in a string by its index. The index is zero-based, meaning that the first character of the string is at index 0, the second character at index 1, and so on. If the specified index is out of bounds (less than 0 or greater than or equal to the length of the string), `charAt` returns an empty string. This method is helpful when you need to retrieve individual characters from strings for processing or manipulation.

def get_first_character():
    text = 'Hello'
    first_character = text[0]  # Using Python indexing to get the first character
    return first_character

print(get_first_character())  # Output: H
def get_character_at_index(text, index):
    if 0 <= index < len(text):
        return text[index]  # Accessing the character at the given index
    else:
        return ''  # Return empty string if index is out of bounds

print(get_character_at_index('Hello', 1))  # Output: e
print(get_character_at_index('Hello', 5))  # Output: ''
def get_last_character(text):
    last_index = len(text) - 1
    return text[last_index] if last_index >= 0 else ''  # Getting the last character

print(get_last_character('Hello'))  # Output: o
print(get_last_character(''))  # Output: ''
