abs

The 'abs' method in Java is used to compute the absolute value of a number. The absolute value of a number is its non-negative value regardless of its sign. This method can be used with different data types including integers, floating-point numbers, and others. The method returns the absolute value of the specified value. It can be particularly useful in mathematical computations where the direction (positive or negative) of the number is not important.

import java.lang.Math;

public class AbsoluteValueExample {
    public static void main(String[] args) {
        int negativeNumber = -10;
        int absoluteValue = Math.abs(negativeNumber);
        System.out.println("Absolute value of " + negativeNumber + " is: " + absoluteValue);
    }
}
// Expected Output: Absolute value of -10 is: 10
