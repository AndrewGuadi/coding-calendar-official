std::vector::push_back

The 'push_back' method of the std::vector class in C++ is used to add a new element to the end of a vector. This method automatically resizes the vector if necessary, allowing it to accommodate the new element. It takes a single argument, which is the value to be added to the vector. If the vector's current size reaches its capacity, the method will allocate a new larger storage and copy existing elements to the new storage, ensuring that the new element can be added.

```cpp
#include 
#include 

int main() {
    std::vector data;
    data.push_back("Hello");
    data.push_back("world!");

    // Display the contents of the vector
    for (const auto& str : data) {
        std::cout << str << ' ';
    }
    return 0;
}
```
