append

The 'append' function in Go is used to add elements to a slice. It takes a slice and one or more values as arguments, and it returns a new slice containing the original elements followed by the new values. If the capacity of the slice is not enough to hold the new elements, the 'append' function automatically allocates a larger array to accommodate the additional elements.

```go
package main

import (
	"fmt"
)

func main() {
	// Create a slice of strings
	names := []string{"Alice", "Bob"}

	// Append a single value
	names = append(names, "Charlie")
	fmt.Println(names) // Output: [Alice Bob Charlie]

	// Append multiple values
	names = append(names, "David", "Eve")
	fmt.Println(names) // Output: [Alice Bob Charlie David Eve]
}
```
```go
package main

import (
	"fmt"
)

func main() {
	// Create a slice of integers
	numbers := []int{1, 2, 3}

	// Append more integers
	numbers = append(numbers, 4, 5, 6)
	fmt.Println(numbers) // Output: [1 2 3 4 5 6]

	// Append another slice
	extraNumbers := []int{7, 8}
	numbers = append(numbers, extraNumbers...)
	fmt.Println(numbers) // Output: [1 2 3 4 5 6 7 8]
}
```
```go
package main

import (
	"fmt"
)

func main() {
	// Initialize a nil slice	numbers := []string{}

	// Append values to a nil slice
	numbers = append(numbers, "One")
	numbers = append(numbers, "Two", "Three")
	fmt.Println(numbers) // Output: [One Two Three]
}
```
