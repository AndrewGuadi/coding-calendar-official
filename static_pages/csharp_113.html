Add

The 'Add' method is commonly used in C# to insert elements into collections such as lists, dictionaries, or sets. It allows you to append an item at the end of the collection or associate a value with a key in a dictionary. The behavior depends on the type of collection being used. Using the 'Add' method is straightforward and requires providing the element or key-value pair that should be added.

```python
# Example of using a list in C# and adding an item using Add()  
myList = []  
myList.Add('Hello')  
myList.Add('World')  
# Now myList contains ['Hello', 'World']
```
```python
# Example of using a dictionary in C# and adding a key-value pair using Add()  
myDict = {}  
myDict.Add('name', 'Alice')  
myDict.Add('age', '30')  
# Now myDict contains {'name': 'Alice', 'age': '30'}
```
```python
# Example of using a HashSet in C# and adding elements using Add()  
mySet = set()  
mySet.Add('Python')  
mySet.Add('CSharp')  
# Now mySet contains {'Python', 'CSharp'}
```
