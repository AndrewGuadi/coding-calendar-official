Add

In C#, the 'Add' method is commonly used in various collections such as lists, dictionaries, and sets to insert a new element into the collection. Depending on the collection type, the method may have different signatures and behaviors. For instance, in a List, the 'Add' method appends an element to the end of the list, while in a Dictionary, 'Add' requires both a key and a value to insert a new entry into the dictionary.

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List fruits = new List();
        fruits.Add("apple"); // Adding 'apple' to the list
        fruits.Add("banana"); // Adding 'banana' to the list

        foreach (var fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}
```
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary capitals = new Dictionary();
        capitals.Add("USA", "Washington, D.C."); // Adding entry for the USA
        capitals.Add("France", "Paris"); // Adding entry for France

        foreach (var capital in capitals)
        {
            Console.WriteLine(capital.Key + " - " + capital.Value);
        }
    }
}
```
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet uniqueNumbers = new HashSet();
        uniqueNumbers.Add(1); // Adding '1'
        uniqueNumbers.Add(2); // Adding '2'
        uniqueNumbers.Add(2); // This will not add '2' again because HashSet only stores unique elements

        foreach (var number in uniqueNumbers)
        {
            Console.WriteLine(number);
        }
    }
}
```
