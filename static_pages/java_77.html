abs

In Java, the method 'abs' is used to compute the absolute value of a given number. The method is defined in the 'java.lang.Math' class and can accept different numeric types including integers, floats, and doubles. The absolute value of a number is its non-negative value irrespective of its sign. For example, the absolute value of -5 is 5, and the absolute value of 5 is also 5. The 'abs' method can be particularly useful in mathematical computations where the magnitude of numbers is required without considering their sign.

import java.lang.Math;

public class AbsoluteValueExample {
    public static void main(String[] args) {
        int negativeNumber = -10;
        int positiveNumber = 10;

        // Using Math.abs to get the absolute value
        int absNegative = Math.abs(negativeNumber);
        int absPositive = Math.abs(positiveNumber);

        System.out.println("The absolute value of " + negativeNumber + " is: " + absNegative);
        System.out.println("The absolute value of " + positiveNumber + " is: " + absPositive);
    }
}
