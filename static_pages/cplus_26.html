std::vector::push_back

The std::vector::push_back method is used to add an element to the end of a vector in C++. This operation increases the size of the vector by one and expands its capacity if necessary to accommodate the new element. The push_back method accepts a single argument, which is the value to be added to the vector.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    for (const auto& str : myVector) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}
```
