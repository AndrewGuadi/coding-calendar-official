acos

The 'acos' method in Java is part of the java.lang.Math class. It is used to compute the arc cosine of a given value, defined in the range [-1, 1]. The return value is expressed in radians, meaning that if a value is outside of the range [-1, 1], the method will return NaN (not a number). This mathematical function is particularly useful in trigonometry where you need the angle corresponding to a specified cosine value.

import math

ac_value = math.acos(1)  # returns 0.0, angle for cos(0) is 1
print(ac_value)
import math

ac_value = math.acos(0)  # returns 1.5707963267948966, angle for cos(pi/2) is 0
print(ac_value)
import math

ac_value = math.acos(-1)  # returns 3.141592653589793, angle for cos(pi) is -1
print(ac_value)
