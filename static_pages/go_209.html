cap

The `cap` function in Go is used to determine the capacity of a given slice, array, or channel. The capacity of a slice is the number of elements in the underlying array, counting from the first element of the slice to the last element in the array. For channels, it returns the buffer size. The capacity is an important parameter, as it can affect the performance of your Go code, allowing you to minimize reallocations by predefining the required capacity.

```go
package main

import "fmt"

func main() {
    // Example 1: Slice
    s := make([]int, 5, 10)
    fmt.Println("Slice capacity:", cap(s)) // Output: 10
    
    // Example 2: Array
    a := [5]int{1, 2, 3, 4, 5}
    fmt.Println("Array capacity:", cap(a)) // Output: 5
    
    // Example 3: Channels
    ch := make(chan int, 3)
    fmt.Println("Channel capacity:", cap(ch)) // Output: 3
}
```
