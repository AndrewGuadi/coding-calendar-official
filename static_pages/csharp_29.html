Add

The 'Add' method in C# is commonly used to add an element to a collection, such as a list, dictionary, or set. This method modifies the collection by including the specified value. In the case of a list, the value is added to the end, while in a dictionary, a key-value pair is added. The Add method will throw an exception if the operation would result in duplication in collections that enforce uniqueness, like HashSet or Dictionary keys.

my_list = []
my_list.append('Python')
my_list.append('Java')
print(my_list) # Output: ['Python', 'Java']
my_dict = {}
my_dict['name'] = 'Alice'
my_dict['age'] = 30
print(my_dict) # Output: {'name': 'Alice', 'age': 30}
my_set = set()
my_set.add('Python')
my_set.add('Java')
print(my_set) # Output: {'Java', 'Python'}}
