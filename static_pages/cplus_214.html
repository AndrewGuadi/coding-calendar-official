std::vector::push_back

The 'push_back' method in C++ is used to add a new element to the end of a vector. A vector is a dynamic array that can grow in size, meaning that as you add elements with 'push_back', the vector will automatically allocate more space if needed. This method increases the size of the vector by one and initializes the new element to the value provided. Since vectors store elements in contiguous memory, 'push_back' is generally efficient, although it may involve reallocations when the internal capacity needs to be increased.

```python
# Example of using a vector to store strings in C++
# We will create a vector, push back some string values, and print them.

#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    myVector.push_back("from");
    myVector.push_back("C++");

    for (const auto& str : myVector) {
        std::cout << str << " ";  // Output will be: Hello World from C++
    }
    return 0;
}
```

```python
# Example of using a vector to store integers in C++
# We will create a vector, push back integer values, sum them, and print result.

#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back(10);
    myVector.push_back(20);
    myVector.push_back(30);

    int sum = 0;
    for (int num : myVector) {
        sum += num;  // Sum will be: 10 + 20 + 30 = 60
    }
    std::cout << "Sum: " << sum << std::endl;  // Output will be: Sum: 60
    return 0;
}
```
