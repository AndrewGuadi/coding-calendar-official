upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a given string to uppercase letters. It returns a new string with all letters in uppercase, leaving other characters unchanged.

original_string = 'hello world' 
capitalized_string = original_string.upcase() 
# Output: 'HELLO WORLD'
input_string = 'Python 3.8' 
uppercase_string = input_string.upcase() 
# Output: 'PYTHON 3.8'
mixed_case_string = 'Ruby Rocks!' 
transformed_string = mixed_case_string.upcase() 
# Output: 'RUBY ROCKS!'
