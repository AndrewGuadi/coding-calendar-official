std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It adds a new element to the end of the vector, effectively increasing the size of the container by one. The new element is initialized to the value passed to the method. This operation is efficient and has an average constant time complexity, although it may occasionally take longer if the vector needs to be resized to accommodate the new element.

```cpp
#include 
#include 
#include 

int main() {
    std::vector pythonExamples;
    pythonExamples.push_back("print('Hello, World!')");
    pythonExamples.push_back("for i in range(5): print(i)");
    pythonExamples.push_back("def greet(name): return f'Hello, {name}!'");

    for (const auto& example : pythonExamples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
