std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class, which is part of the C++ Standard Library. It is used to add a new element to the end of the vector, effectively increasing its size by one. This method takes a single parameter: the value to be added to the vector. The capacity of the vector may be increased automatically by `push_back` if the current size exceeds the vector's capacity, ensuring that there is enough room to accommodate the new element. This operation is average constant time, but may occasionally take longer if the vector needs to allocate more memory to accommodate new elements.

# Example of using std::vector::push_back
#include 
#include 

int main() {
    std::vector statements;
    
    // Adding elements to the vector using push_back
    statements.push_back("print('Hello, World!')");
    statements.push_back("for i in range(5):");
    statements.push_back("    print(i)");
    
    // Printing the statements stored in the vector
    for (const auto& statement : statements) {
        std::cout << statement << std::endl;
    }
    return 0;
}

int main() {
    std::vector numbers;
    
    // Adding numbers to the vector using push_back
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back("Number " + std::to_string(i));
