charAt

The 'charAt' method in JavaScript is used to return the character at a specified index within a string. The index is zero-based, meaning that the first character of the string is at index 0, the second character at index 1, and so on. If the index is out of bounds (i.e., less than 0 or greater than or equal to the string's length), it returns an empty string. This method is particularly useful when you need to access individual characters of a string.

def get_char_at_index(s, index):
    return s[index] if 0 <= index < len(s) else ''

# Example usage:
string = 'Python'
print(get_char_at_index(string, 0))  # Output: 'P'
print(get_char_at_index(string, 3))  # Output: 'h'
print(get_char_at_index(string, 7))  # Output: ''
def extract_initials(name):
    initials = ''
    for i in range(len(name)):
        if i == 0 or name[i - 1] == ' ':
            initials += name.charAt(i)
    return initials

# Example usage:
name = 'John Doe'
print(extract_initials(name))  # Output: 'JD'
def get_last_char(s):
    if len(s) > 0:
        return s.charAt(len(s) - 1)
    return ''

# Example usage:
string = 'JavaScript'
print(get_last_char(string))  # Output: 't'
string = ''
print(get_last_char(string))  # Output: ''
