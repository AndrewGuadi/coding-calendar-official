std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector. When this method is called, it increases the size of the vector by one and initializes the new element with the value provided as an argument. If the vector's capacity is reached, it will automatically allocate more memory to accommodate the new element. This method is commonly used for dynamically managing collections of elements where the total number of elements is not known beforehand.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("hello");
    vec.push_back("world");

    // Output vector contents
    for(const auto& str : vec) {
        std::cout << str << " ";
    }
    return 0;
}
```
```cpp
#include

int main() {
    std::vector numbers;
    for(int i = 0; i < 5; ++i) {
        numbers.push_back(i * 2);
    }

    // Output vector contents
    for(const auto& num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}
```
