cap

The 'cap' function in Go returns the capacity of a given slice, which indicates the maximum number of elements the slice can hold without needing to allocate additional memory. This is important for optimizing performance as it helps manage memory usage when appending new elements to slices. It's worth noting that the capacity can be equal to or greater than the length of the slice. When appending elements, if the length exceeds the capacity, the slice will reallocate memory to accommodate the new elements.

my_slice = []int{1, 2, 3}
print(f"Original slice: {my_slice}")
print(f"Length: {len(my_slice)}")
print(f"Capacity: {cap(my_slice)}")
