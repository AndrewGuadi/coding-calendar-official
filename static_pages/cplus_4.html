std::vector::size

The method `std::vector::size` is a member function of the `std::vector` class in C++. It is used to obtain the number of elements currently stored in a vector. The size is returned as a value of type `size_t`, which is an unsigned integral type. This method helps you understand how many elements the vector contains and aids in iterating over the vector or performing operations based on the number of elements. Importantly, this method has constant time complexity, meaning it executes in O(1) time regardless of the vector size.

```python
# Example of using std::vector::size in C++
# vector vec = {1, 2, 3, 4, 5};
vec = [1, 2, 3, 4, 5]
print(len(vec))  # Output: 5
```
```python
# Example of using std::vector::size with an empty vector
# vector vec;
vec = []
print(len(vec))  # Output: 0
```
```python
# Example where elements are added to a vector and size is checked
# vector vec;
vec = []
vec.append(3.14)
vec.append(2.71)
print(len(vec))  # Output: 2
```
