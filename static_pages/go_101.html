cap

The 'cap' method in Go is used to return the capacity of a slice. The capacity of a slice is the number of elements the slice can hold, which is always greater than or equal to the length of the slice. This method is particularly useful when you want to allocate a slice with an initial capacity and need to check how much memory is optimized for it. In contrast to list in Python, where dynamic resizing can be observed, the 'cap' method directly helps in efficient memory management in Go.

```python
my_slice = [1, 2, 3]
print(len(my_slice))  # Output: 3
print(my_slice.__len__())  # Simulating cap method: approximates slices's length in Go
```
```python
my_slice = ['a', 'b']
my_slice.extend(['c', 'd'])
print(len(my_slice))  # Output: 4
print(my_slice.__len__())  # Simulating cap method
```
```python
my_slice = list(range(100))  # Creating a list of 100 elements
print(len(my_slice))  # Output: 100
print(my_slice.__len__())  # Simulating cap method
```
