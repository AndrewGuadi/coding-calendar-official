std::vector::push_back

The 'std::vector::push_back' method is used to add a new element to the end of a vector in C++. This method modifies the vector to accommodate the new element, increasing its size by one. If the vector's current capacity is exceeded, memory allocation may occur in order to allocate enough space for the new element. It is important to note that this method does not return a value and will throw an exception if memory allocation fails.

```cpp
#include 
#include 

int main() {
    std::vector programmingLanguages;
    programmingLanguages.push_back("Python");
    programmingLanguages.push_back("Java");
    programmingLanguages.push_back("C++");

    for (const auto &lang : programmingLanguages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
