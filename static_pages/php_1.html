strpos

The `strpos` function is used in PHP to find the position of the first occurrence of a substring within a string. It returns the index of the first match or false if the substring is not found. The search is case-sensitive, and the function can take an optional third parameter to specify the starting position for the search in the string.

$string = 'This is a simple PHP example';
$substring = 'PHP';
$position = strpos($string, $substring);
echo 'The position of PHP is: ' . $position;
$string = 'Welcome to the PHP world';
$substring = 'welcome';
$position = strpos($string, $substring);
if ($position === false) {
    echo 'Substring not found!';
} else {
    echo 'Substring found at position: ' . $position;
}
$string = 'Multiple PHP examples';
$substring = 'PHP';
$position = strpos($string, $substring, 10);
echo 'Searching for PHP after position 10: ' . $position;
