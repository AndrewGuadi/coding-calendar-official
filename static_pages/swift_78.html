uppercased

The 'uppercased' method in Swift is used to convert all characters in a given string to uppercase. This transformation affects all characters in the string equally, irrespective of their original case. The result is a new string where every lowercase letter is converted to its uppercase equivalent, while characters that are already uppercase or non-alphabetic characters remain unchanged. This method is commonly utilized to ensure consistency in string comparison or when formatting output to meet certain aesthetic or functional requirements.

input_string = 'Hello, World!'
output_string = input_string.uppercased()
# Result: output_string now is 'HELLO, WORLD!'
input_string = 'python programming'
output_string = input_string.uppercased()
# Result: output_string now is 'PYTHON PROGRAMMING'
input_string = '123abcDEF!'
output_string = input_string.uppercased()
# Result: output_string now is '123ABCDEF!'
