charAt

The `charAt` method in JavaScript returns a new string consisting of the single UTF-16 code unit located at the specified offset. The first position in the string is zero-based, meaning that the first character can be accessed with an index of zero. If the specified index is out of range (i.e., negative or greater than or equal to the string's length), it will return an empty string.

const str = 'Hello World';
console.log(str.charAt(0)); // Output: 'H' - Accessing the first character of the string.
const str = 'JavaScript';
console.log(str.charAt(4)); // Output: 'S' - Accessing the character at index 4 of the string which is 'S'.
const str = 'OpenAI';
console.log(str.charAt(10)); // Output: '' - Attempting to access an out of range index returns an empty string.
