Add

The 'Add' method is used to add an object to a collection. In C#, this method is typically found in collections such as lists, dictionaries, and sets. When you use 'Add', the specified item will be included in the collection, allowing for storage and later access of that item. This method is essential for managing dynamic collections where the number of items can change at runtime.

```python
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
my_set = set()
my_set.add('apple')
my_set.add('banana')
print(my_set)  # Output: {'apple', 'banana'}
```
