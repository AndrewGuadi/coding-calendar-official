abs

The `abs` method in Java is used to return the absolute value of a given number. It is available in the `java.lang.Math` class and can be used for various numeric types including integers, floating-point numbers, and more. The absolute value of a number is its distance from zero on the number line, regardless of direction. For example, the absolute value of both -5 and 5 is 5. Using this method is useful when you need to ensure that a value is always non-negative, such as when calculating distances or differences.

def example_abs_with_integers():
    numbers = [-10, -5, 0, 5, 10]
    absolute_values = [abs(num) for num in numbers]
    return absolute_values  # Output: [10, 5, 0, 5, 10]
def example_abs_with_floats():
    numbers = [-12.3, -5.5, 0.0, 4.2, 11.9]
    absolute_values = [abs(num) for num in numbers]
    return absolute_values  # Output: [12.3, 5.5, 0.0, 4.2, 11.9]
def example_abs_with_differences():
    a = 5
    b = 10
    difference = a - b
    return abs(difference)  # Output: 5
