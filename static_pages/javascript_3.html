endsWith

The endsWith() method in JavaScript checks if a given string ends with the characters of a specified string. It returns true if the string ends with the specified characters, and false otherwise. The method can also accept an optional second parameter that defines the length of the string to be considered, which allows you to restrict the check to a portion of the string.

my_string = 'Hello, World!'; print(my_string.endswith('World!')) # Output: True
file_name = 'report.pdf'; print(file_name.endswith('.pdf')) # Output: True
test_string = 'JavaScript is fun'; print(test_string.endswith('fun')) # Output: True
