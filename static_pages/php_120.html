strlen

The 'strlen' function in PHP is used to get the length of a string. It returns the number of bytes rather than the number of characters, so keep that in mind when working with multibyte character encodings. This function is particularly useful when you need to validate the length of user inputs or when manipulating strings for various purposes. The string length is calculated in bytes, meaning that a character can be counted as one or more bytes depending on its encoding.

my_string = 'Hello World';
length = strlen(my_string)
# Output: 11  # 'Hello World' has 11 characters
my_string = 'Python is fun!';
length = strlen(my_string)
# Output: 16  # Count the number of characters in the string including spaces and punctuation
my_string = 'こんにちは';
length = strlen(my_string)
# Output: 15  # 'こんにちは' contains 5 characters, but the byte count is 15 in UTF-8 encoding
