abs

The 'abs' method in Java is used to return the absolute value of a given number. It can be applied to different numeric types including integers, floating-point numbers, and long integers. The absolute value of a number is its distance from zero on the number line, which is always a non-negative value. If the number is negative, the 'abs' method converts it to its positive counterpart, while if it is already positive or zero, it remains unchanged.

def absolute_value_example():
    num = -10
    return abs(num)  # Output: 10
def float_absolute_value_example():
    num = -5.75
    return abs(num)  # Output: 5.75
def list_absolute_values():
    nums = [-3, 4, -7, 0, 2.5]
    return [abs(n) for n in nums]  # Output: [3, 4, 7, 0, 2.5]
