strlen

The strlen() function in PHP is used to get the length of a string. It counts the number of characters in a string, including spaces and punctuation, returning the total count as an integer. This function is crucial for validating user input, processing strings, and performing various string manipulation tasks. It is important to note that strlen counts the number of bytes in a string if the string contains multibyte characters; in such cases, it might be appropriate to use 'mb_strlen' to get the correct character count for multibyte strings.

def calculate_string_length(input_string):
    return len(input_string)

string_example = 'Hello, World!'
length = calculate_string_length(string_example)
print('Length of the string:', length)  # Output: Length of the string: 13
def is_string_empty(input_string):
    return len(input_string) == 0

input1 = ''
input2 = 'PHP'
print('Is input1 empty?', is_string_empty(input1))  # Output: Is input1 empty? True
print('Is input2 empty?', is_string_empty(input2))  # Output: Is input2 empty? False
def truncate_string(input_string, max_length):
    if len(input_string) > max_length:
        return input_string[:max_length] + '...'
