acos

The acos() method in Java is part of the Math class, which provides various mathematical operations. The acos() method is used to calculate the arc cosine (inverse cosine) of a specified number, returning the result in radians. The argument for this method must be a value between -1 and 1, inclusive. If the value is outside this range, it will return NaN (Not a Number). The return value is used in various applications, including geometry, physics, and engineering, where the angle corresponding to a certain cosine value is needed.

import math

# Calculate the arc cosine of a value
value = 0.5
angle_in_radians = math.acos(value)
angle_in_degrees = math.degrees(angle_in_radians)
print(f'The angle in radians for cos(θ)={value} is: {angle_in_radians}')
print(f'The equivalent angle in degrees is: {angle_in_degrees}')
import math

# Checking input outside the valid range
value = 2
angle = math.acos(value) if -1 <= value <= 1 else float('nan')
print(f'Value: {value}, Result: {angle}')}
