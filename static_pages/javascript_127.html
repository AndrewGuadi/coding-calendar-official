charAt

The `charAt` method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning that the first character has an index of 0. If the index is out of range (less than 0 or greater than the string's length minus one), `charAt` returns an empty string. This method is useful for retrieving specific characters from a string, allowing for character manipulation and analysis.

const text = 'Hello, World!';
const firstChar = text.charAt(0); // 'H'
console.log(firstChar);
const fruit = 'Banana';
const thirdChar = fruit.charAt(2); // 'n'
console.log(thirdChar);
const message = 'JavaScript';
const lastChar = message.charAt(message.length - 1); // 't'
console.log(lastChar);
