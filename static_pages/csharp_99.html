Add

The 'Add' method is used to add elements to a collection, such as a list or a dictionary in C#. This method typically accepts a single argument (the item to be added) and appends it to the end of a list or associates it with a key in a dictionary. It is commonly used in data storage and manipulation scenarios where new items need to be included in an existing dataset.

```python
# Example of using the Add method with a List in C#

# Initialize a List
my_list = []

# Add elements using the Add method
my_list.append('Hello')
my_list.append('World')

# Print the List
print(my_list)  # Output: ['Hello', 'World']
```
```python
# Example of using the Add method with a Dictionary in C#

# Initialize a Dictionary
my_dict = {}

# Add key-value pairs using the Add method
my_dict['name'] = 'John'
my_dict['age'] = 30

# Print the Dictionary
print(my_dict)  # Output: {'name': 'John', 'age': 30}
```
```python
# Example of using the Add method in a custom class in C#

class MyCollection:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

# Create an instance of MyCollection
my_collection = MyCollection()

# Use the Add method to add items
my_collection.add('Item 1')
my_collection.add('Item 2')

# Print the items in the collection
print(my_collection.items)  # Output: ['Item 1', 'Item 2']
```
