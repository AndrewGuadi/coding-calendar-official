charAt

The 'charAt' method in JavaScript returns the character at a specified index in a string. This method takes one argument which is the index (0-based) of the character to be returned. If the index is out of bounds (less than 0 or greater than or equal to the length of the string), it returns an empty string. This method is useful when you need to access individual characters in a string for processing or manipulation.

text = 'Hello, World!'
index = 0
character = text.charAt(index)
print(character)  # Output: 'H'
text = 'Python Programming'
index = 7
character = text.charAt(index)
print(character)  # Output: 'P'
text = 'OpenAI'
index = 10
character = text.charAt(index)
print(character)  # Output: ''
