std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that appends a new element to the end of the vector. This method dynamically increases the size of the vector by one, adding the specified element to the last position. The element passed as an argument can be of any type that is compatible with the vector's type. After the operation, the size of the vector increases by one. It is a commonly used function for building dynamic arrays in C++.

```cpp
#include 
#include 

int main() {
    std::vector animals;
    animals.push_back("cat");
    animals.push_back("dog");
    animals.push_back("bird");

    for (const auto& animal : animals) {
        std::cout << animal << std::endl;
    }
    return 0;
}
```
