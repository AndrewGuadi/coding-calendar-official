std::vector::push_back

The method std::vector::push_back is used to add a new element at the end of the vector, effectively expanding its size by one. This method increases the vector's capacity if it needs to allocate more memory. The new element is initialized to the value provided as an argument.

```python
# Create an empty list to represent a vector
my_vector = []
# Push back several Python code strings into the vector
my_vector.append('print("Hello World")')
my_vector.append('x = 5')
my_vector.append('y = x * 2')

# Print the contents of the vector
for code in my_vector:
    print(code)
```
```python
# Initialize a vector-like list
vector_of_strings = []
# Add a string using push_back equivalent in Python
vector_of_strings.append('def greet():')
vector_of_strings.append('    return "Greetings!"')

# Execute the function and print output
exec("""".join(vector_of_strings))
print(greet())
```
