acos

The 'acos' method in Java's Math class returns the arc cosine of a value. The arc cosine is the inverse function of the cosine function, which means that if you have a cosine value, you can use 'acos' to find the angle (in radians) whose cosine is that value. The input value must be in the range of -1 to 1, as those are the valid outputs of the cosine function. The result is given in radians and can be converted to degrees if necessary by multiplying by (180/Math.PI).

import math

# Example to find the angle whose cosine is 0.5
cos_value = 0.5
angle_in_radians = math.acos(cos_value)
angle_in_degrees = angle_in_radians * (180 / math.pi)

print(f'The angle in radians is: {angle_in_radians}')  # Output: 1.0471975511965979
print(f'The angle in degrees is: {angle_in_degrees}')  # Output: 60.00000000000001
