charAt

The 'charAt' method in JavaScript is used to return the character at a specified index within a string. If the index is out of range (less than 0 or greater than the string length minus one), it will return an empty string. This method is helpful when you need to access individual characters from a string at specific positions.

const str = 'Hello';
const char1 = str.charAt(0);
console.log(char1); // Output: 'H'
// Here, charAt is used to get the first character of the string 'Hello'.
const str = 'JavaScript';
const char2 = str.charAt(4);
console.log(char2); // Output: 'S'
// In this example, charAt is used to get the character at index 4 from the string 'JavaScript', which is 'S'.
const str = 'OpenAI';
const char3 = str.charAt(6);
console.log(char3); // Output: ''
// This example shows charAt with an out of range index (6) for the string 'OpenAI', which returns an empty string.
