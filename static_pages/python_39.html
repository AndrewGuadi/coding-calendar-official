abs

The `abs` method in Python returns the absolute value of a given number. The absolute value of a number is its non-negative value, regardless of its sign. In other words, it turns negative numbers into positive numbers and leaves positive numbers and zero unchanged. The `abs` function can be used with integers, floating-point numbers, and complex numbers. When used with a complex number, it returns the magnitude (or modulus) of the complex number.

result = abs(-10)
print(result)  # Output will be 10
result = abs(3.14)
print(result)  # Output will be 3.14
complex_number = 3 + 4j
result = abs(complex_number)
print(result)  # Output will be 5.0 because sqrt(3^2 + 4^2) is 5
