abs

The `abs` method in Python is used to get the absolute value of a number. The absolute value of a number is its distance from zero, without considering which direction from zero the number lies. This means that the absolute value of both -3 and 3 is 3. The `abs` method can be applied to both integers and floating-point numbers.

print(abs(-5))  # Output: 5
print(abs(3.14))  # Output: 3.14
def calculate_distance(point1, point2):
    return abs(point1 - point2)

print(calculate_distance(10, 3))  # Output: 7
print(calculate_distance(-5, -15))  # Output: 10
