upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to their uppercase counterparts. It does not modify the original string but instead returns a new string with the uppercase letters. If the string already contains uppercase letters or does not have any alphabetic characters, the `upcase` method will return the string unchanged in terms of its existing uppercase letters.

def convert_to_uppercase(input_string):
    # Convert the input string to uppercase
    return input_string.upper()

# Example usage:
string = 'hello world'
print(convert_to_uppercase(string))  # Output: 'HELLO WORLD'
def shout(text):
    # Make the input text in uppercase to simulate shouting
    return text.upper() + '!!!'

# Example usage:
text = 'watch out'
print(shout(text))  # Output: 'WATCH OUT!!!'
def emphasize(text):
    # Convert given text to uppercase to emphasize it
    return text.upper()

# Example usage:
message = 'important notice'
print(emphasize(message))  # Output: 'IMPORTANT NOTICE'
