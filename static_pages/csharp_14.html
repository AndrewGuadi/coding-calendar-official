Add

The Add method in C# is typically used to insert an item into a collection, such as a List or Dictionary. This method allows for adding elements to the collection, expanding its size, and enabling further manipulation of its contents. The method can also vary in implementation depending on the specific collection type being used, such as adding an object to a List or adding a key-value pair to a Dictionary.

C# Code:
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List pythonExamples = new List();
        pythonExamples.Add("print('Hello, World!')");
        pythonExamples.Add("for i in range(5): print(i)");
        pythonExamples.Add("def add(a, b): return a + b");

        foreach (var example in pythonExamples)
        {
            Console.WriteLine(example);
        }
    }
}
C# Code:
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary pythonFunctions = new Dictionary();
        pythonFunctions.Add("Print Function", "print('Hello, World!')");
        pythonFunctions.Add("Range Function", "for i in range(5): print(i)");
        pythonFunctions.Add("Add Function", "def add(a, b): return a + b");

        foreach (var function in pythonFunctions)
        {
            Console.WriteLine(function.Key + ": " + function.Value);
        }
    }
}
