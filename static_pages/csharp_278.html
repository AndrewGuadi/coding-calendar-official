Add

The Add method is a common operation in C# used to insert a new element into a collection such as a List, Dictionary, or a similar data structure. This method typically appends the element to the end of the collection or adds a key-value pair in case of dictionaries. In collections, it expands the size of the data structure to accommodate the new element.

```python
# Example of using the 'Add' method with a List in C#
# Creating a list to store names
names = []
# Adding names using the 'Add' function
names.append('Alice')
names.append('Bob')
names.append('Charlie')
# Print the final list
print(names)  # Output: ['Alice', 'Bob', 'Charlie']
```
```python
# Example of using the 'Add' method with a Dictionary in C#
# Creating a dictionary to store age of persons
ages = {}
# Adding persons and their ages using the 'Add' function
ages['Alice'] = 30
ages['Bob'] = 25
ages['Charlie'] = 35
# Print the final dictionary
print(ages)  # Output: {'Alice': 30, 'Bob': 25, 'Charlie': 35}
```
```python
# Example of using 'Add' method for a List of custom objects in C#
# Defining a class for Person
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

# Creating a list to store Person objects
people = []
# Adding Person objects using the 'Add' function
people.append(Person('Alice', 30))
people.append(Person('Bob', 25))
people.append(Person('Charlie', 35))
# Print the names of persons in the list
for person in people:
    print(person.name)
# Output: 
# Alice
# Bob
# Charlie
```
