uppercased

The `uppercased` method in Swift is used to transform a string into its uppercase representation. This method returns a new string with all the characters converted to uppercase, allowing for easy manipulation of string data where case consistency is required. It does not change the string it is called on; instead, it returns a new string with uppercase characters. This method is particularly useful when you want to ensure that user inputs are validated in a case-insensitive manner or when generating outputs that require uniformity in text case.

string_value = "hello, world"
uppercased_string = string_value.uppercased()
print(uppercased_string)  # Outputs: HELLO, WORLD
user_input = "Good Morning!"
uppercased_input = user_input.uppercased()
print(uppercased_input)  # Outputs: GOOD MORNING!
phrase = "i love programming!"
result = phrase.uppercased()
print(result)  # Outputs: I LOVE PROGRAMMING!
