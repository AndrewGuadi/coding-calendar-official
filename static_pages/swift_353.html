uppercased

The `uppercased` method is a String method in Swift that returns a new string with all the characters of the original string converted to uppercase. This method does not modify the original string; instead, it creates and returns a new string representation. `uppercased` is useful when you want to standardize string values, such as when comparing user input to a predefined set of options. Usage examples include normalizing user input or displaying text in all uppercase for visual emphasis.

original_string = 'hello'
new_string = original_string.uppercased()
print(new_string)  # Output: 'HELLO'
greeting = 'Hello, World!'
uppercased_greeting = greeting.uppercased()
print(uppercased_greeting)  # Output: 'HELLO, WORLD!'
user_input = 'swift programming'
normalized_input = user_input.uppercased()
if normalized_input == 'SWIFT PROGRAMMING':
    print('User input matches the expected value.')  # Output: 'User input matches the expected value.'
