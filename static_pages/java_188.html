acos

The 'acos' method in Java is a part of the 'Math' class that calculates the arc cosine of a given value. The input to this method should be in the range from -1 to 1. The output is returned in radians. If the input is outside this range, it returns NaN (Not a Number). This method is useful in various applications, including trigonometry, physics, and engineering calculations, where it is necessary to determine the angle corresponding to a given cosine value.

import math

# Calculate the arc cosine of a cosine value
cos_value = 0.5
angle_radians = math.acos(cos_value)
angle_degrees = math.degrees(angle_radians)
print(f'Arc cosine of {cos_value} is {angle_radians} radians or {angle_degrees} degrees.')
import math

# Calculate the angle for a negative cosine value
cos_value = -1.0
angle_radians = math.acos(cos_value)
angle_degrees = math.degrees(angle_radians)
print(f'Arc cosine of {cos_value} is {angle_radians} radians or {angle_degrees} degrees.')
import math

# Handling input outside the valid range
cos_value = 1.5  # Invalid input
try:
    angle_radians = math.acos(cos_value)
    print(f'Arc cosine of {cos_value} is {angle_radians} radians.')

    print(f'Input {cos_value} is outside the valid range for arc cosine.')
