Add

The 'Add' method is utilized in C# collections, such as lists or dictionaries, to insert new elements into the collection. In a List, the method adds a new element at the end of the list, allowing for dynamic resizing. In dictionaries, 'Add' inserts a new key-value pair, and if the key already exists, it throws an exception. This method is fundamental for managing collections of data efficiently.

```python
# Example of using the Add method in a List
my_list = []
my_list.append('Python')
my_list.append('C#')
print(my_list)  # Output: ['Python', 'C#']
```
```python
# Example of using the Add method in a Dictionary
my_dict = {}
my_dict['first'] = 'Python'
my_dict['second'] = 'C#'
print(my_dict)  # Output: {'first': 'Python', 'second': 'C#'}
```
```python
# Attempting to use the Add method with a duplicate key in a Dictionary
my_dict = {'first': 'Python'}
try:
    my_dict['first'] = 'C#'
except KeyError:
    print('Key already exists!')
print(my_dict)  # Output: {'first': 'C#'}
```
