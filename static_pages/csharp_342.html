Add

The Add method is a member of various collection classes in C#, such as List, Dictionary, or HashSet. It is used to insert an element into the collection. The specific functionality may vary depending on the type of collection. For example, in a List, the Add method adds an element to the end, while in a Dictionary, it adds a key-value pair. It is important to handle potential exceptions, such as when adding duplicate keys in a Dictionary.

```python
# Example of using the Add method with a List in C#
# In Python: Using a List
my_list = []
my_list.append('Hello')  # Similar to Add in List
print(my_list)  # Output: ['Hello']
```
```python
# Example of using the Add method with a Dictionary in C#
# In Python: Using a Dictionary
my_dict = {}
my_dict['key1'] = 'value1'  # Similar to Add in Dictionary
print(my_dict)  # Output: {'key1': 'value1'}
```
```python
# Example of using the Add method with a HashSet in C#
# In Python: Using a Set
my_set = set()
my_set.add('Hello')  # Similar to Add in HashSet
print(my_set)  # Output: {'Hello'}
```
