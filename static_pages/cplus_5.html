std::vector::push_back

The 'push_back' method in C++ adds a new element to the end of a std::vector. This method dynamically increases the size of the vector to accommodate the new element being added. If the capacity of the vector is reached, it automatically allocates more memory to hold additional elements. This method is part of the standard library and is commonly used for dynamic array implementations where the size of the array is not defined at compile time.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& val : myVector) {
        std::cout << val << ' '; // Output: Hello World
    }
    return 0;
}
```
