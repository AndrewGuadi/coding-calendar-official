std::vector::push_back

The 'push_back' method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one and copies the new element into the added space. It's important to note that if the vector's current capacity is insufficient to accommodate the new element, the vector will automatically reallocate its storage to fit the new data. This method has an average time complexity of O(1), though in the worst case, reallocation can lead to O(n) complexity. It is part of the standard template library (STL) and is most commonly utilized for managing dynamic arrays.

# Example 1: Creating a vector and adding elements
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Python");
    vec.push_back("Java");
    vec.push_back("C++");

    for (const auto& lang : vec) {
        std::cout << lang << std::endl;
    }
    return 0;
}
# Example 2: Using push_back in a loop
#include 
#include 

int main() {
    std::vector languages;
    std::vector new_languages = {"JavaScript", "Ruby", "Go"};

    for (const auto& lang : new_languages) {
        languages.push_back(lang);
    }

    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}

int main() {
    std::vector fruits;
    for (int i = 0; i < 10; ++i) {
        fruits.push_back("Fruit " + std::to_string(i));
        std::cout << "Size: " << fruits.size() << " Capacity: " << fruits.capacity() << std::endl;
    }
    return 0;
}
