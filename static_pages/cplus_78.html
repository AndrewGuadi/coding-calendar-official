std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. This allows dynamic resizing of the vector as elements are added. The method takes a single parameter, which is the value to be added to the vector. If the vector's current capacity is reached, it automatically reallocates its memory to accommodate the new element. This is a key feature of vectors as it enables developers to manage dynamic lists of items efficiently.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("First element");
    myVector.push_back("Second element");

    for (const auto& elem : myVector) {
        std::cout << elem << std::endl;
    }
    return 0;
}
```
