charAt

The charAt() method is used in JavaScript to return the character at a specified index in a string. The index is zero-based, meaning that the first character in the string is at index 0, the second character at index 1, and so on. If the index is out of bounds, it returns an empty string. This method is useful for accessing individual characters in a string without converting it to an array.

string_value = 'python'
char_at_0 = string_value.charAt(0)
print(char_at_0)  # Output: 'p'
string_value = 'javascript'
char_at_5 = string_value.charAt(5)
print(char_at_5)  # Output: 'a'
string_value = 'openai'
char_at_10 = string_value.charAt(10)
print(char_at_10)  # Output: ''
