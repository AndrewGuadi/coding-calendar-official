capitalized

The 'capitalized' property in Swift is used to return a string where the first character of each word is converted to uppercase, and all other characters are converted to lowercase. This is useful for formatting strings where you want to ensure that each word starts with a capital letter, similar to title casing. Note that this method does not alter the original string but creates a new string with the desired formatting.

text = 'hello world'
formatted_text = ' '.join([word.capitalize() for word in text.split()])
# formatted_text will be 'Hello World'
text = 'swift programming language'
formatted_text = ' '.join([word.capitalize() for word in text.split()])
# formatted_text will be 'Swift Programming Language'
text = 'python is an amazing language'
formatted_text = ' '.join([word.capitalize() for word in text.split()])
# formatted_text will be 'Python Is An Amazing Language'
