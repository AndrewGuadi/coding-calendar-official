ascii

The `ascii()` function in Python is used to return a string containing a printable representation of an object, escaping the non-ASCII characters in the string using Unicode escape sequences. This is particularly useful for debugging or displaying strings in contexts where non-ASCII characters need to be represented in a safe format. The output will still be visibly similar to the input string, but any non-ASCII characters will be turned into escape sequences. The function takes a single argument, which is the object (typically a string) you want to convert to an ASCII representation.

string_value = 'Café'
print(ascii(string_value))  # Output: 'Caf\xe9'
string_value = '123 – 456'
print(ascii(string_value))  # Output: '123 \u2013 456'
string_value = 'Hello, 世界'
print(ascii(string_value))  # Output: 'Hello, \u4e16\u754c'
