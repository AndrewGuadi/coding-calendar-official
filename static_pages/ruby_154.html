upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase. It is a part of the String class and does not change the original string; instead, it returns a new string with all letters converted to uppercase. This method is useful when you need to normalize data for case-insensitive comparisons or simply for formatting purposes.

string = "hello world"
upcased_string = string.upcase
print(upcased_string)  # Output: HELLO WORLD
string = "Ruby Programming"
upcased_string = string.upcase
print(upcased_string)  # Output: RUBY PROGRAMMING
string = "data science"
upcased_string = string.upcase
print(upcased_string)  # Output: DATA SCIENCE
