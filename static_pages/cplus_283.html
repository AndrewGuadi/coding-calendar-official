std::vector::push_back

The `std::vector::push_back` method in C++ is used to add a new element to the end of a vector. This operation extends the vector's size by one and increases its capacity if necessary to accommodate the new element. This method is commonly used when the number of elements is not known in advance, providing a dynamic way to manage a list of items. The element can be of any type, and the method ensures that the vector maintains its order by appending the new element at the back.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("apple");
    myVector.push_back("banana");
    myVector.push_back("cherry");

    for (const auto& fruit : myVector) {
        std::cout << fruit << " ";
    }
    return 0;
}
```
