std::vector::push_back

The 'push_back' method is used to add a new element to the end of a vector in C++. It increases the size of the vector by one and adds the new element at the last position. If the vector's capacity is reached, it automatically reallocates memory to accommodate the new element. This method is often used to build up a collection of items dynamically as the exact number of elements may not be known at compile time.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    
    for(const auto& str : myVector) {
        std::cout << str << " "; // Output: Hello World
    }
    return 0;
}
```
