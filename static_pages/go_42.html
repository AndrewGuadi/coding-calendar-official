cap

The 'cap' method in Go is used to obtain the capacity of a slice. The capacity of a slice is the number of elements that the slice can hold in its underlying array. This method is useful for understanding how much more data can be added to a slice before the underlying array needs to be reallocated. It does not modify the slice and is typically used for optimization and memory management.

```python
# Example of using cap in Go

my_slice = []int{1, 2, 3}
print(cap(my_slice))  # Output: 3
```
```python
# Example showing capacity increase when appending elements

my_slice = make([]int, 0, 2)  # Create a slice with capacity of 2
print(cap(my_slice))  # Output: 2

my_slice = append(my_slice, 1)
print(cap(my_slice))  # Output: 2

my_slice = append(my_slice, 2)
print(cap(my_slice))  # Output: 2

my_slice = append(my_slice, 3)  # Appending causes reallocation
print(cap(my_slice))  # Output may vary; typically output: 4
```
```python
# Example demonstrating the relationship between length and capacity

my_slice = make([]int, 3, 5)  # Length 3, Capacity 5
print(len(my_slice))  # Output: 3
print(cap(my_slice))  # Output: 5

my_slice = append(my_slice, 4, 5)
print(len(my_slice))  # Output: 5
print(cap(my_slice))  # Output: 5

my_slice = append(my_slice, 6)  # This will increase capacity
print(len(my_slice))  # Output: 6
print(cap(my_slice))  # Output may vary; typically output: 10
```
