charAt

The charAt() method in JavaScript returns the character at a specified index in a string. If the index is out of bounds (negative or greater than the length of the string), it returns an empty string. This method is commonly used when you want to extract a single character from a string based on its position.

text = 'hello world'
character = text.charAt(0)
print(character)  # Outputs: 'h'
text = 'python programming'
character = text.charAt(4)
print(character)  # Outputs: 'o'
text = 'data science'
character = text.charAt(10)
print(character)  # Outputs: 'e'
