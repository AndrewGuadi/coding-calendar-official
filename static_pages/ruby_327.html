upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method returns a new string with all of its characters in uppercase, leaving non-alphabetic characters unchanged. It is a useful method when you want to standardize text input or display information in a uniform format.

def convert_to_upper(input_string):
    return input_string.upper()

text = 'hello world'
result = convert_to_upper(text)
print(result)  # Output: 'HELLO WORLD'
def shout_message(message):
    return message.upper() + '!'

message = 'hello there'
shouted = shout_message(message)
print(shouted)  # Output: 'HELLO THERE!'
def format_name(name):
    return name.split()[0].upper() + ' ' + name.split()[1].upper()

full_name = 'john doe'
formatted_name = format_name(full_name)
print(formatted_name)  # Output: 'JOHN DOE'
