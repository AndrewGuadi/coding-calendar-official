std::vector::push_back

The 'std::vector::push_back' method is used to add an element to the end of a vector in C++. When you call this method, the specified value is added to the back of the vector, and the vector's size increases by one. If the vector's capacity is reached, it will automatically reallocate to accommodate additional elements. This method is useful for building and managing dynamic arrays, where the number of elements is not known in advance.

```cpp
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    for (const auto &lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```
