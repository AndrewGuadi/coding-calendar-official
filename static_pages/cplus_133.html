std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a `std::vector`. This method increases the size of the vector by one and appends the new element at the end. It automatically manages memory allocation to accommodate the new element. If the capacity of the vector is exceeded, it will allocate more memory to ensure that there is enough space for additional elements. This method is typically used for dynamic arrays where the size can change during runtime.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Python");
    vec.push_back("Java");
    vec.push_back("C++");

    for (const auto& lang : vec) {
        std::cout << lang << std::endl;
    }
    return 0;
}
``` 
The above code demonstrates the usage of `push_back` method to add string elements representing programming languages to a vector and then print them out.
```cpp
#include 
#include 

int main() {
    std::vector languages;
    std::string input;
    std::cout << "Enter programming languages (type 'exit' to stop):" << std::endl;
    while (true) {
        std::getline(std::cin, input);
        if (input == "exit") break;
        languages.push_back(input);
    }

    std::cout << "You entered:
";
    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
``` 
In this example, the program collects user input for programming languages until the user types "exit", using `push_back` to store each language into the vector.
