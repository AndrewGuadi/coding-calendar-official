std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector. This method increases the size of the vector by one and initializes the added element with the value provided. The time complexity is generally amortized constant time, except when the vector requires resizing, in which case it may take linear time. It is important to ensure that the vector can accommodate the new element; if not, it will automatically allocate more memory.

```cpp
#include 
#include 

int main() {
    std::vector greetings;
    greetings.push_back("Hello, world!");
    greetings.push_back("Welcome to C++!");

    for (const auto& greeting : greetings) {
        std::cout << greeting << std::endl;
    }
    return 0;
}```
