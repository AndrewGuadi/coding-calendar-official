Add

The 'Add' method in C# is commonly used to insert an element into a collection, such as a List, Dictionary, or HashSet. This method allows you to add a new item to the collection, increasing its size by one. If the collection does not already contain the item (in the case of a HashSet or Dictionary), it will be added; otherwise, the collection state will remain unchanged. This method is intuitive and aims to provide a straightforward way to manage collections in C#. Itâ€™s important to note that the exact behavior may vary depending on the specific collection type being used.

```python
# Using a list to store names
names = []

# Function to add names to the list
def add_name(name):
    names.append(name)

# Adding names
add_name('Alice')
add_name('Bob')
print(names)  # Output: ['Alice', 'Bob']
```
```python
# Using a set to store unique items
unique_values = set()

# Function to add a value to the set
def add_value(value):
    unique_values.add(value)

# Adding values
add_value(10)
add_value(20)
add_value(10)  # Duplicate, will not be added
print(unique_values)  # Output: {10, 20}
```
```python
# Using a dictionary to store key-value pairs
data = {}

# Function to add a key-value pair to the dictionary
def add_entry(key, value):
    data[key] = value

# Adding entries
add_entry('key1', 'value1')
add_entry('key2', 'value2')
print(data)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
