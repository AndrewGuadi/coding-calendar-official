close

In Go, the 'close' function is used to close a channel, preventing any further sends on that channel. Once a channel is closed, any attempts to send new values to the channel will cause a panic, but you can still receive values from it until it is emptied. Closing a channel is also important to signal to the receiving goroutines that no more values will be sent, allowing them to end or clean up gracefully.

import "fmt"

// Example demonstrating closing a channel
func main() {
    ch := make(chan int)

    go func() {
        for i := 0; i < 5; i++ {
            ch <- i
        }
        close(ch) // Close the channel when done sending
    }()

    // Receive values
    for value := range ch {
        fmt.Println(value) // Prints 0 to 4
    }
}
import "fmt"

// Example showing panic if sending on a closed channel
func main() {
    ch := make(chan int)

    go func() {
        ch <- 1
        close(ch)
    }()

    fmt.Println(<-ch) // Receives 1
    // Uncommenting the next line would cause a panic due to sending on a closed channel
    // ch <- 2 // This will panic
}
import "fmt"

// Example demonstrating range over a channel before and after closing
func main() {
    ch := make(chan string)

    go func() {
        messages := []string{"hello", "world"}
        for _, msg := range messages {
            ch <- msg
        }
        close(ch)
    }()

    for message := range ch {
        fmt.Println(message) // Prints "hello" followed by "world"
    }
}
