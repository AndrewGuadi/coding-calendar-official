abs

The 'abs' method in Java is used to obtain the absolute value of a number. It is part of the java.lang.Math class and can be employed to ensure that the output is non-negative regardless of the sign of the input number. The method can handle integers, floating-point numbers, and other numerical data types. When applied, if the number is positive, it returns the same number; if it is negative, it returns its positive counterpart. This method is widely used in mathematical calculations where the magnitude of a number is required without regard to its sign.

def calculate_distance(x1, y1, x2, y2):
    return abs(x2 - x1) + abs(y2 - y1)

# Example Usage:
distance = calculate_distance(1, 2, 4, 5)
print(distance)  # Output: 6
def temperature_difference(temp1, temp2):
    return abs(temp1 - temp2)

# Example Usage:
difference = temperature_difference(30, 20)
print(difference)  # Output: 10
def value_check(number):
    return abs(number)

# Example Usage:
result = value_check(-15)
print(result)  # Output: 15
