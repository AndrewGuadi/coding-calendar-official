cap

The 'cap' method in Go is primarily used to return the capacity of a slice, which indicates the total amount of space allocated for the slice. The capacity is the total number of elements the slice can hold without needing to allocate additional memory. Understanding the capacity of a slice is important for optimizing memory usage and ensuring performance when appending elements to a slice.

s = []int{1, 2, 3}

# Get the capacity
capacity_of_s = cap(s)

# Display the result
print(f'Capacity of s: {capacity_of_s}')
s = make([]string, 0, 5) # Create a slice with a capacity of 5

# Check its initial capacity
print(f'Initial capacity: {cap(s)}')

# Append elements
s.append('A')
s.append('B')

# Check capacity after adding elements
print(f'Capacity after adding: {cap(s)}')
s = []float64{1.1, 2.2, 3.3}

# Check capacity directly
print(f'Capacity of float slice: {cap(s)}')

# Appending new elements to the slice
for i in range(4, 6):
    s.append(float(i))
    print(f'New capacity after appending {i}: {cap(s)}')
