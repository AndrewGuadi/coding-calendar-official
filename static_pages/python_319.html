ascii

The 'ascii' function in Python is not an in-built function; however, it's often used in the context of returning the ASCII representation of a character using the 'ord' function. It converts a given character to its corresponding ASCII value or converts a string to its ASCII representation as a series of numbers. In Python, you may also use 'repr()' to get a printable representation that encodes non-ASCII characters using escape sequences. This method is useful for debugging, working with binary data, or when you need a human-readable representation of raw byte data.

ascii_value = ord('A')
print(ascii_value)  # Output: 65
ascii_values = [ord(char) for char in 'hello']
print(ascii_values)  # Output: [104, 101, 108, 108, 111]
input_string = 'Hello, World!'
encoded_string = [ord(char) for char in input_string]
print(encoded_string)  # Output: [72, 101, 108, 108, 111, 44, 32, 87, 111, 114, 108, 100, 33]
