Add

The 'Add' method is commonly used to add an item to a collection or data structure in C#. It can be found in various collections like lists, dictionaries, and sets. The method typically takes an argument—the item to be added—and appends it to the end of the collection. Depending on the collection type, the behavior may vary slightly, but the core functionality remains the same: to increase the size of the collection by one.

```python
# Example of using the Add method with a List in C#
class Program:
    def __init__(self):
        self.my_list = []

    def add_item(self, item):
        self.my_list.append(item)


program = Program()
program.add_item('Python')
print(program.my_list)  # Output: ['Python']
```
```python
# Example of using the Add method with a Dictionary in C#
class DictionaryExample:
    def __init__(self):
        self.my_dict = {}

    def add_key_value(self, key, value):
        self.my_dict[key] = value

example = DictionaryExample()
example.add_key_value('language', 'Python')
print(example.my_dict)  # Output: {'language': 'Python'}
```
```python
# Example of using the Add method with a HashSet in C#
class HashSetExample:
    def __init__(self):
        self.my_set = set()

    def add_value(self, value):
        self.my_set.add(value)

set_example = HashSetExample()
set_example.add_value('Python')
set_example.add_value('Java')
print(set_example.my_set)  # Output: {'Java', 'Python'}
```
