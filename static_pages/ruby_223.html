upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method does not modify the original string but returns a new string with all uppercase characters. It is particularly useful for situations where case-insensitivity needs to be handled or for formatting text in a consistent style.

text = 'hello world'
converted_text = text.upcase
# converted_text will be 'HELLO WORLD'
greeting = 'Good Morning!'
uppercase_greeting = greeting.upcase
# uppercase_greeting will be 'GOOD MORNING!'
username = 'user123'
formatted_username = username.upcase
# formatted_username will be 'USER123'
