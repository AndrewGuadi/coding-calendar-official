std::vector::push_back

The method 'std::vector::push_back' is a member function of the std::vector class in C++. It is used to add a new element at the end of the vector, increasing the size of the vector by one. This new element is passed to the function as an argument. If the vector needs more capacity to accommodate the new element, it automatically resizes itself, thus ensuring that all elements are stored without losing any data. This method is generally efficient, though it may involve some overhead when resizing the internal storage of the vector.

In this example, we create a vector of integers and use 'push_back' to add elements to it. Each time we add an element, we print the current contents of the vector:

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back(1);
    myVector.push_back(2);
    myVector.push_back(3);

    std::cout << "Current vector contents:";
    for(int i : myVector) {
        std::cout << ' ' << i;
    }
    return 0;
}
```

In this example, we'll combine integers from a vector into a total sum, using 'push_back' to add results as strings, demonstrating a simple calculation:

```cpp

#include 
#include 

int main() {
    std::vector numbers = {5, 10, 15};
    std::vector results;
    int sum = 0;
    for(int number : numbers) {
        sum += number;
        results.push_back("Current sum after adding " + std::to_string(number) + " is: " + std::to_string(sum));
    }

    std::cout << "Results:";
    for(const std::string& result : results) {
        std::cout << '\n' << result;
    }
    return 0;
}
```
