charAt

The `charAt` method in JavaScript is used to retrieve a character from a string at a specified index (position). The index is zero-based, meaning that the first character of the string is at index 0, the second character is at index 1, and so on. If the index is out of range (less than 0 or greater than or equal to the string length), the method will return an empty string. This method can be particularly useful when you need to access individual characters of a string for manipulation or analysis.

def get_character_at_index(s, index):
    return s[index] if 0 <= index < len(s) else ''

full_string = 'Hello World'
index = 4
character = get_character_at_index(full_string, index)
print(character)  # Output: 'o'
def character_or_empty(s, index):
    return s[index] if 0 <= index < len(s) else 'Empty String'

my_string = 'Python'
index = 10
result = character_or_empty(my_string, index)
print(result)  # Output: 'Empty String'
def extract_first_character(input_string):
    return input_string[0] if len(input_string) > 0 else ''

sample_text = 'Data Science'
first_character = extract_first_character(sample_text)
print(first_character)  # Output: 'D'
