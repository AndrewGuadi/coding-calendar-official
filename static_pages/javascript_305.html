charAt

The `charAt()` method in JavaScript is used to return the character at a specified index position in a string. The index is zero-based, meaning that the first character is at index 0, the second character is at index 1, and so on. If the specified index is out of bounds (negative or greater than the string length), it will return an empty string.

string_1 = 'Hello, World!'

# Get the character at index 0
char_at_index_0 = string_1.charAt(0)
print(char_at_index_0)  # Output: H
string_2 = 'Python Programming'

# Get the character at index 7
char_at_index_7 = string_2.charAt(7)
print(char_at_index_7)  # Output: P
string_3 = 'OpenAI'

# Attempt to get the character at an out-of-bounds index (index 10)
char_at_index_10 = string_3.charAt(10)
print(char_at_index_10)  # Output: ''
