std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a `std::vector`. This method increases the size of the vector by one and stores the new element at the last position. If the vector's capacity is exceeded, a reallocation may occur to accommodate the new element. This operation typically runs in constant time, but may take longer when reallocating storage.

```cpp
#include 
#include 

int main() {
    std::vector words;
    words.push_back("hello");
    words.push_back("world");
    
    for (const auto &word : words) {
        std::cout << word << " "; // Output: hello world
    }
    return 0;
}
```
