uppercased

The 'uppercased' method in Swift is used to convert all the characters in a string to their uppercase equivalents. This method is particularly useful when you want to ensure that a string is treated in a case-insensitive manner or when formatting data for display purposes. The method returns a new string with all characters converted to uppercase, leaving the original string unchanged.

input_string = 'hello world'
uppercased_string = input_string.upper()
print(uppercased_string)  # Output: 'HELLO WORLD'
user_input = 'Python Programming'
formatted_input = user_input.upper()
print(formatted_input)  # Output: 'PYTHON PROGRAMMING'
sentence = 'Data Science is Great!'
uppercased_sentence = sentence.upper()
print(uppercased_sentence)  # Output: 'DATA SCIENCE IS GREAT!'
