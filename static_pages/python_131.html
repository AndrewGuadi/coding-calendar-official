all

The 'all' function in Python is a built-in function that takes an iterable (like a list, tuple, or set) and returns True if all elements in the iterable are truthy (i.e., evaluate to True). If the iterable is empty, 'all' also returns True. If any element in the iterable is falsy (i.e., evaluates to False), 'all' will return False.

numbers = [2, 4, 6, 8]
result = all(num % 2 == 0 for num in numbers)
# result is True, as all numbers in the list are even.
names = ['Alice', 'Bob', 'Charlie']
result = all(name.startswith('A') for name in names)
# result is False, as not all names start with 'A'.
conditions = [True, True, False]
result = all(conditions)
# result is False, as there is one False in the list.
