upcase

The `upcase` method in Ruby is a string method that converts all lowercase letters in a string to uppercase letters. It returns a new string with all the alphabetic characters converted. Characters that are already uppercase or are not alphabetic are unaffected. The original string remains unchanged, as strings in Ruby are immutable. This method does not take any arguments and is often used for normalizing input or preparing text for comparisons.

string = 'hello world'
upcased_string = string.upcase
# Output: 'HELLO WORLD'
input = 'Python Programming'
upper_case_input = input.upcase
# Output: 'PYTHON PROGRAMMING'
text = 'ruby rocks!'
uppercased_text = text.upcase
# Output: 'RUBY ROCKS!'
