strlen

The 'strlen' function in PHP is used to calculate the length of a string. It returns the number of bytes in the string, which is usually the number of characters for single-byte encodings. The function can handle empty strings and can also work correctly with multibyte character encodings as long as the proper encoding is used. If the input is not a string, it is converted to a string before measuring the length.

def get_string_length(input_string):
    return len(input_string)

# Example usage:
string_to_measure = 'Hello, World!'
length = get_string_length(string_to_measure)
print('Length:', length)
