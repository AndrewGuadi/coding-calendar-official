abs

In Java, the 'abs' method is a built-in function that is used to return the absolute value of a number. It can be applied to various numeric data types such as integers, floating-point numbers, and doubles. This method ensures that the returned value is always non-negative, effectively removing any negative sign from the input if present. For example, abs(-5) will return 5, while abs(5) will return 5 as well. This function is useful in mathematical computations where only the magnitude of a value is relevant, regardless of its sign.

def absolute_value_example():
    number = -10
    absolute_value = abs(number)
    print(f'The absolute value of {number} is {absolute_value}.') # Output: The absolute value of -10 is 10.
def list_absolute_values():
    numbers = [-1, 2, -3, 4]
    absolute_values = [abs(n) for n in numbers]
    print(f'Absolute values: {absolute_values}') # Output: Absolute values: [1, 2, 3, 4]
def distance_from_origin(x, y):
    return abs(x) + abs(y)

print(distance_from_origin(-5, 3)) # Output: 8
