upcase

The 'upcase' method in Ruby is used to convert all the characters in a string to uppercase. This is a useful method when you want to standardize string formats, especially when comparing strings or preparing them for display. It does not modify the original string but returns a new string with all the characters converted to uppercase.

def convert_to_uppercase(input_string):
    return input_string.upper()

# Usage
input_str = 'hello world'
output_str = convert_to_uppercase(input_str)
print(output_str)  # Output: 'HELLO WORLD'
def format_name(name):
    return name.upper()

# Usage
name = 'john doe'
formatted_name = format_name(name)
print(formatted_name)  # Output: 'JOHN DOE'
def shout_message(message):
    return message.upper() + '!!!'

# Usage
message = 'this is amazing'
shouted_message = shout_message(message)
print(shouted_message)  # Output: 'THIS IS AMAZING!!!'
