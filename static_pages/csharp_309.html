Add

The 'Add' method in C# is commonly used to add an element to a collection, such as a List or Dictionary. When using the 'Add' method, you specify the element you want to insert into the collection. This method modifies the collection by appending the new element to it, increasing its count by one. The 'Add' method is typically used in scenarios where you need to dynamically build up a collection of items during the execution of your program.

```python
# Example of using the Add method on a List in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List languages = new List();
        languages.Add("Python");
        languages.Add("C#");
        languages.Add("Java");

        foreach (var lang in languages) {
            Console.WriteLine(lang);
        }
    }
}
```
```python
# Example of using the Add method on a Dictionary in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary capitals = new Dictionary();
        capitals.Add("USA", "Washington, D.C.");
        capitals.Add("France", "Paris");
        capitals.Add("Japan", "Tokyo");

        foreach (var country in capitals) {
            Console.WriteLine($"The capital of {country.Key} is {country.Value}");
        }
    }
}
```
```python
# Example of using the Add method on a HashSet in C#

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet numbers = new HashSet();
        numbers.Add(1);
        numbers.Add(2);
        numbers.Add(3);
        numbers.Add(2); // Duplicate, won't be added

        Console.WriteLine("The unique numbers are:");
        foreach (var num in numbers) {
            Console.WriteLine(num);
        }
    }
}
```
