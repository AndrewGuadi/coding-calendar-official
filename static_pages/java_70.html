abs

The abs method in Java is used to calculate the absolute value of a number. The absolute value of a number is its non-negative value, meaning it removes any negative sign. The abs method is overloaded for different numeric types, including int, long, float, and double. It returns the absolute value of the input number passed to it. This method is useful for mathematical calculations where a positive value is needed regardless of the original input's sign.

def calculate_absolute_value():
    number = -10
    absolute_value = abs(number)
    print('The absolute value of', number, 'is', absolute_value)

calculate_absolute_value()
def get_positive_difference(a, b):
    difference = a - b
    return abs(difference)

result = get_positive_difference(5, 15)
print('The positive difference is', result)
def filter_negative_numbers(numbers):
    return [abs(num) for num in numbers]

negative_numbers = [-1, -2, -3, 4, 5]
positive_numbers = filter_negative_numbers(negative_numbers)
print('The positive numbers are', positive_numbers)
