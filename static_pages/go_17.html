append

In Go, the 'append' function is used to add elements to the end of a slice. It takes a slice and one or more elements to append and returns the updated slice. When appending to a slice, if the underlying array is full, a new array will be allocated, and the content will be copied to the new array. This method is commonly used to dynamically grow slices without needing to explicitly manage the capacity of the underlying array.

```
package main

import (
    "fmt"
)

func main() {
    // Initial slice
    words := []string{"
    ", "world"}
    // Append a new word
    words = append(words, "hello")
    // Append multiple words
    words = append(words, "from", "Go")

    // Print the result
    fmt.Println(words)
}
```

package main

func main() {
    // Create a slice of integers
    nums := []int{1, 2, 3}
    // Append more integers
    nums = append(nums, 4, 5, 6)

    // Print the updated slice
    fmt.Println(nums)
}
