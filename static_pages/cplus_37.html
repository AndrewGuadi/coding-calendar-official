std::vector::push_back

The 'push_back' method in C++'s std::vector class is used to add a new element to the end of the vector. As a dynamic array, a vector automatically resizes itself as needed, so you don't have to worry about capacity when adding new elements. The 'push_back' method takes a single argument, which is the value to be added to the vector. After the operation, the vector's size increases by one, and the new element is accessible at the index corresponding to the new size minus one (i.e., size() - 1).

```cpp
#include 
#include 

int main() {
    std::vector string_vector;
    string_vector.push_back("Python");
    string_vector.push_back("JavaScript");
    string_vector.push_back("C++");

    for (const auto& lang : string_vector) {
        std::cout << lang << '\n';
    }
    return 0;
}
``` This program initializes an empty vector of strings and adds "Python", "JavaScript", and "C++" to it using 'push_back'. It then prints each language to the console.
