std::vector::push_back

The method 'std::vector::push_back' is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one and copies the value provided into the new element's position. If the vector's current capacity is reached, the method automatically reallocates memory to accommodate the new element, ensuring that the vector can grow dynamically. This method does not return any value and operates in amortized constant time, making it efficient for adding elements to the end of the vector.

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Adding a string to the vector
    myVector.push_back("World"); // Adding another string

    // Printing the contents of the vector
    for (const auto& str : myVector) {
        std::cout << str << " "; 
    }
    return 0;
}
