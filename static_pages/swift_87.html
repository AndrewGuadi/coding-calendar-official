uppercased

The 'uppercased' method in Swift is used to convert a string to its uppercase equivalent. This method is part of the String type in Swift and can be very useful when you need to standardize user input, prepare strings for case-insensitive comparisons, or format output. The method does not modify the original string but returns a new string with all uppercase letters. The 'uppercased' method does not take any parameters.

def convert_to_uppercase(original_string):
    # Convert the original string to uppercase using the method
    uppercased_string = original_string.upper()
    return uppercased_string

# Usage
input_string = 'hello world'
result = convert_to_uppercase(input_string)
print(result)  # Output: 'HELLO WORLD'
def format_name(name):
    # Format the name to uppercase
    formatted_name = name.upper()
    return formatted_name

# Usage
user_name = 'john doe'
formatted_user_name = format_name(user_name)
print(formatted_user_name)  # Output: 'JOHN DOE'
def shout_message(message):
    # Shout the message by converting it to uppercase
    return message.upper()

# Usage
original_message = 'this is a warning!'
shouted_message = shout_message(original_message)
print(shouted_message)  # Output: 'THIS IS A WARNING!'
