append

The 'append' method in Go is used to add elements to the end of a slice. It takes a slice and one or more values to append and returns a new slice with the appended values. If the capacity of the slice is insufficient, 'append' automatically allocates a larger backing array. This method is typically used for dynamically extending slices without the need to predefine their size.

```go
package main
import (
	"fmt"
)
func main() {
	// Initialize a slice
	numbers := []int{1, 2, 3}
	// Append a single element
	numbers = append(numbers, 4)
	// Append multiple elements
	numbers = append(numbers, 5, 6, 7)
	// Print the slice
	fmt.Println(numbers) // Output: [1 2 3 4 5 6 7]
}
```
```go
package main
import (
	"fmt"
)
func main() {
	// Create a slice of strings
	fruits := []string{"apple", "banana"}
	// Append a single string
	fruits = append(fruits, "cherry")
	// Append multiple strings
	fruits = append(fruits, "date", "elderberry")
	// Print the slice
	fmt.Println(fruits) // Output: [apple banana cherry date elderberry]
}
```
```go
package main
import (
	"fmt"
)
func main() {
	// Initial slice of integers
	values := []int{10, 20}
	// Using 'append' to add more elements
	values = append(values, 30, 40, 50)
	// Print the elements
	for _, v := range values {
		fmt.Println(v) // Output: 10 20 30 40 50
	}
}
```
