upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. It does not modify the original string; instead, it returns a new string with the transformations applied. This method is helpful for standardizing string inputs or for making comparisons case-insensitive.

original_string = 'hello world'
uppercase_string = original_string.upcase
# Output: 'HELLO WORLD'
greeting = 'Good Morning'
shouting_greeting = greeting.upcase
# Output: 'GOOD MORNING'
title = 'the great gatsby'
uppercase_title = title.upcase
# Output: 'THE GREAT GATSBY'
