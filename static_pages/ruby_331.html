upcase

The 'upcase' method in Ruby is used to convert all the lowercase letters in a string into uppercase. It returns a new string with the changes applied, leaving the original string unchanged. This method can be particularly useful when you need to perform case-insensitive comparisons or display text in a specific format.

original_string = 'hello world' 
upcased_string = original_string.upcase 
puts upcased_string # => 'HELLO WORLD'
names = ['alice', 'bob', 'charlie'] 
upcased_names = names.map(&:upcase) 
puts upcased_names.inspect # => ['ALICE', 'BOB', 'CHARLIE']
greeting = 'Good Morning' 
if greeting.upcase == 'GOOD MORNING' 
  puts 'Greeting matches!' 
end # => 'Greeting matches!'
