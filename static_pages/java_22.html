abs

The abs() method in Java is used to return the absolute value of a given number. It can be applied to different types of numbers such as integers, floats, and doubles. This method effectively removes any negative sign from the value, returning a positive number. If the input number is zero, it will return zero as well. The abs() method is part of the java.lang.Math class and can be called statically.

```python
number = -10
absolute_value = abs(number)
print(absolute_value)  # Output: 10```
```python
number = 0
absolute_value = abs(number)
print(absolute_value)  # Output: 0```
```python
number = 4.5
absolute_value = abs(number)
print(absolute_value)  # Output: 4.5```
