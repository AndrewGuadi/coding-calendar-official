std::vector::push_back

The `push_back` method in C++ is used to add an element to the end of a `std::vector`. This method increases the size of the vector by one and is typically used when you want to dynamically add items to a vector without knowing its size at compile time. The time complexity of `push_back` is amortized O(1), which means that while it may occasionally take O(n) time (when reallocation of memory occurs), it generally operates in constant time. The `push_back` method accepts a single parameter which is the element to be added, and it can be of any data type that the vector is designed to hold.

# Example of using std::vector::push_back to collect data

#include 
#include 

int main() {
    std::vector python_codes;
    python_codes.push_back("print('Hello, World!')");
    python_codes.push_back("x = 5");
    python_codes.push_back("for i in range(5): print(i)");

    // Displaying the stored Python code snippets
    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    return 0;
}
# Another example of dynamically adding elements to a vector using push_back

#include 
#include 

int main() {
    std::vector snacks;

    // Adding items to the vector
    snacks.push_back("chips");
    snacks.push_back("cookies");
    snacks.push_back("candy");

    // Printing out the snacks
    std::cout << "List of Snacks:" << std::endl;
    for (size_t i = 0; i < snacks.size(); ++i) {
        std::cout << snacks[i] << std::endl;
    }
    return 0;
}
