append

The append function in Go is used to add elements to a slice, which is a dynamically-sized array. The function takes a slice as the first argument and one or more values to be appended as the subsequent arguments. It returns a new slice containing the original elements and the new elements. This method is efficient for increasing the size of slices as needed without manual memory management.

```go
package main

import "fmt"

func main() {
    original := []string{"python", "java"}
    updated := append(original, "c++", "ruby")
    fmt.Println(updated) // Output: [python java c++ ruby]
}
```
```go
package main

import "fmt"

func main() {
    numbers := []int{1, 2, 3}
    numbers = append(numbers, 4, 5)
    fmt.Println(numbers) // Output: [1 2 3 4 5]
}
```
