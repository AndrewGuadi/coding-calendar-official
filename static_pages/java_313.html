acos

The `acos` method in Java is a mathematical function that computes the arc cosine (inverse cosine) of a given value. The method takes a single argument, which must be in the range of -1.0 to 1.0. It returns the angle in radians between 0 and Ï€ (0 to 3.14159). If the provided argument is outside this range, the method will return NaN (Not a Number). This method is part of the `java.lang.Math` class.

import math

# Compute the arc cosine of 0.5
angle = math.acos(0.5)
# Convert from radians to degrees
angle_in_degrees = math.degrees(angle)
print(f'The angle whose cosine is 0.5 is {angle_in_degrees} degrees.')
