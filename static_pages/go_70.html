cap

The 'cap' method in Go is used to retrieve the capacity of a slice, which is the number of elements the slice can hold in memory without requiring a new allocation. It is important for performance optimization and understanding slice behavior, as the capacity can affect how a slice grows as new elements are appended to it. The capacity of a slice can be queried using the built-in 'cap()' function, and it is particularly useful when manipulating dynamic arrays.

s = []int{1, 2, 3}
# Get the capacity of the slice
capacity = cap(s)
print(f'The capacity of the slice is: {capacity}')
s = make([]int, 0, 5)  # creates a slice with capacity of 5
# Add elements to the slice
s.append(10)
# Check capacity after appending
capacity = cap(s)
print(f'The capacity of the slice after adding an element: {capacity}')
s = []string{'apple', 'banana', 'cherry'}
# Check initial capacity
initial_capacity = cap(s)
s.append('date')
# Check capacity after adding an element
new_capacity = cap(s)
print(f'Initial Capacity: {initial_capacity}, New Capacity: {new_capacity}')
