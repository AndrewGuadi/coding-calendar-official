std::vector::push_back

The method 'std::vector::push_back' is used to add a new element at the end of a vector in C++. This operation increases the size of the vector by one and copies or moves the provided element into the last position. If necessary, it will also handle memory reallocation to accommodate the newly added element, ensuring that the vector can dynamically grow. It's important to note that push_back has average constant time complexity (O(1)), but can be O(n) in cases where reallocation occurs.

```c++
#include 
#include 

int main() {
    std::vector python_examples;
    python_examples.push_back("print('Hello, World!')");
    python_examples.push_back("for i in range(5): print(i)");
    python_examples.push_back("def add(a, b): return a + b");

    for (const auto& example : python_examples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
