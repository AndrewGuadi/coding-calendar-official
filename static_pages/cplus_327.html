std::vector::push_back

The 'std::vector::push_back' method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one and initializes the new element with the value passed to the method. If the vector's storage is already full, it automatically resizes the vector to accommodate the new element. This method is essential for dynamically managing arrays in C++ since vectors are resizable arrays without the need for manual memory management.

```python
# Example of using std::vector::push_back to add elements to a vector
import ctypes

# Simulate the C++ vector functionality in Python using a list
class Vector:
    def __init__(self):
        self.elements = []

    def push_back(self, value):
        # Add value to the end of the vector
        self.elements.append(value)

# Create an instance of our Vector class
vector = Vector()

# Adding elements using push_back
vector.push_back('first')
vector.push_back('second')
vector.push_back('third')

# Checking the elements in the vector
print(vector.elements)  # Output: ['first', 'second', 'third']
```

```python
# Example of using std::vector::push_back with numeric values

# Simulate C++ vector functionality
class Vector:
    def __init__(self):
        self.elements = []

    def push_back(self, value):
        self.elements.append(value)

# Create an instance of Vector
vector = Vector()

# Adding integer elements

    vector.push_back(i)  # Using push_back to add i to the vector
```

```python
# Example of using std::vector::push_back with mixed-type elements

class Vector:
    def __init__(self):
        self.elements = []

    def push_back(self, value):
        self.elements.append(value)

# Create an instance of Vector
vector = Vector()

# Adding different types of elements
vector.push_back(42)            # Integer
vector.push_back('hello')       # String
vector.push_back(3.14)          # Float
vector.push_back(True)          # Boolean

# Displaying the current elements in the vector
print(vector.elements)  # Output: [42, 'hello', 3.14, True]
```
