Add

The 'Add' method is commonly used to insert an element into a collection, such as a List or Dictionary, in C#. This method typically takes a single parameter representing the item to be added and updates the collection accordingly. It is essential to ensure that the collection can accommodate the new item, as attempting to add to a full collection (like an array with fixed size) will result in an exception.

```python
# This example demonstrates how to use the Add method with a List in C#.

# Create a List to hold strings
my_list = []

# Add strings to the List
my_list.append('Hello')
my_list.append('World')

# Display the List
print(my_list)  # Output: ['Hello', 'World']
```
```python
# This example shows how to use the Add method with a Dictionary in C#.

# Create a dictionary to hold key-value pairs
my_dict = {}

# Add key-value pairs to the Dictionary
my_dict['key1'] = 'Value1'
my_dict['key2'] = 'Value2'

# Display the Dictionary
print(my_dict)  # Output: {'key1': 'Value1', 'key2': 'Value2'}
```
```python
# This example illustrates how to use the Add method with a HashSet in C#.

# Create a set to hold unique elements
my_set = set()

# Add unique elements to the set
my_set.add('Element1')
my_set.add('Element2')

# Display the set
print(my_set)  # Output: {'Element1', 'Element2'}
```
