std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class, which allows you to add a new element to the end of the vector. This method increases the size of the vector by one and initializes the added element to the specified value. It is widely used for dynamic array manipulation, allowing for the efficient addition of elements as needed.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    
    // Push back 'hello' and 'world'
    myVector.push_back("hello");
    myVector.push_back("world");
    
    // Output the vector contents
    for (const auto &s : myVector) {
        std::cout << s << " ";
    }
    
    return 0;
}
```
