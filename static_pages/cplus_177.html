std::vector::push_back

The std::vector::push_back method is a member function of the std::vector class in C++, which adds a new element to the end of the vector. It automatically resizes the vector if necessary to accommodate the new element. This function is commonly used when you want to append items to a dynamic array. It does not return any value and its time complexity is typically O(1), though it may be O(n) in the case of resizing. This method is particularly useful in scenarios where the size of the data structure is not known at compile time and must be modified during runtime.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& item : myVector) {
        std::cout << item << " ";
    }
    return 0;
}
```
