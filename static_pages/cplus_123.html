std::vector::push_back

The 'push_back' method is a member function of the std::vector class in C++. It is used to add a new element to the end of the vector. When an element is added, the size of the vector increases by one. If the vector's storage capacity is insufficient to hold the new element, it will automatically allocate more memory to accommodate it. The complexity of this operation is amortized constant time, meaning that while individual calls can be linear in the worst case, most of the time it will be constant time. This method can be used to build a vector dynamically, especially useful when the number of elements is not known at compile time.

```cpp
#include 
#include 

int main() {
    std::vector stringVector;
    
    // Adding elements using push_back
    stringVector.push_back("Hello");
    stringVector.push_back("World");
    stringVector.push_back("from");
    stringVector.push_back("C++");
    
    // Displaying the contents of the vector
    for (const std::string& str : stringVector) {
        std::cout << str << " ";
    }
    return 0;
}
```
