Add

The 'Add' method is commonly used in various collections in C#, such as lists, dictionaries, and sets. This method typically allows you to add an element to a collection. For example, in a List, the Add method appends a new element to the end of the list, while in a Dictionary, it adds a new key-value pair. The method can also raise exceptions if the collection has constraints (like capacity limits or key uniqueness in dictionaries).

my_list = []  # Create an empty list
my_list.append('print("Hello, World!")')  # Using 'Add' in a list context
print(''.join(my_list))  # Output: print("Hello, World!")
my_dict = {}  # Create an empty dictionary
my_dict['key1'] = 'print("Goodbye, World!")'  # Using 'Add' in a dictionary context
exec(my_dict['key1'])  # Execute the code stored in the dictionary, Output: Goodbye, World!
my_set = set()  # Create an empty set
my_set.add('print("Hello from set!")')  # Using 'Add' in a set context
for code in my_set:
    exec(code)  # Execute the code stored in the set, Output: Hello from set!
