uppercased

In Swift, the 'uppercased' method is used to convert all characters in a string to their uppercase equivalent. This method is a part of the String type and returns a new string with all characters converted to uppercase. This can be particularly useful when normalizing user input or preparing strings for display in a case-insensitive manner.

def convert_to_uppercase(text):
    return text.upper()

text = 'hello world'
print(convert_to_uppercase(text))  # Output: 'HELLO WORLD'
def shout(text):
   return text.upper() + '!!!'

message = 'make it loud'
print(shout(message))  # Output: 'MAKE IT LOUD!!!'
def filter_uppercase_words(words):
    return [word for word in words if word.upper() == word]

word_list = ['Hello', 'WORLD', 'example', 'TEST']
print(filter_uppercase_words(word_list))  # Output: ['WORLD', 'TEST']
