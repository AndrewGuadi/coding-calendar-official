std::vector::push_back

The 'push_back' method is a member function of the 'std::vector' class in C++. It is used to add a new element at the end of the vector, which automatically increases the vector's size by one. This function allocates memory if necessary, and it accepts a value as an argument, which is copied into the vector. If the vector needs more capacity, it will typically allocate additional memory to accommodate future elements. It is a convenient way to grow a vector dynamically during runtime.

#include 
#include 

int main() {
    std::vector fruits;
    
    // Adding elements using push_back
    fruits.push_back("Apple");
    fruits.push_back("Banana");
    fruits.push_back("Cherry");
    
    // Displaying vector elements
    for (const auto &fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}
