Add

The 'Add' method is used to insert a new element into a collection, such as a list, a dictionary, or a set. This method typically takes an item as a parameter and adds that item to the collection, allowing for dynamic growth and management of the collection's elements. The behavior can vary depending on the type of collection being used. In C#, the 'Add' method is commonly found in collections like List, Dictionary, and HashSet.

```python
# Example of using Add with a List
my_list = []
my_list.append('Python code example 1')
my_list.append('Python code example 2')
print(my_list)  # Output: ['Python code example 1', 'Python code example 2']
```
```python
# Example of using Add with a Dictionary
my_dict = {}
my_dict['key1'] = 'Python value 1'
my_dict['key2'] = 'Python value 2'
print(my_dict)  # Output: {'key1': 'Python value 1', 'key2': 'Python value 2'}
```
```python
# Example of using Add with a Set
my_set = set()
my_set.add('Python set value 1')
my_set.add('Python set value 2')
print(my_set)  # Output: {'Python set value 1', 'Python set value 2'}
```
