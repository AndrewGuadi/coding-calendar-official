strlen

The `strlen()` function in PHP is used to obtain the length of a given string. It counts the number of characters in the string, including spaces and special characters, and returns it as an integer value. This method is particularly useful when you need to validate inputs, manipulate string data, or understand the size of data being handled. Note that `strlen` does not count UTF-8 multibyte characters accurately; for multibyte character sets, consider using `mb_strlen()` instead.

def count_length_of_string(s):
    return len(s)

string = 'Hello, World!'
length = count_length_of_string(string)
print(f'The length of the string is: {length}')
def is_string_longer_than_min(s, min_length):
    return len(s) > min_length

string = 'Python programming'
min_length = 10
result = is_string_longer_than_min(string, min_length)
print(f'Is the string longer than {min_length} characters? {result}')
def get_string_lengths(strings):
    return [len(s) for s in strings]

strings = ['apple', 'banana', 'cherry']
lengths = get_string_lengths(strings)
print(f'Lengths of the strings are: {lengths}')
