std::vector::push_back

The `push_back` method is used to add a new element at the end of a vector in C++. This method increases the size of the vector by one and initializes the added element with the value provided as the parameter. If the vector's capacity is reached, it will automatically allocate more memory to accommodate the new element. This is a commonly used method for dynamically building an array-like structure where the size can change during runtime.

```c++
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Hello");
    vec.push_back("World");

    for(const auto& str : vec) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}
```
