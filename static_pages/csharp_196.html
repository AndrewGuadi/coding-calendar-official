Add

The 'Add' method in C# is commonly used to add an element to a collection, such as a list, dictionary, or other data structures. Depending on the type of collection, this method appends the element at the end of the collection or inserts it at a specific index. This method is vital in managing dynamic collections where the count of elements isnâ€™t predetermined. In the context of a List, for example, it increases the size of the list by one as it adds a new element.

list = []
list.append('print("Hello, World!")')
print(exec(list[0]))  # Outputs: Hello, World!
dictionary = {}
dictionary['key1'] = 'print("Hello from key1!")'
exec(dictionary['key1'])  # Outputs: Hello from key1!
set_of_code = set()
set_of_code.add('print("Hello from the set!")')
for code in set_of_code:
    exec(code)  # Outputs: Hello from the set!
