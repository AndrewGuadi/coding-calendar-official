Add

The 'Add' method is used to add an element to a collection, such as a list, dictionary, or set in C#. When this method is called on a specific collection type, it appends the specified element to the end of the collection (for lists) or adds the key-value pair (for dictionaries), or simply adds the item (for sets). The method typically does not return any value (void) and may throw exceptions if the collection is read-only or if an invalid operation is attempted.

```python
# Example of adding a number to a list in C#
my_list = []  # initializing an empty list
# Using Add method to add an element
my_list.Add(1)
my_list.Add(2)
my_list.Add(3)
print(my_list)  # Output: [1, 2, 3]
```
```python
# Example of adding a key-value pair to a dictionary in C#
my_dict = {}
# Using Add method to add a key-value pair
my_dict.Add('a', 1)
my_dict.Add('b', 2)
print(my_dict)  # Output: {'a': 1, 'b': 2}
```
```python
# Example of adding an element to a set in C#
my_set = set()
# Using Add method to add an element
my_set.Add(10)
my_set.Add(20)
my_set.Add(10)  # This won't add another '10'
print(my_set)  # Output: {10, 20}
```
