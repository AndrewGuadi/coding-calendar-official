Any

The 'Any' method in C# is a LINQ extension method that determines whether any elements in a collection satisfy a specified condition or whether any elements exist in a collection. This method returns a boolean value: true if any of the elements match the condition or if the collection is not empty, and false otherwise.

fruits = ['apple', 'banana', 'cherry']
any_fruit_start_with_a = any(fruit.startswith('a') for fruit in fruits)  # Returns True
numbers = [1, 2, 3, 4, 5]
any_even_numbers = any(number % 2 == 0 for number in numbers)  # Returns True
mixed_data = [1, 2, 3, 'a', 'b']
any_strings = any(isinstance(item, str) for item in mixed_data)  # Returns True
