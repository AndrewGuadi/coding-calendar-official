uppercased

The 'uppercased' method is a built-in function in Swift that transforms a given string into its uppercase equivalent. This method is commonly used when the need arises to ensure that strings are in a consistent format for comparison, display, or processing, particularly in cases where case sensitivity might affect outcomes. The function does not mutate the original string but instead returns a new string that is entirely in uppercase letters.

original_string = 'hello world'
uppercased_string = original_string.uppercased()  # Result: 'HELLO WORLD'
name = 'Alice'
uppercased_name = name.uppercased()  # Result: 'ALICE'
phrase = 'this is a Test'
uppercased_phrase = phrase.uppercased()  # Result: 'THIS IS A TEST'
