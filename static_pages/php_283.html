strlen

The strlen() function in PHP is used to calculate the length of a string, returning the total number of characters contained in that string. This includes all types of characters such as letters, numbers, symbols, and whitespace. It is commonly used in various programming tasks where string manipulation is necessary, such as validating input lengths or generating messages based on string size. The function returns an integer representing the length of the given string.

def example_strlen_1():
    my_string = 'Hello, World!'
    length = len(my_string)
    print('The length of the string is:', length)

example_strlen_1()
def example_strlen_2():
    user_input = input('Enter a string: ')
    length = len(user_input)
    print('You entered a string of length:', length)

example_strlen_2()
def example_strlen_3():
    empty_string = ''
    length = len(empty_string)
    if length == 0:
        print('The string is empty.')
    else:
        print('The string has a length of:', length)

example_strlen_3()
