std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. This method increases the size of the vector by one, and copies the provided value to the new element. If the vector's capacity is exceeded, it will automatically allocate more memory to accommodate the new element. It is important to note that the value being added does not have to be of a type that is already in the vector. This method is a member function of the std::vector class, and it avoids the manual memory management typically associated with dynamic arrays in C++. The operation has an average time complexity of O(1), though it can be O(n) in the case of reallocation.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Python");
    vec.push_back("Java");
    vec.push_back("C++");

    // Displaying the elements in the vector
    for (const auto &lang : vec) {
        std::cout << lang << " ";
    }
    return 0;
}
```

```cpp
int main() {
    std::vector languages;

    languages.push_back("C#");
    languages.push_back("PHP");
    languages.push_back("Swift");

    // Using push_back in a loop to add multiple items
    std::string additionalLanguages[] = {"Kotlin", "Rust", "Scala"};
    for (const auto &lang : additionalLanguages) {
        languages.push_back(lang);
    }

    // Displaying all languages
    for (const auto &lang : languages) {
        std::cout << lang << " ";
    }
    return 0;
}
```
