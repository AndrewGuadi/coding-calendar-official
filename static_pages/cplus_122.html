std::vector::push_back

The `push_back` method in C++ is used to add an element to the end of a vector, which is a dynamic array provided by the Standard Template Library (STL). This method increases the size of the vector by one and initializes the new element to the value passed as an argument. If the capacity of the vector is exceeded, it will automatically allocate more memory to accommodate the new element. This method is typically used when you want to build a vector by appending elements sequentially.

```cpp
#include 
#include 

int main() {
    std::vector python_codes;
    python_codes.push_back("print('Hello, World!')");
    python_codes.push_back("for i in range(5): print(i)");
    python_codes.push_back("def add(a, b): return a + b");
    
    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    
    return 0;
}
```
