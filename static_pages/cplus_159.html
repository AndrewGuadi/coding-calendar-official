std::vector::push_back

The `push_back` method in C++ adds a new element to the end of a std::vector. This method increases the size of the vector by one, and the new element is initialized to the value provided in the method call. If the vector needs to reallocate memory to accommodate the new element, it will do so automatically. It's important to understand that using `push_back` might lead to performance implications when called multiple times in a loop, as the vector may require several reallocations. To avoid performance overhead, itâ€™s often better to use `reserve` to allocate enough memory ahead of time when the number of elements to be added is known.`
