charAt

The charAt() method in JavaScript returns the character at a specified index (position) in a string. The index starts from 0, meaning that the first character is at index 0, the second character at index 1, and so on. If the specified index is out of range, charAt() returns an empty string. This method is useful for retrieving a specific character from a string when the exact position of that character is known.

my_string = 'Hello, world!'

char_at_seven = my_string.charAt(7)
# This will return 'w', which is the 7th character in 'Hello, world!'.
my_string = 'JavaScript is fun!'

first_character = my_string.charAt(0)
# This will return 'J', which is the first character in 'JavaScript is fun!'.
my_string = 'OpenAI'

last_character = my_string.charAt(5)
# This will return 'I', which is the last character in 'OpenAI' since it is a 6-letter string.
