ascii

The `ascii()` function in Python returns a string containing a printable representation of an object. This representation is similar to using `repr()`, but it escapes non-ASCII characters so that the output can be safely encoded in ASCII. This is especially useful for debugging or logging purposes when you want to see the string representation of an object that may contain non-ASCII characters, while ensuring that the output is in a compatible format.

data = {'name': 'Alice', 'age': 30, 'city': 'New York'}
result = ascii(data)
print(result)  # Output: {'name': 'Alice', 'age': 30, 'city': 'New York'}
