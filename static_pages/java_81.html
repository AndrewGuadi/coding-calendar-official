abs

The `abs` method in Java is used to return the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, regardless of direction. This method can handle various data types: `int`, `long`, `float`, and `double`. When the method is called, it removes any negative sign from the input number, resulting in a non-negative value.

num = -5
result = abs(num)
print(result)  # Output will be 5
num = 3.14
result = abs(num)
print(result)  # Output will be 3.14
num = -42
result = abs(num)
print(result)  # Output will be 42
