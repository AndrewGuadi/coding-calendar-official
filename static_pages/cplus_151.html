std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This method automatically increases the size of the vector to accommodate the new element. It is a member function of the `std::vector` class, which is part of the C++ Standard Library. The `push_back` operation is generally efficient and allows dynamic resizing of the underlying array that the vector uses to store its elements. However, in some cases, it may require allocating new memory if the current capacity is exceeded. This function does not return a value.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& str : myVector) {
        std::cout << str << " ";
    }
    return 0;
}
``` In this example, we create a vector of strings, add two values ('Hello' and 'World') to the vector using `push_back`, and then print them out.

#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i);
    }

    std::cout << "Numbers in vector: ";
    for (const auto& number : numbers) {
        std::cout << number << " ";
    }
    return 0;
}
``` In this example, we populate a vector with integers from 1 to 5 using `push_back` in a loop and then print them out.
