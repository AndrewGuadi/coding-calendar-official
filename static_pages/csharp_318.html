Add

The 'Add' method is a commonly used function in C# collections such as lists, dictionaries, and sets. It is used to add a new item to the collection. For instance, if you have a list of integers, using the 'Add' method allows you to insert an integer at the end of the list. The method does not return a value and typically modifies the collection in place. If the collection is full or immutable, it may throw an exception.

```python
# Example of adding elements to a list in Python
my_list = [1, 2, 3]
my_list.append(4)  # Equivalent to Add in C#
print(my_list)  # Output: [1, 2, 3, 4]```
```python
# Example of adding key-value pairs to a dictionary in Python
dictionary = {'a': 1, 'b': 2}
dictionary['c'] = 3  # Adding new key-value pair, similar to Add in C#
print(dictionary)  # Output: {'a': 1, 'b': 2, 'c': 3}```
```python
# Example of adding items to a set in Python
my_set = {1, 2, 3}
my_set.add(4)  # Similar to Add method in C# for a set
print(my_set)  # Output: {1, 2, 3, 4}```
