Add

The 'Add' method in C# is commonly used in collections to add a new element to a collection such as lists, dictionaries, or sets. It appends the specified value to the collection, increasing its size by one. This method is essential in data structures for managing collections of objects, allowing dynamic modification of contents. For example, in a List, the Add method can be used to insert a new item at the end of the list.

my_list = []
my_list.append('python')
print(my_list)  # Output: ['python']
my_dict = {}
my_dict['key'] = 'python'
print(my_dict)  # Output: {'key': 'python'}
my_set = set()
my_set.add('python')
print(my_set)  # Output: {'python'}
