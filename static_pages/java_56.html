abs

The 'abs' method in Java is used to return the absolute value of a given number. It can be applied to integers, floating-point numbers, and long integers. The absolute value of a number is defined as the non-negative value of that number without regard to its sign. If the number is positive, the absolute value is the number itself. If the number is negative, the absolute value is the opposite of the number (i.e., the negation of its value). This method is useful in various mathematical calculations where the direction does not matter and only the magnitude of the number is important.

def demonstrate_abs_function():
    num1 = -10
    num2 = 5
    print('Absolute value of', num1, 'is', abs(num1))
    print('Absolute value of', num2, 'is', abs(num2))

demonstrate_abs_function()
def calculate_distance(x1, y1, x2, y2):
    distance_x = abs(x2 - x1)
    distance_y = abs(y2 - y1)
    return distance_x + distance_y

print('Distance:', calculate_distance(-5, -5, 5, 5))
def temperature_difference(temp1, temp2):
    return abs(temp1 - temp2)

print('Temperature Difference:', temperature_difference(30, 20))
print('Temperature Difference:', temperature_difference(20, 30))
