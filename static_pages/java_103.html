abs

The `abs` method in Java is used to calculate the absolute value of a given number. It is a part of the `java.lang.Math` class and can accept various numeric types, including integers, floats, doubles, and long integers. This method returns the non-negative value of the input number, meaning if the input is negative, it will convert it to positive, and if positive or zero, it will return the number unchanged.

result = Math.abs(-5)  # result will be 5
result = Math.abs(3.14)  # result will be 3.14
result = Math.abs(-10.5)  # result will be 10.5
