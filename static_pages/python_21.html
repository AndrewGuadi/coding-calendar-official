abs

The `abs` method in Python is a built-in function that returns the absolute value of a given number. The absolute value of a number is its distance from zero on the number line, regardless of direction. This function can handle integers, floating-point numbers, and complex numbers. If a complex number is provided, it returns its magnitude.

```python
# Example 1: Using abs with an integer
number = -10
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
# Output: The absolute value of -10 is 10
```
```python
# Example 2: Using abs with a floating-point number
floating_number = -3.14
absolute_value = abs(floating_number)
print('The absolute value of', floating_number, 'is', absolute_value)
# Output: The absolute value of -3.14 is 3.14
```
```python
# Example 3: Using abs with a complex number
complex_number = 3 - 4j
absolute_value = abs(complex_number)
print('The magnitude of', complex_number, 'is', absolute_value)
# Output: The magnitude of (3-4j) is 5.0
```
