str_replace

The 'str_replace' function in PHP is used to replace all occurrences of a given search string with a replacement string in a specified input string. It can be particularly useful for modifying text data, such as user inputs, configuration values, or any other string manipulation tasks. The function is case-sensitive, meaning it will differentiate between uppercase and lowercase letters in the search string. It can also handle arrays as both the search and replacement parameters, allowing for multiple substitutions in one call.

original_string = 'The quick brown fox jumps over the lazy dog.'
search = 'fox'
replace = 'cat'
result = str_replace(search, replace, original_string)
# Output: 'The quick brown cat jumps over the lazy dog.'
original_string = 'I like to code in Python. Python is great!'
search = 'Python'
replace = 'PHP'
result = str_replace(search, replace, original_string)
# Output: 'I like to code in PHP. PHP is great!'
original_string = 'Hello World!'
search = ['Hello', 'World']
replace = ['Hi', 'Everyone']
result = str_replace(search, replace, original_string)
# Output: 'Hi Everyone!'
