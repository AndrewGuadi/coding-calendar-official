Add

The 'Add' method in C# is commonly used to add elements to a collection, such as a list or a dictionary. This method typically accepts an individual item or key-value pair and appends it to the appropriate data structure. The specific implementation can vary depending on the collection type and its underlying data structure. For example, the 'List' collection uses 'Add' to append an item to the end, while in a 'Dictionary', 'Add' is used to insert a new key-value pair. Using this method is essential for managing collections dynamically during the runtime.

```python
# Using a list to store values
my_list = []

# Function to add an item to the list
def add_item(item):
    my_list.append(item)

# Adding items
add_item('apple')
add_item('banana')
print(my_list)  # Output: ['apple', 'banana']
```
```python
# Using a dictionary to store key-value pairs
my_dict = {}

# Function to add key-value pairs to the dictionary
def add_entry(key, value):
    my_dict[key] = value

# Adding entries
add_entry('name', 'John Doe')
add_entry('age', 30)
print(my_dict)  # Output: {'name': 'John Doe', 'age': 30}
```
```python
# Using a set to store unique items
my_set = set()

# Function to add an item to the set
def add_unique_item(item):
    my_set.add(item)

# Adding items
add_unique_item('red')
add_unique_item('blue')
add_unique_item('red')  # Duplicate, won't be added
print(my_set)  # Output: {'red', 'blue'}
```
