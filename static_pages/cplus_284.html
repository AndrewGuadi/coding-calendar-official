std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This method increases the size of the vector by one and initializes the new element with the specified value. The complexity of `push_back` is amortized O(1), meaning that in most cases, it operates in constant time, although occasionally it may require reallocation and copying of elements when the vector capacity is exceeded.

```cpp
#include 
#include 

int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");

    for (const auto &fruit : fruits) {
        std::cout << fruit << '\n';
    }
    return 0;
}
```
