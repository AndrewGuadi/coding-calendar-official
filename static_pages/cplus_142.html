std::vector::push_back

The std::vector::push_back method in C++ is used to add an element to the end of a vector. A vector is a dynamic array that allows for the storage and retrieval of a collection of elements. The push_back method increases the size of the vector by one and initializes the new element with the value provided. This method allows vectors to dynamically store an increasing number of elements without needing to specify an initial size, as the vector will handle memory allocation internally. It is important to note that if the current capacity of the vector is insufficient to accommodate the new element, it will automatically resize the vector to ensure that it can hold the new amount of data.

```python
my_vector = []
my_vector.append('apple')
my_vector.append('banana')
my_vector.append('cherry')
print(my_vector)  # Output will be ['apple', 'banana', 'cherry']
```
```python
my_vector = []
for fruit in ['grape', 'orange', 'kiwi']:
    my_vector.append(fruit)
print(my_vector)  # Output will be ['grape', 'orange', 'kiwi']
```
```python
my_vector = []
items = ['car', 'bike', 'bus']
for item in items:
    my_vector.append(item)

if 'train' not in my_vector:
    my_vector.append('train')
print(my_vector)  # Output will be ['car', 'bike', 'bus', 'train']
```
