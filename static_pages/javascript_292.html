charAt

The `charAt` method in JavaScript is used to retrieve the character at a specified index in a string. If the index is out of bounds, it returns an empty string. This method is particularly useful for accessing individual characters in a string for various string manipulation tasks.

```python
# Example: Accessing the first character of the string
my_string = 'Hello, World!'
first_character = my_string[0]  # Output: 'H'
```
```python
# Example: Accessing a character at a specific index
my_string = 'Hello, World!'
character_at_index_7 = my_string[7]  # Output: 'W'
```
```python
# Example: Attempting to access a character out of bounds
my_string = 'Hello'
out_of_bounds_character = my_string[10] if len(my_string) > 10 else ''  # Output: '' (an empty string)
```
