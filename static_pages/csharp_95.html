Add

The 'Add' method in C# is typically used to insert an item into a collection, such as a list, dictionary, or any other collection that allows adding elements. This method is defined within various generic and non-generic collections in the .NET framework. The 'Add' method modifies the collection by appending the new item to it, thus increasing its size by one. It's a straightforward operation that usually results in O(1) time complexity for most collections (e.g. List, HashSet). However, the complexity might vary depending on the underlying data structure and the conditions of the collection (like resizing) when adding elements.

```python
my_list = []
my_list.append('Hello')
my_list.append('World')
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}
my_dict['key1'] = 'value1'
my_dict['key2'] = 'value2'
print(my_dict)  # Output: {'key1': 'value1', 'key2': 'value2'}
```
```python
class CustomCollection:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

collection = CustomCollection()
collection.add('Item1')
collection.add('Item2')
print(collection.items)  # Output: ['Item1', 'Item2']
```
