ascii

The 'ascii' function is used to return a string representation of an object, escaping non-ASCII characters. It is mainly used to ensure that the output contains only printable ASCII characters. In Python, it is often used to convert non-ASCII characters to a readable format. This function typically works well with strings, lists, and dictionaries, where any non-ASCII characters are replaced with their escape sequences.

text = 'Hello, мир!'  # A string with non-ASCII characters
ascii_text = ascii(text)  # Converts non-ASCII characters to escape sequences
print(ascii_text)  # Output: 'Hello, \u043c\u0438\u0440!'
data = {'name': 'José', 'age': 30}  # Dictionary containing non-ASCII character
ascii_data = ascii(data)  # Converts non-ASCII characters to escape sequences
print(ascii_data)  # Output: "{'name': 'Jos\u00e9', 'age': 30}"
characters = ['a', 'b', '©', 'd']  # List with a non-ASCII character
ascii_characters = [ascii(char) for char in characters]  # Convert each character using ascii
print(ascii_characters)  # Output: ['a', 'b', '\xa9', 'd']
