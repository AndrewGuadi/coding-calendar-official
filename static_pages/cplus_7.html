std::vector::push_back

The method 'push_back' is a member function of the C++ Standard Library's 'std::vector' class. It is used to add a new element to the end of the vector, thereby increasing its size by one. The element is copied or moved into the vector, and if there is not enough space in the current memory allocation, it automatically reallocates memory to accommodate the new element and maintain continuity of storage. This function can be particularly useful for dynamically building a vector as data becomes available at runtime.

```cpp
#include 
#include 

int main() {
    std::vector python_codes;
    
    // Adding Python code snippets to the vector
    python_codes.push_back("print('Hello, World!')");
    python_codes.push_back("for i in range(5): print(i)");
    python_codes.push_back("def greet(name): return 'Hello ' + name");
    
    // Display vector contents
    for (const auto& code : python_codes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```

```cpp
        std::cout << "Script " << i+1 << ": " << python_scripts[i] << std::endl;
    }
    return 0;
}
```
