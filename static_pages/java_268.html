acos

The `acos` method in Java is a static method of the `java.lang.Math` class that returns the arc cosine (inverse cosine) of a given number. The input value should be in the range of -1.0 to 1.0. The method returns the angle in radians. If the argument is outside this range, it will return NaN, indicating that the result is not a number (not a valid result). This method is useful in fields such as mathematics, physics, and computer graphics where angle calculations are necessary.

import math

# Calculate the arc cosine of 0.5
result = math.acos(0.5)
print('The arc cosine of 0.5 is:', result)
