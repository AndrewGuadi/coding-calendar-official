std::vector::push_back

The `push_back` method in C++ standard library's vector class adds a new element to the end of the vector. This method increases the size of the vector by one, and the new element is initialized with the value passed to the method. If necessary, the vector will dynamically allocate more memory to accommodate this new element. It allows for efficient management of elements in a dynamic array, making it a fundamental part of the vector's functionality.

```cpp
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("first");
    vec.push_back("second");
    vec.push_back("third");

    for (const auto& str : vec) {
        std::cout << str << " ";
    }
    return 0;
}
``` The example demonstrates creating a vector of strings, adding elements using `push_back`, and then iterating through the vector to print the elements.
```cpp
#include 
#include 

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i);
    }

    std::cout << "Numbers in the vector: ";
    for (const auto& num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}
``` This example uses a loop to push integers from 1 to 5 into a vector and then prints the contents of the vector.
