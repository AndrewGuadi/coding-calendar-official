Add

The 'Add' method is commonly used in C# collections, such as lists, dictionaries, or sets, to add an element to the data structure. When the 'Add' method is called on a collection, it typically inserts the specified element into the collection, increasing its size and making the new element accessible for future retrievals or manipulations.

list_example = []
list_example.append('Hello, world!')
print(list_example)  # Output: ['Hello, world!']
dict_example = {}
dict_example['key'] = 'value'
print(dict_example)  # Output: {'key': 'value'}
set_example = set()
set_example.add('item1')
set_example.add('item2')
print(set_example)  # Output: {'item1', 'item2'}}
