Add

The 'Add' method is commonly used in various collection types in C#, such as List and Dictionary. This method adds a specified element to the collection. For lists, it appends the element to the end of the list, while for dictionaries, it adds a key-value pair. It's essential to ensure that the key does not already exist in the dictionary, as this will result in an exception.

```python
# Example of using Add method with a list in C#

# Create a list of strings
List items = new List();

# Add items to the list
items.Add("apple");
items.Add("banana");
items.Add("cherry");

# Print all items in the list
foreach (string item in items) 
{
    Console.WriteLine(item);
}
```
```python
# Example of using Add method with a dictionary in C#

// Create a dictionary to store fruit prices
dictionary fruitPrices = new Dictionary();

// Add key-value pairs using Add method
fruitPrices.Add("apple", 0.60m);
fruitPrices.Add("banana", 0.50m);
fruitPrices.Add("cherry", 1.75m);

// Print out the fruit prices
foreach (var fruit in fruitPrices) 
{
    Console.WriteLine($"{fruit.Key}: {fruit.Value}");
}
```
```python
# Example of using Add method in a custom class collection in C#

// Define a class to represent a person
class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

// Create a list to hold persons
List people = new List();

// Add new persons using Add method
people.Add(new Person { Name = "John", Age = 30 });
people.Add(new Person { Name = "Jane", Age = 25 });
people.Add(new Person { Name = "Doe", Age = 40 });

// Print all persons
foreach (var person in people) 
{
    Console.WriteLine($"{person.Name}, Age: {person.Age}");
}
```
