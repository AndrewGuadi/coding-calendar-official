Add

The 'Add' method is often used to insert elements into a collection such as a list, dictionary, or set in C#. In C#, the 'Add' method can be called on various collections like List, Dictionary, HashSet, etc. This method appends a new element to the collection. If the collection does not allow duplicates (like HashSet or Dictionary), trying to add an element that already exists will result in an error or fail silently, depending on the collection type. The 'Add' method does not return a value; it modifies the collection in place.

```python
my_list = []  # Initialize an empty list
my_list.append('Hello')  # Add 'Hello' to the list
my_list.append('World')  # Add 'World' to the list
print(my_list)  # Output: ['Hello', 'World']
```
```python
my_dict = {}  # Initialize an empty dictionary
my_dict['name'] = 'Alice'  # Add key 'name' with value 'Alice'
my_dict['age'] = 30  # Add key 'age' with value 30
print(my_dict)  # Output: {'name': 'Alice', 'age': 30}
```
```python
my_set = set()  # Initialize an empty set
my_set.add('Python')  # Add 'Python' to the set
my_set.add('C#')  # Add 'C#' to the set
print(my_set)  # Output: {'C#', 'Python'}
```
