Add

The 'Add' method is utilized in various collections in C#, such as List, Dictionary, and others, to insert an element or a key-value pair into the collection. When used in a List, it appends the specified item at the end of the list, while in a Dictionary, it allows for the addition of a key-value pair, where the key must be unique. The method typically does not return a value and modifies the collection in place.

```python
# Adding elements to a list
my_list = []
my_list.append('Python')
my_list.append('JavaScript')
print(my_list)  # ['Python', 'JavaScript']
```
```python
# Adding key-value pairs to a dictionary
my_dict = {}
my_dict['language'] = 'Python'
my_dict['framework'] = 'Flask'
print(my_dict)  # {'language': 'Python', 'framework': 'Flask'}
```
```python
# Adding multiple elements to a list in a loop
my_list = []
for i in range(5):
    my_list.append(f'Item {i}')
print(my_list)  # ['Item 0', 'Item 1', 'Item 2', 'Item 3', 'Item 4']
```
