std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This dynamic array allows elements to be added without worrying about the underlying memory management, as the vector will automatically handle resizing as needed. The method modifies the vector by increasing its size by one, and the new element is inserted at the position defined by the new size (i.e., the last index of the vector). The time complexity of the `push_back` operation is amortized constant time, meaning that while individual calls may be slower due to reallocations, on average, the operation is efficient.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    myVector.push_back("!");

    for(const auto &str : myVector) {
        std::cout << str << ' ';
    }
    return 0;
}
```

```cpp
int main() {
    std::vector numbers;
    for(int i = 1; i <= 5; ++i) {
        numbers.push_back(i);
    }

    int sum = 0;
    for(auto number : numbers) {
        sum += number;
    }

    std::cout << "Sum: " << sum;
    return 0;
}
```
