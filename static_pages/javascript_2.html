concat

The concat() method in JavaScript is used to merge two or more arrays. This method does not change the existing arrays but instead returns a new array containing the values of the joined arrays. The syntax for this method is `array1.concat(array2, array3, ..., arrayN)`, where `array1` is the base array and the remaining arrays are the ones to be concatenated. This method can also take in values (like numbers or objects) that are not arrays. The resulting array is a combination of the arrays or values passed to the method.

array1 = ['a', 'b', 'c']
array2 = ['d', 'e']
result = array1.concat(array2) # result: ['a', 'b', 'c', 'd', 'e'] 
array1 = [1, 2, 3]
array2 = [4, 5]
array3 = [6, 7]
result = array1.concat(array2, array3) # result: [1, 2, 3, 4, 5, 6, 7]
array1 = ['apple', 'banana']
values = [1, 2, 3]
result = array1.concat(values) # result: ['apple', 'banana', 1, 2, 3]
