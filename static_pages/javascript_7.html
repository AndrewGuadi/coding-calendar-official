charAt

The `charAt` method in JavaScript is used to access a specific character in a string based on its index. The method takes one parameter, which is the index position of the character you want to retrieve. Indexing in JavaScript is zero-based, meaning that the first character of the string is at index 0, the second at index 1, and so on. If the specified index is out of range (negative or greater than the string length), the method will return an empty string.

string = 'Hello, World!'

# Access the first character
result = string.charAt(0)  # H
string = 'Python Programming'

# Access the fifth character
result = string.charAt(4)  # o
string = 'Accessing Characters'

# Accessing out of range index
result = string.charAt(25)  # '' (empty string)
