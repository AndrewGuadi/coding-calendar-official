strlen

The strlen function in PHP is used to compute the length of a given string. It returns the number of bytes rather than characters, making it important to note when dealing with multi-byte character encodings. This function can be particularly useful for validation purposes, such as ensuring that input strings meet certain length requirements before processing.

def get_string_length(input_string):
    # PHP equivalent: $length = strlen($input_string);
    return len(input_string)

# Example usage:
result = get_string_length('Hello World')
print(result)  # Output: 11
def check_username_length(username):
    # PHP equivalent: if (strlen($username) < 5) {
    #   return 'Username too short';
    # }
    if len(username) < 5:
        return 'Username too short'
    return 'Username is valid'

# Example usage:
result = check_username_length('user')
print(result)  # Output: Username too short
def count_chars_and_spaces(input_string):
    # PHP equivalent: $count = strlen($input_string);
    #   return count;
    return len(input_string)

# Example usage:
result = count_chars_and_spaces('Count this!')
print(result)  # Output: 12
