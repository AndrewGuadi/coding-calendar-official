upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase. This method does not modify the original string; instead, it returns a new string with all letters in uppercase. If the string contains characters that are not letters, those characters remain unchanged.

str = 'hello world'
result = str.upcase
# result will be 'HELLO WORLD'
print(result)
greeting = 'Good Morning!'
new_greeting = greeting.upcase
# new_greeting will be 'GOOD MORNING!'
print(new_greeting)
message = 'Welcome to Ruby programming. Let's CODE!'
uppercase_message = message.upcase
# uppercase_message will be 'WELCOME TO RUBY PROGRAMMING. LET'S CODE!'
print(uppercase_message)
