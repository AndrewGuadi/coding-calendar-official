std::vector::push_back

The method 'std::vector::push_back' is used to add a new element at the end of a std::vector. This method automatically resizes the vector if necessary, allocating more memory to accommodate the new element. Using 'push_back' ensures that the elements are always stored in a contiguous block of memory. This method takes a single argument, which is the value to be added to the vector, and modifies the vector in place. It is important to note that this method has an average time complexity of O(1), but in cases where reallocation is needed, it can take longer.

```
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    for (const auto& value : myVector) {
        std::cout << value << " ";
    }
    return 0;
}
```
