std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that adds a new element at the end of the vector, expanding its size by one. This function is typically used when you want to dynamically add elements to a vector without having to specify the size in advance. The `push_back` operation usually has an average time complexity of O(1), but in cases of resizing when the underlying array is full, the complexity can temporarily increase to O(n).

```cpp
#include 
#include 

int main() {
    std::vector languages;
    languages.push_back("Python"); // Adding 'Python' to the vector.
    languages.push_back("Java");   // Adding 'Java' to the vector.
    languages.push_back("C++");    // Adding 'C++' to the vector.

    for (const auto& lang : languages) {
        std::cout << lang << std::endl; // Output all languages in the vector.
    }
    return 0;
}
```

#include

int main() {
    std::vector numbers;
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i); // Adding integers from 1 to 5 to the vector.
    }

    // Display the numbers in the vector
    for (const auto& number : numbers) {
        std::cout << number << " "; // Output all numbers in the vector.
    }
    return 0;
}
