acos

The acos() method in Java is a part of the java.lang.Math class and is used to compute the arc cosine (inverse cosine) of a value. It takes a double value between -1 and 1 as an argument and returns the angle in radians whose cosine is the specified value. If the value is outside the range of -1 to 1, the method will return NaN (Not a Number). The result is expressed in radians, so it may be converted to degrees if needed.

import math

# Compute the arc cosine of 0.5
value = 0.5
result = math.acos(value)
print('The arc cosine of', value, 'is', result, 'radians')
