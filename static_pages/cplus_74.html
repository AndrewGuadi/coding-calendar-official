std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class. It is used to add a new element to the end of the vector, effectively increasing the vector's size by one. This method requires the element to be passed as an argument. If the current vector size exceeds its allocated capacity, `push_back` will automatically allocate more memory to accommodate the new element. The time complexity of the `push_back` operation is amortized O(1), meaning that while adding elements is generally constant time, occasional reallocation can make some calls take longer.

```python
# Example: Using std::vector::push_back to create a list of Python strings

# Define a list to simulate a C++ vector
vector = []

# Use push_back equivalent: append elements representing programming languages
vector.append('Python')  # Similar to vector.push_back('Python') in C++
vector.append('C++')
vector.append('Java')

# Print the resultant list
print(vector)  # Output: ['Python', 'C++', 'Java']
```
```python
# Example: Using std::vector::push_back to build a dynamic list of numbers

# Define an empty list
vector = []

# Use push_back equivalent: add numbers to the list
for i in range(5):
    vector.append(i * 10)  # Here simulating vector.push_back(i * 10)

# Print the resultant list
print(vector)  # Output: [0, 10, 20, 30, 40]
```
```python
# Example: Using std::vector::push_back to concatenate strings

# Define an empty list
vector = []

# Use push_back equivalent: adding sentences to the list
vector.append('Hello')  # Equivalent to vector.push_back('Hello') in C++
vector.append('World')
vector.append('from OpenAI')

# Print the resultant list
print(' '.join(vector))  # Output: 'Hello World from OpenAI'
```
