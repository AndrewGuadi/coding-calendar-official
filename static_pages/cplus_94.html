std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class, which adds a new element to the end of the vector. This method increases the size of the vector by one and appends the given value to the vector. It is commonly used to dynamically grow the vector as more elements are needed. The operation is generally efficient, but if the current memory allocation is full, it may require reallocation to accommodate the new element, which can impact performance. Overall, `push_back` is an essential function to manage dynamic arrays in C++ using the Standard Template Library (STL).

int main() {
    std::vector languages;
    
    // Adding programming languages to the vector
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");
    
    // Output the elements in the vector
    for (const auto& lang : languages) {
        std::cout << lang << " ";
    }
    return 0;
} // Output: Python Java C++
# Using push_back with a loop
#include 
#include 

int main() {
    std::vector fruits;
    std::string fruitList[] = {"Apple", "Banana", "Cherry", "Date"};

    // Adding fruits to the vector using push_back in a loop
    for (const auto& fruit : fruitList) {
        fruits.push_back(fruit);
    }

    // Displaying the fruit vector
    for (const auto& fruit : fruits) {
        std::cout << fruit << " ";
    }
    return 0;
} // Output: Apple Banana Cherry Date
