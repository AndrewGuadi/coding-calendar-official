std::vector::push_back

The std::vector::push_back method is used to add a new element at the end of a vector, effectively increasing the size of the vector by one. When push_back is called, it appends the specified value to the end of the current vector. If the vector's current capacity is reached, it automatically reallocates memory for the additional element. This method is part of the C++ Standard Library and is crucial when you need dynamic storage of elements in a sequential manner.

```python
# Example 1: Creating a vector and adding elements using push_back

import ctypes

class Vector:
    def __init__(self):
        self.size = 0
        self.capacity = 1
        self.data = (ctypes.py_object * self.capacity)()

    def push_back(self, value):
        if self.size == self.capacity:
            self.capacity *= 2
            new_data = (ctypes.py_object * self.capacity)()
            for i in range(self.size):
                new_data[i] = self.data[i]
            self.data = new_data
        self.data[self.size] = value
        self.size += 1

vector = Vector()

vector.push_back('Hello')
vector.push_back('World')

# Displaying the elements in the vector
for i in range(vector.size):
    print(vector.data[i])  # Output: Hello World
```

# Example 3: Building a list of favorite movies using push_back

favorite_movies = Vector()

favorite_movies.push_back('Interstellar')

# Displaying the favorite movies
total_movies = "Favorite Movies: " + ', '.join([favorite_movies.data[i] for i in range(favorite_movies.size)])
