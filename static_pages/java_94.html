abs

The 'abs' method in Java is used to calculate the absolute value of a number. It can operate on various numeric types including integers, floats, and doubles. The absolute value of a number is its distance from zero on the number line, disregarding whether it is positive or negative. In Java, the 'abs' method is available in the 'Math' class, and can be utilized to ensure that calculations remain non-negative.

def calculate_distance(x: float, y: float) -> float:
    return abs(x - y)

distance = calculate_distance(-3.5, 2.5)
print(distance)  # Output: 6.0
def temperature_difference(temp1: float, temp2: float) -> float:
    return abs(temp1 - temp2)

result = temperature_difference(-10.2, 5.4)
print(result)  # Output: 15.6
def profit_or_loss(cost_price: float, selling_price: float) -> float:
    return abs(selling_price - cost_price)

result = profit_or_loss(50.0, 30.0)
print(result)  # Output: 20.0
