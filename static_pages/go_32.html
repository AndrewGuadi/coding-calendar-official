append

The `append` function in Go is used to add elements to the end of a slice. If the slice has enough capacity to accommodate the new elements, they are added without allocating a new array. If the capacity is insufficient, a new underlying array is allocated and the elements are copied over. The method returns the updated slice.

```go
package main
import (
	"fmt"
)

func main() {
	numbers := []int{1, 2, 3}
	numbers = append(numbers, 4, 5)
	fmt.Println(numbers) // Output: [1 2 3 4 5]
}```
```go
package main
import (
	"fmt"
)

func main() {
	letters := []string{"a", "b"}
	letters = append(letters, "c", "d")
	fmt.Println(letters) // Output: [a b c d]
}```
```go
package main
import (
	"fmt"
)

func main() {
	fruits := []string{"apple", "banana"}
	moreFruits := []string{"orange", "grape"}
	fruits = append(fruits, moreFruits...)
	fmt.Println(fruits) // Output: [apple banana orange grape]
}```
