std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a vector. This function increases the size of the vector by one, and the new element is initialized to the value provided as an argument. If the vector's current capacity is reached, it automatically reallocates memory to accommodate the new element. This method is a member function of the `std::vector` class and is a crucial operation when dynamically managing a list of items.

```c++
#include 
#include 

int main() {
    std::vector vec;
    vec.push_back("Hello");
    vec.push_back("World");

    for (const auto& str : vec) {
        std::cout << str << ' '; // Output: Hello World
    }
    return 0;
}
```
