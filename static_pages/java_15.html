abs

The 'abs' method in Java is a static method provided in the Math class of the Java standard library. It is used to return the absolute value of a given number. The method can take various forms depending on the type of the input: it can take an int, long, float, or double value. The absolute value of a number is its non-negative value, meaning that if the number is negative, this method will return the positive equivalent.

```python
# Example 1: Using abs with an integer
number = -10
absolute_value = abs(number)
print(absolute_value)  # Output: 10
```
```python
# Example 2: Using abs with a float
number = -5.75
absolute_value = abs(number)
print(absolute_value)  # Output: 5.75
```
```python
# Example 3: Using abs with a list of numbers
numbers = [-3, -7.2, 4, 8, -2]
absolute_values = [abs(num) for num in numbers]
print(absolute_values)  # Output: [3, 7.2, 4, 8, 2]
```
