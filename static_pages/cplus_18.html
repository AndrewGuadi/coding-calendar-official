std::vector::push_back

The 'push_back' method in C++'s Standard Library vector allows you to add a new element to the end of the vector. This method automatically increases the size of the vector by one, allowing a dynamic increase in capacity as necessary. It is commonly used to build dynamic arrays where the size of the data is not known during compile time. The new element is copied or moved into the vector, and this method is typically efficient in terms of time complexity, operating in amortized constant time (O(1)).

```c++
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& str : myVector) {
        std::cout << str << ' '; // Outputs: Hello World 
    }
    return 0;
}
```
