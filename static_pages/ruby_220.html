upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to their uppercase equivalents. This method does not modify the original string but returns a new string with the converted format. If the string is already uppercased or contains no alphabetic characters, it will remain unchanged. Itâ€™s commonly used for standardizing input to ensure case-insensitivity in string comparisons.

text = 'hello world' 
 upper_text = text.upcase 
 print(upper_text)  # Output: 'HELLO WORLD'
text = 'Ruby is fun' 
 upper_text = text.upcase 
 print(upper_text)  # Output: 'RUBY IS FUN'
text = 'Python 3.8' 
 upper_text = text.upcase 
 print(upper_text)  # Output: 'PYTHON 3.8'
