std::vector::push_back

The 'std::vector::push_back' method in C++ is used to add a new element to the end of a std::vector. It automatically resizes the vector if needed to accommodate the new element, and it preserves the order of elements. This is a commonly used method for dynamically growing arrays in C++. The complexity of this operation is amortized O(1), but in some cases, it can take O(n) time due to reallocation.

```cpp
#include 
#include 

int main() {
    std::vector languages;

    // Adding programming languages using push_back
    languages.push_back("Python");
    languages.push_back("Java");
    languages.push_back("C++");

    // Display the elements in the vector
    for (const auto& lang : languages) {
        std::cout << lang << std::endl;
    }
    return 0;
}
```

```cpp
#include 
#include 

int main() {
    std::vector numbers;

    // Add numbers to the vector
    for (int i = 1; i <= 5; ++i) {
        numbers.push_back(i * 10);  // Pushing multiples of 10
    }

    // Print the numbers in vector
    for (int number : numbers) {
        std::cout << number << ' ';
    }
    return 0;
}
```
