std::vector::push_back

The `push_back` method in C++ is a member function of the `std::vector` class that adds a new element to the end of the vector. This method resizes the vector to accommodate the new element, ensuring that the vector's capacity is sufficient. If the current capacity is exceeded, the vector automatically allocates more memory to accommodate the additional elements. `push_back` can be used to build a collection of items dynamically at runtime. It is an efficient way to append elements as it amortizes the cost of dynamic memory allocation over multiple insertions.

```cpp
#include 
#include 

int main() {
    std::vector pythonExamples;
    pythonExamples.push_back("print('Hello, World!')");
    pythonExamples.push_back("for i in range(5): print(i)");
    pythonExamples.push_back("def add(a, b): return a + b");
    
    for (const auto& example : pythonExamples) {
        std::cout << example << std::endl;
    }
    return 0;
}
```
