abs

The `abs` function in Python returns the absolute value of a number. The absolute value of a number is the non-negative value of that number without regard to its sign. For instance, the absolute value of both -5 and 5 is 5. The `abs` function can be used with integers, floating-point numbers, or complex numbers. For complex numbers, `abs` returns the magnitude of the number.

# Example 1: Using abs with an integer
negative_integer = -10
absolute_value = abs(negative_integer)
print('Absolute value of -10 is:', absolute_value)
# Example 2: Using abs with a floating-point number
negative_float = -7.25
absolute_value = abs(negative_float)
print('Absolute value of -7.25 is:', absolute_value)
# Example 3: Using abs with a complex number
complex_number = 3 - 4j
magnitude = abs(complex_number)
print('Magnitude of 3 - 4j is:', magnitude)
