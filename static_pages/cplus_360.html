std::vector::push_back

The method 'std::vector::push_back' is used to add a new element at the end of a vector in C++. This function increases the size of the vector by one and assigns the value of the new element to the specified value. If the vector exceeds its current capacity, it will allocate additional memory and copy its existing elements to this new memory. The time complexity of push_back is amortized O(1).

```cpp
#include 
#include 
using namespace std;

int main() {
    vector vec;
    vec.push_back("print('Hello, World!')");
    vec.push_back("x = 5");
    vec.push_back("print(x + 3)");

    // Display the contents of the vector
    for (const auto& code : vec) {
        cout << code << endl;
    }
    return 0;
}
```
