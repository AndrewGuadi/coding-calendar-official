Add

The 'Add' method is typically used to insert a new element into a collection such as a list, dictionary, or set in C#. In the context of data structures, this method adds a new item to the collection, increasing its count and ensuring that the item can later be accessed, modified, or removed as needed. The usage of Add may vary slightly depending on the type of collection being utilized, but the core functionality remains the same across different collections.

```python
# Example of using Add with a List in C#
# This example demonstrates how to create a list of strings and add items to it.

names = List()
names.Add('Alice')
names.Add('Bob')
names.Add('Charlie')

print(names)  # Output: ['Alice', 'Bob', 'Charlie']
```
```python
# Example of using Add with a Dictionary in C#
# Here, we create a dictionary to map names to ages and use Add to insert key-value pairs.

ages = Dictionary()
ages.Add('Alice', 30)
ages.Add('Bob', 25)
ages.Add('Charlie', 35)

print(ages)  # Output: {'Alice': 30, 'Bob': 25, 'Charlie': 35}
```
```python
# Example of using Add with a HashSet in C#
# This example illustrates how to add unique items into a HashSet.

uniqueNames = HashSet()
uniqueNames.Add('Alice')
uniqueNames.Add('Bob')
uniqueNames.Add('Alice')  # This will not create a duplicate

print(uniqueNames)  # Output: {'Alice', 'Bob'}
```
