acos

The 'acos' method in Java is a static method of the Math class that returns the arc cosine (inverse cosine) of a value. This method takes a single argument of type double, which must be in the range of -1.0 to 1.0. The result is given in radians, with a return value of 0 for input 1, π/2 for input 0, and π for input -1. It’s useful for determining the angle in a triangle given the length of two sides, among other applications in engineering and graphics.

import math

# Calculate the angle (in radians) whose cosine is 0.5
cos_value = 0.5
angle = math.acos(cos_value)
print('The angle in radians is:', angle) # This should return approximately 1.0472 radians (60 degrees)
