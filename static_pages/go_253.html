close

The 'close' method is used to close a resource, such as a file or a network connection. In Go, it is typically used to free up system resources and ensure that all pending operations for the resource are completed before it is released. Closing a resource is an important operation and should always be done when the resource is no longer needed to avoid resource leaks.

import os

# Open a file for writing
file = os.open('example.txt', os.O_WRONLY | os.O_CREAT)

# Write some data to the file
os.write(file, b'This is an example.')

# Close the file
os.close(file)
import socket

# Create a socket
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# Connect to a server
s.connect(('localhost', 8080))

# Send some data
s.sendall(b'Hello, server')

# Close the socket
s.close()
import sqlite3

# Connect to a database
conn = sqlite3.connect('example.db')

# Create a cursor and execute a query
cursor = conn.cursor()
cursor.execute('CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)')

# Close the connection to the database
conn.close()
