std::vector::push_back

The 'push_back' method in C++ is a member function of the 'std::vector' class that is used to add elements to the end of the vector. When using 'push_back', the vector automatically resizes itself to accommodate the new element if necessary. This method is commonly used when you want to dynamically increase the size of a vector to store additional items. The time complexity of this method is amortized constant time, meaning that it typically operates in O(1) time, although in some cases may take longer when the underlying array needs to be resized.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello"); // Adding a string element
    myVector.push_back("World"); // Adding another string element

    for (const auto& str : myVector) {
        std::cout << str << ' '; // Output: Hello World 
    }
}
```
