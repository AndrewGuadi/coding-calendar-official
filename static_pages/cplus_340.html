std::vector::push_back

The `push_back` method in C++ is used to add an element to the end of a vector. It takes the value as a parameter and appends it to the back of the vector, effectively increasing the size of the vector by one. This method automatically handles memory management for the vector as needed, ensuring that there is enough space to accommodate new elements. If the vector's current storage cannot accommodate the new element, it will allocate new storage and copy the existing elements to this new storage before adding the new element.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    myVector.push_back("from");
    myVector.push_back("C++");

    for (const auto& str : myVector) {
        std::cout << str << " ";
    }
    return 0;
}
```

```cpp
        numbers.push_back(i * 10);
    }

    std::cout << "Numbers in the vector:

    for (const auto& num : numbers) {

}
```
