abs

The 'abs' method in Java is used to return the absolute value of a specified number. This method is part of the 'Math' class and can handle several numeric types including int, long, float, and double. The absolute value of a number is the non-negative value of that number without regard to its sign. For example, the absolute value of -5 is 5, and the absolute value of 3 is 3. The method is typically used in mathematical calculations where the sign of the number does not matter but the magnitude does.

def calculate_distance(point_a, point_b):
    x1, y1 = point_a
    x2, y2 = point_b
    distance_x = abs(x2 - x1)
    distance_y = abs(y2 - y1)
    return distance_x + distance_y

# Usage
point_a = (2, 3)
point_b = (5, 1)
distance = calculate_distance(point_a, point_b)
print(f'Distance: {distance}')
def temperature_difference(temp1, temp2):
    return abs(temp1 - temp2)

# Usage
today_temp = 20.5
yesterday_temp = 15.0
difference = temperature_difference(today_temp, yesterday_temp)
print(f'Temperature difference: {difference}')
def net_balance(deposit, withdrawal):
    return abs(deposit - withdrawal)

# Usage
deposit = 500
withdrawal = 300
balance = net_balance(deposit, withdrawal)
print(f'Net balance: {balance}')
