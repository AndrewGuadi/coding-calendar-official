addAll

The 'addAll' method in Java is used to add all elements from a specified collection (such as a list or set) to another collection. This method is particularly useful when you want to combine multiple collections into one without requiring a loop to add each element individually. The method alters the collection it is called on by adding all the elements from the specified collection. It's commonly found in classes that implement the 'Collection' interface, like 'ArrayList', 'HashSet', etc.

list1 = [1, 2, 3]
list2 = [4, 5, 6]
list1.extend(list2)  # Adding all elements from list2 to list1
print(list1)  # Output: [1, 2, 3, 4, 5, 6]
set1 = {'apple', 'banana'}
set2 = {'cherry', 'date'}
set1.update(set2)  # Adding all elements from set2 to set1
print(set1)  # Output: {'apple', 'banana', 'cherry', 'date'}
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}
dict1.update(dict2)  # Adding all key-value pairs from dict2 to dict1
print(dict1)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
