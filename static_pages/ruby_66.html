upcase

The `upcase` method in Ruby is used to convert all lowercase letters in a string to uppercase letters. If the string contains no lowercase letters or is already in uppercase, the method returns the string unchanged. This method is case-sensitive and does not affect non-alphabetic characters.

text = 'hello world'
uppercase_text = text.upcase
puts uppercase_text  # Output: 'HELLO WORLD'
sentence = 'Hello, Ruby!'
upcased_sentence = sentence.upcase
puts upcased_sentence  # Output: 'HELLO, RUBY!'
mixed_case = 'hElLo WoRlD!'
result = mixed_case.upcase
puts result  # Output: 'HELLO WORLD!'
