Add

The 'Add' method is commonly used in C# collections, such as List, Dictionary, or other data structures, to add an element to a collection. When using 'Add', it dynamically increases the size of the collection and allows insertion of new data at the end or in key-value pairs for dictionaries. This method may throw exceptions if the collection has reached its capacity or if you are trying to add a duplicate key in case of dictionaries.

```python
# Example of using a List in C# to add elements

# Create a List of strings
my_list = []

# Add elements using the Add method
my_list.append('Hello')
my_list.append('World')

# Print the list to show the result
print(my_list)  # Output: ['Hello', 'World']
```
```python
# Example of using a Dictionary in C# to add key-value pairs

# Create a Dictionary in C# to store student names and their grades
my_dict = {}

# Add key-value pairs using the Add method
my_dict['Alice'] = 90
my_dict['Bob'] = 85

# Print the dictionary to show the result
print(my_dict)  # Output: {'Alice': 90, 'Bob': 85}
```
```python
# Example of using a HashSet in C# to add unique elements

# Create a HashSet
my_set = set()

# Add elements using the Add method
my_set.add('Python')
my_set.add('C#')
my_set.add('Python')  # Duplicate, this will not be added

# Print the HashSet to show the result
print(my_set)  # Output: {'Python', 'C#'}
```
