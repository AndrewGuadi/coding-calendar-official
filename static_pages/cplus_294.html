std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one and copies the new element into the newly allocated space at the end. The `push_back` method is efficient but can occasionally involve reallocating the underlying array if the current capacity is exceeded. This method is part of the C++ Standard Library and is typically used when the size of the vector is not known beforehand or when elements need to be added dynamically.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    for (const auto& str : myVector) {
        std::cout << str << ' ';
    }
    return 0;
}
``` 
This example creates a vector of strings, adds the strings 'Hello' and 'World' using `push_back`, and then prints them out.
