std::vector::push_back

The `push_back` method in C++ is used to add a new element at the end of a vector. This method increases the size of the vector by one, and the new element is initialized with the value provided. If the vector's current capacity is reached, its capacity is increased to accommodate the new element. This method ensures that vectors can dynamically resize to store an arbitrary number of elements, unlike arrays which have a fixed size. Using `push_back` is efficient and allows for easy management of collections of data.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("print('Hello World')");
    std::cout << "First element: " << myVector[0] << std::endl;
    return 0;
}
```

#include 
#include 

int main() {
    std::vector commandList;
    commandList.push_back("for i in range(10):");
    commandList.push_back("    print(i)");
    for (const auto& command : commandList) {
        std::cout << command << std::endl;
    }
    return 0;
}
