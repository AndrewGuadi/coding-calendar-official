strlen

The strlen function in PHP is used to get the length of a string. It returns the number of bytes in the string, which typically represents the number of characters in the string for most character encodings. It is particularly useful when you need to validate input, determine string formatting, or simply obtain the size of a string for processing.

def get_string_length(input_string):
    return len(input_string)

# Example usage:
input_string = 'Hello, World!'

length_of_string = get_string_length(input_string)
print(f'The length of the string is: {length_of_string}')  # Output: The length of the string is: 13
def is_string_longer_than(input_string, threshold):
    return len(input_string) > threshold

# Example usage:
input_string = 'Python Programming'
threshold = 10

if is_string_longer_than(input_string, threshold):
    print('The string is longer than the threshold.')  # Output: The string is longer than the threshold.
def get_length_and_check(input_string):
    length = len(input_string)
    if length < 5:
        return 'String is too short.'
    elif length > 10:
        return 'String is too long.'
    return 'String length is acceptable.'

# Example usage:
input_string = 'Hi!'

result = get_length_and_check(input_string)
print(result)  # Output: String is too short.
