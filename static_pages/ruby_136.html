upcase

The 'upcase' method in Ruby is used to convert a given string into uppercase letters. This method does not modify the original string but returns a new string with all lowercase letters converted to uppercase characters. This is particularly useful when you need to ensure that a string is in a consistent uppercase format, such as when processing user input or formatting strings for output.

input_string = 'hello world'
uppercased = input_string.upcase
puts uppercased  # Output: 'HELLO WORLD'
greeting = 'Good Morning!'
upper_case_greeting = greeting.upcase
print upper_case_greeting  # Output: 'GOOD MORNING!'
error_message = 'file not found'
formatted_error = error_message.upcase
puts formatted_error  # Output: 'FILE NOT FOUND'
