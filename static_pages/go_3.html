append

In Go, the 'append' function is used to add elements to a slice. It takes a slice and one or more elements to be added to that slice, returning a new slice that includes the additional elements. This method is essential for dynamically managing collections of data as it allows for resizing of slices without needing to manually manage underlying arrays.

mySlice := []string{"apple", "banana"}
mySlice = append(mySlice, "cherry")
// Now mySlice contains: ["apple", "banana", "cherry"]
numbers := []int{1, 2, 3}
numbers = append(numbers, 4, 5)
// Now numbers contains: [1, 2, 3, 4, 5]
names := []string{}
names = append(names, "John")
names = append(names, "Jane", "Doe")
// Now names contains: ["John", "Jane", "Doe"]
