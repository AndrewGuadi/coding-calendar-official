strlen

The `strlen` function in PHP is used to get the length of a string, which is the number of characters in it, including spaces and special characters. The function returns an integer value representing the string length. If the input is not a valid string, it will convert it to a string and then calculate its length.

$exampleString = 'Hello, World!';
$length = strlen($exampleString);
echo $length; // Output: 13
$exampleString = '   Spaces before and after   ';
$length = strlen($exampleString);
echo $length; // Output: 30
$exampleString = '1234567890!@#$%^&*()';
$length = strlen($exampleString);
echo $length; // Output: 20
