Add

The 'Add' method in C# is typically used to append an element to a collection or data structure, such as a list, dictionary, or set. When utilized in a list, 'Add' adds the specified object to the end of the list. In the context of dictionaries, it adds a key-value pair. This method is a vital component of the C# collection framework, allowing dynamic data manipulation and management.

```python
# Example 1: Adding numbers to a Python list
nums = []

# Function to add number to the list
def add_to_list(num):
    nums.append(num)

add_to_list(10)
add_to_list(20)
add_to_list(30)
print(nums)  # Output: [10, 20, 30]
```
```python
# Example 2: Adding key-value pairs to a Python dictionary
my_dict = {}

# Function to add entries to the dictionary
def add_to_dict(key, value):
    my_dict[key] = value

add_to_dict('name', 'Alice')
add_to_dict('age', 30)
print(my_dict)  # Output: {'name': 'Alice', 'age': 30}
```
```python
# Example 3: Adding elements to a Python set
my_set = set()

# Function to add elements to a set
def add_to_set(element):
    my_set.add(element)

add_to_set(1)
add_to_set(2)
add_to_set(2)  # Duplicate value is ignored
print(my_set)  # Output: {1, 2}
```
