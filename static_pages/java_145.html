acos

The 'acos' method in Java is used to compute the arc cosine (inverse cosine) of a specified value, returning the result in radians. The input value must be in the range of -1.0 to 1.0. If the input is outside this range, a runtime exception will be thrown. The method is part of the Java Math class and can be useful in various applications, such as calculating angles in trigonometric problems.

import math

# Example: Calculate arc cosine of 0.5
value = 0.5
angle_rad = math.acos(value)
angle_deg = math.degrees(angle_rad)
print(f"Arc Cosine of {value} in radians: {angle_rad}")
print(f"Arc Cosine of {value} in degrees: {angle_deg}")
import math

import math

# Example: Handling out of range value
try:
    value = 1.5  # This is outside the range of -1 to 1
    angle_rad = math.acos(value)
except ValueError as e:
    print(f"Error: {e}")
