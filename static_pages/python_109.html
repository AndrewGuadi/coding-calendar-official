abs

The `abs()` function in Python is used to calculate the absolute value of a number. This means that it returns the positive magnitude of the given number regardless of whether the original number was positive or negative. The function can be applied to both integer and floating-point numbers.

# Example 1: Finding the absolute value of an integer
number = -10
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
# Output: The absolute value of -10 is 10
# Example 2: Finding the absolute value of a floating-point number
number = -7.25
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
# Output: The absolute value of -7.25 is 7.25
# Example 3: Absolute value of a complex expression
number = -4 + 3
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
# Output: The absolute value of -1 is 1
