upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method returns a new string and does not modify the original string. It's particularly useful when you need to standardize the casing of text, such as when comparing strings or preparing data for display.

original_string = 'hello world'
uppercase_string = original_string.upcase
print(uppercase_string)  # Output: 'HELLO WORLD'
greeting = 'Good Morning!'
upper_greeting = greeting.upcase
print(upper_greeting)  # Output: 'GOOD MORNING!'
name = 'aLiCe'
upper_name = name.upcase
print(upper_name)  # Output: 'ALICE'
