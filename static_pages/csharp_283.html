Add

The 'Add' method is used in various contexts in C# to add an element to a collection, such as a list, a dictionary, or a set. For instance, when dealing with a List object, the Add method appends the specified object to the end of the list. Similarly, the method can be used in dictionaries to add a key-value pair. The method is common in multiple collection types. The following examples illustrate the use of the Add method in a List and a Dictionary.

```python
# Example 1: Using Add with a List in Python

# Initialize an empty list
my_list = []

# Add elements to the list
my_list.append("Python")
my_list.append("Java")
my_list.append("C#")

# Print the updated list
print(my_list)  # Output: ['Python', 'Java', 'C#']
```
```python
# Example 2: Using Add with a Dictionary in Python

# Initialize an empty dictionary
my_dict = {}

# Add key-value pairs to the dictionary
my_dict['language_1'] = 'Python'
my_dict['language_2'] = 'Java'
my_dict['language_3'] = 'C#'

# Print the updated dictionary
print(my_dict)  # Output: {'language_1': 'Python', 'language_2': 'Java', 'language_3': 'C#'}
```
```python
# Example 3: Using Add in a Set in Python

# Initialize an empty set
my_set = set()

# Add elements to the set
my_set.add("Python")
my_set.add("Java")
my_set.add("C#")
my_set.add("Python")  # This will not add a duplicate

# Print the updated set
print(my_set)  # Output: {'Java', 'Python', 'C#'}  # 'Python' will only appear once
```
