Add

The 'Add' method is commonly used in C# to add an item to a collection, such as a list or a dictionary. When using the Add method, the item is appended at the end of the collection if itâ€™s a list, or it is added as a key-value pair in a dictionary. This method is essential for managing dynamic collections where the size may vary during runtime.

```python
# Example of using 'Add' with a list in C#
# Suppose we have a list of strings, and we want to add a new string to it.
strings = List[str]()  # Create a new list
strings.Add('Hello')  # Add 'Hello' to the list
strings.Add('World')  # Add 'World' to the list

# Now the list contains: ['Hello', 'World']
print(strings)  # Output: ['Hello', 'World']
```
```python
# Example of using 'Add' with a Dictionary in C#
# Consider a dictionary where we store a person's age based on their name.
ages = Dictionary[str, int]()  # Create a new dictionary
ages.Add('Alice', 30)  # Add key 'Alice' with value 30
ages.Add('Bob', 25)  # Add key 'Bob' with value 25

# Now the dictionary contains: {'Alice': 30, 'Bob': 25}
print(ages)  # Output: {'Alice': 30, 'Bob': 25}
```
```python
# Example of using 'Add' with a list in C# to manage a shopping cart.
shopping_cart = List[str]()  # Create an empty shopping cart
shopping_cart.Add('Apples')  # Add Apples to the cart
shopping_cart.Add('Bananas')  # Add Bananas to the cart
shopping_cart.Add('Oranges')  # Add Oranges to the cart

# Now the shopping cart contains: ['Apples', 'Bananas', 'Oranges']
print(shopping_cart)  # Output: ['Apples', 'Bananas', 'Oranges']
```
