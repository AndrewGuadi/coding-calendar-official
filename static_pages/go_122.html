cap

In Go, the 'cap' method is used to retrieve the capacity of a slice, which represents the number of elements the slice can hold without requiring a reallocation. This is useful for optimizing memory usage and performance when working with slices. The capacity is determined by the size of the array that the slice points to. When initializing a slice, you can define its capacity for better performance during operations that may modify its size.

def main():
    # Initialize a slice with a predefined capacity
    fruits = ['apple', 'banana', 'cherry']
    print('Fruits:', fruits)
    print('Capacity:', len(fruits))  # Using len() in place of cap() for demonstration

main()
def main():
    # Initialize a list with a specific capacity
    numbers = [x for x in range(10)]  # List comprehension
    print('Numbers:', numbers)
    print('Capacity:', len(numbers))  # Using len() in place of cap() for demonstration

main()
def main():
    # Adding elements to a list dynamically
    colors = []
    for color in ['red', 'green', 'blue']:
        colors.append(color)
    print('Colors:', colors)
    print('Capacity:', len(colors))  # Using len() in place of cap() for demonstration
