std::vector::push_back

The 'push_back' method in C++ is used to add a new element to the end of a 'std::vector'. This method dynamically increases the size of the vector by one, allowing the user to store more elements. When called, 'push_back' takes a single argument, which is the value to be added to the vector. It is important to note that the vector may need to allocate more memory if it exceeds its current capacity, which can result in a reallocation of existing elements. This method is often used for building collections of data in dynamic arrays.

In this example, we will create a vector of strings and use 'push_back' to add Python code snippets to it:

```cpp
#include 
#include 
#include 

int main() {
    std::vector pythonCodes;
    
    pythonCodes.push_back("def greet():\n    print('Hello, World!')");
    pythonCodes.push_back("for i in range(5):\n    print(i)");
    pythonCodes.push_back("class Dog:\n    def bark(self):\n        print('Woof!')");
    
    for (const auto& code : pythonCodes) {
        std::cout << code << std::endl;
    }
    return 0;
}
```

```cpp
int main() {
    std::vector factorialFunctions;
    
    factorialFunctions.push_back("def factorial(n):\n    if n == 0:\n        return 1\n    else:\n        return n * factorial(n-1)");
    factorialFunctions.push_back("def main():\n    print(factorial(5))");
    factorialFunctions.push_back("if __name__ == '__main__':\n    main()");
    
    for (const auto& function : factorialFunctions) {
        std::cout << function << std::endl;
    }
    return 0;
}
```
