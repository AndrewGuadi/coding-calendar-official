append

The 'append' method in Go is used to add elements to the end of a slice. This method can take one or more values to append to a slice and can dynamically increase the capacity of the slice if necessary. It is an in-built function that makes it easy to extend slices without having to manually manage the underlying array. When calling 'append', if the combined length of the original slice and new elements exceeds the capacity, a new slice will be allocated with a bigger size, and the elements will be copied over.

```go
package main
import (
    "fmt"
)
func main() {
    numbers := []int{1, 2, 3}
    numbers = append(numbers, 4, 5)
    fmt.Println(numbers) // Output: [1, 2, 3, 4, 5]
}
```
```go
package main
import (
    "fmt"
)
func main() {
    fruits := []string{"apple", "banana"}
    fruits = append(fruits, "orange")
    fmt.Println(fruits) // Output: [apple banana orange]
}
```
```go
package main
import (
    "fmt"
)
func main() {
    chars := []rune{‘a’, ‘b’}
    chars = append(chars, ‘c’, ‘d’, ‘e’)
    fmt.Println(chars) // Output: [97 98 99 100 101]
}
```
