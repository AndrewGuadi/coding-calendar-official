acos

The 'acos' method in Java is a part of the Math class and is used to compute the arc cosine of a value. This method returns the angle in radians whose cosine is the specified value. The input value must be in the range of -1.0 to 1.0, as these are the valid outputs of the cosine function. If the provided argument is outside this range, it will return NaN (Not a Number). The value returned can be converted to degrees if needed, by multiplying the result by (180/PI).

import math

# Angle in radians for cos(45 degrees) = 0.7071
value = 0.7071
angle = math.acos(value)
# Print angle in radians
print('Angle in radians:', angle)
# Convert radians to degrees
angle_degrees = angle * (180 / math.pi)
print('Angle in degrees:', angle_degrees)
