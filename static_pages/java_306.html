acos

The 'acos' method in Java is used to compute the arc cosine (inverse cosine) of a specified value, returning the angle in radians. The argument passed to 'acos' must be in the range of -1.0 to 1.0, where the result will be between 0 and Ï€ radians (0 to 180 degrees). If the argument is outside of this range, the method will return NaN (not a number). This method is commonly used in trigonometry calculations and when converting between different coordinate systems.

import math

# Given a cosine value
cos_value = 0.5

# Calculate the arc cosine
angle_radians = math.acos(cos_value)
angle_degrees = math.degrees(angle_radians)

# Print results
print(f'Arc cosine of {cos_value} is {angle_radians:.2f} radians or {angle_degrees:.2f} degrees')
