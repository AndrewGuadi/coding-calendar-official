acos

The acos method in Java is part of the java.lang.Math class and is used to compute the arc cosine (inverse cosine) of a specified value. It returns the angle in radians whose cosine is the specified number. The input value must be in the range of -1.0 to 1.0, inclusive. If the input is outside of this range, the method will return NaN (Not a Number). The output represents the angle in radians; to convert it to degrees, multiply by (180/PI).

import math

# Define a value between -1 and 1
value1 = 0.5
# Calculate the arc cosine
angle1 = math.acos(value1)
# Convert to degrees
angle1_degrees = math.degrees(angle1)
print('The arc cosine of {} is {} radians or {} degrees'.format(value1, angle1, angle1_degrees))
