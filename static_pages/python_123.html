abs

The 'abs' method in Python is a built-in function that returns the absolute value of a number. This means it will convert any negative number to its positive equivalent while leaving positive numbers and zero unchanged. The 'abs' function works with integers, floating-point numbers, and even complex numbers (though for complex numbers, it returns the magnitude).

```python
# Example 1: Absolute value of an integer
number = -10
absolute_value = abs(number)
print(absolute_value)  # Output: 10
```

```python
# Example 2: Absolute value of a floating-point number
float_number = -23.67
absolute_value = abs(float_number)
print(absolute_value)  # Output: 23.67
```

```python
# Example 3: Absolute value (magnitude) of a complex number
complex_number = 3 + 4j
absolute_value = abs(complex_number)
print(absolute_value)  # Output: 5.0
```
