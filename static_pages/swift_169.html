uppercased

The 'uppercased' method in Swift is an instance method available on the String type that returns a new string with all the characters in uppercase. It does not alter the original string but creates a modified copy. This method is useful when you want to standardize casing or format text for display or processing purposes. Itâ€™s particularly helpful in scenarios like formatting user input, generating report headers, or preparing data for case-sensitive comparisons.

def to_uppercase(text):
    return text.upper()

# Example usage:
input_text = 'hello world'
result = to_uppercase(input_text)
print(result)  # Output: 'HELLO WORLD'
def format_title(title):
    return title.upper()

# Example usage:
title_text = 'the great gatsby'
formatted_title = format_title(title_text)
print(formatted_title)  # Output: 'THE GREAT GATSBY'
