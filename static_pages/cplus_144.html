std::vector::push_back

The 'std::vector::push_back' method is used to add a new element at the end of a vector in C++. This operation increases the size of the vector by one, and the new element is placed at the position indicated by the current size of the vector. The method can be used to insert elements of any data type that can be stored in the vector. If the vector's current capacity is exceeded, the method automatically allocates more memory and copies the existing elements to the new memory location. This operation has an average time complexity of O(1) but can take O(n) time in the worst case when reallocating memory.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");  // Adding a string element
    myVector.push_back("World");  // Adding another string element
    std::cout << myVector[0] << " " << myVector[1]; // Output: Hello World
    return 0;
}
```
