upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method does not alter any characters that are already in uppercase or non-letter characters. It returns a new string with the desired transformations. This can be useful for standardizing user inputs, formatting text, or creating case-insensitive comparisons.

def convert_to_uppercase(input_string):
    return input_string.upper()

# Example usage:
input_string = 'hello world'
print(convert_to_uppercase(input_string))  # Output: 'HELLO WORLD'
def shout(text):
    return text.upper() + '!'

# Example usage:
message = 'this is a shout'
print(shout(message))  # Output: 'THIS IS A SHOUT!'
def normalize_name(name):
    return name.upper().strip()

# Example usage:
user_input = '   john doe  '
print(normalize_name(user_input))  # Output: 'JOHN DOE'
