charAt

The `charAt` method in JavaScript returns the character at a specified index (position) in a string. The index is zero-based, meaning the first character of the string is at index 0. If the index is out of bounds (less than 0 or greater than the string length), an empty string is returned. This method is useful for accessing individual characters of a string for manipulation or examination.

str = 'Hello World';
char_at_index_4 = str.charAt(4);  # 'o'
# This example retrieves the character at index 4 from the string 'Hello World', which is 'o'.
str = 'Python Programming';
char_at_index_0 = str.charAt(0);  # 'P'
# This example retrieves the first character of the string 'Python Programming', which is 'P'.
str = 'OpenAI';
char_at_index_6 = str.charAt(6);  # ''
# This example attempts to access index 6 of the string 'OpenAI', which is out of bounds, thus it returns an empty string.
