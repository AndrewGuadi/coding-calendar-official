cap

The 'cap' function in Go is used to return the capacity of a slice. The capacity is the number of elements that the slice can hold before needing to allocate more memory. This is particularly useful for optimizing performance and memory management in Go programs, especially when working with dynamically sized arrays or slices. It allows developers to understand the underlying memory allocation and adjust their slices accordingly.

```python
# Example 1: Basic usage of cap in a slice
# Create a slice with an initial capacity
my_slice = [1, 2, 3]
print(f'Capacity of my_slice: {len(my_slice)}')  # Prints the number of elements currently in it

# To mimic Go's cap, let's expand 
# with appending values to see capacity management
for i in range(4, 11):
    my_slice.append(i)  # Expanding the slice
print('Final Slice:', my_slice)  # Final values in the slice
# Note: Python lists do not expose a capacity, but this simulates the idea.
```
```python
# Example 2: Expanding a slice and tracking capacity
# Function to mimic growing a slice
def grow_slice(start, end):
    my_slice = []  # Create an empty slice
    for i in range(start, end + 1):
        my_slice.append(i)
        print(f'Current Slice: {my_slice}, Length: {len(my_slice)}')

# Call the function to see how the slice grows
grow_slice(1, 5)
```
```python
# Example 3: Understanding slice expansion
# Function demonstrating similar capacity concept with Python lists
def demo_capacity():
    initial_capacity = 5  # Mimic initial capacity
    my_slice = list(range(initial_capacity))  # Pre-fill with 0 to 4

    print(f'Initial Slice: {my_slice}')
    print(f'Length: {len(my_slice)}')  # Length of current slice
    
    # Expand it to exceed capacity
    for i in range(5, 15):
        my_slice.append(i)
        print(f'Adding {i}, New Length: {len(my_slice)}')  # Printing the current length in each addition

# Call the function
demo_capacity()
```
