swapcase

The 'swapcase' method in Ruby is used to return a new string where all uppercase letters are converted to lowercase and all lowercase letters are converted to uppercase. This is useful for toggling the case of the characters in a string, allowing for easy text manipulation and transformation.

input_string = 'Hello World'
result = input_string.swapcase()
# result will be 'hELLO wORLD'
input_string = 'Ruby Programming'
result = input_string.swapcase()
# result will be 'rUBY pROGRAMMING'
input_string = 'OpenAI'
result = input_string.swapcase()
# result will be 'oPENai'
