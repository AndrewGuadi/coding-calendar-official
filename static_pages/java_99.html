abs

The 'abs' method in Java is used to calculate the absolute value of a given number. The absolute value of a number is its distance from zero, regardless of the number's sign. The method can accept different data types such as int, long, float, or double, and returns the absolute value in the same data type. This is particularly useful when you need to ensure that a value is non-negative before performing operations that require it, such as calculations involving distances or differences.

def calculate_distance(x1, y1, x2, y2):
    distance = ((x2 - x1) ** 2 + (y2 - y1) ** 2) ** 0.5
    return distance

# Example usage
print(calculate_distance(3, 4, 7, 1))  # Output: 5.0
def ensure_non_negative(value):
    abs_value = abs(value)
    return abs_value

# Example usage
print(ensure_non_negative(-25))  # Output: 25
print(ensure_non_negative(10))   # Output: 10
def percentage_difference(value1, value2):
    difference = abs(value1 - value2)
    average = (value1 + value2) / 2
    return (difference / average) * 100

# Example usage
print(percentage_difference(40, 60))  # Output: 50.0
