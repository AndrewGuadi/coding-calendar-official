std::vector::push_back

The std::vector::push_back method is used to add a new element to the end of a vector in C++. This method modifies the vector by increasing its size by one, and it can accept any data type that the vector holds. If the vector's capacity is reached, it automatically reallocates memory to accommodate the new element. It is an efficient way to append elements to a dynamic array, making it very useful in scenarios where the size of the array is not known at compile time.

```cpp
#include 
#include 

int main() {
    std::vector strVec;
    strVec.push_back("Hello");
    strVec.push_back("World");
    
    for (const auto &str : strVec) {
        std::cout << str << ' '; // Outputs: Hello World 
    }
    return 0;
}```

#include 
#include 

int main() {
    std::vector intVec;
    for (int i = 0; i < 5; i++) {
        intVec.push_back(i * 2); // Appending even numbers 0, 2, 4, 6, 8
    }
    
    for (const auto &num : intVec) {
        std::cout << num << ' '; // Outputs: 0 2 4 6 8 
    }
    return 0;
}```
