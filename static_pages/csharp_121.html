Add

The 'Add' method is used to add an element to a collection, such as a list, dictionary, or set. In C#, this is often seen in classes like List, Dictionary, and HashSet. The method typically takes one or more parameters, which specify the item or items to be added to the collection. After invocation, the collection will include the newly added element(s). If the collection does not allow duplicates (e.g., HashSet), adding a duplicate item will be ignored.

```python
# Using a list to demonstrate Add method
my_list = []

# Add method to add items to the list
my_list.append('Python')
my_list.append('C#')

print(my_list)  # Output: ['Python', 'C#']
```
```python
# Using a dictionary to demonstrate Add method
my_dict = {}

# Using keys and values to add items to the dictionary
my_dict['language'] = 'Python'
my_dict['version'] = '3.9'

print(my_dict)  # Output: {'language': 'Python', 'version': '3.9'}
```
```python
# Using a set to demonstrate Add method
my_set = set()

# Add method to add items to the set
my_set.add('Python')
my_set.add('C#')
my_set.add('Python')  # Duplicates will be ignored

print(my_set)  # Output: {'Python', 'C#'}
```
