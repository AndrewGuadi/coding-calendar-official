acos

The 'acos' method in Java is a part of the Math class that returns the arc cosine of a specified number in radians. The arc cosine is the inverse cosine function that returns the angle whose cosine is the specified number. The input value must be in the range of -1 to 1, inclusive. If the argument is outside this range, the method will return NaN (Not-a-Number). This method can be used to find the angles in various applications, such as physics simulations or computer graphics.

import math

# Calculate the arc cosine of a value
value = 0.5
angle_radians = math.acos(value)
angle_degrees = math.degrees(angle_radians)
print(f'The arc cosine of {value} is {angle_radians} radians or {angle_degrees} degrees.')

import math

# Using acos to determine angle between two vectors
# Vector A
A = (1, 0)
# Vector B
B = (0, 1)

# Calculate the dot product of A and B
dot_product = A[0] * B[0] + A[1] * B[1]  
# Calculate the magnitudes of vector A and B
magnitude_A = math.sqrt(A[0]**2 + A[1]**2)
magnitude_B = math.sqrt(B[0]**2 + B[1]**2)

# Calculate the cosine of the angle
cosine_angle = dot_product / (magnitude_A * magnitude_B)

# Now use acos to find the angle
angle_radians = math.acos(cosine_angle)
angle_degrees = math.degrees(angle_radians)
print(f'The angle between the two vectors is {angle_radians} radians or {angle_degrees} degrees.')
