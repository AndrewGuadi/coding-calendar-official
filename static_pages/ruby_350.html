upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase. It returns a new string with all characters transformed to their uppercase counterparts, leaving characters that are already uppercase and non-alphabetical characters unchanged. This method is useful when you want to standardize text for comparison or display purposes.

def convert_to_uppercase(input_string):
    return input_string.upper()

text = 'hello world'
result = convert_to_uppercase(text)
print(result)  # Output: 'HELLO WORLD'
def emphasize_message(message):
    return message.upper()

msg = 'this is important!'
print(emphasize_message(msg))  # Output: 'THIS IS IMPORTANT!'
def shout(text):
    return text.upper()

user_input = 'please pay attention!'
print(shout(user_input))  # Output: 'PLEASE PAY ATTENTION!'
