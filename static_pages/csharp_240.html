Add

The Add method is used to add an element to a collection in C#. This method can be found in various collection classes, such as List, Dictionary, and HashSet. It modifies the collection by inserting the provided element, which can subsequently be retrieved or manipulated based on its type and position within the collection. The Add method may also have different overloads depending on the specific collection type it is associated with.

```python
# Adding elements to a list in C#
myList = List[int]()
myList.Add(1)
myList.Add(2)
myList.Add(3)
print(myList)  # Output: [1, 2, 3]
```
```python
# Adding key-value pairs to a dictionary in C#
myDict = Dictionary[str, int]()
myDict.Add('one', 1)
myDict.Add('two', 2)
myDict.Add('three', 3)
print(myDict)  # Output: {'one': 1, 'two': 2, 'three': 3}
```
```python
# Adding unique elements to a HashSet in C#
myHashSet = HashSet[int]()
myHashSet.Add(1)
myHashSet.Add(2)
myHashSet.Add(2)  # Duplicate, won't be added
print(myHashSet)  # Output: {1, 2}
```
