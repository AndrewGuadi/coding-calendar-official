ascii

The `ascii()` function in Python takes an object as an argument and returns a string containing a printable representation of that object. This representation is the same as that produced by the built-in `repr()` function, except it replaces non-ASCII characters in the string with their escape sequences. This is particularly useful for displaying strings that include non-printable or special characters, making it easier to understand the output without executing the actual content.

print(ascii('Hello, 世界'))  # Output: 'Hello, \u4e16\u754c'
print(ascii({'key': 'value', 'non_ascii': 'ñ'}))  # Output: '{'key': 'value', 'non_ascii': '\u00f1'}'
print(ascii([1, 2, 'text', 'üñico']))  # Output: '[1, 2, 'text', '\xfc\u00f1ico']'
