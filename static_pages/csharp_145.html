Add

The 'Add' method is used to insert a new element into a collection such as a list, dictionary, or other data structures in C#. It typically appends the element to the end of the collection or associates it with a key in the case of dictionaries. The 'Add' method is crucial for dynamically modifying collections during runtime.

```python
# Example of using Add method with List in C#

# Create a List to store integers
my_list = []

# Function to add an item to the list
def add_item(item):
    my_list.append(item)

# Add items to the list using the Add method
add_item(1)
add_item(2)
add_item(3)

# Print the list
print(my_list)  
# Output: [1, 2, 3]
```
```python
# Example of using Add method with Dictionary in C#

# Create a Dictionary to store key-value pairs
my_dict = {}

# Function to add a key-value pair to the dictionary
def add_item(key, value):
    my_dict[key] = value

# Add key-value pairs to the dictionary using the Add method
add_item('a', 1)
add_item('b', 2)
add_item('c', 3)

# Print the dictionary
print(my_dict)  
# Output: {'a': 1, 'b': 2, 'c': 3}
```
```python
# Example of using Add method with a custom class in C#

# Create a custom class to hold items
class Item:
    def __init__(self, name):
        self.name = name

# Create a List to store items
my_items = []

# Function to add an item to the list
def add_item(item):
    my_items.append(item)

# Create instances of Item and add them to the list
item1 = Item('Item1')
item2 = Item('Item2')
add_item(item1)
add_item(item2)

# Print the names of the items
for item in my_items:
    print(item.name)
# Output:
# Item1
# Item2
```
