std::vector::push_back

The std::vector::push_back method is part of the C++ Standard Library and is used to add a new element to the end of a vector. This method increases the vector's size by one, and the new element is initialized with the value provided. If the current storage capacity is insufficient to accommodate the new element, the vector automatically reallocates memory to ensure that there is enough space. This method is crucial for dynamically managing lists of data, allowing for efficient insertion as the vector grows.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");

    // Display elements in the vector
    for(const auto& str : myVector) {
        std::cout << str << std::endl;
    }
    return 0;
}
```
