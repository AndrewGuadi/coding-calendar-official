charAt

The `charAt` method in JavaScript is used to return the character at a specific index in a string. It takes a single argument, which is the index of the character you want to retrieve. If the index is out of bounds (less than 0 or greater than the string length), it returns an empty string. The indexing is zero-based, meaning the first character of the string is at index 0.

```python
# Example of using charAt to access characters of a string
sample_string = 'Hello, World!'
first_character = sample_string[0]  # Gets 'H'
seventh_character = sample_string[6]  # Gets ','
print(first_character, seventh_character)
```
```python
# Example of handling out of bounds with charAt
sample_string = 'Hello'
negative_index = sample_string[-1:]  # This will fail in JavaScript but returns '' in Python
out_of_bounds_index = sample_string[10:11]  # Also returns ''
print(negative_index, out_of_bounds_index)
```
```python
# Example of iterating over a string
sample_string = 'chatgpt'
for i in range(len(sample_string)):
    character = sample_string[i]  # Retrieving each character using indexing
    print('Character at index', i, 'is', character)
```
