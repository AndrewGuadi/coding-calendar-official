std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element to the end of a vector. This method resizes the vector if necessary, allocating additional memory if the current capacity is exceeded. The new element is copied or moved into the vector. This is a fundamental operation for dynamic arrays provided by the C++ Standard Library. The 'push_back' function provides a simple and efficient way to grow the vector while maintaining order.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Hello");
    myVector.push_back("World");
    
    for(const auto& str : myVector) {
        std::cout << str << ' '; // Outputs: Hello World
    }
    return 0;
}
```

    std::vector numbers;
    numbers.push_back(10);
    numbers.push_back(20);
    numbers.push_back(30);
    
    int sum = 0;
    for(int num : numbers) {
        sum += num; // Sums up the numbers: 10 + 20 + 30
    }
    std::cout << "Sum: " << sum << std::endl; // Outputs: Sum: 60
    return 0;
}
