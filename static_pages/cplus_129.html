std::vector::push_back

The std::vector::push_back method in C++ is used to add an element at the end of a vector. It resizes the vector if needed to accommodate the new element. The method does not return any value. It is commonly used when you need to store a list of elements dynamically and want to add elements to that list. The complexity of the operation is amortized constant time, meaning that in general, adding an element takes constant time, except when the vector needs to be resized, which can take linear time.

```
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("Python");
    myVector.push_back("Java");
    myVector.push_back("C++");

    std::cout << "Contents of myVector:\n";
    for(const auto &language : myVector) {
        std::cout << language << std::endl;
    }
    return 0;
}
```
