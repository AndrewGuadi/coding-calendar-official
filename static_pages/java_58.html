abs

The 'abs' method in Java returns the absolute value of a given number. It is a static method of the Math class, which means it can be called without creating an instance of the Math class. The 'abs' method is overloaded, allowing it to accept different primitive data types such as int, long, float, and double. The absolute value of a number is its distance from zero on the number line, regardless of the number's sign (positive or negative).

number = -10
result = abs(number)
print(result)  # Output: 10
number = -5.5
result = abs(number)
print(result)  # Output: 5.5
number = -20000
result = abs(number)
print(result)  # Output: 20000
