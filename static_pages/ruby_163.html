upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. It does not modify the original string but returns a new string with all characters converted to uppercase. This method is frequently used in scenarios where case-insensitive comparisons or uniform text formatting is needed.

def convert_to_uppercase(text):
    # Converting the input text to uppercase using the upcase method
    return text.upper()

# Example usage
input_text = 'hello world'
result = convert_to_uppercase(input_text)
print(result)  # Output: HELLO WORLD
def format_name(name):
    # Format name to uppercase
    return name.upper()

# Example usage
user_name = 'john doe'
formatted_name = format_name(user_name)
print(formatted_name)  # Output: JOHN DOE
def shout_message(message):
    # Return the message in all uppercase!
    return message.upper()

# Example usage
shout = shout_message('this is a warning')
print(shout)  # Output: THIS IS A WARNING
