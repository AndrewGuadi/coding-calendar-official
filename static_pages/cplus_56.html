std::vector::push_back

The std::vector::push_back method in C++ is used to add a new element at the end of a vector, which dynamically increases the size of the vector. This method takes a value as an argument and appends it to the vector's existing elements in the order they were added. If the vector needs to grow beyond its current capacity, it automatically allocates more memory and moves existing elements to the new location. This makes it an essential method for managing a dynamic array in C++. It operates in amortized constant time, ensuring efficient performance for sequence modifications.

```cpp
#include 
#include 
int main() {
    std::vector fruits;
    fruits.push_back("apple");
    fruits.push_back("banana");
    fruits.push_back("cherry");

    for (const auto& fruit : fruits) {
        std::cout << fruit << std::endl;
    }
    return 0;
}```
