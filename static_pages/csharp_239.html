Add

The 'Add' method in C# is commonly used to insert an element into a collection. This method can be found in various collection types, such as lists, dictionaries, or sets. When using 'Add', it adds the specified object to the end of the collection, and for dictionaries, it associates the object with a given key. This method typically does not return a value but may throw exceptions if there are constraints, such as duplicate keys in a dictionary or exceeding capacity in a list.

```python
# Example 1: Using Add with a List
def example_add_list():
    my_list = []
    my_list.append('python')  # Simulates Add in C#
    my_list.append('javascript')
    print(my_list)  # Output: ['python', 'javascript']

example_add_list()```
```python
# Example 2: Using Add with a Dictionary
def example_add_dictionary():
    my_dict = {}
    my_dict['a'] = 'hello'  # Simulates Add in C#
    my_dict['b'] = 'world'
    print(my_dict)  # Output: {'a': 'hello', 'b': 'world'}

example_add_dictionary()```
```python
# Example 3: Using Add with a Set
def example_add_set():
    my_set = set()
    my_set.add('python')  # Simulates Add in C# in terms of Sets
    my_set.add('java')
    my_set.add('python')  # No effect since 'python' is already in the set
    print(my_set)  # Output: {'python', 'java'}

example_add_set()```
