std::vector::push_back

The `push_back` method in C++ is used to add a new element to the end of a `std::vector`. The new element is added after the current last element and the size of the vector increases by one. This method takes a single argument, which is the value to be added. If the vector needs to resize to accommodate the new element, it will handle it automatically. The new element is copied (or moved) into the container.

# Example of using std::vector::push_back to add strings to a vector.
#include 
#include 

int main() {
    std::vector stringVector;

    // Adding elements to the vector
    stringVector.push_back("Python");
    stringVector.push_back("Java");
    stringVector.push_back("C++");

    // Displaying the elements of the vector
    for (const auto& str : stringVector) {
        std::cout << str << std::endl;
    }
    return 0;
}
