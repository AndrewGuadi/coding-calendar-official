append

The 'append' function in Go is used to add elements to the end of a slice. It takes a slice and one or more values of the same type as the slice, and it returns a new slice with the added elements. If the capacity of the slice is exceeded, 'append' will allocate a new slice with double the capacity and copy the existing elements over, ensuring that the operation remains efficient. It is important to note that slices in Go are reference types, meaning that if you append elements to a slice, you may end up modifying the original slice if you are working with the same reference.

```go
package main
import "fmt"
func main() {
    slice := []string{"apple", "banana"}
    slice = append(slice, "cherry")
    fmt.Println(slice) // Output: [apple banana cherry]
}
```
```go
package main
import "fmt"
func main() {
    slice := []int{1, 2, 3}
    slice = append(slice, 4, 5, 6)
    fmt.Println(slice) // Output: [1 2 3 4 5 6]
}
```
```go
package main
import "fmt"
func main() {
    originalSlice := []float64{1.1, 2.2}
    newSlice := append(originalSlice, 3.3)
    fmt.Println(originalSlice) // Output: [1.1 2.2]
    fmt.Println(newSlice)      // Output: [1.1 2.2 3.3]
}
```
