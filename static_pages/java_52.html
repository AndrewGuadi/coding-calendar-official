abs

The 'abs' method in Java is a static method defined in the 'Math' class that returns the absolute value of a given number. It can accept different types of numeric values including int, long, float, and double. The absolute value of a number is its non-negative value without regard for its sign. So the absolute value of both -10 and 10 is 10. This method is particularly useful in mathematical computations where the magnitude of the number is more important than its sign.

num = -5.5
absolute_value = Math.abs(num)
System.out.println("The absolute value of " + num + " is: " + absolute_value);
num = -42
absolute_value = Math.abs(num)
System.out.println("The absolute value of " + num + " is: " + absolute_value);
num = 15
absolute_value = Math.abs(num)
System.out.println("The absolute value of " + num + " is: " + absolute_value);
