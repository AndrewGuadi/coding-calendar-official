abs

The 'abs' function in Python returns the absolute value of a given number. The absolute value of a number is its non-negative value without regard to its sign. This function can be applied to integers, floating-point numbers, and complex numbers. For complex numbers, the magnitude (modulus) is returned.

# Example 1: Getting the absolute value of a negative integer
number = -10
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')  # Output: The absolute value of -10 is 10
# Example 2: Getting the absolute value of a floating-point number
number = -4.5
absolute_value = abs(number)
print(f'The absolute value of {number} is {absolute_value}')  # Output: The absolute value of -4.5 is 4.5
# Example 3: Getting the magnitude of a complex number
number = complex(-3, 4)
absolute_value = abs(number)
print(f'The magnitude of {number} is {absolute_value}')  # Output: The magnitude of (-3+4j) is 5.0
