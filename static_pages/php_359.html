strlen

The `strlen` function in PHP is used to get the length of a given string, which includes all characters such as letters, digits, spaces, and special characters. It returns the number of bytes in the string; for single-byte character encodings like ISO-8859-1, this is equivalent to the number of characters, while for multibyte encodings like UTF-8, it may return a higher byte count. This function is often employed in form validation and string manipulation scenarios to ensure the expected content length or to dynamically adjust outputs or processes based on string length.

def get_string_length(input_str):
    return len(input_str)

string_input = 'Hello, World!'
length = get_string_length(string_input)
print(f'Length of the input string: {length}')
def validate_username(username):
    if 3 <= len(username) <= 15:
        return True
    return False

username_input = 'user123'
if validate_username(username_input):
    print('Username is valid.')
else:
    print('Username must be between 3 and 15 characters.')
def truncate_string(input_str, max_length):
    if len(input_str) > max_length:
        return input_str[:max_length] + '...'
    return input_str

text_input = 'This is a long string that needs to be truncated.'
truncated_text = truncate_string(text_input, 20)
print(f'Truncated text: {truncated_text}')
