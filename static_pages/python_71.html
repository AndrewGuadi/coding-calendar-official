abs

The `abs` method in Python returns the absolute value of a given number. This number can be a float, integer, or a complex number. The absolute value of a number is its distance from zero on the number line, without considering the direction.

number = -10
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
number = -25.67
absolute_value = abs(number)
print('The absolute value of', number, 'is', absolute_value)
complex_number = 3 - 4j
absolute_value = abs(complex_number)
print('The absolute value (or magnitude) of', complex_number, 'is', absolute_value)
