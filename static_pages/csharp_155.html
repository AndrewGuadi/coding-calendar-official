Add

The 'Add' method is commonly used in C# to add an element to a collection, such as a list, set, or dictionary. This method allows you to insert a new item and typically returns a boolean indicating whether the item was successfully added. For instances such as lists, the item is added to the end, while in dictionaries, a key-value pair is added.

The following example shows how to use the 'Add' method to add elements to a list of strings. This demonstrates creating a list, adding elements, and iterating through the list to display the added elements:

```python
my_list = []  # Create an empty list

# Use the Add method to add Python code examples to the list
my_list.append('print("Hello, World!")')
my_list.append('for i in range(5): print(i)')
my_list.append('def greet(name): return f"Hello, {name}!"')

# Display the contents of the list
for item in my_list:
    print(item)  # Output each added Python code
```
In this example, we will demonstrate how to use the 'Add' method in a dictionary. This illustrates how to add key-value pairs:

```python
dictionary = {}  # Create an empty dictionary

# Use the Add method to add example functions to the dictionary
# Here we use the keys as names of functions
key1 = 'hello_function'
value1 = 'def hello(): print("Hello, World!")'  
dictionary[key1] = value1

key2 = 'sum_function'
value2 = 'def sum(a, b): return a + b'  
dictionary[key2] = value2

# Display the dictionary's contents
for key, value in dictionary.items():
    print(f'{key}: {value}')  # Output each function example
```
This example illustrates the use of the 'Add' method to build a set. Sets automatically handle duplicate entries, hence only unique items are stored:

```python
my_set = set()  # Create an empty set

# Use the Add method to add different Python code snippets
my_set.add('def multiply(a, b): return a * b')
my_set.add('def divide(a, b): return a / b')
my_set.add('def multiply(a, b): return a * b')  # Duplicate; will not be added

# Display the contents of the set
for item in my_set:
    print(item)  # Output unique Python code examples
```
