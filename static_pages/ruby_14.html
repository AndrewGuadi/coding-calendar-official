upcase

The 'upcase' method is a built-in String method in Ruby that converts all lowercase letters in a string to uppercase. This method does not modify the original string; instead, it returns a new string with all characters in uppercase. It is particularly useful for formatting strings consistently or for case-insensitive comparisons.

def convert_to_uppercase(input_string):
    return input_string.upper()

# Sample usage:
print(convert_to_uppercase('hello world'))  # Outputs: HELLO WORLD
def shout_message(message):
    return message.upper() + '!!!'

# Sample usage:
print(shout_message('ruby is fun'))  # Outputs: RUBY IS FUN!!!
def emphasize_text(text):
    return text.upper()

# Sample usage:
print(emphasize_text('keep trying'))  # Outputs: KEEP TRYING
