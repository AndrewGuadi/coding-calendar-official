ascii

The `ascii()` function in Python takes an object as an argument and returns a string containing a printable representation of the object, where non-ascii characters (i.e., characters with codes greater than 127) are escaped. It is useful for debugging and logging when you want to visualize the representation of the object in a safe form. Unlike `repr()`, which returns the `repr` representation of the object, `ascii()` guarantees that the result is a string containing only ASCII characters.

data = {'name': 'céline', 'age': 30}
print(ascii(data))  # Output: '{'name': 'c\xe9line', 'age': 30}'
text = 'Python is fun!'
print(ascii(text))  # Output: 'Python is fun!'
symbols = '©, ®, ™'
print(ascii(symbols))  # Output: '\xe2\x82\xa9, \xe2\xae\xa2, \xe2\x84\xa2'
