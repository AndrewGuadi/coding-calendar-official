Add

The 'Add' method is commonly used in C# collections (such as List or Dictionary) to insert a new element into the collection. The method takes a single argument, which is the element to be added. If the collection supports duplicate entries, it will allow adding the same element multiple times. However, some collections (like dictionaries) require a unique key for each element.

```python
# Example using a list in Python to demonstrate Add method logic

# Define a list
my_list = []

# Function to simulate adding an element
def add_element(element):
    my_list.append(element)

# Using the function to add elements
add_element('apple')
add_element('banana')

# Display the list
print(my_list)  # Output: ['apple', 'banana']
```
```python
# Example using a dictionary in Python to demonstrate Add method logic

# Define a dictionary
my_dict = {}

# Function to simulate adding a key-value pair
def add_pair(key, value):
    my_dict[key] = value

# Using the function to add key-value pairs
add_pair('name', 'John')
add_pair('age', 30)

# Display the dictionary
print(my_dict)  # Output: {'name': 'John', 'age': 30}
```
```python
# Example of adding items in a set using Python to demonstrate Add method logic

# Define a set
my_set = set()

# Function to simulate adding an element to a set
def add_to_set(element):
    my_set.add(element)

# Using the function to add elements
add_to_set('apple')
add_to_set('banana')
add_to_set('apple')  # Attempt to add a duplicate element

# Display the set
print(my_set)  # Output: {'apple', 'banana'}  # Note: 'apple' only appears once as sets do not allow duplicates
```
