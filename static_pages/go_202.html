cap

The 'cap' function in Go is used to return the capacity of a slice, array, or channel. The capacity refers to the maximum number of elements that the collection can hold without needing to allocate additional memory. This is particularly useful for optimizing memory usage and performance when working with slices in Go.

```go
package main
import (
	"fmt"
)

func main() {
	// Create a slice with given length and capacity
	slice := make([]string, 3, 10) // length 3, capacity 10

	fmt.Println("Length of slice:", len(slice)) // Output: Length of slice: 3
	fmt.Println("Capacity of slice:", cap(slice)) // Output: Capacity of slice: 10
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Create a slice with default length and capacity
	slice := []int{1, 2, 3, 4, 5}

	fmt.Println("Length of slice:", len(slice)) // Output: Length of slice: 5
	fmt.Println("Capacity of slice:", cap(slice)) // Output: Capacity of slice: 5
}
```
```go
package main
import (
	"fmt"
)

func main() {
	// Using a channel with a specified buffer size
	ch := make(chan int, 3) // channel buffer capacity of 3

	ch <- 1
	ch <- 2
	ch <- 3

	fmt.Println("Capacity of channel:", cap(ch)) // Output: Capacity of channel: 3
}
```
