close

The 'close' method in Go is used to close resources such as files, network connections, or any stream that implements the 'io.Closer' interface. Closing a resource is important as it releases any resources associated with it and prevents memory leaks. When a resource is closed, it typically ensures that any buffered data is flushed and will not cause any unintended access after closure.

with open('example.txt', 'r') as file:
    data = file.read()
    print(data)  # Reads and prints the content of the file
# Upon exiting the with-block, the file is automatically closed.
import json

# Simulating a file operation
with open('data.json', 'w') as json_file:
    json.dump({'name': 'John', 'age': 30}, json_file)
# The file is closed automatically here after writing to it
import sqlite3

# Connecting to a database
connection = sqlite3.connect('example.db')

try:
    cursor = connection.cursor()
    cursor.execute('CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT)')
finally:
    connection.close()  # Ensure that the connection is closed after the operations
