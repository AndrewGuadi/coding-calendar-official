abs

The 'abs' method in Java is used to compute the absolute value of an integer, a floating-point number, or a long value. The absolute value of a number is its non-negative value, meaning that if the number is negative, it is multiplied by -1 to make it positive. This method is part of the Math class and can be used with different numerical types such as int, double, and long. It is particularly useful in scenarios where you need to measure the distance from zero, regardless of the sign of the number.

def compute_absolute_values():
    values = [-10, 0, 15, -3.5, 2.9]
    absolute_values = [abs(value) for value in values]
    return absolute_values

result = compute_absolute_values()
print(result)  # Output: [10, 0, 15, 3.5, 2.9]
def distance_from_zero(value):
    return abs(value)

value1 = -42
value2 = 42
print(distance_from_zero(value1))  # Output: 42
print(distance_from_zero(value2))  # Output: 42
def compare_absolute_values(num1, num2):
    return abs(num1) == abs(num2)

print(compare_absolute_values(-10, 10))  # Output: True
print(compare_absolute_values(-10, -5))  # Output: False
