strlen

The `strlen` function in PHP is used to get the length of a string, measured in bytes. It counts all characters in the string but does not account for multibyte characters that may consume more than one byte. This function can be particularly useful when validating string lengths for user inputs, enforcing character limits, or performing checks in your application.

string1 = 'Hello, World!'
length1 = len(string1)
print(length1)  # Output will be 13, as there are 13 characters in 'Hello, World!'
string2 = 'こんにちは'
length2 = len(string2)
print(length2)  # Output will be 5, as the string contains 5 characters but may take more bytes
string3 = ''
length3 = len(string3)
print(length3)  # Output will be 0, as the string is empty
