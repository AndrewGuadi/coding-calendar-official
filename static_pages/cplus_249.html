std::vector::push_back

The method 'std::vector::push_back' is used in C++ to add a new element to the end of a vector. The vector is a dynamic array that can grow in size automatically as elements are added. When you use 'push_back', the element is copied or moved into the vector, increasing the current size by one. It's important to note that this operation may sometimes require reallocation of memory if the current capacity of the vector is exceeded. This method is typically used when you need to build a collection of items dynamically and do not know the size of your collection in advance.

```cpp
#include 
#include 
#include 

int main() {
    std::vector python_examples;

    // Adding Python code examples to the vector
    python_examples.push_back("print('Hello, World!')");
    python_examples.push_back("x = 10\ny = 20\nprint(x + y)");
    python_examples.push_back("for i in range(5):\n    print(i)");

    // Outputting the examples
    for (const auto &example : python_examples) {
        std::cout << example << std::endl;
    }

    return 0;
}
```
