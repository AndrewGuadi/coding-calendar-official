upcase

The 'upcase' method in Ruby is used to convert all lowercase letters in a string to uppercase letters. This method returns a new string with all the characters transformed to uppercase, leaving non-letter characters unchanged. It does not modify the original string, making it a non-destructive method. It is particularly useful for normalizing user input or for ensuring a consistent format when comparing strings.

def convert_to_uppercase(input_str):
    # Using a hypothetical upcase method to demonstrate functionality
    return input_str.upper()

result = convert_to_uppercase('hello world')  # This will return 'HELLO WORLD'
print(result)

    # This function mimics the behavior of Ruby's upcase
