Add

In C#, the 'Add' method is commonly used to insert an element into a collection. This method is often found in data structures such as List, Dictionary, and HashSet. The 'Add' method typically takes one or more arguments, which are the items to be added, and it modifies the collection in place, ensuring that the new item is included in subsequent operations. Using this method is essential for dynamically managing collections during runtime.

```python
# Python code to demonstrate adding values to a list.
# Create an empty list
my_list = []

# Function to add elements to the list
def add_to_list(element):
    my_list.append(element)

# Adding elements
add_to_list(10)
add_to_list(20)
add_to_list(30)

# Result
print(my_list)  # Output: [10, 20, 30]
```
```python
# Python code to demonstrate adding key-value pairs to a dictionary.
# Create an empty dictionary
my_dict = {}

# Function to add key-value pairs to the dictionary
def add_to_dict(key, value):
    my_dict[key] = value

# Adding key-value pairs
add_to_dict('name', 'Alice')
add_to_dict('age', 30)

# Result
print(my_dict)  # Output: {'name': 'Alice', 'age': 30}
```
```python
# Python code to demonstrate adding elements to a set.
# Create an empty set
my_set = set()

# Function to add elements to the set
def add_to_set(element):
    my_set.add(element)

# Adding elements
add_to_set(1)
add_to_set(2)
add_to_set(3)

# Adding a duplicate element.
add_to_set(2)  # This will not change the set

# Result
print(my_set)  # Output: {1, 2, 3} - Note: order is not guaranteed
```
