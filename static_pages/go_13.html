append

The `append` function in Go is used to add elements to a slice. Slices in Go are dynamic, meaning they can grow in size when new elements are added using `append`. The function takes a slice and one or more values to be appended to that slice. It returns a new slice that includes the original elements along with the newly added values. If the underlying array of the slice does not have enough capacity to accommodate the new elements, `append` will allocate a new array, copy the original elements over, and then add the new elements.

```python
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)
# Output: [1, 2, 3, 4]```
```python
my_list = ['a', 'b']
my_list.append('c')
my_list.append('d')
print(my_list)
# Output: ['a', 'b', 'c', 'd']```
```python
my_list = []
for i in range(5):
    my_list.append(i)
print(my_list)
# Output: [0, 1, 2, 3, 4]```
