std::vector::push_back

The std::vector::push_back method adds a new element to the end of a vector in C++. This function increases the size of the vector by one and initializes the new element with the provided value. If the current capacity of the vector is insufficient to accommodate the new element, the vector will automatically allocate more memory to hold the additional elements. It is important to note that after this operation, the size of the vector is increased while the original vector elements are preserved.

```cpp
#include 
#include 

int main() {
    std::vector myVector;
    myVector.push_back("hello");
    myVector.push_back("world");

    for (const auto &str : myVector) {
        std::cout << str << ' ';
    }
    // Output: hello world
    return 0;
}
```

int main() {

    languages.push_back("Java");
    languages.push_back("C++");

    for (size_t i = 0; i < languages.size(); ++i) {
        std::cout << languages[i] << '\n';
    }
    // Output: 
    // Python
    // Java
    // C++
    return 0;
}
