abs

The 'abs' method in Java is used to obtain the absolute value of a number. It can take arguments of various types, including integers, floating-point numbers, and doubles. The method returns the non-negative value of the passed number. If the number is already positive, it returns the same number. If the number is negative, it converts it to positive. This method is useful in mathematical calculations where the magnitude of a value is needed without regards to its sign.

def calculate_distance(x1, y1, x2, y2):
    return abs((x1 - x2)**2 + (y1 - y2)**2)

# Calculate distance between two points (5, 5) and (1, 1)
distance = calculate_distance(5, 5, 1, 1)
print(distance)  # Output will show the squared distance
def adjust_balance(balance):
    return abs(balance)

# Adjust user's negative balance
user_balance = -150
adjusted_balance = adjust_balance(user_balance)
print(adjusted_balance)  # Output will show 150
def get_absolute_value(num):
    return abs(num)

# Get the absolute value of a negative number
number = -42
absolute_value = get_absolute_value(number)
print(absolute_value)  # Output will show 42
