strlen

The `strlen` function in PHP is used to measure the length of a string. It returns the number of bytes (not characters) in the string provided as input. This is particularly useful for validating user input, checking password lengths, or manipulating strings based on their size.

To find the length of a sample string, you can use the `strlen` method as follows:

```php
$string = 'Hello, World!';
$length = strlen($string);
echo 'The length of the string is: ' . $length;
// Output: The length of the string is: 13
```
In a scenario where you need to validate a password to ensure it meets certain length criteria:

```php
$password = 'MySecureP@ssw0rd';
if (strlen($password) < 8) {
    echo 'Password must be at least 8 characters long.';
} else {
    echo 'Password length is valid.';
}
// Output: Password length is valid.
```
You can also use `strlen` to check the length of user input before processing it:

```php
$userInput = 'Sample text';
if (strlen($userInput) === 0) {
    echo 'Input cannot be empty.';
} else {
    echo 'Input received with length: ' . strlen($userInput) . ' characters.';
}
// Output: Input received with length: 11 characters.
```
