strlen

The `strlen` function in PHP is used to calculate the length of a string. It returns the number of bytes (not characters) in the string, which is particularly relevant for multi-byte character encodings like UTF-8. The result can be useful for validating the length of a string or for processing strings where the length is required for subsequent operations.

Using `strlen` to find the length of a simple string:
```php
$string = 'Hello, World!';
echo strlen($string);  // Outputs: 13
```
Finding the length of a string with special characters:
```php
$string = 'こんにちは'; // Japanese for 'Hello'
echo strlen($string);  // Outputs: 15 (because of multibyte)
```
Validating user input by checking the length of a password:
```php
$password = 'myp@ssw0rd';
if (strlen($password) < 8) {
    echo 'Password must be at least 8 characters long.';
} else {
    echo 'Password is valid.';
}
```
