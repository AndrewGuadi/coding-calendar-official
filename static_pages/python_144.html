ascii

The `ascii` function in Python returns a string containing a printable representation of an object, but it escapes the non-ASCII characters. Any character that is not in the ASCII range (0 to 127) will be replaced with its respective Unicode escape sequence, making it useful for debugging and logging, especially when dealing with strings containing special or non-ASCII characters.
