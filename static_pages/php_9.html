strlen

The `strlen` function in PHP is used to calculate the length of a string, returning the number of bytes or characters contained in the string. It is particularly useful for validating input data, ensuring that strings meet expected length criteria before processing them further.

```python
# Define a string
my_string = 'Hello, World!'
# Calculate length of the string
length = len(my_string)
# Output the result
print('Length of my_string:', length)
```
```python
# Function to check the length of a string
def check_string_length(input_string):
    return len(input_string) > 10

# Check various strings
print(check_string_length('Short'))  # Output: False
print(check_string_length('This is a longer string'))  # Output: True
```
```python
# Function to format a message based on string length
def format_message(input_string):
    if len(input_string) < 5:
        return 'Input too short!'
    return 'Input is acceptable!'

# Test the function
print(format_message('Hi'))  # Output: Input too short!
print(format_message('Hello'))  # Output: Input is acceptable!
```
