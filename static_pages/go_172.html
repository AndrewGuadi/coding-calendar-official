cap

In Go, the 'cap' function is used to return the capacity of a slice, which is the maximum number of elements that the slice can hold without reallocating. This function is particularly useful when working with slices of variable sizes, as it helps you understand how much space is allocated for your slice to avoid unnecessary allocations.

```python
# Create a slice with initial capacity
my_slice = [0] * 5  # Creating a slice of capacity 5

# Use a method to get capacity (simulating Go "cap")

def cap(slice):
    return len(slice)

print(cap(my_slice))  # Output: 5
```,
```python
# Create a larger slice
my_slice = [0] * 10  # Creating a slice of capacity 10

# Function to simulate capacity
def cap(slice):
    return len(slice)

print(cap(my_slice))  # Output: 10
```,
```python
# Append elements to a slice while checking capacity

my_slice = [0] * 3  # Initially, create a slice with capacity 3
print(cap(my_slice))  # Output: 3

def cap(slice):
    return len(slice)

# Simulated append functionality
my_slice.append(4)
my_slice.append(5)
print('New capacity: ', cap(my_slice))  # Output: 5 (after appending)
