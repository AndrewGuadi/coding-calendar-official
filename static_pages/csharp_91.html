Add

The 'Add' method is used to add an element to a collection or a specific data structure, such as a list, dictionary, or set. It is commonly found in collections within .NET frameworks, allowing programmers to dynamically expand their collections as needed. This method typically requires at least one argument that represents the element to be added.

```python
# Example of using the Add method with a list
def main():
    numbers = []  # Initialize an empty list
    numbers.append(1)  # Add integer 1
    numbers.append(2)  # Add integer 2
    print(numbers)  # Output: [1, 2]

main()
```
```python
# Example of using the Add method with a set
def main():
    unique_numbers = set()  # Initialize an empty set
    unique_numbers.add(1)  # Add integer 1
    unique_numbers.add(2)  # Add integer 2
    unique_numbers.add(1)  # Try to add integer 1 again (duplicate not added)
    print(unique_numbers)  # Output: {1, 2}

main()
```
```python
# Example of using the Add method with a dictionary
def main():
    scores = {}  # Initialize an empty dictionary
    scores['Alice'] = 90  # Add entry for Alice
    scores['Bob'] = 85  # Add entry for Bob
    print(scores)  # Output: {'Alice': 90, 'Bob': 85}

main()
```
