abs

The 'abs' method in Java is used to return the absolute value of a given number. This means that it converts a negative number to its positive equivalent while leaving positive numbers unchanged. The 'abs' method can handle various data types, including integers, floats, and doubles. It's essential for calculations where you need to ensure that distance or any measure remains non-negative.

def absolute_value_demo():
    num = -5
    abs_value = abs(num)
    print('The absolute value of', num, 'is', abs_value)

absolute_value_demo()
def absolute_float_demo():
    num = -3.14
    abs_value = abs(num)
    print('The absolute value of', num, 'is', abs_value)

absolute_float_demo()
def absolute_list_demo():
    numbers = [-1, -2, 3, -4, 5]
    abs_values = [abs(num) for num in numbers]
    print('The absolute values of', numbers, 'are', abs_values)

absolute_list_demo()
