Add

The 'Add' method in C# is commonly used to add an item to a collection, such as a list or a dictionary. In the context of lists, it appends an element to the end of the list, while in dictionaries, it adds a new key-value pair. The method typically returns a void type, meaning it does not return any value after execution. The 'Add' method is very useful for dynamic data structures where the quantity of items cannot be predetermined, allowing for the growth of collections at runtime.

```python
# Example of adding an item to a list
my_list = []
my_list.append('Python')  # Adding a string to the list
print(my_list)  # Output: ['Python']
```
```python
# Example of adding a key-value pair to a dictionary
dictionary = {}
dictionary['language'] = 'Python'  # Adding a new key-value pair
print(dictionary)  # Output: {'language': 'Python'}
```
```python
# Example of adding multiple items to a list
years = []
years.append(2020)
years.append(2021)
years.append(2022)
print(years)  # Output: [2020, 2021, 2022]
```
