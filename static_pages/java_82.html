abs

The 'abs' method in Java is used to obtain the absolute value of numerical types such as integers, floats, and doubles. The absolute value is the non-negative value of a number without regard to its sign. For example, the absolute value of -3 is 3, and the absolute value of 3 is also 3. In Java, this method is found in the Math class: Math.abs(value).

def calculate_distance(point1, point2):
    return abs(point1 - point2)

# Example usage:
# Distance between points 5 and -2
print(calculate_distance(5, -2))  # Output will be 7

def get_positive_value(value):
    return abs(value)

# Example usage:
# Get absolute value of -42
print(get_positive_value(-42))  # Output will be 42

def difference_in_temperatures(temp1, temp2):
    return abs(temp1 - temp2)

# Example usage:
# Difference between temperatures 98.6 and 100.4
print(difference_in_temperatures(98.6, 100.4))  # Output will be 1.8
