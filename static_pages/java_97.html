abs

The `abs` method in Java is used to return the absolute value of a given number. This method can operate on both integers and floating-point types. For an integer input, it returns the integer value without change if it is positive or zero; if it is negative, it returns the positive equivalent. For floating-point values, it behaves similarly, returning the positive representation of the number. The method is a vital tool in mathematical computations where the sign is not relevant, and only the magnitude of the number matters.

number = -5
result = abs(number)
print(result)  # Output: 5
number = 3.14
result = abs(number)
print(result)  # Output: 3.14
number = -42.7
result = abs(number)
print(result)  # Output: 42.7
