uppercased

The 'uppercased' method in Swift is used to convert all the characters in a given string to uppercase. This method does not change the original string but returns a new string with the uppercase version. It is particularly useful when you want to display text in uppercase format for emphasis or consistency in UI elements.

```swift
let originalString = "hello, world!"
let uppercasedString = originalString.uppercased()
print(uppercasedString) // Output: "HELLO, WORLD!"
```
```swift
let pythonCode = "print('Hello, World!')"
let uppercasedCode = pythonCode.uppercased()
print(uppercasedCode) // Output: "PRINT('HELLO, WORLD!')"
```
```swift
let sentence = "this is a test."
let uppercasedSentence = sentence.uppercased()
print(uppercasedSentence) // Output: "THIS IS A TEST."
```
