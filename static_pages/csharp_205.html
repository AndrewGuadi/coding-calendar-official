Add

The `Add` method is used to add an element or item to a collection, such as a list, set, or dictionary in C#. This method allows you to dynamically modify collections during runtime by appending new items. It is commonly used across various types of collections in the .NET framework like List, HashSet, and Dictionary.

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creating a new list of strings
        List pythonVersions = new List();

        // Adding Python versions using the Add method
        pythonVersions.Add("Python 2.7");
        pythonVersions.Add("Python 3.6");
        pythonVersions.Add("Python 3.9");

        // Outputting the list contents
        foreach (var version in pythonVersions)
        {
            Console.WriteLine(version);
        }
    }
}
```
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creating a new HashSet to ensure unique versions of Python
        HashSet pythonUniqueVersions = new HashSet();

        // Adding unique Python versions using the Add method
        pythonUniqueVersions.Add("Python 2.7");
        pythonUniqueVersions.Add("Python 3.8");
        pythonUniqueVersions.Add("Python 3.8"); // This will not be added again due to uniqueness constraint

        // Outputting the unique versions
        foreach (var version in pythonUniqueVersions)
        {
            Console.WriteLine(version);
        }
    }
}
```
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creating a dictionary to map Python versions to their release years
        Dictionary pythonReleases = new Dictionary();

        // Adding Python versions and their release years using the Add method
        pythonReleases.Add("Python 2.7", 2010);
        pythonReleases.Add("Python 3.6", 2016);
        pythonReleases.Add("Python 3.9", 2020);

        // Outputting the versions and their release years
        foreach (var kvp in pythonReleases)
        {
            Console.WriteLine($"{kvp.Key} was released in {kvp.Value}");
        }
    }
}
```
