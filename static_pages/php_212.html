strlen

The `strlen` function in PHP is used to determine the length of a given string. It returns the number of bytes in a string. For multi-byte character encodings such as UTF-8, the number of characters may differ from the number of bytes. Therefore, `strlen` is best suited for single-byte character encodings (like ISO-8859-1). For multi-byte character encodings, the `mb_strlen` function is more appropriate. The resulting length excludes any null terminators and is sensitive to the string's encoding.

string_to_check = 'Hello World!'
strlen_result = len(string_to_check)
print(f'The length of the string is: {strlen_result}')  # Output will be: The length of the string is: 12
string_with_spaces = '  This is a test string.  '
strlen_result = len(string_with_spaces)
print(f'The length of string with spaces is: {strlen_result}')  # Output will be: The length of string with spaces is: 27
multi_byte_string = 'こんにちは'
strlen_result = len(multi_byte_string)
print(f'The length of the multi-byte string is: {strlen_result}')  # Output will be: The length of the multi-byte string is: 15 (number of bytes)
