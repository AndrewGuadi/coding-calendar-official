Add

The 'Add' method in C# is commonly used for adding elements to collections such as lists, dictionaries, or sets. Its primary function is to insert a new item, increasing the size of the collection by one. When working with different data structures, the specifics of how 'Add' operates can vary, but its main purpose remains the same: to append new data into a collection.

In this example, we create a List of integers and use the 'Add' method to insert a series of numbers into it. The list is then printed to show the contents.
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        List numbers = new List();
        numbers.Add(10);
        numbers.Add(20);
        numbers.Add(30);

        Console.WriteLine("Numbers in the list:");
        foreach (var number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}
```
In this example, we utilize a Dictionary to associate string keys with integer values. The 'Add' method is used to insert several key-value pairs, which can then be accessed by their keys.
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Dictionary ages = new Dictionary();
        ages.Add("Alice", 30);
        ages.Add("Bob", 25);
        ages.Add("Charlie", 35);

        Console.WriteLine("Ages:");
        foreach (var entry in ages)
        {
            Console.WriteLine(entry.Key + " - " + entry.Value);
        }
    }
}
```
In this example, we demonstrate the use of the 'Add' method to insert elements into a HashSet. This collection only allows unique elements, so trying to add duplicates will not change the size of the collection.
```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        HashSet fruits = new HashSet();
        fruits.Add("Apple");
        fruits.Add("Banana");
        fruits.Add("Apple"); // Duplicate, will not be added

        Console.WriteLine("Fruits:");
        foreach (var fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}
```
