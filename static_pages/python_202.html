ascii

The `ascii()` method in Python takes an object (like a string, list, or dictionary) and returns a string containing a printable representation of the object, where non-ASCII characters are escaped using their Unicode code points. This is particularly useful when you want to represent an object that may contain non-ASCII characters in a format that can be easily displayed or logged.

print(ascii(my_dict))  # Outputs: '{\'name\': \'Alice\', \'emoji\': \'\ud83d\ude0a\'}'
