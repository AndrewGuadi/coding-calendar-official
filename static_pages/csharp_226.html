Add

The 'Add' method is commonly used in collection classes such as List, Dictionary, or HashSet in C#. This method is employed to insert a new element into the collection. For example, when using a List, the 'Add' method appends an item to the end of the list. It can also be employed in dictionaries to add a key-value pair. The 'Add' method increases the size of the collection by one. It is important to note that if the key already exists in a dictionary, attempting to use 'Add' will throw an exception; however, when adding to a List, duplicates are allowed.

list_of_codes = []
list_of_codes.append("print('Hello World')")
list_of_codes.append("def add(a, b): return a + b")
print(list_of_codes)  # Output: ["print('Hello World')", "def add(a, b): return a + b"]
codes_dict = {}
codes_dict['print_hello'] = "print('Hello World')"
codes_dict['add_function'] = "def add(a, b): return a + b"
print(codes_dict)  # Output: {'print_hello': "print('Hello World')", 'add_function': "def add(a, b): return a + b"}
codes_set = set()
codes_set.add("print('Hello World')")
codes_set.add("def add(a, b): return a + b")
print(codes_set)  # Output: {"print('Hello World')", "def add(a, b): return a + b"}
