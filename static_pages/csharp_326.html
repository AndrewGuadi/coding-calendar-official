Add

The 'Add' method in C# is commonly used to insert an element into a collection, such as a List, Dictionary, or HashSet. This method typically appends the provided value to the end of the collection or registers it with a specific key in the case of a dictionary. Using the 'Add' method is straightforward; simply specify the item you want to add. This method increases the size of the collection and may also involve reallocating memory if the collection's capacity is exceeded.

```python
# Example of using a list in Python to demonstrate the Add method logic.
my_list = []

# Function to simulate adding an item to a collection using append() method
def add_item(item):
    my_list.append(item)

# Adding Python code as strings
add_item("print('Hello, World!')")
add_item("def add(x, y): return x + y")

# Display the list after adding items
print(my_list)  # Output: ["print('Hello, World!')", "def add(x, y): return x + y"]
```
```python
# Example of using a dictionary in Python to demonstrate the Add method logic.
my_dict = {}

# Function to simulate adding a key-value pair to a dictionary
def add_entry(key, value):
    my_dict[key] = value

# Adding Python code as strings
add_entry('example_1', "print('Hello from example 1')")
add_entry('example_2', "print('Hello from example 2')")

# Display the dictionary after adding entries
print(my_dict)  # Output: {'example_1': "print('Hello from example 1')", 'example_2': "print('Hello from example 2')"}
```
```python
# Example of using a set in Python to demonstrate the Add method logic.
my_set = set()

# Function to simulate adding an item to a set
def add_unique_item(item):
    my_set.add(item)

# Adding unique Python code as strings
add_unique_item("print('Adding first item')")
add_unique_item("print('Adding second item')")
add_unique_item("print('Adding first item')")  # Duplicate entry

# Display the set after adding items
print(my_set)  # Output will show unique entries only
```
